# <a name="imachinedebugmanager-interface"></a>Interfejs IMachineDebugManager
Podstawowy interfejs do Machine Manager debugowania. Ten interfejs jest podobny do `IMachineDebugManagerCookie` interfejsu.  
  
 Oprócz dziedziczone z metody `IUnknown`, `IMachineDebugManager` interfejsu udostępnia następujące metody.  
  
## <a name="methods-in-vtable-order"></a>Metody w kolejności Vtable  
  
|Metoda|Opis|  
|------------|-----------------|  
|[IMachineDebugManager::AddApplication](../../winscript/reference/imachinedebugmanager-addapplication.md)|Dodaje aplikację do uruchamiania liście aplikacji.|  
|[IMachineDebugManager::RemoveApplication](../../winscript/reference/imachinedebugmanager-removeapplication.md)|Usuwa z uruchomionym aplikacji listy aplikacji.|  
|[IMachineDebugManager::EnumApplications](../../winscript/reference/imachinedebugmanager-enumapplications.md)|Zwraca moduł wyliczający bieżącą listę uruchomionych aplikacji.|
