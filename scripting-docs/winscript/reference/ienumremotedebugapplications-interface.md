# <a name="ienumremotedebugapplications-interface"></a>Interfejs IEnumRemoteDebugApplications
Wylicza obiektów aplikacji. Ten interfejs może służyć do wyliczenia uruchamianie aplikacji na komputerze dla okna dialogowego "Dołączanie do aplikacji".  
  
 Oprócz dziedziczone z metody `IUnknown`, `IEnumRemoteDebugApplications` interfejsu udostępnia następujące metody.  
  
## <a name="methods-in-vtable-order"></a>Metody w kolejności Vtable  
  
|Metoda|Opis|  
|------------|-----------------|  
|[IEnumRemoteDebugApplications::Clone](../../winscript/reference/ienumremotedebugapplications-clone.md)|Tworzy moduł wyliczający, który zawiera stan tego samego jako bieżący modułu wyliczającego.|  
|[IEnumRemoteDebugApplications::Next](../../winscript/reference/ienumremotedebugapplications-next.md)|Pobiera określoną liczbę segmentów w kolejności wyliczenia.|  
|[IEnumRemoteDebugApplications::Reset](../../winscript/reference/ienumremotedebugapplications-reset.md)|Resetuje sekwencję wyliczenia na początku.|  
|[IEnumRemoteDebugApplications::Skip](../../winscript/reference/ienumremotedebugapplications-skip.md)|Pomija określoną liczbę segmentów w kolejności wyliczenia.|
