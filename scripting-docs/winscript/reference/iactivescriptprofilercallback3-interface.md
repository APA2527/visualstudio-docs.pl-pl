# <a name="iactivescriptprofilercallback3-interface"></a>Interfejs IActiveScriptProfilerCallback3
Udostępnia metodę, aby powiadomić profiler, który profilowanych wykonań są wykonywane w kontekście internetowy proces roboczy.  
  
## <a name="syntax"></a>Składnia  
  
```cpp
interface IActiveScriptProfilerCallback3 : IActiveScriptProfilerCallback2  
```  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[IActiveScriptProfilerCallback3::SetWebWorkerId, metoda](../../winscript/reference/iactivescriptprofilercallback3-setwebworkerid-method.md)|Powiadamia program profilujący o identyfikatorze procesów roboczych dla tej sesji profilowania.|
