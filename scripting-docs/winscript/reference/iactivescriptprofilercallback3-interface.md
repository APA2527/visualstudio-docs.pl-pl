# <a name="iactivescriptprofilercallback3-interface"></a>Interfejs IActiveScriptProfilerCallback3
Udostępnia metodę powiadomiono profiler PROFILOWANEGO wykonaniami są wykonywane w kontekście pracownika w sieci web.  
  
## <a name="syntax"></a>Składnia  
  
```  
interface IActiveScriptProfilerCallback3 : IActiveScriptProfilerCallback2  
```  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[Metoda IActiveScriptProfilerCallback3::SetWebWorkerId](../../winscript/reference/iactivescriptprofilercallback3-setwebworkerid-method.md)|Powiadamia profilera o identyfikatorze procesu roboczego do użycia dla tej sesji profilowania.|
