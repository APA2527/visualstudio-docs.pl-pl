# <a name="iactivescriptgarbagecollector-interface"></a>Interfejs IActiveScriptGarbageCollector
Udostępnia metodę uruchamiania odzyskiwania pamięci. Ten interfejs powinien być zaimplementowany przez aparaty aktywnych skryptów, które chcą oczyścić swoje zasoby.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|  
|------------|-----------------|  
|[IActiveScriptGarbageCollector::CollectGarbage](../../winscript/reference/iactivescriptgarbagecollector-collectgarbage.md)|Uruchamia odzyskiwanie pamięci.|
