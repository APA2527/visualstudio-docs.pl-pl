# <a name="ijsdebugprocessperformasyncbreak-method"></a>IJsDebugProcess::PerformAsyncBreak — Metoda
Umieszcza aparat skryptu w trybie przerwania powodowania podział na następną instrukcję skryptu.  
  
## <a name="syntax"></a>Składnia  
  
```  
HRESULT PerformAsyncBreak(  
   DWORD threadId  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `threadId`  
 [in] Identyfikator wątku.  
  
## <a name="return-value"></a>Wartość zwracana
