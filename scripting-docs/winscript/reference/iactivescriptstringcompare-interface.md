# <a name="iactivescriptstringcompare-interface"></a>Interfejs IActiveScriptStringCompare
Interfejs `IActiveScriptCompare` służy do definiowania metody porównywania ciągów dla aparatu obsługi skryptów.  
  
## <a name="methods-in-vtable-order"></a>Metody w kolejności tablic wirtualnych  
  
|Metoda|Opis|  
|------------|-----------------|  
|[IActiveScriptStringCompare::StrComp](../../winscript/reference/iactivescriptstringcompare-strcomp.md)|Określa metodę porównywania ciągów dla aparatu obsługi skryptów.|
