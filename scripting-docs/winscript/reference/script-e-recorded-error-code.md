# <a name="script_e_recorded-error-code"></a>Kod błędu SCRIPT_E_RECORDED
Przekazano błąd między aparatem skryptu i hostem. Host musi przekazać kod błędu do obiektu wywołującego.  
  
## <a name="syntax"></a>Składnia  
  
```cpp  
SCRIPT_E_RECORDED   0x86664004L  
```
