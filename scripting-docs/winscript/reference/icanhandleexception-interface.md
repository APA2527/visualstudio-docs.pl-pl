# <a name="icanhandleexception-interface"></a>Interfejs ICanHandleException
Umożliwia obiektowi wywołującemu obiektowi wywołującemu aparat skryptu określenie wyjątki, które obsługuje.  
  
## <a name="methods"></a>Metody  
 Oprócz metod odziedziczone `IUnknown`, `ICanHandleException` interfejsu udostępnia następujące metody.  
  
|Metoda|Opis|  
|------------|-----------------|  
|[ICanHandleException::CanHandleException](../../winscript/reference/icanhandleexception-canhandleexception.md)|Określa, jeśli obiekt wywołujący aparat skryptu może obsłużyć określony wyjątek.|
