# <a name="icanhandleexception-interface"></a>Interfejs ICanHandleException
Umożliwia obiekt wywołujący aparatu skryptu, aby określić, które wyjątki wywołującego uchwytów.  
  
## <a name="methods"></a>Metody  
 Oprócz dziedziczone z metody `IUnknown`, `ICanHandleException` interfejsu udostępnia następujące metody.  
  
|Metoda|Opis|  
|------------|-----------------|  
|[ICanHandleException::CanHandleException](../../winscript/reference/icanhandleexception-canhandleexception.md)|Określa, czy element wywołujący aparat skryptu może obsłużyć określony wyjątek.|
