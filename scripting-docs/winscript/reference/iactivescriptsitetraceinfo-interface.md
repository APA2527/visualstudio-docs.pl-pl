# <a name="iactivescriptsitetraceinfo-interface"></a>Interfejs IActiveScriptSiteTraceInfo
Udostępnia metodę wysyłania informacji śledzenia skryptu.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Opis|
|-|-|
|[IActiveScriptSiteTraceInfo::SendScriptTraceInfo, metoda](../../winscript/reference/iactivescriptsitetraceinfo-sendscripttraceinfo-method.md)|Wysyła informacje o śledzeniu, które obejmują typ zdarzenia, kontekst i instrukcję skryptu.|
