## <a name="methods"></a>Metody  
  
|||  
|-|-|  
|Metoda|Opis|  
|[IActiveScriptSiteTraceInfo::SendScriptTraceInfo, metoda](../../winscript/reference/iactivescriptsitetraceinfo-sendscripttraceinfo-method.md)|Wysyła informacje o śledzeniu, zawierający typ zdarzenia, kontekstu i instrukcji skryptu.|
