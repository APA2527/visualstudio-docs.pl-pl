# <a name="ibindeventhandler-interface"></a>Interfejs IBindEventHandler
Włącza obiekt wywołujący, aby zapewnić obiekt, który obsługuje określona procedura obsługi zdarzeń.  
  
## <a name="methods"></a>Metody  
 Oprócz metod odziedziczone `IUnknown`, `IBindEventHandler` interfejsu udostępnia następujące metody.  
  
|Metoda|Opis|  
|------------|-----------------|  
|[IBindEventHandler::BindHandler](../../winscript/reference/ibindeventhandler-bindhandler.md)|Wiąże zdarzenia do obiektu.|
