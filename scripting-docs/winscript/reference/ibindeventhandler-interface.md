# <a name="ibindeventhandler-interface"></a>Interfejs IBindEventHandler
Umożliwia obiekt wywołujący, aby zapewnić obiekt, który obsługuje obsługi określonego zdarzenia.  
  
## <a name="methods"></a>Metody  
 Oprócz dziedziczone z metody `IUnknown`, `IBindEventHandler` interfejsu udostępnia następujące metody.  
  
|Metoda|Opis|  
|------------|-----------------|  
|[IBindEventHandler::BindHandler](../../winscript/reference/ibindeventhandler-bindhandler.md)|Wiąże zdarzenia obiektu.|
