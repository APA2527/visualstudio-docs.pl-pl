# <a name="iactivescriptauthorprocedure-interface"></a>Interfejs IActiveScriptAuthorProcedure
Oprócz dziedziczone z metody `IUnknown`, `IScriptAuthorProcedure` interfejsu udostępnia następujące metody.  
  
## <a name="methods-in-vtable-order"></a>Metody w kolejności Vtable  
  
|Metoda|Opis|  
|------------|-----------------|  
|[IActiveScriptAuthorProcedure::ParseProcedureText](../../winscript/reference/iactivescriptauthorprocedure-parseproceduretext.md)|Analizuje procedury kodu, dodaje tekst procedury kodu do skryptu tworzenia aparatu i tworzy `IScriptEntry` obiekt, który odpowiada procedury kodu.|
