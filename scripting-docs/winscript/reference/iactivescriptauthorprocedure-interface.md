# <a name="iactivescriptauthorprocedure-interface"></a>Interfejs IActiveScriptAuthorProcedure
Oprócz metod dziedziczonych z `IUnknown` interfejs `IScriptAuthorProcedure` uwidacznia następującą metodę.  
  
## <a name="methods-in-vtable-order"></a>Metody w kolejności tablic wirtualnych  
  
|Metoda|Opis|  
|------------|-----------------|  
|[IActiveScriptAuthorProcedure::ParseProcedureText](../../winscript/reference/iactivescriptauthorprocedure-parseproceduretext.md)|Analizuje procedurę kodu, dodaje tekst procedury kodu do aparatu tworzenia skryptów i tworzy obiekt `IScriptEntry`, który odnosi się do procedury kodu.|
