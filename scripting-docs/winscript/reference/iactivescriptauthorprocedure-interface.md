# <a name="iactivescriptauthorprocedure-interface"></a>Interfejs IActiveScriptAuthorProcedure
Oprócz metod odziedziczone `IUnknown`, `IScriptAuthorProcedure` interfejsu udostępnia następujące metody.  
  
## <a name="methods-in-vtable-order"></a>Metody w Vtable kolejności  
  
|Metoda|Opis|  
|------------|-----------------|  
|[IActiveScriptAuthorProcedure::ParseProcedureText](../../winscript/reference/iactivescriptauthorprocedure-parseproceduretext.md)|Analizuje kod procedury, dodaje tekst procedury kodu do skryptu tworzenia aparatu i tworzy `IScriptEntry` obiekt, który odnosi się do procedury kodu.|
