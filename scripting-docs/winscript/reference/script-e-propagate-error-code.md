# <a name="scriptepropagate-error-code"></a>Kod błędu SCRIPT_E_PROPAGATE
Błąd skryptu jest przekazywana dalej do elementu wywołującego, która może być w innym wątku. Host należy przekazać kod błędu do obiektu wywołującego.  
  
## <a name="syntax"></a>Składnia  
  
```cpp  
SCRIPT_E_PROPAGATE  0x80020102L  
```
