# <a name="javascript-fundamentals"></a>Podstawowe założenia JavaScript
[!INCLUDE[javascript](../javascript/includes/javascript-md.md)]jest interpretowany, obiektowy język skryptów.  
  
 [!INCLUDE[javascript](../javascript/includes/javascript-md.md)] Języka używa składni podobny do C i obsługuje strukturę konstrukcje, takich jak `if...else`, `for`, i `do...while`. Nawiasy klamrowe ({}) służą do ograniczania bloków instrukcji. Język obsługuje różne typy danych, w tym `String`, `Number`, `Boolean`, `Object`, i `Array`. Zawiera obsługę funkcji rozszerzonej daty, funkcje trygonometryczne i wyrażenia regularne.  
  
 [!INCLUDE[javascript](../javascript/includes/javascript-md.md)]używa prototypy zamiast klasy. Tworząc konstruktora funkcji można określić obiektu.  
  
 [!INCLUDE[javascript](../javascript/includes/javascript-md.md)]to język słabo typizowana, co oznacza, że nie jawnie deklarować typów zmiennych. W wielu przypadkach [!INCLUDE[javascript](../javascript/includes/javascript-md.md)] wykonuje konwersje automatycznie, gdy są potrzebne. Na przykład jeśli dodasz wiele pozycji, która składa się z tekstu (ciąg), liczba jest konwertowana na tekst.
