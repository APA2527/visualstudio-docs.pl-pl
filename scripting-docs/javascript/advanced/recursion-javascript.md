# <a name="recursion-javascript"></a>Rekursja (JavaScript)
Rekursja jest ważne technik programowania, w którym funkcja wywołuje się.  
  
## <a name="an-example-of-recursion"></a>Przykład rekursji  
 Przykładem jest obliczenie silni. Silnię liczby  *n*  jest obliczany przez pomnożenie 1 \* 2 \* 3 \*... *n*. Poniższy przykład przedstawia sposób obliczania silni wielokrotnie powtarzane, oznacza to, za pomocą `while` pętli w jest obliczana wynik.  
  
```JavaScript  
function factorial(num)  
{  
    // If the number is less than 0, reject it.  
    if (num < 0) {  
        return -1;  
    }  
    // If the number is 0, its factorial is 1.  
    else if (num == 0) {  
        return 1;  
    }  
    var tmp = num;  
    while (num-- > 2) {  
        tmp *= num;  
    }  
    return tmp;  
}  
  
var result = factorial(8);  
document.write(result);  
  
// Output: 40320  
```  
  
 Istnieje możliwość cyklicznego przykład bardzo prosty. Zamiast `while` pętli do obliczenia wartości, możesz po prostu wywołać `factorial` ponownie, przekazując następnej najmniejszej wartości. Rekursja zatrzymuje, gdy wartość jest równa 1.  
  
```JavaScript  
function factorial(num)  
{  
    // If the number is less than 0, reject it.  
    if (num < 0) {  
        return -1;  
    }  
    // If the number is 0, its factorial is 1.  
    else if (num == 0) {  
        return 1;  
    }  
    // Otherwise, call this recursive procedure again.  
    else {  
        return (num * factorial(num - 1));  
    }  
}  
  
var result = factorial(8);  
document.write(result);  
  
// Output: 40320  
```
