# <a name="expected-"></a>Oczekiwano znaku „:"
Podjęto próbę utworzenia wyrażenia za pomocą operatora warunkowego Trzyelementowy, ale nie zawiera on dwukropka między drugim i trzecim argumentem operacji. Operator warunkowy Trzyelementowy (trzy operandy) wymaga dwukropka między drugim (true) i trzecimi operandami (false).  
  
### <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Wstaw dwukropek między drugim i trzecim argumentem operacji.
