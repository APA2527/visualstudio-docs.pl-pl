# <a name="expected-3939"></a>Oczekiwano &#39;: &#39;
Podjęto próbę utworzenia wyrażenia przy użyciu operatora warunkowego trójargumentowy, ale nie zawiera dwukropek między drugiego i trzeciego argumentu operacji. Operator warunkowy (trzy argumenty) trójargumentowy wymaga dwukropkiem między sekundy (true) oraz innych (false) argumentów operacji.  
  
### <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Wstaw dwukropkiem między drugiego i trzeciego argumentu operacji.
