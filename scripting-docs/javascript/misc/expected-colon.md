# <a name="expected-"></a>Oczekiwano znaku „:"
Podjęto próbę utworzenia wyrażenia używającego trójargumentowy operatora warunkowego, ale nie zawiera dwukropka między drugi i trzeci operand. Operator warunkowy trójargumentowy (trzy operandy) wymaga dwukropek między sekundy (true) oraz innych (false) argumentów.  
  
### <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Wstaw dwukropek między drugi i trzeci operand.
