# <a name="the-use-of-a-future-reserved-word-for-an-identifier-is-invalid"></a>Zastosowanie przyszłego słowa zastrzeżonego w roli identyfikatora jest nieprawidłowe
1049: użycie zarezerwowanych na przyszłość słów jako identyfikator jest nieprawidłowy  
  
 Nie można użyć zarezerwowanych na przyszłość słów jako nazwę zmiennej, funkcji lub innego identyfikatora użytkownika.  
  
 Aby uzyskać więcej informacji na temat przyszłych słów zastrzeżonych, zobacz [JavaScript przyszłe słowa zastrzeżone](../../javascript/reference/javascript-future-reserved-words.md).
