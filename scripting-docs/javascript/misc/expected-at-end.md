# <a name="expected-end"></a>Oczekiwano "\@end"
Podjęto próbę utworzenia skompilowanego warunkowo bloku kodu, ale nie uwzględniono instrukcji <strong>@end</strong> na końcu. instrukcje [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] mogą być kompilowane warunkowo, umieszczając je w bloku <strong>@if/ @end</strong> .  
  
### <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj odpowiednią instrukcję <strong>@end</strong> .
