# <a name="expected-3939"></a>Oczekiwano &#39; {&#39;
Nie dołączono lewy nawias klamrowy, który oznacza początek treści funkcji. Kod, który stanowi treści funkcji, nawet jeśli jest jednym wierszu, musi być zawarty w nawiasy lewe i prawe. Należy zauważyć, że użycie nawiasów klamrowych w treści funkcji jest bardziej restrykcyjne niż w pętli.  
  
### <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj lewy nawias klamrowy, który oznacza początek treści funkcji.
