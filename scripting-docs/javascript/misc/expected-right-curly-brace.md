# <a name="expected-3939"></a>Oczekiwano &#39;} &#39;
Nie zawierają prawy nawias klamrowy, oznacza koniec treści funkcji, pętla, blok kodu, ani inicjatora obiektu. Przykładem tego błędu może być **dla** pętli z tylko lewego nawiasu oznaczenie treści pętli.  
  
### <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj prawy nawias klamrowy, oznaczającą zakończenie funkcja pętli, bloku albo inicjatora obiektu.
