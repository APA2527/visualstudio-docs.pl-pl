# <a name="expected-3939-javascript"></a>Oczekiwano &#39; = &#39; (JavaScript)
Podjęto próbę utworzenia zmiennej ma być używany z instrukcji kompilacja warunkowa, ale nie umieścić znaku równości między zmienną i wartość, którą chcesz przypisać do niej.  
  
### <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj znak równości. Na przykład:  
  
    ```JavaScript  
    @set @myvar1 = 1  
    ```
