# <a name="label-not-found"></a>Nie odnaleziono etykiety
Wprowadzone odniesienie do etykiety, który nie istnieje. Etykiety mogą być używane do oznaczania bloki kodu, ale w określonym zakresie, musi być unikatowa.  
  
### <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, upewnij się, że nie błędnie nazwę etykiety.  
  
-   Upewnij się, wszystkich odwołań do etykiet zostały wprowadzone do etykiet, które zostały zdefiniowane w bieżącym zakresie (dotyczy to również do przodu definicje).
