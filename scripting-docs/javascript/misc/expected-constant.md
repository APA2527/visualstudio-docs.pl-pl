# <a name="expected-constant"></a>Oczekiwana stała
Próbowano użyć zmiennej (z systemem innym niż warunkowego kompilacji) w instrukcji testów kompilacji warunkowej. Kompilacja warunkowa instrukcje testu musi być stałą.  
  
### <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zastąp zmienną literału.  
  
-   Zastąp zmienną zmienną kompilacji warunkowej.
