# <a name="mathsin-function-javascript"></a>Math.sin — Funkcja (JavaScript)
Zwraca sinus liczby.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
Math.sin(radians)   
```  
  
## <a name="remarks"></a>Uwagi  
 `radians` Argument jest wyrażenia liczbowego, który zawiera kąt w radianach.  
  
 Wartość zwracana jest sinus argument numeryczny.
