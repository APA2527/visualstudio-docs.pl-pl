# <a name="tostring-method-string-1"></a>toString — metoda (ciąg) 1
Zwraca reprezentację ciągu.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
string.toString()  
```  
  
## <a name="parameters"></a>Parametry  
 `string`  
 Wymagany. Tablica do reprezentowania jako ciąg.  
  
## <a name="return-value"></a>Wartość zwracana  
 Reprezentacja ciągu ciągu.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład przedstawia użycie **toString** metody z ciągiem.  
  
```JavaScript  
var string = "this is a test";  
var strStr = string.toString();  
document.write(strStr);  
  
// Output: this is a test  
  
```
