# <a name="tostring-method-weakmap-javascript"></a>toString — Metoda (WeakMap) (JavaScript)
Zwraca reprezentację ciągu `WeakMap` obiektu.  
  
## <a name="syntax"></a>Składnia  
  
```JavaScript  
weakmapObj.toString()  
```  
  
#### <a name="parameters"></a>Parametry  
 `weakmapObj`  
 Wymagany. A `WeakMap` obiektu.  
  
## <a name="property-valuereturn-value"></a>Wartość właściwości/Zwracana wartość  
 Reprezentacja ciągu `WeakMap`.
