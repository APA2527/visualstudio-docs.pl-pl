# <a name="resolvedoptions-method-intlnumberformat"></a>resolvedOptions — Metoda (Intl.NumberFormat)
Zwraca obiekt, który zawiera właściwości i wartości `NumberFormat` obiektem programu formatującego.  
  
## <a name="syntax"></a>Składnia  
  
```  
numberFormatObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>Parametry  
 `numberFormatObj`  
 `NumberFormat` Obiekt do pobrania informacji z.  
  
## <a name="remarks"></a>Uwagi  
 Właściwości obiektu zwróconego odpowiada właściwości obliczanej `NumberFormat` obiektu.
