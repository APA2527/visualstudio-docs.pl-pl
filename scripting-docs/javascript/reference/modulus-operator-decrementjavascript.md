# <a name="modulus-operator--javascript"></a>Operator modulo (JavaScript)
Dzieli wartość wyrażenia liczbowego przez wartość innego wyrażenia liczbowego i tworzy resztę.  
  
## <a name="syntax"></a>Składnia  
  
```  
result = expression1 % expression2  
```  
  
## <a name="arguments"></a>Argumenty  
 `result`  
 Dowolnej zmiennej.  
  
 `expression1`  
 Dowolne wyrażenie liczbowe.  
  
 `expression2`  
 Dowolne wyrażenie liczbowe.
