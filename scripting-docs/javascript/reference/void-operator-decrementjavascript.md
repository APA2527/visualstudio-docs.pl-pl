# <a name="void-operator-javascript"></a>void — Operator (JavaScript)
Zapobiega, zwraca wartość wyrażenia.  
  
## <a name="syntax"></a>Składnia  
  
```  
void expression   
```  
  
## <a name="remarks"></a>Uwagi  
 `expression` Argument jest dowolny prawidłowy [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] wyrażenia.  
  
 `void` Operator ocenia jej wyrażenia i zwraca `undefined`. Jest to przydatne w sytuacjach, w którym powinny być oceniane wyrażenia, ale nie ma wyników widoczne z resztą skryptu.
