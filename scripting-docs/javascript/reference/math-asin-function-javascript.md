# <a name="mathasin-function-javascript"></a>Math.asin — Funkcja (JavaScript)
Zwraca arcus sinus liczby.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
Math.asin(number)   
```  
  
## <a name="remarks"></a>Uwagi  
 Wymagane `number` argument jest wyrażenia liczbowego, dla którego sinus jest wymagana.  
  
 Wartość zwracana jest arcus sinus `number` argument w radianach.
