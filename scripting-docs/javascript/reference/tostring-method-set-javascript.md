# <a name="tostring-method-set-javascript"></a>toString — Metoda (Set) (JavaScript)
Zwraca reprezentację ciągu zestawu.  
  
## <a name="syntax"></a>Składnia  
  
```JavaScript  
setObj.toString()  
```  
  
#### <a name="parameters"></a>Parametry  
 `setObj`  
 Wymagany. A `Set` obiektu.  
  
## <a name="property-valuereturn-value"></a>Wartość właściwości/Zwracana wartość  
 Reprezentacja ciągu w zestawie.
