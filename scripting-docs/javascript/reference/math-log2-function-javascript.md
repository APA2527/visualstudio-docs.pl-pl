# <a name="mathlog2-function-javascript"></a>Math.log2 — funkcja (JavaScript)
Zwraca logarytm 2 dla danej liczby.  
  
## <a name="syntax"></a>Składnia  
  
```  
Math.log2(x)   
```  
  
## <a name="return-value"></a>Wartość zwracana  
 Jeśli `x` jest `NaN`, funkcja zwraca `NaN`. Jeśli `x` jest mniejsza niż zero (0), funkcja zwraca `NaN`.
