# <a name="assignment-operator--javascript"></a>Operator przypisania (=) (JavaScript)
Przypisuje wartość do zmiennej.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
result = expression  
```  
  
## <a name="parameters"></a>Parametry  
 `result`  
 Dowolnej zmiennej.  
  
 `expression`  
 Dowolne wyrażenie liczbowe.  
  
## <a name="remarks"></a>Uwagi  
 = — Operator zachowuje się jak innych operatorów, więc wyrażeń, które zawierają jego wartość. Oznacza to, że tworzenia łańcucha operatory przypisania w następujący sposób: `j = k = l = 0`. W takim przypadku `j`, `k`, i `l` równy zero.
