# <a name="global-object-javascript"></a>Obiekt Global (JavaScript)
Obiektu wewnętrznego, których celem jest zbieranie funkcji globalnych i stałych do jednego obiektu.  
  
## <a name="remarks"></a>Uwagi  
 `Global` Obiekt nie jest nigdy używana bezpośrednio i nie można utworzyć przy użyciu `new` operatora. Jest on tworzony podczas inicjowania aparat skryptów, dzięki czemu jego funkcji i stałe dostępne natychmiast.
