# <a name="mathcos-function-javascript"></a>Math.cos — Funkcja (JavaScript)
Zwraca cosinus liczby.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
Math.cos(radians)   
```  
  
## <a name="remarks"></a>Uwagi  
 Wymagane `radians` argument jest wyrażenia liczbowego, który zawiera kąt w radianach.  
  
 Wartość zwracana jest cosinus liczbowych argument `radians`.
