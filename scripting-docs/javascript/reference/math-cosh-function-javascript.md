# <a name="mathcosh-function-javascript"></a>Math.cosh — funkcja (JavaScript)
Zwraca cosinus hiperboliczny liczby.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
Math.cosh(x)   
```  
  
## <a name="remarks"></a>Uwagi  
 Zwracana wartość jest taka sama jak `(exp(x) + exp(-x))/2`. Jeśli `x` jest `NaN`, wynikiem jest `NaN`.
