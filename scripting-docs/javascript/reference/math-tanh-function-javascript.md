# <a name="mathtanh-function-javascript"></a>Math.tanh — funkcja (JavaScript)
Zwraca tangens hiperboliczny liczby.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
Math.tanh(  
x  
)  
  
```  
  
## <a name="remarks"></a>Uwagi  
 Zwracana wartość jest taka sama jak `(exp(x) - exp(-x))/(exp(x) + exp(-x))`. Jeśli `x` jest `NaN`, wynikiem jest `NaN`.
