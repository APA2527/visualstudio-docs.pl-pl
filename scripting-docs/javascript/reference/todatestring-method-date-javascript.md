# <a name="todatestring-method-date-javascript"></a>toDateString — Metoda (Data) (JavaScript)
Zwraca daty w postaci wartości ciągu.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
objDate.toDateString( )  
```  
  
## <a name="remarks"></a>Uwagi  
 Wymagane *objDate* odwołanie jest `Date` obiektu.  
  
 `toDateString` Metoda zwraca ciąg zawierający Data, w bieżącej strefy czasowej w dogodnym, łatwo odczytać formatu.
