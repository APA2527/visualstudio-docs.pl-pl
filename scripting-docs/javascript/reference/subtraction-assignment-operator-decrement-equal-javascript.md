# <a name="subtraction-assignment-operator---javascript"></a>Operator przypisania odejmowania (-=) (JavaScript)
Odejmuje wartość wyrażenia z wartości zmiennej, a następnie przypisuje wynik do zmiennej.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
result -= expression  
```  
  
## <a name="parameters"></a>Parametry  
 *wynik*  
 Wszelkie liczbowa zmiennej.  
  
 *wyrażenie*  
 Dowolne wyrażenie liczbowe.  
  
## <a name="remarks"></a>Uwagi  
 Przy użyciu `-=` operator jest dokładnie taka sama jak wykonanie następujących czynności:  
  
```JavaScript  
result = result - expression  
```
