# <a name="valueof-method-set-javascript"></a>valueOf — Metoda (Set) (JavaScript)

## <a name="syntax"></a>Składnia  
  
```JavaScript  
setObj.valueOf()  
```  
  
#### <a name="parameters"></a>Parametry  
 `setObj`  
 Wymagany. A `Set` obiektu.
