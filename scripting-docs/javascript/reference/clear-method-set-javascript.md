# <a name="clear-method-set-javascript"></a>clear — Metoda (Set) (JavaScript)
Usuwa wszystkie elementy z zestawu.  
  
## <a name="syntax"></a>Składnia  
  
```JavaScript  
setObj.clear()  
```  
  
#### <a name="parameters"></a>Parametry  
 `setObj`  
 Wymagany. Zestaw, aby wyczyścić.
