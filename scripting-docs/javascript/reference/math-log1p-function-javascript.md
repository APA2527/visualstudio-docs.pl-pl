# <a name="mathlog1p-function-javascript"></a>Math.log1p — funkcja (JavaScript)
Zwraca logarytm naturalny 1 + liczbą.  
  
## <a name="syntax"></a>Składnia  
  
```  
Math.log1p(x)   
```  
  
## <a name="return-value"></a>Wartość zwracana  
 Jeśli `x` jest `NaN`, funkcja zwraca `NaN`. Jeśli `x` jest mniejsza niż zero (0), funkcja zwraca `NaN`.
