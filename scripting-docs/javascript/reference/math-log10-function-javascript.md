# <a name="mathlog10-function-javascript"></a>Math.log10 — funkcja (JavaScript)
Zwraca logarytm 10 dla danej liczby.  
  
## <a name="syntax"></a>Składnia  
  
```  
Math.log10(x)   
```  
  
## <a name="return-value"></a>Wartość zwracana  
 Jeśli `x` jest `NaN`, funkcja zwraca `NaN`. Jeśli `x` jest mniejsza niż zero (0), funkcja zwraca `NaN`.
