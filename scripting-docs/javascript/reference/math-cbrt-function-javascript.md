# <a name="mathcbrt-function-javascript"></a>Math.cbrt — funkcja (JavaScript)
Zwraca pierwiastek modułu liczby.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
Math.cbrt(  
number  
)   
```  
  
## <a name="remarks"></a>Uwagi  
 Jeśli `number` jest `NaN`, wynikiem będzie `NaN`.
