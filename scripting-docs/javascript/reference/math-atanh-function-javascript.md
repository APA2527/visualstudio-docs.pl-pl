# <a name="mathatanh-function-javascript"></a>Math.atanh — funkcja (JavaScript)
Zwraca odwrotny tangens hiperboliczny liczby.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
Math.atanh(x)   
```  
  
## <a name="remarks"></a>Uwagi  
 Jeśli `x` jest `NaN`, wynikiem jest `NaN`.
