# <a name="size-property-set-javascript"></a>size — Właściwość (Set) (JavaScript)
Zwraca liczbę elementów w zestawie.  
  
## <a name="syntax"></a>Składnia  
  
```JavaScript  
sizeVar = setObj.size  
```  
  
#### <a name="parameters"></a>Parametry  
 `sizeVar`  
 Wymagany. Dowolna liczba.  
  
 `setObj`  
 Wymagany. Wszelkie `Set` obiektu.
