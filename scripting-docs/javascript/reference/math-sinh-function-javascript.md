# <a name="mathsinh-function-javascript"></a>Math.sinh — funkcja (JavaScript)
Zwraca odwrotny sinus hiperboliczny liczby.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
Math.sinh(x)   
```  
  
## <a name="remarks"></a>Uwagi  
 Zwracana wartość jest taka sama jak `(exp(x) - exp(-x))/2`. Jeśli x to `NaN`, wynikiem jest `NaN`.
