# <a name="mathsqrt-function-javascript"></a>Math.sqrt — Funkcja (JavaScript)
Zwraca pierwiastek kwadratowy z liczby.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
Math.sqrt(  
number  
)   
```  
  
## <a name="remarks"></a>Uwagi  
 Wymagane `number` argument jest wyrażenia liczbowego.  
  
 Jeśli `number` jest ujemna, jest zwracana wartość `NaN`.
