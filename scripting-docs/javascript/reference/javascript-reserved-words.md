# <a name="javascript-reserved-words"></a>JavaScript Słowa zastrzeżone
[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]zawiera liczbę słów zastrzeżonych, których nie można używać jako identyfikatorów. Słowa zastrzeżone mają określone znaczenie w celu [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] języka, ponieważ są one częścią składni języka. Przy użyciu słowem zastrzeżonym powoduje błąd kompilacji podczas ładowania skryptu.  
  
 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]zawiera także listę przyszłe słowa zastrzeżone. Te słowa, które nie są obecnie częścią [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] języka, chociaż są one zarezerwowane do użytku w przyszłości. Aby uzyskać więcej informacji na temat przyszłych zastrzeżonych słów kluczowych w języku JavaScript, zobacz [JavaScript przyszłe słowa zastrzeżone](../../javascript/reference/javascript-future-reserved-words.md).  
  
 W przypadku wybrania identyfikatory ważne jest również w celu uniknięcia dowolnego słowa, które są już nazwy wewnętrznej [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] obiekty lub funkcje, takie jak `String` lub `parseInt`.
