# <a name="mathatan2-function-javascript"></a>Math.atan2 — Funkcja (JavaScript)
Zwraca kąt (w radianach) osi X do punktu (*y*,*x*).  
  
## <a name="syntax"></a>Składnia  
  
```  
  
Math.atan2(y, x)  
```  
  
## <a name="parameters"></a>Parametry  
 *x*  
 Wymagany. Wyrażenie liczbowe reprezentujący kartezjańskimi współrzędną x.  
  
 *y*  
 Wymagany. Wyrażenie liczbowe reprezentujący kartezjańskimi współrzędną y.  
  
## <a name="remarks"></a>Uwagi  
 Wartość zwracana jest między *-pi* i *pi*. Reprezentuje kąt podane (*y*,*x*) punkt w radianach.
