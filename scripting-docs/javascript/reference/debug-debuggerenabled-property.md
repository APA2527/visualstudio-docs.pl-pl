# <a name="debugdebuggerenabled-property"></a>Debug.debuggerEnabled — właściwość
Określa, czy włączone jest debugowanie skryptu kontekstu. Debugowanie może być włączona lub wyłączona lub nie jest dołączony debuger.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
var dbgEnabled = Debug.debuggerEnabled;  
```
