# <a name="mathexp-function-javascript"></a>Math.exp — Funkcja (JavaScript)
Zwraca wartość liczby e (podstawę logarytmów naturalnych) podniesionej do potęgi.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
Math.exp(number)   
```  
  
## <a name="remarks"></a>Uwagi  
 Wymagane `number` argument jest wyrażenia liczbowego reprezentujący możliwości *e*.  
  
 Zwracana wartość jest liczbą. Stała *e* jest Eulera liczba, jest równa około 2.71828 i `number` jest podany argument.
