# <a name="mathfloor-function-javascript"></a>Math.floor — Funkcja (JavaScript)
Zwraca największą liczbę całkowitą mniejszą niż jej argument numeryczny.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
Math.floor(number)   
```  
  
## <a name="remarks"></a>Uwagi  
 Wymagane *numer* argument jest wyrażenia liczbowego.  
  
 Wartość zwracana jest wartość całkowitą równą największą liczbę całkowitą mniejszą lub równą jej argument numeryczny.
