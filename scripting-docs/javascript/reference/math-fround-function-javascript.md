# <a name="mathfround-function-javascript"></a>Math.fround — funkcja (JavaScript)
Zwraca najbliższej formatu liczb zmiennoprzecinkowych pojedynczej precyzji liczby.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
Math.fround(  
number  
)   
```  
  
## <a name="remarks"></a>Uwagi  
 Jeśli `number` jest `NaN`, wynikiem jest `NaN`.
