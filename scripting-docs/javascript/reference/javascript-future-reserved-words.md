# <a name="javascript-future-reserved-words"></a>JavaScript Przyszłe słowa zastrzeżone
W języku JavaScript przyszłych zastrzeżonych słów kluczowych nie można używać jako identyfikatorów, mimo że mają specjalnego znaczenia w bieżącej wersji.  
  
 Aby uzyskać listę słowa zastrzeżone w języku JavaScript, zobacz [JavaScript słowa zastrzeżone](../../javascript/reference/javascript-reserved-words.md).  
  
## <a name="future-reserved-words"></a>Przyszłe słowa zastrzeżone  
 Przyszłe słowa zastrzeżone są następujące:  
  
1.  class  
  
2.  enum  
  
3.  export  
  
4.  Rozszerza  
  
5.  import  
  
6.  Super  
  
## <a name="future-reserved-words-in-strict-mode"></a>Przyszłe słowa zastrzeżone w trybie z ograniczeniami  
 Poniżej przedstawiono przyszłe słowa zastrzeżone tylko w trybie z ograniczeniami. Aby uzyskać więcej informacji o trybie z ograniczeniami, zobacz [use strict — dyrektywa](../../javascript/reference/use-strict-directive.md).  
  
1.  implements  
  
2.  interface  
  
3.  Pakiet  
  
4.  private  
  
5.  protected  
  
6.  public  
  
7.  static  
  
8.  yield
