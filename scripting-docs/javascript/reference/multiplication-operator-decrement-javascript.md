# <a name="multiplication-operator--javascript"></a>Operator mnożenia (*) (JavaScript)
Mnoży wartości dwóch wyrażeń.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
result = number1*number2  
```  
  
## <a name="parameters"></a>Parametry  
 *wynik*  
 Dowolnej zmiennej.  
  
 `number1`  
 Dowolne wyrażenie.  
  
 `number2`  
 Dowolne wyrażenie.
