# <a name="resolvedoptions-method-intldatetimeformat"></a>resolvedOptions — Metoda (Intl.DateTimeFormat)
Zwraca obiekt, który zawiera właściwości i wartości daty/godziny obiektem programu formatującego.  
  
## <a name="syntax"></a>Składnia  
  
```  
dateTimeFormatObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>Parametry  
 `dateTimeFormatObj`  
 `DateTimeFormat` Obiekt do pobrania informacji z.  
  
## <a name="remarks"></a>Uwagi  
 Właściwości obiektu zwróconego odpowiada właściwości obliczanej `DateTimeFormat` obiektu.
