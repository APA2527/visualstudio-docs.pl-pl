# <a name="mathpow-function-javascript"></a>Math.pow — Funkcja (JavaScript)
Zwraca wartość wyrażenia podstawowe przekierowanie do określonej potęgi.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
Math.pow(  
base, exponent)   
```  
  
## <a name="parameters"></a>Parametry  
 *Podstawa*  
 Wymagany. Podstawową wartość wyrażenia.  
  
 *wykładnik*  
 Wymagany. Wykładnika wartość wyrażenia.  
  
## <a name="example"></a>Przykład  
 W poniższym przykładzie wyrażenia liczbowego równa baseexponent zwraca 1000.  
  
```JavaScript  
Math.pow(10,3);  
```
