# <a name="mathrandom-function-javascript"></a>Math.random — Funkcja (JavaScript)
Zwraca pseudolosowych liczbę z przedziału od 0 do 1.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
Math.random( )  
  
```  
  
## <a name="remarks"></a>Uwagi  
 Numer pseudolosowych generowany jest z zakresu od 0 (włącznie) do 1 (wyłączne), oznacza to, zwrócony kod może być równa zero, ale zawsze będzie mniejsza niż jedna. Generatora liczb losowych jest rozpoczęta automatycznie po [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] jest ładowana jako pierwsza.
