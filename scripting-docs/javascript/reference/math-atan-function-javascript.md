# <a name="mathatan-function-javascript"></a>Math.atan — Funkcja (JavaScript)
Zwraca arcus tangens liczby.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
Math.atan(number)   
```  
  
## <a name="remarks"></a>Uwagi  
 Wymagane `number` argument jest wyrażenia liczbowego, dla którego tangens jest wymagana.  
  
 Wartość zwracana jest arcus tangens `number` argument w radianach.
