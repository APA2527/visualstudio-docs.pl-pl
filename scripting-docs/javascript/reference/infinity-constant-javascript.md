# <a name="infinity-constant-javascript"></a>Stała Infinity (JavaScript)
Liczba, która jest większa niż największa liczba zmiennoprzecinkowa. Zwraca wartość początkową `Number.POSITIVE_INFINITY`. Minus nieskończoność (— wartości Infinity) jest mniejsza niż wartość zmiennoprzecinkowa najmniejszy numer punktu.  
  
## <a name="syntax"></a>Składnia  
  
```  
Infinity   
```  
  
## <a name="remarks"></a>Uwagi  
 `Infinity` Stałej jest elementem członkowskim `Global` obiektu i ma zostać udostępnione po zainicjowaniu aparatu skryptów.
