# <a name="mathtan-function-javascript"></a>Math.tan — Funkcja (JavaScript)
Zwraca tangens liczby.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
Math.tan(  
radians  
)  
  
```  
  
## <a name="remarks"></a>Uwagi  
 Wymagane `radians` argument jest wyrażenia liczbowego, który zawiera kąt w radianach.  
  
 Wartość zwracana jest tangens liczbowych argument `radians`.
