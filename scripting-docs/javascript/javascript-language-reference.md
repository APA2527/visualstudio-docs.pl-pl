# <a name="javascript-language-reference"></a>Dokumentacja języka JavaScript
[!INCLUDE[javascript](../javascript/includes/javascript-md.md)]jest językiem skryptowym, który można ją osadzić w innych aplikacjach i stron sieci web.  
  
 W niniejszej dokumentacji opisano implementację JavaScript przez firmę Microsoft. Implementacja jest zgodna z 5 wydniem specyfikacji języka ECMAScript (ECMAScript Language Specification 5th Edition). Udostępnia także dodatkowe funkcje, które nie znajdują się w normach ECMA.  
  
 Można użyć [!INCLUDE[javascript](../javascript/includes/javascript-md.md)] kodu w aplikacji przeglądarki HTML, CSS i modelu DOM (Document Object), reprezentujący HTML i przeglądarka obiektów.  
  
-   Aby dowiedzieć się, HTML, zobacz [odwołania HTML/XHTML](http://go.microsoft.com/fwlink/p/?LinkId=251007).  
  
-   Aby uzyskać informacje o CSS, zobacz [kaskadowych arkuszy stylów](http://go.microsoft.com/fwlink/p/?LinkId=251008).  
  
-   Aby uzyskać informacje o modelu DOM, zobacz [modelu DOM (Document Object)](http://go.microsoft.com/fwlink/p/?LinkId=251009).  
  
 Można również użyć [!INCLUDE[javascript](../javascript/includes/javascript-md.md)] kodu w aplikacji Sklepu Windows, które można uruchamiać na [!INCLUDE[win8](../javascript/includes/win8-md.md)] i [!INCLUDE[win81](../javascript/includes/win81-md.md)]i w aplikacjach systemu Windows platformy Uniwersalnej, które są uruchamiane w systemie Windows 10. Aplikacje ze Sklepu Windows mogą być opracowane w [!INCLUDE[vs_dev11_long](../javascript/includes/vs-dev11-long-md.md)], [!INCLUDE[vs_dev12](../javascript/includes/vs-dev12-md.md)], i [!INCLUDE[vs_dev14](../javascript/includes/vs-dev14-md.md)]; Aplikacji platformy uniwersalnej systemu Windows mogą być opracowane w [!INCLUDE[vs_dev14](../javascript/includes/vs-dev14-md.md)].  
  
-   Informacje o JavaScript w [!INCLUDE[win8_appname_long](../javascript/includes/win8-appname-long-md.md)] aplikacji, zobacz [plan JavaScript](https://msdn.microsoft.com/en-us/library/windows/apps/hh465037.aspx).  
  
-   Informacje o HTML i CSS w [!INCLUDE[win8_appname_long](../javascript/includes/win8-appname-long-md.md)] aplikacji, zobacz [HTML/CSS w przypadku aplikacji ze Sklepu Windows](http://go.microsoft.com/fwlink/p/?LinkId=250939).  
  
-   Aby uzyskać informacje o środowiska wykonawczego systemu Windows i biblioteka systemu Windows interfejsów API języka JavaScript, zobacz [dokumentacja interfejsu API dla środowiska wykonawczego systemu Windows i biblioteka systemu Windows dla języka JavaScript](http://go.microsoft.com/fwlink/p/?LinkID=250938).  
  
-   Aby dowiedzieć się, jak za pomocą interfejsu API środowiska wykonawczego systemu Windows z użyciem języka JavaScript, zobacz [przy użyciu środowiska uruchomieniowego systemu Windows w języku JavaScript](../jswinrt/using-the-windows-runtime-in-javascript.md).  
  
 Edytor kodu JavaScript w programie Visual Studio zapewnia obsługę technologii IntelliSense. Aby uzyskać więcej informacji, zobacz [IntelliSense dla JavaScript](http://go.microsoft.com/fwlink/p/?LinkId=256499).
