# <a name="javascript-language-reference"></a>JavaScript Odwołanie językowe

JavaScript jest językiem skryptowym, który może być osadzony w stronach sieci Web i innych aplikacjach.  
  
Implementacje języka JavaScript firmy Microsoft są zgodne z specyfikacją języka ECMAScript w wersji 5. Udostępnia także dodatkowe funkcje, które nie znajdują się w normach ECMA.

> [!IMPORTANT]
> Firma Microsoft dołączyła do społeczności użytkowników, aby [powiadomienia MDN w witrynie internetowej dokumenty](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference) z przewodnikiem dotyczącym programowania w sieci Web, zaczynając od przekierowania wszystkich 500 + stron dokumentacji interfejsu API języka JavaScript firmy Microsoft z [docs.Microsoft.com](https://docs.microsoft.com) do ich odpowiedników powiadomienia MDN i przenoszenia naszych starszych rozszerzeń js do celów historycznych i obsługi.
