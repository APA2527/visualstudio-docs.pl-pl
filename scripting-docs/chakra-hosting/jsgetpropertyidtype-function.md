# <a name="jsgetpropertyidtype-function"></a>JsGetPropertyIdType — funkcja
Pobiera typ właściwości.  
  
## <a name="syntax"></a>Składnia  
  
```  
STDAPI_(JsErrorCode) JsGetPropertyIdType(  
   _In_ JsPropertyIdRef propertyId,  
   _Out_ JsPropertyIdType* propertyIdType  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `propertyId`  
 Identyfikator właściwości można pobrać typu.  
  
 `propertyIdType`  
 `JsPropertyIdType` Dana właściwość identyfikatora.  
  
## <a name="return-value"></a>Wartość zwracana  
 Kod `JsNoError` Jeśli operacja powiodła się, kod błędu w przeciwnym razie wartość.  
  
## <a name="remarks"></a>Uwagi  
 Wymaga kontekstu aktywnego skryptu.  
  
 Ten interfejs API jest obsługiwana tylko w trybie krawędzi.  
  
## <a name="requirements"></a>Wymagania  
 **Nagłówek:** jsrt.h
