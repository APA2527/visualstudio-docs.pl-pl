# <a name="jsgetexternaldata-function"></a>JsGetExternalData — Funkcja
Pobiera dane z obiektu zewnętrznego.  
  
## <a name="syntax"></a>Składnia  
  
```  
STDAPI_(JsErrorCode) JsGetExternalData(  
   _In_ JsValueRef object,  
   _Out_ void **externalData  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `object`  
 Obiekt zewnętrzny.  
  
 `externalData`  
 Zewnętrzne dane przechowywane w obiekcie. Może mieć wartości null, jeśli nie dane zewnętrzne jest przechowywana w obiekcie.  
  
## <a name="return-value"></a>Wartość zwracana  
 Kod `JsNoError` Jeśli operacja powiodła się, kod błędu w przeciwnym razie wartość.  
  
## <a name="remarks"></a>Uwagi  
 Wymaga kontekstu aktywnego skryptu.  
  
## <a name="requirements"></a>Wymagania  
 **Nagłówek:** jsrt.h
