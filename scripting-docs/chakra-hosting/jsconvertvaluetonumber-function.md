# <a name="jsconvertvaluetonumber-function"></a>JsConvertValueToNumber — Funkcja
Konwertuje wartość na numer przy użyciu standardowa semantyka języka JavaScript.  
  
## <a name="syntax"></a>Składnia  
  
```  
STDAPI_(JsErrorCode) JsConvertValueToNumber(  
   _In_ JsValueRef value,  
   _Out_ JsValueRef *numberValue  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `value`  
 Wartość do konwersji.  
  
 `numberValue`  
 Przekonwertowana wartość.  
  
## <a name="return-value"></a>Wartość zwracana  
 Kod `JsNoError` Jeśli operacja powiodła się, kod błędu w przeciwnym razie wartość.  
  
## <a name="remarks"></a>Uwagi  
 Wymaga kontekstu aktywnego skryptu.  
  
## <a name="requirements"></a>Wymagania  
 **Nagłówek:** jsrt.h
