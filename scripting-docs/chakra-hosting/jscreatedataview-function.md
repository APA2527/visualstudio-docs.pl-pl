# <a name="jscreatedataview-function"></a>JsCreateDataView — funkcja
Tworzy Javascript `DataView` obiektu.  
  
## <a name="syntax"></a>Składnia  
  
```  
JsErrorCode  JsCreateDataView(  
   _In_ JsValueRef arrayBuffer,  
   _In_ unsigned int byteOffset,  
   _In_ unsigned int byteLength,  
   _Out_ JsValueRef *result  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `arrayBuffer`  
 Istniejące `ArrayBuffer` obiekt ma być używany jako magazyn dla wyniku `DataView` obiektu.  
  
 `byteOffset`  
 Przesunięcie w bajtach od początku `arrayBuffer` dla wyniku `DataView` do odwołania.  
  
 `byteLength`  
 Liczba bajtów w ArrayBuffer dla wyniku DataView do odwołania.  
  
 `result`  
 Nowy obiekt DataView.  
  
## <a name="return-value"></a>Wartość zwracana  
 Kod `JsNoError` Jeśli operacja powiodła się, kod błędu w przeciwnym razie wartość.  
  
## <a name="remarks"></a>Uwagi  
 Wymaga kontekstu aktywnego skryptu.  
  
 Ten interfejs API jest obsługiwana tylko w trybie krawędzi.

 **Nagłówek:** jsrt.h
