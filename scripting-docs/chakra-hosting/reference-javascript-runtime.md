# <a name="reference-javascript-runtime"></a>Odwołanie (środowisko uruchomieniowe JavaScript)

 Jeśli chcesz osadzić [ChakraCore](https://github.com/Microsoft/ChakraCore) w aplikacji, zapoznaj się z [ChakraCore Wiki](http://aka.ms/corejsrtref) dla JSRT zamiast tego odwołuje.
