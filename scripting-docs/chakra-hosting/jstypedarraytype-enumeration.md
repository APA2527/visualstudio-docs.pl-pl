# <a name="jstypedarraytype-enumeration"></a>JsTypedArrayType — wyliczenie
Typ maszynowy tablicy JavaScript.  
  
## <a name="syntax"></a>Składnia  
  
```cpp  
enum JsTypedArrayType;  
```  
  
## <a name="members"></a>Elementy członkowskie  
  
### <a name="values"></a>Wartości  
  
|Nazwa|Opis|  
|----------|-----------------|  
|`JsArrayTypeInt8`|Tablica int8.|  
|`JsArrayTypeUint8`|Tablica uint8.|  
|`JsArrayTypeUint8Clamped`|Uint8 zablokowane za pomocą tablicy.|  
|`JsArrayTypeInt16`|Tablica wartości int16.|  
|`JsArrayTypeUint16`|Tablica uint16.|  
|`JsArrayTypeInt32`|Tablica int32.|  
|`JsArrayTypeUint32`|Tablica uint32.|  
|`JsArrayTypeFloat32`|Tablica float32.|  
|`JsArrayTypeFloat64`|Tablica float64.|  
  
## <a name="requirements"></a>Wymagania  
 **Nagłówek:** jsrt.h
