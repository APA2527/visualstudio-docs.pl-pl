# <a name="jsvalueref-typedef"></a>JsValueRef — Typedef
Odwołanie do wartości JavaScript.  
  
## <a name="syntax"></a>Składnia  
  
```  
typedef JsRef JsValueRef;  
```  
  
## <a name="remarks"></a>Uwagi  
 Wartość JavaScript jest jednym z następujących typów wartości: niezdefiniowana, Null, Boolean, ciąg, numer lub obiekt.  
  
## <a name="requirements"></a>Wymagania  
 **Nagłówek:** jsrt.h
