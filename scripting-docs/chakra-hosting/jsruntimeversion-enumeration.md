# <a name="jsruntimeversion-enumeration"></a>JsRuntimeVersion — Wyliczenie
Wersja środowiska uruchomieniowego.  
  
## <a name="syntax"></a>Składnia  
  
```  
enum JsRuntimeVersion;  
```  
  
## <a name="members"></a>Elementy członkowskie  
  
### <a name="values"></a>Wartości  
  
|Nazwa|Opis|  
|----------|-----------------|  
|`JsRuntimeVersion10`|Tworzenie środowiska uruchomieniowego z wersją programu IE10.|  
|`JsRuntimeVersion11`|Utworzono wersji IE11 środowiska wykonawczego.|  
|`JsRuntimeVersionEdge`|Tworzenie środowiska wykonawczego o najwyższej wersji na komputerze w czasie wykonywania.|  
  
## <a name="requirements"></a>Wymagania  
 **Nagłówek:** jsrt.h
