# <a name="how-to-report-a-problem-in-visual-studio-for-mac"></a>Jak zgłosić problem w programie Visual Studio dla komputerów Mac

Jeśli wystąpi problem z programem Visual Studio dla komputerów Mac, chcemy wiedzieć o go tak, aby firma Microsoft może zdiagnozować i rozwiązać ten problem. 

## <a name="how-to-report-a-problem"></a>Jak zgłosić problem

Aby zgłosić problem z programem Visual Studio dla komputerów Mac, wybierz **Pomoc > Zgłoś Problem** na pasku menu:

![Zgłoś problem łącze](media/report-problem-image1.png)

Można również użyć **zgłosić Problem** łącze znajduje się na stronie powitalnej.

Spowoduje to wyświetlenie **zgłosić Problem** strony sieci web. Ta strona służy do wyświetlania wszystkich istniejących problemów, oddawać głosy na podobnych problemów i przesyłanie rozwiązania problemów zgłoszonych przez inne osoby:

![Zgłoś problem strony sieci Web](media/report-problem-image2.png)

Wybierz **zgłosić Problem** przycisk na stronie sieci web. 

Jeśli nie zostało już zarejestrowane w społeczność deweloperów aplikacji, zostanie wyświetlony monit to na następującej stronie. Wybierz **Akceptuj** przycisk, aby zezwolić aplikacji społeczność deweloperów dla Twojego konta Microsoft:

![Zgłoś problem strony sieci Web](media/report-problem-image3.png)

Nadaj opisowy tytuł problemu wraz z żadnych dodatkowych informacji na temat tego problemu, który uważasz, że mogą być przydatne. Jeśli to możliwe, podaj nam kroki odtwarzania jako pozwala zrozumieć, i rozwiązać problem tak szybko jak to możliwe.

![Zgłoś problem strony sieci Web](media/report-problem-image4.png)

## <a name="provide-a-suggestion"></a>Podaj propozycję

Użyj **Podaj sugestię** łącze na stronie powitalnej kierujące do [programu Visual Studio for Mac na stronie UserVoice](https://visualstudio.uservoice.com/forums/563332-visual-studio-for-mac). W tym miejscu można dodawać nowych żądań i zagłosować na istniejących żądań.
