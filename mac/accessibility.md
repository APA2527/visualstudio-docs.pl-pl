# <a name="accessibility"></a>Ułatwienia dostępu

Istnieje wiele narzędzi i funkcji ułatwień dostępu wbudowanych bezpośrednio w system macOS, które ułatwiają korzystanie z programu Visual Studio dla komputerów Mac.  Oprócz tych program Visual Studio dla komputerów Mac ma również następujące funkcje, aby uczynić go bardziej dostępnym dla osób o różnych umiejętnościach:

* Rozszerzenie tekstu edytora
* Powiększanie tekstu w panelu
* Dostosowanie motywu kolorowego edytora kodu
* Dostosowywanie skrótów klawiaturowych
* Nawigacja przy użyciu klawiatury

> [!NOTE]
> Ten przewodnik odnosi się do programu Visual Studio dla komputerów Mac. W programie Visual Studio w systemie Windows zobacz [Funkcje ułatwień dostępu programu Visual Studio](/visualstudio/ide/reference/accessibility-features-of-visual-studio).
