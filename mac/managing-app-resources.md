# <a name="managing-app-resources"></a>Zarządzanie zasobami aplikacji

Pliki zasobów aplikacji, takich jak obrazy, pliki tekstowe i pliki dźwiękowe są integralne elementy aplikacji, które nie są kompilowane przy użyciu aplikacji. Każdej z platform, która jest obsługiwana przez program Visual Studio for Mac obsługuje następujące typy plików na różne sposoby. Odwołuje się do prowadnic poniżej, aby uzyskać więcej informacji o obsłudze zasobów aplikacji w programie Visual Studio dla komputerów Mac.

## <a name="xamarinforms"></a>Platformy Xamarin.Forms

Kod platformy Xamarin.Forms działa na wielu platformach — każdy z nich ma własny system plików. Oznacza to, że jest Odczyt i zapis plików może odbywać się przy użyciu macierzystych interfejsów API plików na każdej platformie lub dodając pliki jako Osadzanie zasobów.

* [Praca z obrazami](https://developer.xamarin.com/guides/xamarin-forms/user-interface/images/)
* [Praca z plikami]( https://developer.xamarin.com/guides/xamarin-forms/application-fundamentals/files/)
