# <a name="managing-app-resources"></a>Zarządzanie zasobami aplikacji

Pliki zasobów aplikacji, takie jak obrazy, pliki tekstowe i pliki audio, są niezbędne dla aplikacji, ale nie są kompilowane przy użyciu aplikacji. Każda platforma obsługiwana przez Visual Studio dla komputerów Mac obsługuje te zasoby na różne sposoby, jak wyjaśniono w następujących przewodnikach:

## <a name="xamarinforms"></a>Xamarin.Forms

Kod Xamarin. Forms działa na wielu platformach — z których każdy ma własny system plików, a każdy z nich określa, jak odczytywać i zapisywać pliki. W programie Xamarin. Forms można zarządzać zasobami aplikacji przy użyciu natywnych interfejsów API plików na każdej platformie lub przez dodawanie plików jako zasobów osadzonych.

* [Praca z obrazami](https://developer.xamarin.com/guides/xamarin-forms/user-interface/images/)
* [Praca z plikami]( https://developer.xamarin.com/guides/xamarin-forms/application-fundamentals/files/)
