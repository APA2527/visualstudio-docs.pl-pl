# <a name="aspnet-core-web-app-deployment"></a>ASP.NET Core wdrażanie aplikacji sieci Web

Ważnym krokiem podczas opracowywania aplikacji sieci Web jest publikowanie (wdrażanie). Możesz publikować w środowisku zdalnym, aby inne osoby mogły korzystać z aplikacji lub do celów testowych. Visual Studio dla komputerów Mac oferuje dwie opcje publikowania aplikacji. Publikowanie w usłudze Azure App Service umożliwia opublikowanie aplikacji sieci Web w programie Azure App Service. Publikowanie w folderze umożliwia opublikowanie aplikacji w folderze.

## <a name="publish-to-azure-app-service"></a>Publikowanie w usłudze Azure App Service

Za pomocą Visual Studio dla komputerów Mac można opublikować aplikację sieci Web bezpośrednio w programie Azure App Service. Aby opublikować ASP.NET Core aplikację sieci Web w celu Azure App Service zapoznaj się z artykułem [Publikowanie w Azure App Service](publish-app-svc.md) .

## <a name="publish-to-folder"></a>Publikowanie w folderze

Możesz również opublikować aplikacje sieci Web ASP.NET Core w folderze. Aby dowiedzieć się więcej o sposobach publikowania aplikacji sieci Web w folderze, zobacz [Publikowanie w folderze](publish-folder.md).
