# <a name="aspnet-core-web-app-deployment"></a>ASP.NET podstawowe wdrażanie aplikacji sieci Web

Ważnym krokiem w tworzeniu aplikacji sieci web jest publikowanie (wdrażanie). Można opublikować w środowisku zdalnym, tak aby inni mogli korzystać z aplikacji lub do celów testowych. Program Visual Studio dla komputerów Mac oferuje dwie opcje publikowania aplikacji. Publikuj w usłudze Azure App Service umożliwia publikowanie aplikacji sieci web w usłudze Azure App Service. Publikuj w folderze umożliwia opublikowanie aplikacji w folderze.

## <a name="publish-to-azure-app-service"></a>Publikowanie w usłudze Azure App Service

Za pomocą programu Visual Studio dla komputerów Mac można opublikować aplikację sieci web bezpośrednio w usłudze Azure App Service. Aby opublikować aplikację sieci web ASP.NET Core w usłudze Azure App Service, zapoznaj się z artykułem [Publikuj w usłudze Azure App Service.](publish-app-svc.md)

## <a name="publish-to-folder"></a>Publikowanie w folderze

Można również opublikować ASP.NET podstawowe aplikacje internetowe w folderze. Aby dowiedzieć się więcej o tym, jak opublikować aplikację sieci Web w folderze, zobacz [Publikuj w folderze](publish-folder.md).
