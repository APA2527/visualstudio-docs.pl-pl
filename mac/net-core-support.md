# <a name="net-core-support"></a>Obsługa platformy .NET Core

W poniższej tabeli opisano wersje programu .NET Core obsługiwane przez stabilne i w wersji zapoznawczej programu Visual Studio dla komputerów Mac:

| Wersja .NET Core SDK |Visual Studio dla komputerów Mac 8.1 (stabilny) | Visual Studio dla komputerów Mac 8.2 (stabilny) | Visual Studio dla komputerów Mac 8.3 (stabilny) | Visual Studio dla komputerów Mac 8.4 (stabilny)
|---------|---------|---------|---------|---------|
|wersja 2.1.0 - v2.1.5xx | | | | |
|wersja 2.1.600 + |✔︎|✔︎|✔︎|✔︎|
|wersja 2.2.1 - v2.2.1xx | | | | |
|wersja 2.2.200 + |✔︎|✔︎|✔︎|✔︎|
|wersja 3.0 | | |✔︎|✔︎|
|wersja 3.1 | | | |✔︎|

> [!IMPORTANT]
> Wersje w wersji zapoznawczej sdk .NET Core nie są obsługiwane; należy zaktualizować do wydanej wersji. Podczas instalowania programu Visual Studio dla komputerów Mac 8.4 zostanie zainstalowana wydana wersja programu .NET Core w wersji 3.1.

> [!IMPORTANT]
> Jeśli wcześniej używałeś programu .NET Core w wersji 2.2.1xx w programie Visual Studio dla komputerów Mac 8.0, należy ręcznie zaktualizować do obsługiwanej wersji programu .NET Core, zgodnie z powyższą tabelą. Zalecamy [2.1.700](https://dotnet.microsoft.com/download/dotnet-core/2.1) lub [2.2.300](https://dotnet.microsoft.com/download/dotnet-core/2.2)

* Program .NET Core w wersji 3.1 jest domyślnie zainstalowany dla wersji 8.4.
* Program .NET Core w wersji 3.0 jest domyślnie zainstalowany dla wersji 8.3.
* Program .NET Core v2.1.701 (wersja 2.1.700 dla wersji 8.1) jest instalowany domyślnie z instalatorem.
* Aby pobrać dowolną inną wersję programu .NET Core, odwiedź [stronę dotnet](https://dotnet.microsoft.com/download/dotnet-core).
* Podczas korzystania z .NET Core 3.0, C# wersja 8 będzie używana domyślnie. C# 7.3 jest domyślny podczas korzystania z .NET Core 2.x. Aby uzyskać więcej informacji, zobacz [wersja językowa języka C#.](/dotnet/csharp/language-reference/configure-language-version)
* Aby uzyskać informacje na temat instalowania wersji zapoznawczej programu Visual Studio dla komputerów Mac, zobacz przewodnik [Instalowanie wersji w wersji zapoznawczej.](/visualstudio/mac/install-preview)
