# <a name="visual-studio-for-mac-workloads"></a>Visual Studio for Mac obciążeń

Visual Studio for Mac jest wstępnie zainstalowane z obciążeniami dla następujących:

* Aplikacje mobilne i Platform z platformą .NET: Android, iOS, systemu tvOS, watchOS
* Aplikacje komputerowe Mac
* Aplikacje .NET core
* Aplikacje ASP.NET Core sieci Web
* Gry środowiska Unity i platform

## <a name="xamarin"></a>Xamarin

Xamarin umożliwia tworzenie wieloplatformowych aplikacji mobilnych z platformą .NET dla systemu Android, iOS systemu tvOS, watchOS i Mac aplikacji klasycznych.

Informacji na temat używania platformy Xamarin w programie Visual Studio dla komputerów Mac, można znaleźć w [Centrum deweloperów Xamarin](https://developer.xamarin.com/) i zapoznaj się z aplikacji mobilnych [laboratorium praktycznego](https://github.com/Microsoft/vs4mac-labs/tree/master/Mobile/Getting-Started).

## <a name="net-core-applications"></a>Aplikacje .NET core

Aby uzyskać więcej informacji na temat używania platformy .NET Core, zapoznaj się [dokumentacji](https://docs.microsoft.com/dotnet/core/).

## <a name="aspnet-core-web-applications"></a>Aplikacje sieci Web platformy ASP.NET Core

Aby uzyskać informacji na temat pobierania uruchomiony przy użyciu platformy ASP.NET Core w programie Visual Studio dla komputerów Mac, zobacz [wprowadzenie do platformy ASP.NET Core](~/asp-net-core.md) przewodnik lub zapoznaj się z aplikacji sieci web platformy ASP.NET Core [laboratorium praktycznego](https://github.com/Microsoft/vs4mac-labs/tree/master/Web/Getting-Started).

## <a name="cross-platform-unity-game-development"></a>Tworzenie gier środowiska Unity i platform

Aby zacząć tworzyć gry z Unity, zobacz [strony Instalatora](~/setup-vsmac-tools-unity.md) i [przewodnik przy użyciu narzędzi platformy Unity](~/using-vsmac-tools-unity.md), lub sprawdź jednolitości [laboratorium praktycznego](https://github.com/Microsoft/vs4mac-labs/tree/master/Unity/Getting-Started).
