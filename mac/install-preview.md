# <a name="installing-preview-releases"></a>Zwalnia instalowania wersji zapoznawczej

Przed oficjalnie wydaniu nowej wersji programu Visual Studio dla komputerów Mac, zwolnieniu jako podgląd. Wersja zapoznawcza daje możliwość wypróbowanie nowych funkcji i uzyskać najnowsze poprawki błędów, aby w pełni są włączone do produktu.

Wersje zapoznawcze dla programu Visual Studio for Mac są dystrybuowane jako aktualizacja, a nie za pomocą osobny plik do pobrania. Visual Studio for Mac kanałami trzy updater, zgodnie z opisem w [aktualizacji](update.md) artykuł: stabilny, w wersji Beta i alfa.

Większość wersji zapoznawczej będzie dostępna za pośrednictwem zarówno **Beta** i **alfa** kanały, ale sprawdzać [informacje o wersji zapoznawczej](/visualstudio/releasenotes/vs2017-mac-preview-relnotes) najbardziej dokładnych informacji.

Aby zainstalować w wersji zapoznawczej programu Visual Studio dla komputerów Mac, użyj następujących kroków:

1. Przejdź do **programu Visual Studio > sprawdzić aktualizacje...** .
2. W polu kombi kanału aktualizacji, zaznacz **Beta**.
3. Wybierz **kanału przełącznika** przycisk, aby przełączyć się do wybranego kanału i rozpocząć pobieranie nowe aktualizacje.
4. Wybierz **ponowne uruchomienie i zainstaluj aktualizacje** przycisk, aby rozpocząć instalowanie aktualizacji.

Aby uzyskać więcej informacji na temat aktualizowania w programie Visual Studio dla komputerów Mac, zobacz [aktualizacji](update.md) artykułu.
