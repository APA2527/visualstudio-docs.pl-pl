# <a name="signing-in-to-visual-studio-for-mac"></a>Logowanie do programu Visual Studio dla komputerów Mac

Po uruchomieniu programu Visual Studio for Mac po raz pierwszy, zostanie wyświetlony monit o zalogowanie się przy użyciu konta Microsoft. Użyj konta Microsoft, które jest podłączony do licencji, do której chcesz użyć. Jeśli nie masz konta Microsoft, zobacz [jak zarejestrować do konta](https://support.microsoft.com/instantanswers/d18cc497-d839-cf50-dea8-f99c95f2bd16/sign-up-for-a-microsoft-account). 

Jeśli nie chcesz zalogować razu, będzie można użyć kopii ewaluacyjnej przez 30 dni. Po 30 dniach należy zalogować się na używanie kopii programu Visual Studio dla komputerów Mac 

## <a name="how-to-sign-in-to-visual-studio-for-mac"></a>Jak zarejestrować się w programie Visual Studio dla komputerów Mac

Upewnij się, że nawiązano połączenie z Internetem przed zarejestrowaniem się do programu Visual Studio dla komputerów Mac. Subskrypcje można uaktywniać tylko online.
 
Aby zalogować się do programu Visual Studio dla komputerów Mac, wykonaj następujące czynności:

1. Kliknij znak łącza na stronie powitalnej, aby wyświetlić okno dialogowe kont i naciśnij klawisz **logowania** przycisk:

    ![](media/signing-in-image12.png)

2. Wprowadź poświadczenia firmy Microsoft:

    ![](media/signing-in-image13.png)

    Nie można użyć poświadczeń Xamarin, aby zalogować się do programu Visual Studio dla komputerów Mac.

3.  Gdy użytkownik zalogował się pomyślnie, zostanie wyświetlony danego typu licencji. W tym oknie dialogowym można dodać dodatkowych kont Microsoft, które mogą się, że używasz:

    ![](media/signing-in-image14.png)

## <a name="adding-multiple-user-accounts"></a>Dodawanie wielu kont użytkowników

Visual Studio for Mac obsługuje dodawanie wielu kont do swojego konta personalizacji. Te dodatkowe konta pozwoli na dostęp do zasobów, takich jak Azure, z Każde dodane konto. 

Aby dodać dodatkowe konta użytkowników, postępuj zgodnie z instrukcjami [jak zarejestrować się w programie Visual Studio for Mac](#How_to_sign_in_to_Visual_Studio_for_Mac) sekcji.
