# <a name="sign-in-to-visual-studio-for-mac"></a>Logowanie się do programu Visual Studio dla komputerów Mac

Logowanie się do programu Visual Studio dla komputerów Mac to [sposób aktywowania subskrypcji](enable-subscription.md). Po pobraniu programu Visual Studio dla komputerów Mac zawsze otrzymasz wersję społeczności domyślnie. Jeśli masz [licencję Professional lub Enterprise,](https://visualstudio.microsoft.com/vs/compare/)powinieneś zalogować się przy pierwszym uruchomieniu, aby odblokować dodatkowe funkcje. Ponadto otrzymasz również następujące informacje:

* **Dostęp do programu Visual Studio Dev Essentials** — ten program zawiera oferty wolnego oprogramowania, szkolenia, wsparcie i inne. Aby uzyskać więcej informacji, zobacz [Podstawowe informacje dotyczące programu Visual Studio Dev.](https://visualstudio.microsoft.com/dev-essentials/)

* **Automatyczne łączenie się z platformą Azure** w ide bez monitowania ponownie o poświadczenia dla tego samego konta.

Po uruchomieniu programu Visual Studio dla komputerów Mac po raz pierwszy zostanie wyświetlony monit o zalogowanie się przy użyciu konta Microsoft. Użyj konta Microsoft połączonego z licencją, której chcesz użyć. Jeśli nie masz konta Microsoft, zobacz [Jak założyć konto](https://support.microsoft.com/instantanswers/d18cc497-d839-cf50-dea8-f99c95f2bd16/sign-up-for-a-microsoft-account).

Jeśli zdecydujesz się nie logować od razu, będziesz mógł użyć kopii ewaluacyjnej przez 30 dni. Po 30 dniach musisz się zalogować, aby kontynuować korzystanie z kopii programu Visual Studio dla komputerów Mac.

## <a name="how-to-sign-in-to-visual-studio-for-mac"></a>Jak zalogować się do programu Visual Studio dla komputerów Mac

> [!TIP]
> Przed zalogowaniem się do programu Visual Studio dla komputerów Mac upewnij się, że masz połączenie z Internetem. > Subskrypcje można aktywować tylko online. Jeśli nie masz połączenia, wybierz opcję "Zrobię to później" i zaloguj się w menu po nawiązaniu połączenia.

Aby zalogować się do programu Visual Studio dla komputerów Mac przy pierwszym uruchomieniu, wykonaj następujące czynności:

1. Kliknij przycisk **Zaloguj się za pomocą firmy Microsoft** w oknie logowania:

    ![Okno dialogowe Konta w programie Visual Studio dla komputerów Mac](media/ide-tour-2019-start-signin.png)

2. Wprowadź poświadczenia firmy Microsoft:

    ![Okno dialogowe poświadczenia firmy Microsoft](media/signing-in-image13.png)

4. Po zalogowaniu się zostanie wyświetlony opcja wyboru skrótów klawiaturowych. Wybierz opcję, której chcesz użyć, i **kontynuuj**. Zostanie wyświetlony monit z oknem startowym programu Visual Studio 2019 dla komputerów Mac. W tym miejscu można otworzyć lub utworzyć nowy projekt:

    ![Logowanie się pomyślnie](media/signing-in-image14.png)

Alternatywnie można użyć **programu Visual Studio > logowanie...** pozycji menu, aby zalogować się i wylogować w dowolnym momencie.

## <a name="adding-multiple-user-accounts"></a>Dodawanie wielu kont użytkowników

Program Visual Studio dla komputerów Mac obsługuje dodawanie wielu kont do konta personalizacji. Te dodatkowe konta umożliwiają dostęp do zasobów, takich jak Azure, z dowolnego konta dodanego.

Aby dodać dodatkowe konta użytkowników, wybierz menu **Konto > programu Visual Studio z** programu Visual Studio dla komputerów Mac. Kliknij przycisk **Dodaj...,** aby wprowadzić dodatkowe poświadczenia konta.

![Zarządzanie kontami](media/signing-in-image15.png)

## <a name="view-or-change-your-profile-information"></a>Wyświetlanie lub zmienianie informacji profilowych

1. Przejdź do **konta > programu Visual Studio...** i wybierz przycisk **Mój profil.**

2. W oknie przeglądarki wybierz pozycję **Edytuj profil** i zmień żądane ustawienia.

3. Po zakończeniu wybierz pozycję **Zapisz zmiany**.
