# <a name="troubleshooting-why-can-i-not-see-all-available-templates"></a>Rozwiązywanie problemów: Dlaczego nie widzę wszystkich dostępnych szablonów?

W oknie dialogowym Nowy projekt są wyświetlane tylko szablony składników, które zostały zainstalowane. Jeśli brakuje następujących składników, upewnij się, że są zainstalowane odpowiednie składniki, jak opisano poniżej:

|Brakujący szablon  |Składnik do zainstalowania  |
|---------|---------|
|Android     |Android        |
|iOS     |iOS         |
|tvOS     |iOS         |
|watchOS     |iOS         |
|Wieloplatformowy     |Android i iOS         |
|Mac     |macOS (Kakao)         |
|.NET Core     |.NET Core         |

Uruchom VisualStudioForMacInstaller.dmg ponownie zainstalować składniki. Jeśli już go nie masz, możesz go pobrać ze strony sieci Web [programu Visual Studio dla komputerów Mac.](https://visualstudio.microsoft.com/vs/mac/)
