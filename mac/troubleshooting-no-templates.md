# <a name="troubleshooting-why-can-i-not-see-all-available-templates"></a>Rozwiązywanie problemów: Dlaczego nie można wyświetlić wszystkie dostępne szablony?

W oknie dialogowym Nowy projekt widoczne są tylko szablonów elementów, które zostały zainstalowane. Jeśli brakuje dowolnej z następujących składników, upewnij się, że poprawne składniki są zainstalowane, zgodnie z poniższym opisem:


|Brak szablonu  |Składnik do zainstalowania  |
|---------|---------|
|Android     |Android        |
|iOS     |iOS         |
|tvOS     |iOS         |
|watchOS     |iOS         |
|Multiplatform     |Systemy android i iOS         |
|Mac     |macOS (Cocoa)         |
|.NET Core     |.NET Core         |

Uruchom VisualStudioForMacInstaller.dmg ponownie, aby zainstalować składniki. Jeśli masz już go, możesz pobrać go z [programu Visual Studio dla komputerów Mac](https://aka.ms/vsmac) strony sieci Web.
