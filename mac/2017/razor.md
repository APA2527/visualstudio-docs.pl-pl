# <a name="razor-support"></a>Wsparcie brzytwy

Program Visual Studio dla komputerów Mac obsługuje edycję razor, w tym intellisense i wyróżnianie składni w plikach *cshtml.*

![edycja brzytwy w studio wizualnym dla komputerów Mac](media/razor-image1.png)

## <a name="getting-started-with-razor-in-visual-studio-for-mac"></a>Wprowadzenie do razor w programie Visual Studio dla komputerów Mac

Istnieją dwie opcje, które należy wziąć pod uwagę podczas rozpoczynania pracy z razor w programie Visual Studio dla komputerów Mac: Razor Pages w ASP.NET Core i ASP.NET Core MVC. Aby uzyskać samouczki i więcej informacji na temat obu opcji, odwiedź jeden z poniższych przewodników:

- [Wprowadzenie do aplikacji Razor Pages w ASP.NET Core w systemie macOS w programie Visual Studio dla komputerów Mac](/aspnet/core/tutorials/razor-pages-mac/razor-pages-start?view=aspnetcore-2.1)
- [Wprowadzenie do ASP.NET Core MVC i visual studio dla komputerów Mac](/aspnet/core/tutorials/first-mvc-app-mac/start-mvc?view=aspnetcore-2.1)
