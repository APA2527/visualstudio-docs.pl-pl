# <a name="razor-support"></a>Obsługa Razor

Visual Studio dla komputerów Mac obsługuje edytowanie Razor, w tym funkcji IntelliSense i wyróżniania składni w plikach *. cshtml* .

![Edytowanie Razor w programie Visual Studio dla komputerów Mac](media/razor-image1.png)

## <a name="getting-started-with-razor-in-visual-studio-for-mac"></a>Wprowadzenie ze składnią Razor w Visual Studio dla komputerów Mac

Istnieją dwie opcje, które należy wziąć pod uwagę przy rozpoczynaniu pracy z Razor w Visual Studio dla komputerów Mac: Razor Pages w ASP.NET Core i ASP.NET Core MVC. Samouczki i więcej informacji na temat obu opcji można znaleźć w jednym z poniższych przewodników:

- [Rozpocznij pracę z Razor Pages w ASP.NET Core na macOS z Visual Studio dla komputerów Mac](/aspnet/core/tutorials/razor-pages-mac/razor-pages-start?view=aspnetcore-2.1)
- [Wprowadzenie do ASP.NET Core MVC i Visual Studio dla komputerów Mac](/aspnet/core/tutorials/first-mvc-app-mac/start-mvc?view=aspnetcore-2.1)
