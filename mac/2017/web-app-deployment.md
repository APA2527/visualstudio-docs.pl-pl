# <a name="aspnet-core-web-app-deployment"></a>Wdrażanie aplikacji sieci Web platformy ASP.NET Core

Ważnym krokiem w tworzeniu aplikacji sieci web jest publikowanie (wdrażanie). Możesz opublikować w zdalnym środowisku tak, aby inne osoby za pomocą tej aplikacji lub do celów testowych. Program Visual Studio for Mac oferuje dwie opcje, aby opublikować aplikację. Opublikuj umożliwia usłudze Azure App Service pozwala publikować aplikację sieci web w usłudze Azure App Service. Opublikuj umożliwia folderu można opublikować aplikację w folderze.

## <a name="publish-to-azure-app-service"></a>Publikowanie w usłudze Azure App Service

Za pomocą programu Visual Studio for Mac można opublikować aplikację sieci web bezpośrednio w usłudze Azure App Service. Publikowanie aplikacji sieci web w usłudze Azure App Service można znaleźć ASP.NET Core [Publikuj w usłudze Azure App Service](publish-app-svc.md) artykułu.

## <a name="publish-to-folder"></a>Publikowanie w folderze

Można również publikować aplikacje sieci web ASP.NET Core w folderze. Aby dowiedzieć się więcej na temat jak opublikować aplikację sieci web do folderu się [publikowania do folderu](publish-folder.md).
