# <a name="update-visual-studio-2017-for-mac-version-7-to-visual-studio-2019-for-mac-version-80"></a>Aktualizacja programu Visual Studio 2017 dla komputerów Mac (wersja 7) do 2019 usługi Visual Studio dla komputerów Mac (wersja 8.0)

Visual Studio 2019 r dla komputerów Mac oferuje wiele nowych ulepszeń: zupełnie nowe C# edytora, uruchamianie nowego środowiska, obsługa wielu wystąpień i hostem nowe ulepszenia dla deweloperów platformy Xamarin, Unity i platformy .NET Core. Aby uzyskać więcej informacji na temat wszystko jest zawarte w programie Visual Studio for Mac 2019 zapoznaj się następujące wpisy na blogu:

- [Visual Studio 2019 r dla komputerów Mac jest teraz dostępna](https://devblogs.microsoft.com/visualstudio/visual-studio-2019-for-mac-is-now-available/)
- [Visual Studio 2019 r dla komputerów Mac w wersji 8.1 są dostępne (i w wersji zapoznawczej dla 8.2)](https://devblogs.microsoft.com/visualstudio/visual-studio-2019-for-mac-version-8-1-is-now-available-and-a-preview-for-8-2/)

> [!IMPORTANT]
> Jest to znany problem w programie Visual Studio 2017 dla komputerów Mac **wersji 7.7.3**, w którym aktualizacji będzie stale "Błąd podczas pobierania informacji o aktualizacji". W takim przypadku [Pobierz program Visual Studio dla komputerów Mac](https://visualstudio.microsoft.com/vs/mac/) i postępuj zgodnie z instrukcjami, aby [zainstalować](https://docs.microsoft.com/visualstudio/mac/installation?view=vsmac-2019) Visual Studio 2019 r dla komputerów Mac. Należy pamiętać, że spowoduje to zastąpienie istniejącego wystąpienia programu Visual Studio 2017 dla komputerów Mac.

Aby uzyskać najnowszą wersję można użyć [aktualizatora](https://docs.microsoft.com/visualstudio/mac/update?view=vsmac-2017) , przechodząc do **programu Visual Studio > sprawdzać dostępność aktualizacji...** :

![okno dialogowe Updater](media/update-vsmac-updater.png)

Lub, jeśli chcesz użyć lub występują problemy z aktualizacji, [Pobierz](https://visualstudio.microsoft.com/vs/mac/) całkowicie nowego Instalatora 2019 r i get działanie szybko:

![ekranie instalacji](media/update-vsmac-installer.png)
