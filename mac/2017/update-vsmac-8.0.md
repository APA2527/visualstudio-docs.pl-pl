# <a name="update-visual-studio-2017-for-mac-version-7-to-visual-studio-2019-for-mac-version-80"></a>Aktualizowanie programu Visual Studio 2017 dla komputerów Mac (wersja 7) do programu Visual Studio 2019 dla komputerów Mac (wersja 8.0)

Visual Studio 2019 dla komputerów Mac wprowadza wiele nowych ulepszeń: zupełnie nowy edytor Języka C#, nowe środowisko uruchamiania, obsługę wielu wystąpień i wiele nowych ulepszeń dla deweloperów platformy Xamarin, Unity i .NET Core. Aby uzyskać więcej informacji na temat wszystkiego, co znajduje się w programie Visual Studio 2019 dla komputerów Mac, zapoznaj się z następującymi wpisami w blogu:

- [Visual Studio 2019 dla komputerów Mac jest już dostępny](https://devblogs.microsoft.com/visualstudio/visual-studio-2019-for-mac-is-now-available/)
- [Visual Studio 2019 dla komputerów Mac w wersji 8.1 jest już dostępny (i wersja zapoznawcza dla wersji 8.2)](https://devblogs.microsoft.com/visualstudio/visual-studio-2019-for-mac-version-8-1-is-now-available-and-a-preview-for-8-2/)

> [!IMPORTANT]
> Istnieje znany problem z programem Visual Studio 2017 dla komputerów Mac **w wersji 7.7.3**, gdzie updater będzie stale wyświetlać "Błąd pobierania informacji o aktualizacji". W takim przypadku [pobierz program Visual Studio dla komputerów Mac](https://visualstudio.microsoft.com/vs/mac/) i wykonaj kroki, aby [zainstalować](/visualstudio/mac/installation?view=vsmac-2019) program Visual Studio 2019 dla komputerów Mac. Należy zauważyć, że spowoduje to zastąpienie istniejącego wystąpienia programu Visual Studio 2017 dla komputerów Mac.

Aby uzyskać najnowszą wersję, możesz użyć [aktualizatora,](/visualstudio/mac/update?view=vsmac-2017) przeglądając **program Visual Studio > Sprawdź dostępność aktualizacji...**:

![Okno dialogowe programu updater](media/update-vsmac-updater.png)

Lub, jeśli wolisz lub masz problemy z updater, [pobierz](https://visualstudio.microsoft.com/vs/mac/) nowy instalator 2019 i szybko wstań i działa:

![ekran instalacji](media/update-vsmac-installer.png)
