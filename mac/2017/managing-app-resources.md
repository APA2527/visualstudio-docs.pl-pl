# <a name="managing-app-resources"></a>Zarządzanie zasobami aplikacji

Pliki zasobów aplikacji, takie jak obrazy, pliki tekstowe i pliki audio, są niezbędne do aplikacji, ale nie są kompilowane z aplikacją. Każda platforma obsługiwana przez program Visual Studio dla komputerów Mac obsługuje te zasoby na różne sposoby, jak wyjaśniono w poniższych przewodnikach:

## <a name="xamarinforms"></a>Xamarin.Forms

Kod Xamarin.Forms działa na wielu platformach - z których każdy ma swój własny system plików, a każdy system plików dyktuje sposób odczytu i zapisu do plików. W języku Xamarin.Forms można zarządzać zasobami aplikacji przy użyciu natywnych interfejsów API plików na każdej platformie lub przez dodanie plików jako zasobów osadzonych.

* [Praca z obrazami](https://developer.xamarin.com/guides/xamarin-forms/user-interface/images/)
* [Praca z plikami]( https://developer.xamarin.com/guides/xamarin-forms/application-fundamentals/files/)
