# <a name="xamarin-mobile-app-development"></a>Tworzenie aplikacji mobilnych platformy Xamarin

Najwyższej jakości pomoc techniczna dla platformy [Xamarin](/xamarin) umożliwia tworzenie rozbudowanych natywnych środowisk dla systemów Android, macOS, iOS, tvOS i watchOS. Aplikacje dla wielu platform oparte na platformie Xamarin.Forms ułatwiają współużytkowanie kodu interfejsu użytkownika opartego na języku XAML między systemami Android, iOS i macOS bez ograniczania dostępu do funkcji natywnych.

## <a name="android"></a>Android

Visual Studio dla komputerów Mac ma swój własny zintegrowany menedżer zestawów SDK systemu Android, umożliwiając dostęp do zestawów SDK, które mają być kierowane przez aplikację.

W przypadku aplikacji dla systemu Android visual studio dla `.axml` komputerów Mac zawiera własny projektant, który współpracuje z plikami systemu Android, aby wizualnie konstruować interfejsy użytkownika. Program Visual Studio dla komputerów Mac otworzy te pliki w swoim projektancie systemu Android, jak pokazano na poniższej ilustracji:

![Projektant interfejsu użytkownika systemu Android](media/intro-image31.png)

Aby uzyskać więcej informacji na temat projektanta systemu Android, zobacz [Xamarin.Android Designer Omówienie przewodnika.](/xamarin/android/user-interface/android-designer/index)

## <a name="ios"></a>iOS

Projektant systemu iOS jest w pełni zintegrowany z programem Visual Studio dla komputerów Mac i umożliwia wizualną edycję plików .xib i storyboard w celu utworzenia interfejsów użytkownika iOS, tvOS i WatchOS. Cały interfejs użytkownika można zbudować przy użyciu funkcji przeciągania i upuszczania między przybornikami a powierzchnią projektową, przy jednoczesnym intuicyjnym podejściu do obsługi zdarzeń. Projektant systemu iOS obsługuje również [formanty niestandardowe](/xamarin/ios/user-interface/designer/ios-designable-controls-overview) z dodatkową zaletą renderowania w czasie projektowania.

![Projektant scenochorystury systemu iOS](media/intro-image30.png)

Aby uzyskać więcej informacji na temat korzystania z projektanta systemu iOS, zobacz [przewodniki projektanta.](/xamarin/ios/user-interface/designer/?tabs=macos)

### <a name="mac"></a>Mac

Xamarin udostępnia natywne powiązania interfejsu API dla komputerów Mac, które umożliwiają tworzenie pięknych aplikacji mac.

Aby uzyskać więcej informacji na temat pisania aplikacji dla komputerów Mac za pomocą programu Visual Studio dla komputerów Mac, zapoznaj się z przewodnikami [po platformie Xamarin.Mac.](/xamarin/mac/get-started/index)

## <a name="xamarin-enterprise-features"></a>Funkcje platformy Xamarin Enterprise

> [!Note]
> Tych produktów można używać tylko z subskrypcją programu Visual Studio Enterprise.

### <a name="profiler"></a>Profiler

Xamarin Profiler posiada trzy instrumenty dostępne do profilowania. Wprowadzenie do przewodnika [Xamarin Profiler](/xamarin/tools/profiler/index?tabs=macos) bada, co te instrumenty mierzą i jak analizują aplikację i wyjaśnia znaczenie danych prezentowanych na każdym ekranie.

### <a name="inspector"></a>Inspector

Inspektor platformy Xamarin udostępnia interaktywną konsolę języka C# z narzędziami użytkownika. Może służyć jako pomoc debugowania lub diagnostyki podczas sprawdzania aplikacji na żywo, jako narzędzie nauczania, jako narzędzie dokumentacji lub narzędzie do eksperymentowania.

![Xamarin Inspector](media/intro-inspector.png)

Składa się z autonomicznej aplikacji, która zapewnia bogatą konsolę języka C#, która może być kierowana na różne platformy programowania (Android, iOS, Mac i Windows) i integrować się z przepływem pracy debugowania identyfikatorów.

Aby uzyskać więcej informacji, zobacz przewodnik [Inspektora platformy Xamarin.](/xamarin/tools/inspector/)
