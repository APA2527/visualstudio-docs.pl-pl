# <a name="visual-studio-customer-experience-improvement-program"></a>Program poprawy jakości obsługi klienta systemu Visual Studio

Visual Studio Program poprawy jakości obsługi klienta (VSCEIP) to program zaprojektowany, aby pomóc firmie Microsoft w ulepszaniu Visual Studio dla komputerów Mac z upływem czasu. Ten program zbiera informacje o sprzęcie komputerowym i sposobie używania Visual Studio dla komputerów Mac przez osoby, bez zakłócania pracy użytkowników na komputerze. Zbierane informacje ułatwiają firmie Microsoft identyfikację funkcji, które należy poprawić. W tym artykule opisano, jak wybrać lub z VSCEIP.

Aby uzyskać więcej informacji na temat informacji zbieranych, przetwarzanych lub przesyłanych przez VSCEIP, zobacz [zasady zachowania poufności informacji firmy Microsoft](https://privacy.microsoft.com/privacystatement).

## <a name="choice-and-control-over-the-visual-studio-customer-experience-improvement-program"></a>Wybór i kontrola nad Program poprawy jakości obsługi klienta programu Visual Studio

VSCEIP jest domyślnie włączona. Można ją wyłączyć lub ponownie włączyć, postępując zgodnie z poniższymi instrukcjami.

1. Otwórz Visual Studio dla komputerów Mac.

1. Na pasku menu wybierz pozycję **preferencje > programu Visual Studio > inne > Opinie**.

1. Wybierz preferencję udziału:
