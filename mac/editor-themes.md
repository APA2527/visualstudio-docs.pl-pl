# <a name="editor-themes"></a>Edytor motywów
Motywami edytora może być zmieniona na wszystkich systemów dziesięć kolor domyślnie w programie Visual Studio dostępne dla komputerów Mac za pomocą **programu Visual Studio > Preferencje... > Edytor tekstu > motywy kolorów** opcji menu:

 ![Wybór motywu kolorów](media/source-editor-image17.png)

## <a name="adding-new-themes"></a>Dodawanie nowej kompozycji

Można dodawać nowe motywy kolorów, zakładając, że są one w jednym z obsługiwanych formatów. Są to:

* Visual Studio (`.vssettings`)
* Program Xamarin Studio (`.json`)
* TextMate (`.tmTheme`)

Istnieje wiele zasobów, pod warunkiem online zostanie umożliwiające tworzenie kompozycji i tworzy plik w obsługiwanym formacie. Istnieje wiele galerii online zapewniające bezpłatny dostęp do programu Visual Studio i TextMate kompozycji.

Po znaleziono lub utworzyć nowy motyw i pobrana na komputer lokalny, dodaj go w następujący sposób:

1. Wybierz **Dodaj** przycisk w oknie dialogowym opcja motywy kolorów:   
    ![Dodaj motywu kolorów](media/source-editor-image20.png)
2. Przeglądanie w poszukiwaniu motywu na komputerze lokalnym i naciśnij klawisz **Otwórz**.
3. Spowoduje to dodanie motyw do dolnej części listy wstępnie zainstalowane kompozycje. Aby wyświetlić jego podgląd, zaznacz go. Spowoduje to zaktualizowanie edytora w tle: ![podglądu motywu kolorów](media/source-editor-image21.png)
4. Aby zastosować go, naciśnij klawisz **OK** przycisku.
