# <a name="editor-themes"></a>Motywy edytora

Motywy edytora można zmienić na dowolny z 10 schematów kolorów dostępnych domyślnie w programie Visual Studio dla komputerów Mac za pośrednictwem menu **Visual Studio > Preferences > Text Editor > opcje menu Motywy kolorów:**

![Wybór motywu koloru](media/source-editor-image17.png)

## <a name="adding-new-themes"></a>Dodawanie nowych motywów

Można dodać nowe motywy kolorów, pod warunkiem, że znajdują się one w jednym z obsługiwanych formatów:

* Visual Studio`.vssettings`( )
* Xamarin Studio`.json`( )
* TextMate`.tmTheme`( )

Po znalezieniu lub utworzeniu nowego motywu i pobraniu go do komputera lokalnego dodaj go, wykonując następujące czynności:

1. Wybierz przycisk **Dodaj** w oknie dialogowym ![Opcja Motywy kolorów: Dodaj motyw kolorystyczny](media/source-editor-image20.png)
2. Wyszukaj motyw na lokalnym komputerze i naciśnij **przycisk Otwórz**.
3. Motyw zostanie dodany do dolnej części wstępnie zainstalowanej listy motywów. Aby wyświetlić podgląd motywu, wybierz go: ![Podgląd motywu koloru](media/source-editor-image21.png)
4. Aby zastosować motyw, naciśnij przycisk **Okay.**
