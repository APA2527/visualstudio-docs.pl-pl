# <a name="keyboard-navigation-in-visual-studio-for-mac"></a>Nawigacja za pomocą klawiatury w programie Visual Studio dla komputerów Mac

Program Visual Studio dla komputerów Mac jest dostępny dla użytkowników klawiatury korzystających z większości typowych skrótów klawiaturowych systemu macOS, a także wielu, które są specyficzne dla programu Visual Studio dla komputerów Mac. W tym miejscu dowiesz się o kilku podstawowych poleceniach klawiatury dla obu tych poleceń oraz o konfigurowaniu skrótów klawiaturowych programu Visual Studio dla komputerów Mac.

## <a name="use-keyboard-navigation"></a>Korzystanie z nawigacji za pomocą klawiatury

Obsługa nawigacji klawiatury jest wbudowana w system macOS. Aby mieć najbardziej wszechstronne środowisko, ustaw system macOS, aby przenieść fokus klawiatury między **wszystkimi elementami sterującymi**. Dzięki temu można poruszać się po wszystkich formantów w oknie lub oknie dialogowym.

![Zrzut ekranu przedstawiający opcje klawiatury z wybraną wybraną wybraną wybraną kontrolką Wszystkie](media/accessibility-preferences-keyboard.png)

## <a name="customize-keyboard-shortcuts"></a>Dostosowywanie skrótów klawiaturowych

Program Visual Studio dla komputerów Mac ma szereg skrótów klawiaturowych do różnych zadań. Skróty można wyświetlać i edytować w następujący sposób:

1. Wybierz **pozycję Visual Studio (menu)** > **Preferences (&#8984;.)**.

1. Przejdź do **powiązania klucza** > **środowiska**.

1. Przewiń listę lub wyszukaj skrót lub polecenie.

1. Wybierz polecenie.

1. Wybierz **opcję Powiązanie klawiszy**i wpisz kombinację klawiszy.

1. Wybierz przycisk **Zastosuj**.

> [!TIP]
> Możesz wybrać z listy schematów wiązania klawiszy, aby najlepiej odpowiadać twoim potrzebom, jeśli znasz inne typowe skróty klawiaturowe (na przykład Visual Studio w systemie Windows).
