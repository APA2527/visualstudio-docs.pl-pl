# <a name="user-accounts"></a>Konta użytkowników

Po pobraniu i zainstalowaniu programu Visual Studio dla komputerów Mac należy zalogować się, aby aktywować subskrypcję, aby uzyskać dostęp do wszystkich funkcji programu Visual Studio dla komputerów Mac.

Logując się, można włączyć dostęp do wszystkich usług skojarzonych z kontem użytkownika, a także wszystkich kont, które są połączone z tym kontem. Ustawienia konta są przemieszczane z Tobą wszędzie tam, gdzie korzystasz z programu Visual Studio. Można skonfigurować poziomy uprawnień dla określonego wystąpienia programu Visual Studio dla poszczególnych użytkowników.
