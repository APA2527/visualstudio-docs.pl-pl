# <a name="user-accounts"></a>Konta użytkowników

Po pobraniu i zainstalowaniu Visual Studio dla komputerów Mac musisz się zalogować, aby aktywować subskrypcję w celu uzyskania dostępu do wszystkich funkcji Visual Studio dla komputerów Mac.

Logując się, włączasz dostęp do wszystkich usług skojarzonych z kontem użytkownika, a także do wszystkich kont, które są połączone z tym kontem. Twoje ustawienia konta są przekazywane do Ciebie wszędzie tam, gdzie używasz programu Visual Studio. Można skonfigurować poziomy uprawnień dla określonego wystąpienia programu Visual Studio dla poszczególnych użytkowników.
