# <a name="user-accounts"></a>Konta użytkowników

Po pobraniu i zainstalowany program Visual Studio dla komputerów Mac, musisz zarejestrować się aktywować subskrypcję, aby uzyskać dostęp do wszystkich funkcji programu Visual Studio dla komputerów Mac.

Logowanie, umożliwia dostęp do wszystkich usług, które są skojarzone z kontem użytkownika, a także wszystkie konta, które są połączone z tym kontem. Ustawienia konta są przenoszone z Tobą, wszędzie tam, gdzie możesz użyć programu Visual Studio. Poziomy uprawnień można skonfigurować na określonym wystąpieniu programu Visual Studio dla poszczególnych użytkowników.
