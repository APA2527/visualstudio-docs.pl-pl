# <a name="user-accounts"></a>Konta użytkowników

Po pobraniu i zainstalować program Visual Studio dla komputerów Mac, należy zalogować się do aktywowania licencji i uzyskać dostęp do jednej wersji Community, Professional lub Enterprise programu Xamarin Studio.

Logowanie, umożliwia dostęp do wszystkich usług, które są skojarzone z kontem użytkownika, a także wszystkich kont, które są połączone z tym kontem. Ustawienia konta są przenoszone z Tobą wszędzie tam, gdzie możesz użyć programu Visual Studio. Poziomy uprawnień można skonfigurować na określonym wystąpieniu programu Visual Studio dla poszczególnych użytkowników.
