Poprzedni krok: [Instalowanie narzędzi](get-started-netcore-visualstudio-01.md)

## <a name="create-an-aspnet-web-app"></a>Tworzenie aplikacji sieci web platformy ASP.NET
W programie Visual Studio 2017 utworzyć nowy projekt, to musi być obecnie **aplikacji sieci Web platformy ASP.NET Core**. Nazwij projekt "**webfrontend**".

![](images/NewProjectDialog1.png)

Wybierz **aplikacji sieci Web (Model-View-Controller)** szablonu i pamiętaj, że ma być przeznaczona dla **.NET Core** i **ASP.NET Core 2.0** w dwóch list rozwijanych w górnej części okno dialogowe. Kliknij przycisk **OK** Aby utworzyć projekt.

![](images/NewProjectDialog2.png)
