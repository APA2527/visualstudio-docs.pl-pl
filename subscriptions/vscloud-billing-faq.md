# <a name="visual-studio-cloud-subscriptions-billing-faq"></a>Chmura subskrypcji programu Visual Studio często zadawane pytania dotyczące rozliczeń

Upewnij się, że [Porównaj korzyści z subskrypcji chmury i ceny](https://visualstudio.microsoft.com/vs/pricing/) Aby zrozumieć zalety każdego programu Visual Studio subskrypcji z porównania między chmurą i standardowa subskrypcji programu Visual Studio, szczegółowe informacje o korzyści dla subskrybenta i nie tylko.

## <a name="general-purchasing-questions"></a>Pytania ogólne zakupu

### <a name="q-can-i-buy-visual-studio-cloud-subscriptions-using-a-purchase-order"></a>P: czy można kupić subskrypcje chmury programu Visual Studio przy użyciu zamówienie zakupu?
Odpowiedź: nie. Wszystkie subskrypcje chmury programu Visual Studio należy zakupić przy użyciu subskrypcji platformy Azure. (Go traktować jako konta rozliczeniowego Azure.)

### <a name="q-what-types-of-azure-subscriptions-can-be-used-to-buy-visual-studio-cloud-subscriptions"></a>Pytanie: jakie typy platformy Azure subskrypcji można kupić subskrypcje chmury programu Visual Studio?
Odp.: najbardziej subskrypcje mogą być używane — obsługujemy subskrypcji platformy Azure, połączone usługi [Enterprise Agreement (EA)](https://azure.microsoft.com/pricing/enterprise-agreement/), subskrypcji platformy Azure przez dostawców rozwiązań w chmurze (CSP), subskrypcje platformy Azure, skonfigurować za pomocą programu Microsoft Otwórz u odsprzedawców licencji i subskrypcji platformy Azure rozliczana według bieżącego użycia.

Niektóre typy subskrypcji platformy Azure, w tym [bezpłatnej wersji próbnej Azure](https://azure.microsoft.com/pricing/free-trial/) i subskrypcje uwzględniona, zgodnie z korzyściami dla subskrybenta w subskrypcji programu Visual Studio nie można użyć.

### <a name="q-am-i-required-to-buy-other-azure-services"></a>P: jestem wymagane kupowania innych usług systemu Azure?
Odp.: wcale. Jeśli chcesz kupić subskrypcje chmury programu Visual Studio za pośrednictwem platformy Azure, możesz to zrobić.

## <a name="enterprise-agreement-ea-customers"></a>Klienci z umową Enterprise Agreement (EA)

### <a name="q-can-i-use-an-enterprise-agreement-to-buy-visual-studio-cloud-subscriptions"></a>P: czy mogę używać umowy Enterprise Agreement kupić subskrypcje chmury programu Visual Studio?

Odp. tak, możesz to zrobić. Musisz być właścicielem lub współautorem subskrypcji platformy Azure, który został utworzony dla Twojej umowy EA. Upewnij się, wprowadź zakupów dla subskrypcji programu Visual Studio w chmurze bezpośrednio w witrynie Visual Studio Marketplace. Nie można kupić subskrypcje chmury programu Visual Studio przy użyciu zamówienie zakupu.

### <a name="q-how-can-i-tell-whether-i-have-the-necessary-privileges-to-buy-services-in-the-visual-studio-marketplace-through-my-organizations-enterprise-agreement"></a>P: jak rozpoznać, czy mogę ma wystarczających uprawnień, aby kupić usługi Visual Studio Marketplace w ramach umowy Enterprise Agreement mojej organizacji?

Odp.: to najłatwiejsza metoda, aby określić, jeśli masz odpowiednich uprawnień dostępu jest kliknięcie **Kup** przycisk usługi oferowane w Visual Studio Marketplace.
Musisz wybrać subskrypcję platformy Azure (jest to konto rozliczeniowe) z prezentowanych listę subskrypcji platformy Azure, które są aktualnie połączone na potrzeby logowania.
Ponieważ nazwa subskrypcji platformy Azure, wartość domyślna to typ konta rozliczeniowego ("Płatność za rzeczywiste użycie", "Enterprise Agreement" itp.), często jest jasne, czy subskrypcja platformy Azure jest częścią umowy Enterprise Agreement.

Innym rozwiązaniem jest próba odwiedź [Azure Enterprise Portal](http://ea.azure.com).  Jeśli użytkownik mógł go osiągnąć pomyślnie, następnie masz już administratorem przedsiębiorstwa lub roli właściciela konta. Tylko właściciele kont można zdefiniować nowe konta rozliczeniowego Azure w ramach umowy Enterprise Agreement. Jeśli nie masz dostępu do portalu Azure Enterprise, następnie sprawdź zapytanie w Twojej organizacji, aby dowiedzieć się, kto jest administratorem przedsiębiorstwa i poproś tę osobę, aby dodał Cię jako właściciel konta w portalu Azure Enterprise.  Jeśli nie możesz odnaleźć tę osobę mogą [Wyślij bilet pomocy technicznej](http://aka.ms/AzureEntSupport) i poproś o informacje kontaktowe.  Potrzebujesz nazwę swojej organizacji oraz numer rejestracji umowy Enterprise Agreement bilet pomocy technicznej.

### <a name="q-can-i-use-the-azure-monetary-commitment-funds-from-my-enterprise-agreement-to-buy-visual-studio-cloud-subscriptions"></a>P: czy mogę używać środków zobowiązania pieniężnego platformy Azure z moją umową Enterprise Agreement kupić subskrypcje chmury programu Visual Studio?

Odp.: nie tych przedpłaconych środków pieniężnych nie kwalifikują się do zakupu subskrypcji chmury programu Visual Studio. Po wybraniu subskrypcji platformy Azure, który został utworzony dla Twojej umowy EA w celu zakupu subskrypcji programu Visual Studio w chmurze, opłaty będą wyświetlane na następnej fakturze "nadwyżkowe". Zwykle dzieje się tak co miesiąc, ale z powodu historycznych reguł dla niektórych klientów EA, faktury za użycie nadmiarowe może nie być wystawiane przez kilka miesięcy. Zapoznaj się licencjonowania specjalista ds. umowy EA, jeśli użytkownik chce wiedzieć, jakie ilość dodatkowe zakupy (zakupy, które nie są uprawnione do zobowiązań pieniężnych platformy Azure) spowoduje wyzwolenie nadwyżkowe faktury.

## <a name="how-charges-are-processed"></a>Jak są przetwarzane opłaty

### <a name="q-how-are-monthly-cloud-subscription-charges-processed"></a>P: w jaki sposób są **miesięczne** opłaty za subskrypcję przetwarzania w chmurze?
Odp.: w przypadku pierwszego zakupu opłaty są naliczane proporcjonalnie do liczby na pokrycie dni pozostałych w bieżącym miesiącu. Na przykład jeśli zakup 10 programu Visual Studio Professional, miesięczne subskrypcje chmury zostało wprowadzone od 15 kwietnia, następnie czy opłaty są naliczane 5 jednostek ponieważ 50% miesiąca pozostaje (15 dni lub 30-dniowego miesiąca).
Na pierwszym maja, a każdy kolejny miesiąc do momentu anulowania, pełnej 10 jednostek będą naliczane.

Później zwiększyć ilość płatnego możemy również prorate zwiększone jednostki, które obejmują dni pozostałych w bieżącym miesiącu. Dlatego jeśli zakupiono 1 więcej subskrypcję Visual Studio Professional miesięczne chmury 10 maja, czy opłaty około 0.677 jednostki (21 dni pozostałej w miesiącu mającym 31 dni maja).

### <a name="q-how-are-annual-cloud-subscription-charges-processed"></a>P: w jaki sposób są **roczne** opłaty za subskrypcję przetwarzania w chmurze?
Odp.: w przypadku każdego zakupu opłaty są naliczane całej ilości zakupione od razu. Opłaty nie są rozłożone w ciągu roku i nie bez określenia proporcjonalnego. Jeśli kupisz rocznych subskrypcji chmury w różnym czasie, w tym roku masz subskrypcje odnawiania w różnych miesiącach. Firma Microsoft nie należy wprowadzać wszystkich klientów rocznych subskrypcji chmury coterminous, co jest często spotykane z Microsoft wolumin licencjonowania umowy zakupów.

### <a name="q-how-do-cancelations-work"></a>P: jak działa anulowanie?
Odp.: Jeśli anulujesz subskrypcję chmury programu Visual Studio, że anulujesz automatycznego odnawiania. Subskrypcja jest kontynuowany aż do daty jego normalnej odnowienia i po prostu wygasa.
Po wygaśnięciu subskrybent programu Visual Studio nie mogą już używać programu Visual Studio lub wszelkie inne korzyści z subskrypcji.

Za pomocą miesięczne subskrypcje chmury anulowanie obowiązywać pierwszego dnia następnego miesiąca. Tylko anulowanie niektórych miesięczne subskrypcje w chmurze, należy usunąć użytkowników z pierwszego dnia następnego miesiąca, aby upewnić się, że właściwych osób w dalszym ciągu ma przypisanych aktywnych subskrypcji.

W przypadku rocznych subskrypcji chmury anulowanie zaczęły obowiązywać, pierwszego dnia miesiąca, po 12 miesięcy od pierwotnego zakupu lub opłata w wysokości 12 miesięcy od ostatniego odnowienia roczne. Na przykład jeśli zakupiono rocznej subskrypcji chmury programu Visual Studio Enterprise 3 stycznia 2018 r. następnie pozostanie aktywne do 1 lutego maja 2019 r po jego automatycznie odnawia na następny rok. Jeśli anulujesz w dowolnej chwili między następnie i 1 lutego 2020 r, a następnie subskrypcja wygaśnie w dniu 1 lutego 2020 r. Nie ma żadnego rabatu część sposób anuluje się za pośrednictwem rok subskrypcji za pomocą rocznych subskrypcji chmury.

### <a name="q-what-kind-of-volume-discounts-are-available-for-visual-studio-subscriptions"></a>P: jakiego rodzaju rabaty hurtowe są dostępne dla subskrypcji programu Visual Studio?

Odp.: otrzymujesz rabat w wysokości 5% 6. i wszystkie kolejne subskrypcje *w przypadku każdego typu* subskrypcji:

* Visual Studio Professional miesięczna
* Visual Studio Professional roczna
* Visual Studio Enterprise miesięczna
* Visual Studio Enterprise annual

Na przykład jeśli kupisz 6 miesięczne subskrypcje programu Visual Studio Professional i 5 miesięczne subskrypcje programu Visual Studio Enterprise, zapłacisz regularne cena 5 Professional uzyskać rabat w wysokości 5% 6. Professional i płacić ceny wszystkich 5 Enterprise Subskrypcje.

Ponadto rabat dotyczy tylko opłat w danym miesięcznym okresie rozliczeniowym. Dlatego jeśli kupisz 5 rocznych subskrypcji programu Visual Studio Professional w ciągu jednego miesiąca, a następnie Kup 5 więcej następnego miesiąca, zapłacisz regularne ceny na wszystkie 10 subskrypcje.
