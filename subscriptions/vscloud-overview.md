# <a name="buy-visual-studio-professional-and-visual-studio-enterprise-cloud-subscriptions"></a>Kupowanie subskrypcji w chmurze programów Visual Studio Professional i Visual Studio Enterprise
[Subskrypcje programów Visual Studio Professional i Visual Studio Enterprise](https://visualstudio.microsoft.com/subscriptions/) można kupić w portalu Visual Studio [Marketplace.](https://marketplace.visualstudio.com) Są one nazywane **"subskrypcjami w chmurze".**

[Porównanie korzyści z subskrypcji w chmurze i cen](https://visualstudio.microsoft.com/vs/pricing/)

> [!NOTE]
> Firma Microsoft nie oferuje już rocznych subskrypcji programu Visual Studio Professional i rocznych subskrypcji programu Visual Studio Enterprise w subskrypcjach w chmurze. Nie będzie żadnych zmian w istniejącym doświadczeniu klientów i możliwości odnawiania, zwiększania, zmniejszania lub anulowania subskrypcji. Zachęcamy nowych klientów, [https://visualstudio.microsoft.com/vs/pricing/](https://visualstudio.microsoft.com/vs/pricing/) aby przejść do eksplorowania różnych opcji zakupu programu Visual Studio. Nadal możesz kupować nowe miesięczne subskrypcje w chmurze.

## <a name="before-you-start"></a>Przed rozpoczęciem
Aby nali produktem rozliczeniowym były zakupy, potrzebna jest [subskrypcja platformy Azure.](https://azure.microsoft.com/pricing/purchase-options/) Możesz [zarejestrować się](https://portal.azure.com) przed pierwszym zakupem lub podczas pierwszego zakupu w witrynie Visual Studio Marketplace.

## <a name="who-can-buy-visual-studio-cloud-subscriptions"></a>Kto może kupić subskrypcje w chmurze programu Visual Studio?
Każdy, kto [ma właściciel,](https://docs.microsoft.com/azure/role-based-access-control/built-in-roles#owner) [administratora usługi lub współadministratora,](https://docs.microsoft.com/azure/billing/billing-add-change-azure-subscription-administrator#assign-a-subscription-administrator) może zakupić subskrypcje w chmurze.

## <a name="how-to-buy-cloud-subscriptions"></a>Jak kupować subskrypcje w chmurze

1. Zaloguj się do [usługi Visual Studio Marketplace > subskrypcje](https://marketplace.visualstudio.com/subscriptions).

2. Wybierz subskrypcję, którą chcesz kupić, na przykład — Visual Studio Professional — subskrypcja miesięczna.

3. Wybierz subskrypcję platformy Azure do użycia w rozliczeniach. Możesz utworzyć nową subskrypcję platformy Azure, jeśli jej nie masz.
    > [!div class="mx-imgBorder"]
    > ![Wybieranie subskrypcji platformy Azure do rozliczeń](_img/buy-vs-subscriptions/buy-vs-sub-Azure-sub.png)

4. Wybierz liczbę subskrypcji do kupienia.
    > [!div class="mx-imgBorder"]
    > ![Wybierz liczbę subskrypcji](_img/buy-vs-subscriptions/buy-vs-sub-users.png)

    Jeśli nie chcesz, aby subskrypcje były przypisane do siebie, **wyczyść wyczyść wyczyść tę subskrypcję**do mnie .

5. Potwierdź i zakończ zakup. Jeśli przypisano subskrypcję do siebie, można odwiedzić [portal subskrybenta programu Visual Studio,](https://my.visualstudio.com) aby rozpocząć pobieranie oprogramowania i przy użyciu innych korzyści subskrybenta.

> [!div class="mx-imgBorder"]
> ![Zakończ zakup pomyślnie](_img/buy-vs-subscriptions/buy-vs-sub-success.png)

> [!div class="mx-imgBorder"]
> ![Przejdź do portalu subskrybenta programu Visual Studio](_img/buy-vs-subscriptions/view-subscription-benefits-subscriptions-portal.png)

Jeśli zakupiono więcej niż jedną subskrypcję, przejdź do [portalu administracji subskrypcjami programu Visual Studio,](https://manage.visualstudio.com) aby przypisać subskrypcje innym osobom.

> [!div class="mx-imgBorder"]
> ![Przypisywanie subskrypcji innym osobom](_img/buy-vs-subscriptions/buy-vs-sub-success-many.png)

> [!div class="mx-imgBorder"]
> ![Wyświetlanie subskrypcji](_img/buy-vs-subscriptions/assign-subscriptions.png)

* Aby zmienić liczbę zakupionych subskrypcji:

> [!div class="mx-imgBorder"]
> ![Zmienianie subskrypcji](_img/buy-vs-subscriptions/manage-subscriptions.png)

## <a name="frequently-asked-questions"></a>Często zadawane pytania

### <a name="for-new-customers"></a>Dla nowych klientów:
#### <a name="q--i-was-considering-the-annual-cloud-subscription-and-now-i-cant-find-it-what-should-i-do"></a>P: Rozważałem subskrypcję Rocznej Chmury, a teraz nie mogę jej znaleźć.Co mam zrobić?
Odp.: [https://visualstudio.microsoft.com/vs/pricing/](https://visualstudio.microsoft.com/vs/pricing/) Przejdź do innych opcji zakupu.

#### <a name="q-what-purchasing-options-are-available-to-me"></a>P: Jakie opcje zakupu są dla mnie dostępne?
O: Zapewniamy miesięczne subskrypcje i standardowe subskrypcje w naszych kanałach sprzedaży bezpośredniej. Aby uzyskać więcej [https://visualstudio.microsoft.com/vs/pricing/](https://visualstudio.microsoft.com/vs/pricing/)informacji, sprawdź .
Istnieje również wiele opcji za pośrednictwem naszego programu licencjonowania zbiorowego. Możesz zbadać opcje i [https://www.microsoft.com/en-us/licensing/how-to-buy/how-to-buy](https://www.microsoft.com/en-us/licensing/how-to-buy/how-to-buy)ceny na .

#### <a name="q-the-price-for-a-retail-subscription-is-higher-what-additional-benefits-am-i-receiving"></a>P: Cena za subskrypcję detaliczną jest wyższa. Jakie dodatkowe korzyści otrzymuję?
Odp.: Otrzymasz prawa do licencji wieczystej dla ide programu Visual Studio, który został dostarczone z subskrypcją. Kiedy nadszedł czas, aby odnowić, płacisz tylko cenę ubezpieczenia oprogramowania, która jest znacznie mniejsza.

### <a name="for-existing-annual-cloud-subscribers"></a>Dla istniejących rocznych subskrybentów chmury
#### <a name="q--how-will-this-change-impact-me"></a>P: Jak ta zmiana wpłynie na mnie?
Odp.: Ta zmiana nie wpłynie na istniejących subskrybentów.

#### <a name="q--do-i-need-to-do-anything-as-a-result-of-these-changes"></a>P: Czy muszę coś zrobić w wyniku tych zmian?
Odp.: Nie.  Twoje subskrypcje pozostaną na swoim miejscu i będziesz mógł nimi zarządzać tak jak dzisiaj.

#### <a name="q-what-if-i-want-to-increase-the-number-of-subscriptions-for-my-customers"></a>P: Co zrobić, jeśli chcę zwiększyć liczbę subskrypcji dla moich klientów?
O: Roczne subskrypcje zakupione w danym miesiącu są naliczane natychmiast przez cały rok i są ważne przez cały rok. Z tego powodu można zmodyfikować liczbę licencji tylko w bieżącym miesiącu zakupu (będą one ważne przez jeden rok i naliczane przez jeden rok). Poza miesiącem zakupu nie jest już możliwe zwiększenie liczby subskrypcji.

#### <a name="q-what-if-i-want-to-decrease-the-number-of-subscriptions-for-my-customers"></a>P: Co zrobić, jeśli chcę zmniejszyć liczbę subskrypcji dla moich klientów?
Odp.: Administrator może zmniejszyć liczbę subskrypcji, https://manage.visualstudio.com odwiedzając i postępując zgodnie z instrukcjami zawartymi w tym artykule, aby zmniejszyć liczbę subskrypcji. System *nie* wygeneruje proporcjonalnego kredytu. Aby uzyskać zwrot pieniędzy, skontaktuj się z zespołem rozliczeniowym platformy Azure.

#### <a name="q-if-i-cancel-my-subscription-will-i-be-able-to-buy-another-annual-cloud-subscription-later"></a>P: Czy jeśli anuluję subskrypcję, będę mógł później kupić kolejną subskrypcję rocznej chmury?
Odp.: Chociaż nie będzie można kupić innej rocznej subskrypcji chmury, istnieje wiele opcji zakupu subskrypcji programu Visual Studio.  Dowiedz się [https://visualstudio.microsoft.com/vs/pricing/](https://visualstudio.microsoft.com/vs/pricing/)więcej na temat .

### <a name="for-cloud-solution-providers-csp"></a>Dla dostawców rozwiązań w chmurze (CSP)
#### <a name="q-how-will-this-change-impact-my-customers"></a>P: Jak ta zmiana wpłynie na moich klientów?
Odp.: Klienci nie będą mogli kupować nowych rocznych subskrypcji usługi Visual Studio Cloud.

#### <a name="q-do-my-customers-need-to-do-anything-because-of-these-changes"></a>P: Czy moi klienci muszą coś zrobić z powodu tych zmian?
Odp.: Nie będzie to miało wpływu na istniejących klientów. Nowi klienci będą musieli zdecydować, czy chcą kupić miesięczne subskrypcje programu Visual Studio, czy przejść do innego kanału, takiego jak licencjonowanie zbiorowe, aby kupić subskrypcje programu Visual Studio.

#### <a name="q-what-if-i-want-to-increase-the-number-of-subscriptions-for-my-customers"></a>P: Co zrobić, jeśli chcę zwiększyć liczbę subskrypcji dla moich klientów?
O: Roczne subskrypcje zakupione w danym miesiącu są naliczane natychmiast przez cały rok i są ważne przez cały rok. Z tego powodu administrator może zmodyfikować tylko liczbę licencji w bieżącym miesiącu zakupu (będą one ważne przez 1 rok i naliczane przez 1 rok). Poza miesiącem zakupu nie jest już możliwe zwiększenie liczby subskrypcji.

#### <a name="q-what-if-i-want-to-decrease-the-number-of-subscriptions-for-my-customers"></a>P: Co zrobić, jeśli chcę zmniejszyć liczbę subskrypcji dla moich klientów?
Odp.: Administrator może zmniejszyć liczbę subskrypcji, [https://manage.visualstudio.com](https://manage.visualstudio.com) odwiedzając i postępując zgodnie z instrukcjami zawartymi w tym artykule, aby zmniejszyć liczbę subskrypcji. System **nie** wygeneruje proporcjonalnego kredytu. Aby uzyskać zwrot pieniędzy, skontaktuj się z zespołem rozliczeniowym platformy Azure.

#### <a name="q-if-my-customers-cancel-their-subscriptions-will-they-be-able-to-buy-more-annual-cloud-subscriptions-later"></a>Pyt.: Jeśli moi klienci anulują swoje subskrypcje, czy będą mogli później kupić więcej subskrypcji w chmurze rocznej?
Odp.: Mimo że nie będą mogli kupić innej rocznej subskrypcji w chmurze, istnieje wiele opcji zakupu subskrypcji programu Visual Studio.  Dowiedz się [https://visualstudio.microsoft.com/vs/pricing/](https://visualstudio.microsoft.com/vs/pricing/)więcej na temat .
