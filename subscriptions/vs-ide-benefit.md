# <a name="using-the-visual-studio-2017-benefit"></a>Za pomocą programu Visual Studio 2017 korzyści
Jako subskrybent Visual Studio masz dostęp do środowiska IDE programu Visual Studio.  Wersja zależy od poziomu subskrypcji.  Korzyści będą wyświetlane jako Kafelek w kategorii narzędzi na [korzyści](https://my.visualstudio.com/benefits) strony portal subskrybenta.  

(Visual Studio Enterprise będzie można używać w tym przykładzie.). 

Aby pobrać IDE:
1. Kliknij przycisk **Pobierz** łącza w lewym dolnym kafelka korzyści programu Visual Studio. Można również pobrać ją z [pobiera](https://my.visualstudio.com) strony. 

![Visual Studio Enterprise kafelka](_img\vs-ide-experience\vs-ide-tile.png)

2.  Kliknij łącze "Pobierz" u dołu kafelka korzyści.  Użytkownik zostanie przekierowany do strony pobierania wyników dla programu Visual Studio Enterprise, której będziesz mieć możliwość pobrania IDE i uzyskać klucz produktu, aby aktywować go. Mogą również podawać klucza produktu, klikając niebieski **uzyskać klucz** przycisk lub nowszego oświadczeń klucz [klucze produktu](https://my.visualstudio.com/productkeys) strony.
3.  Na karcie Szczegóły strony pobierania wyników:
- Wybierz żądaną architektury (x86, x64)
- Wybierz preferencje językowe. 
4.  Kliknij przycisk Pobierz zielony.
![Szczegóły pobierania programu Visual Studio Enterprise](_img\vs-ide-experience\vs-ide-download-details-cropped.png)
5.  Aby zainstalować teraz, kliknij przycisk "Uruchom".
6.  W oknie dialogowym Kontrola konta użytkownika kliknij przycisk "Tak".
7.  Kliknij przycisk **Kontynuuj** do akceptowania postanowień licencyjnych.
8.  Po uruchomieniu Instalatora, wybierz żądany obciążeń, poszczególne składniki i pakiety językowe, wybierając je na odpowiednich kart.  
![Dostosowywanie programu Visual Studio Enterprise](_img\vs-ide-experience\vs-ide-customize-install-cropped.png)
9.  Kliknij przycisk **zainstalować**. 
10. Po zakończeniu instalacji będziesz mieć opcję, aby zmodyfikować instalację lub uruchom program Visual Studio.  Kliknij przycisk **uruchamianie** rozpocząć pracę.  
![Uruchom program Visual Studio Enterprise](_img\vs-ide-experience\vs-ide-launch-cropped.png)
11. Zaloguj się przy użyciu konta skojarzonego z subskrypcją programu Visual Studio. 
12. Wybierz preferowany kompozycji, a następnie kliknij przycisk **Uruchom Visual Studio**.
13. Wyświetli się Strona początkowa.  Możesz rozpocząć korzystanie z programu Visual Studio!
![Rozpoczynanie korzystania z programu Visual Studio Enterprise](_img\vs-ide-experience\vs-ide-start-cropped.png)
