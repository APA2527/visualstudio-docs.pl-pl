# <a name="enterprise-mobility--security-ems-in-visual-studio-subscriptions"></a>Enterprise Mobility + Security (EMS) w ramach subskrypcji programu Visual Studio

Roczne chmury programu Visual Studio Enterprise i standard subskrypcje obejmują Enterprise Mobility + Security (EMS).  Zabezpieczenie i zarządzanie nimi tożsamości, urządzeń, aplikacji i danych za pomocą usług EMS — Azure Active Directory, Microsoft Intune, Azure Information Protection, Microsoft Advanced Threat Analytics i Microsoft Cloud App Security.  

## <a name="activation-steps"></a>Kroki aktywacji

Aby użyć pakietu EMS, należy skonfigurować konto lub zaloguj się przy użyciu istniejącego konta. 

1.  Zaloguj się do [ https://my.visualstudio.com/benefits ](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs).

2.  Znajdź Kafelek pakietu Enterprise Mobility i zabezpieczeń w sekcji narzędzia i kliknij na **Pobierz** łącze w dolnej części fragmentu.   Otrzymasz powiadomienie, że kod został pomyślnie pobrany.  Kliknij przycisk **aktywować**. 

    ![Kafelek korzyści EMS](_img\vs-ems\vs-ems-tile.png)

2.  Na Enterprise Mobility + strony E5 zabezpieczeń należy zalogować się przy użyciu istniejącej subskrypcji lub utworzyć nowy.  Aby utworzyć nową, wykonać żądane informacje i kliknij **dalej**. 

    ![Korzyści EMS Tworzenie konta](_img\vs-ems\vs-ems-create-account-cropped.png)

3. Utwórz swoją nazwę użytkownika i hasło.  Kliknij przycisk **Utwórz moje konto** aby kontynuować.

    ![Korzyści EMS Utwórz identyfikator użytkownika](_img\vs-ems\vs-ems-userID-cropped.png)

4.  Następnie podaj numer telefonu, aby uzyskać kod weryfikacyjny przez telefon lub tekst.  Wprowadź kod, a następnie kliknij przycisk **dalej**.

    ![Weryfikacja korzyści EMS](_img\vs-ems\vs-ems-robot-cropped.png)

5.  Następna strona będzie wyświetlana Twojej identyfikator logowania stron i użytkowników.  Zapisz te informacje do późniejszego użycia.  Kliknij przycisk **wszystko jest gotowe**.

    ![Utworzono konto świadczenia usług EMS](_img\vs-ems\vs-ems-save-info-cropped.png)

6.  Pakiet EMS wymaga dostępu do programu Microsoft [Azure Portal](https://azure.microsoft.com).  Należy wykonać trzy czynności na stronie początkowej pakietu EMS do aktywowania subskrypcji platformy Azure, dodanie użytkowników i konfigurowania pakietu EMS.
