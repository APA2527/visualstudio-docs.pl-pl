# <a name="buying-visual-studio-cloud-subscriptions---making-your-first-purchase"></a>Kupowanie subskrypcji chmury programu Visual Studio — Tworzenie pierwszego zakupu

Zakupu subskrypcji chmury programu Visual Studio jest proste.  Sprawdź poniższe czynności, aby szybko zapoznać.

1.  Odwiedź stronę https://marketplace.visualstudio.com/subscriptions i zaloguj się.

2.  Wybierz subskrypcję miesięcznego lub rocznego, które chcesz kupić.

3.  Na stronie Przegląd subskrypcji kliknij **uzyskać**.

4.  Użytkownik zostanie przekierowany do uruchomienia procesu zakupu i system będzie wyszukiwać w Twojej subskrypcji platformy Azure
    -  Jeśli nie ma subskrypcji platformy Azure, możesz utworzyć [subskrypcji płatność za rzeczywiste użycie](https://azure.microsoft.com/en-us/offers/ms-azr-0003p/) klikając **Utwórz nową subskrypcję Azure**.

    **Uwaga:** ten proces wymaga karty kredytowej, o ile wcześniej zatwierdzone do faktury
    -  Jeśli oczekiwane wyszukiwania można znaleźć subskrypcji platformy Azure, a żadnej nie znaleziono, sprawdź, czy konto użyte do logowania.  Jeśli masz wiele kont, spróbuj zalogować się za innego konta, a następnie uruchomiona ponownie na powyżej kroku 1.  

5.  Po zidentyfikowaniu subskrypcji platformy Azure zobaczysz jego nazwę w polu listy rozwijanej.   W tym momencie zakończył wymagania rozliczeń i kliknąć przycisk **Kontynuuj**.

6.  Wybierz liczbę subskrypcji, które chcesz kupić.

7.  Przejrzeć zamówienie, kliknij przycisk **Potwierdź**, a wszystko gotowe!

Następny krok: Zapoznaj się z naszym [dokumentacji](/visualstudio/subscriptions/) Aby dowiedzieć się więcej o przypisanie oraz zarządzanie subskrypcjami.

## <a name="faq"></a>FAQ:
### <a name="q--can-i-use-my-azure-monetary-credits-to-purchase-a-visual-studio-cloud-subscription"></a>Pytanie: czy można za pomocą mojej środków pieniężnych subskrypcji platformy Azure można kupić subskrypcję chmury programu Visual Studio?
Odpowiedź: nie można użyć środków zobowiązania pieniężnego Azure, z umową Enterprise kupić subskrypcję chmury programu Visual Studio.  [Dowiedz się więcej](/vsts/billing/faq-azure-billing#billing) o subskrypcje platformy Azure, które można użyć dla Visual Studio Marketplace zakupów.
### <a name="q--are-the-monthly-visual-studio-cloud-subscriptions-prorated"></a>Pytanie: czy obliczone proporcjonalnie miesięczne subskrypcji chmury programu Visual Studio?
Odpowiedź: tak miesięczne subskrypcje są obliczone proporcjonalnie za pierwszy miesiąc.  Następnie konto zostanie naliczona opłata, pierwszego dnia każdego miesiąca.
