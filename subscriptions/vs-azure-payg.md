# <a name="azure-devtest-pay-as-you-go-subscriptions"></a>Subskrypcje płatności zgodnie z rzeczywistymi usługami Dewelopertest platformy Azure
Współpracuj z zespołem w chmurze.  Tworzenie wielu deweloperów i testowania subskrypcji płatności zgodnie z rzeczywistymi przyuśmiakami, umożliwiając współpracę z innymi subskrybentami, zapewniając jednocześnie izolację między zespołami i środowiskami.  Otrzymujesz ekskluzywne zniżki stawek godzinowych za prawa do tworzenia/testowania i korzystania z chmury dla oprogramowania subskrypcyjnego na platformie Azure.

## <a name="set-up-a-subscription"></a>Konfigurowanie subskrypcji
1. Aby korzystać z korzyści subskrypcyjnej usługi [https://my.visualstudio.com/benefits](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs) Azure Pay-As-You-Go, zaloguj się i kliknij łącze **Aktywuj** u dołu kafelka korzyści.
   > [!div class="mx-imgBorder"]
   > ![Kafelek usługi Azure PAYG](_img/vs-azure-payg/vs-azure-payg-tile.png)

2. Jeśli konto platformy Azure zostało już aktywowane w ramach korzyści z subskrypcji (na przykład jeśli masz subskrypcję Enterprise i aktywowałeś 150 USD miesięcznie za pomocą indywidualnych kredytów usługi Azure DevTest), przejdź do kroku 5.

3. Na stronie Rejestracji platformy Microsoft Azure możesz skonfigurować ofertę dewelopera/testu płatności zgodnie z rzeczywistym pozycją (PAYG) lub uaktualnić do oferty programu Visual Studio Enterprise.  Kliknij **przycisk Kontynuuj z bieżącą ofertą (Pay-As-You-Go Dev/Test),** jeśli nie chcesz uaktualnić.

4. Aby zweryfikować swoją tożsamość, wprowadź numer telefonu i wybierz "Wyślij wiadomość TEKSTOWĄ" lub "Zadzwoń do mnie".  Wprowadź otrzymany kod.  Kliknij **pozycję Zweryfikuj kod**.

5. Następnie podaj informacje rozliczeniowe dla swojego konta i kliknij **przycisk Dalej**.  (Będzie to karta kredytowa, która jest naliczana za wszystkie zakupy na platformie Azure).

6. W sekcji Umowa kliknij pole wyboru, aby zaakceptować umowę subskrypcyjną, szczegóły oferty i zasady zachowania poufności informacji, a następnie kliknij pozycję **Kup**.

7. Na stronie Witamy na **platformie Microsoft Azure,** możesz kupić plan pomocy technicznej lub rozpocząć korzystanie z subskrypcji.

8. Jeśli zdecydujesz się rozpocząć korzystanie z subskrypcji, zobaczysz [witrynę Microsoft Azure portal,](https://portal.azure.com) a jeśli jest to twoja pierwsza wizyta, otrzymasz krótką wycieczkę po niektórych funkcjach.  Po zakończeniu lub zrezygnowaniu z wycieczki zobaczysz pulpit nawigacyjny.  Możesz zacząć korzystać z platformy Azure!

## <a name="frequently-asked-questions"></a>Często zadawane pytania
### <a name="q--what-if-i-want-to-make-changes-to-this-subscription"></a>Pyt.: Co zrobić, jeśli chcę wprowadzić zmiany w tej subskrypcji?
Odp.: Aby wprowadzić zmiany w dowolnej subskrypcji platformy Azure:
1. Zaloguj się do [Portalu Azure](https://portal.azure.com).
2. Na liście w lewym okienku wybierz pozycję **Zarządzanie kosztami + Rozliczenia**.
3. Lista subskrypcji zostanie wyświetlona w polu **Moje subskrypcje.** Kliknij żądaną subskrypcję.
4. Teraz będziesz mieć opcje zarządzania lub anulowania subskrypcji, zmiany jej nazwy, zmiany katalogu lub przeniesienia własności rozliczeniowej.
