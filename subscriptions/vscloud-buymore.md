# <a name="buying-visual-studio-cloud-subscriptions---buying-additional-subscriptions"></a>Kupowanie subskrypcji chmury programu Visual Studio — kupowania dodatkowego subskrypcji

Aby kupić dodatkowe subskrypcji chmury programu Visual Studio, zaloguj się do programu Visual Studio [portalu administracyjnego subskrypcji](https://manage.visualstudio.com/) przy użyciu konta, które ma dostęp do subskrypcji platformy Azure Umożliwia pokazanie pierwotnego zakupu.

1.  W lewym górnym rogu Sprawdź, czy wybrano odpowiednią subskrypcję platformy Azure.
2.  W obszarze nawigacji po lewej stronie wybierz **kupić więcej** w ramach zakupu subskrypcji
3.  Zwiększyć liczbę subskrypcji do żądanej ilości, a następnie kliknij przycisk **Kontynuuj**.
4.  Przejrzyj Twoje zamówienie, a następnie kliknij przycisk **Potwierdź**.
5.  Po powrocie do portalu administracyjnego subskrypcji, należy zauważyć zwiększająca się liczba dostępnych subskrypcji.

Aby uzyskać więcej informacji, zapoznaj się z naszym [dokumentacji](/visualstudio/subscriptions/) na przypisanie oraz zarządzanie subskrypcjami

## <a name="faq"></a>FAQ:
### <a name="q--how-long-do-i-have-to-wait-before-i-can-assign-subscriptions"></a>Pytanie: jak długo trzeba czekać przed można przypisać subskrypcji?
A: subskrypcji będą natychmiast dostępne do przypisania w portalu administracyjnym subskrypcji.
