# <a name="buying-visual-studio-cloud-subscriptions---cancelling-subscriptions"></a>Kupowanie subskrypcji chmury programu Visual Studio — anulowanie subskrypcji

Aby anulować subskrypcję chmury programu Visual Studio, zaloguj się do programu Visual Studio [portalu administracyjnego subskrypcji](https://manage.visualstudio.com/) przy użyciu konta, które ma dostęp do subskrypcji platformy Azure Umożliwia pokazanie pierwotnego zakupu.
1.  W lewym górnym rogu Sprawdź, czy wybrano odpowiednią subskrypcję platformy Azure.
2.  W obszarze nawigacji po lewej stronie wybierz **zmniejszyć** w ramach subskrypcji, aby anulować.
3.  Użytkownik zostanie wyświetlone okno, aby dostosować liczbę subskrypcji skojarzonych z Twoim kontem.
4.  Zmniejsz liczbę subskrypcji wymagana ilość lub zero (0) do anulowania, a następnie kliknij przycisk**Kontynuuj**.
5.  Przejrzeć zamówienie, kliknij przycisk **Potwierdź**i you'redone!
## <a name="faq"></a>FAQ:
### <a name="q--can-i-still-use-a-subscription-after-i-cancel"></a>Pytanie: czy można nadal korzystać z subskrypcji po anulować?
A: tak, może nadal dostęp swoje korzyści dla pozostałego czasu trwania subskrypcji.
