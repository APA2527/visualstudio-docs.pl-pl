# <a name="using-the-subscriber-portal---myvisualstudiocom"></a>Korzystanie z portalu subskrybenta - my.visualstudio.com

Portal subskrypcji programu Visual [https://my.visualstudio.com](https://my.visualstudio.com?wt.mc_id=o~msft~docs)Studio to zasób typu "wszystko w jednym", który zapewnia pełny zakres korzyści oferowanych za pośrednictwem subskrypcji programu Visual Studio.

Korzystanie z portalu wymaga, aby zalogować się przy użyciu tożsamości skojarzonej z subskrypcją.  Może to być konto Microsoft (MSA) lub adres e-mail podany przez organizację, taką jak pracodawca lub szkoła.  Dowiedz się więcej o [logowaniu](signing-in.md).

Portal jest podzielony na sześć stron.  Każda strona zapewnia dostęp do różnych produktów lub usług dołączonych do subskrypcji.  Sześć stron to korzyści, pliki do pobrania, klucze produktów, subskrypcje, pomoc techniczna i Marketplace.

## <a name="benefits"></a>[Korzyści](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs)
Narzędzia i usługi zawarte w subskrypcji są wyświetlane jako "kafelki" na stronie Korzyści.  Nie wszystkie korzyści są uwzględnione we wszystkich subskrypcjach. Kafelki są pogrupowane w trzy kategorie: Narzędzia, Rozwój zawodowy i Wsparcie.  

### <a name="tools"></a>Narzędzia
W kategorii Narzędzia znajdziesz podstawowe narzędzia programistyczne, takie jak IDE programu Visual Studio, Azure DevOps, a także narzędzia dostarczane przez naszych partnerów.  Zobaczysz również usługi, takie jak Azure i Visual Studio App Center.

### <a name="professional-development"></a>Profesjonalne programowanie
Ta kategoria zawiera korzyści, takie jak szkolenia online firmy Microsoft i naszych partnerów, a także subskrypcję msdn magazine, wszystko po to, aby pomóc Ci rozwijać swoje umiejętności rozwoju i być na bieżąco ze wszystkimi rzeczami kodu.

### <a name="support"></a>Pomoc techniczna
Masz pytanie dotyczące platformy Azure?  Uzyskaj pomoc od społeczności platformy Azure za pośrednictwem forów lub skorzystaj z czatu doradczego platformy Azure (wersja zapoznawcza).  Usługa Czat concierge jest dostępna, aby pomóc w znalezieniu dokumentacji technicznej i informacji o produkcie oraz pomóc w zadawanych pytaniach związanych z subskrypcją.  Wybrane subskrypcje obejmują również pomoc techniczną opartą na zdarzeniach, która ułatwia rozwiązywanie problemów z szeroką gamą produktów firmy Microsoft działających w środowiskach deweloperskich/testowych.

## <a name="downloads"></a>[Pliki do pobrania](https://my.visualstudio.com/downloads?wt.mc_id=o~msft~docs)
W zależności od posiadanej subskrypcji lub programu masz dostęp do pobierania oprogramowania, od aktualizacji i poprawek po obszerną bibliotekę bieżących produktów, a także starsze wersje, dzięki czemu możesz testować aplikacje w wielu różnych scenariuszach.

Aby ułatwić subskrybentom, którzy mają wiele subskrypcji skojarzonych z jednym kontem, podczas odwiedzania strony Pobrane pliki, zobaczysz pliki do pobrania dostępne dla najwyższej posiadanych subskrypcji.  W ten sposób możesz zobaczyć wszystkie pliki do pobrania, niezależnie od tego, której subskrypcji aktualnie używasz.

## <a name="product-keys"></a>[Klucze produktu](https://my.visualstudio.com/productkeys?wt.mc_id=o~msft~docs)
Odwiedź stronę Klucze produktu, aby uzyskać klucze do instalowania i aktywowania produktów oraz aby zobaczyć, jakie klucze zostały już odebrane.  Możesz nawet wyeksportować listę wszystkich kluczy do przechowywania.

## <a name="subscriptions"></a>[Subskrypcje](https://my.visualstudio.com/subscriptions?wt.mc_id=o~msft~docs)
Na stronie Subskrypcje udostępnia możliwości wyświetlania, które subskrypcje są własnością, przełączania się między subskrypcjami, odnawiania ich i aktywowania nowych subskrypcji. Można również skojarzyć subskrypcje, do których uzyskujesz dostęp przy użyciu konta Microsoft, z kontem alternatywnym, które jest już skojarzone z istniejącymi zasobami IDE programu Visual Studio, azure devops lub platformy Azure.

## <a name="support"></a>Pomoc techniczna

Oprócz indywidualnych korzyści znalezionych w kategorii pomocy technicznej na korzyści strony, pomoc techniczna strona zapewnia dostęp do zasobów, które pomogą Ci uzyskać jak najwięcej z subskrypcji programu Visual Studio. Uzyskaj dostęp do [zasobów pomocy technicznej](https://visualstudio.microsoft.com/subscriptions/support/) w celu rozwiązania problemów technicznych lub pytań związanych ze sprzedażą subskrypcji, kontami i rozliczeniami.

## <a name="marketplace"></a>[Rynek](https://marketplace.visualstudio.com/)

Chcesz odnowić subskrypcję w chmurze programu Visual Studio?  Chcesz pobrać najnowsze rozszerzenia dla rodziny produktów programu Visual Studio?  Portal Marketplace pomaga użytkownikom znaleźć rozszerzenia, narzędzia i usługi, które sprawiają, że usługi Azure DevOps, Azure DevOps Server i Visual Studio Code są jeszcze lepsze. Możesz nawet zostać wydawcą i publikować własne rozszerzenia w Marketplace.
