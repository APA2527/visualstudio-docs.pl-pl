# <a name="visual-studio-subscriptions-are-available-through-the-microsoft-store"></a>Subskrypcje programu Visual Studio są dostępne za pośrednictwem sklepu Microsoft Store
Subskrypcje programu Visual Studio są dostępne za pośrednictwem różnych kanałów, w tym [licencjonowania zbiorowego,](https://www.microsoft.com/licensing/default)dostawców rozwiązań w chmurze, [portalu Visual Studio Marketplace](https://marketplace.visualstudio.com/subscriptions)i microsoft store.  W tym artykule omówiono zakup i aktywację subskrypcji za pośrednictwem [sklepu Microsoft Store](https://www.microsoft.com/search?q=visual+studio+subscription).  

## <a name="how-to-buy-subscriptions"></a>Jak kupować subskrypcje
Potrzebujesz pomocy w podjęciu decyzji, która subskrypcja jest dla Ciebie odpowiednia?  Sprawdź naszą [stronę korzyści z subskrypcji](https://visualstudio.microsoft.com/vs/benefits/).  

Subskrypcje programu Visual Studio dostępne w sklepie Microsoft Store obejmują:
- [Visual Studio Enterprise](https://www.microsoft.com/p/visual-studio-enterprise-subscription/dg7gmgf0dst4?activetab=pivot%3aoverviewtab)
- [Visual Studio Professional](https://www.microsoft.com/p/visual-studio-professional-subscription/dg7gmgf0dst3?activetab=pivot%3aoverviewtab)
- [Visual Studio Test Professional](https://www.microsoft.com/p/visual-studio-test-professional-subscription/dg7gmgf0dst6?activetab=pivot%3aoverviewtab)

Każda z tych subskrypcji oferuje możliwość dokonania zakupu początkowego lub odnowienia istniejącej subskrypcji.  Ceny odnowienia, które odzwierciedlają znaczną zniżkę, są dostępne tylko dla istniejących subskrypcji. 

> [!IMPORTANT]
> Aby kwalifikować się do odnawiania subskrypcji po obniżonej cenie odnowienia, subskrypcje powinny zostać odnowione przed ich wygaśnięciem.  

## <a name="how-to-activate-subscriptions"></a>Jak aktywować subskrypcje
Po zakupie subskrypcji programu Visual Studio za pośrednictwem sklepu Microsoft Store można rozpocząć proces aktywacji od płatności "Strona potwierdzenia" lub "Wiadomość e-mail z potwierdzeniem", która została otrzymana po zakupie.

1. Kliknij **kliknij tutaj, aby zalogować się** w wiadomości e-mail z potwierdzeniem lub na stronie potwierdzenia zakupu.
2. Zostaniesz przekierowany [https://my.visualstudio.com/subscriptions/activate](https://my.visualstudio.com/subscriptions/activate?wt.mc_id=o~msft~docs)do .
3. Wprowadź adres e-mail, który wymaga dostępu do subskrypcji programu Visual Studio, a następnie kliknij przycisk **Kontynuuj**.
4. Możesz zostać przekierowany do jednej z dwóch różnych stron logowania w zależności od typu używanego konta:
    - Subskrybenci korzystający z osobistego "konta Microsoft" (MSA) zobaczą stronę hasła do konta Microsoft.
    - Subskrybenci korzystający z kont służbowych zostaną przekierowani na stronę logowania w organizacji.
5. Wprowadź hasło lub wypełnij wymagania dotyczące logowania w organizacji.
6. Po pomyślnym zalogowaniu pojawi się okno "Aktywuj subskrypcję programu Visual Studio".
7. Wprowadź informacje o kluczu produktu dostępne z płatności "Strona potwierdzenia" lub "E-mail z potwierdzeniem" i kliknij **Aktywuj**.
8. Subskrypcja powinna być teraz widoczna na stronie [Subskrypcje](https://my.visualstudio.com/subscriptions?wt.mc_id=o~msft~docs) i powinieneś mieć dostęp do wszystkich korzyści z subskrypcji.
