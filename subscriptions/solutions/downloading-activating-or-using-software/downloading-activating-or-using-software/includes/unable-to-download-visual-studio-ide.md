## <a name="im-unable-to-download-or-activate-visual-studio-enterprise-or-professional"></a>Nie mogę pobrać ani aktywować Visual Studio Enterprise lub wersji Professional

Poziom subskrypcji identyfikuje poziom programu Visual Studio IDE, do którego masz dostęp. Na przykład subskrypcja Visual Studio Professional nie ma dostępu do programu Visual Studio Enterprise. Jeśli nie masz pewności, którego poziomu używasz, zapoznaj się z dostępnymi subskrypcjami, sprawdzając [stronę subskrypcji](https://my.visualstudio.com/subscriptions) w portalu.    

## <a name="how-to-download-and-activate"></a>Jak pobrać i aktywować 

1. Upewnij się, że zalogowano się do [portalu subskrypcji](https://my.visualstudio.com/benefits).  
1. W najnowszej wersji Znajdź kafelek IDE programu Visual Studio w sekcji Tools (narzędzia), a następnie kliknij link Download (Pobierz) w dolnej części kafelka korzyści. 
1. Nastąpi przekierowanie do strony Wyniki pobierania dla programu Visual Studio, na której będzie można pobrać środowisko IDE.
1. Postępuj zgodnie z instrukcjami pobierania i instalowania.
1. Przy użyciu poczty e-mail, do której przypisano subskrypcję, będziesz mieć możliwość natychmiastowego zalogowania się do środowiska IDE po instalacji, ale nie jest wymagany żaden klucz produktu. [Dowiedz się, jak przejąć klucz produktu](https://docs.microsoft.com/visualstudio/subscriptions/find-keys) , jeśli jesteś w środowisku w trybie offline. 

## <a name="find-an-older-version"></a>Znajdowanie starszej wersji

Możesz również wyszukać i pobrać poprzednie wersje ze [strony pliki do pobrania](https://my.visualstudio.com/Downloads?PId=6545) w portalu subskrypcji.
