## <a name="what-if-you-cant-find-the-ide-feature-you-want-to-use"></a>Co zrobić, jeśli nie możesz znaleźć funkcji IDE, której chcesz użyć? 

Wybrana wersja środowiska IDE i dostępne funkcje będą zależeć od poziomu subskrypcji. Program Visual Studio będzie wyświetlany jako kafelek w kategorii narzędzia na stronie korzyści [portalu subskrypcje](https://my.visualstudio.com/benefits). 

## <a name="compare-supported-visual-studio-features"></a>Porównaj obsługiwane funkcje programu Visual Studio 

Po zanotowaniu, która wersja programu Visual Studio jest dołączona do subskrypcji, [PORÓWNAJ obsługiwane funkcje](https://visualstudio.microsoft.com/vs/compare/) , aby sprawdzić, czy ta funkcja jest dostępna w używanym środowisku IDE. Jeśli używasz wersji środowiska IDE, która nie zawiera funkcji, musisz pobrać odpowiednie środowisko IDE. 

## <a name="upgrade-your-subscription-level"></a>Uaktualnij poziom subskrypcji  

Jeśli funkcja jest specyficzna dla innego poziomu subskrypcji, należy zmienić poziom subskrypcji.  

Jeśli Twoja subskrypcja programu Visual Studio jest świadczona przez służbowe lub szkolne, prawdopodobnie została przypisana do Ciebie przez administratora subskrypcji. Skontaktuj się z administratorem organizacji przy użyciu przycisku "kontakt z administratorem" w prawym górnym rogu nawigacji w [portalu subskrypcje](https://my.visualstudio.com/benefits).  

Jeśli subskrypcja została zakupiona bezpośrednio lub nie masz administratora, musisz zakupić nową subskrypcję na wymaganym poziomie.  

## <a name="how-to-contact-your-admin"></a>Jak skontaktować się z administratorem 

Uwzględniając rozmiar i złożoność wielu organizacji, możesz nie wiedzieć o tym, kto jest administratorem, ale w portalu subskrypcji jest dostępna funkcja ułatwiająca kontaktowanie się z administratorem. 

1. Zaloguj się do [portalu subskrypcji](https://my.visualstudio.com/benefits).  
1. Kliknij przycisk skontaktuj się z moim administratorem w prawym górnym rogu. 
1. Wprowadź komunikat dla administratora. 
1. Jeśli chcesz otrzymać kopię wiadomości wysłanej do administratora, kliknij pole wyboru Wyślij do mnie kopię. Na koniec kliknij pozycję Wyślij.
