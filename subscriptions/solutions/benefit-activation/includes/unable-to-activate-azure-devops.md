## <a name="im-unable-to-activate-my-azure-devops-benefit"></a>Nie mogę aktywować korzyści z korzystania z platformy Azure DevOps.

Jako subskrybent programu Visual Studio możesz użyć szeregu funkcji w usłudze Azure DevOps. Będzie można korzystać z tych samych funkcji w każdej organizacji usługi Azure DevOps, w której jesteś członkiem, niezależnie od tego, czy została utworzona organizacja, czy dodana przez kogoś innego.  

## <a name="sign-in"></a>Zaloguj
1. Zaloguj się do [portalu subskrypcji](https://my.visualstudio.com/benefits) i kliknij link wprowadzenie w dolnej części kafelka korzyści z usługi Azure DevOps.
1. Nastąpi przekierowanie do usługi Azure DevOps, w której należy wyrazić zgodę na warunki korzystania z usługi. 
1. Zaloguj się do usługi Azure DevOps za pomocą wiadomości e-mail, która jest używana do logowania się do subskrypcji programu Visual Studio. Jeśli wolisz używać innego adresu e-mail do logowania, możesz [utworzyć alternatywne konto](https://docs.microsoft.com/visualstudio/subscriptions/vs-alternate-identity) , aby połączyć swoją subskrypcję, umożliwiając dostęp do usługi Azure DevOps przy użyciu obu wiadomości e-mail.
