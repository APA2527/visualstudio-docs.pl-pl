# <a name="search-and-filter-subscribers-in-visual-studio-subscriptions"></a>Wyszukiwanie i filtrowanie subskrybentów w subskrypcjach programu Visual Studio
Portal administracyjny subskrypcji programu Visual Studio umożliwia znalezienie określonego podzbioru subskrybentów na podstawie różnych kryteriów. Listę subskrybentów można filtrować według nazwy, adresu e-mail, poziomu subskrypcji i innych informacji.

## <a name="to-filter-the-subscriber-list"></a>Aby filtrować listę subskrybentów
1. Aby wyszukać określoną grupę subskrybentów, wybierz kartę **Filtr.**
   > [!div class="mx-imgBorder"]
   > ![Wybierz filtr, aby przeszukać listę subskrybentów](_img/search-filter/filter-list.png)

2. Wprowadź kryteria, których chcesz użyć jako podstawy wyszukiwania, a następnie wybierz ponownie **pozycję Filtruj.**
   > [!div class="mx-imgBorder"]
   > ![Filtrowanie danych subskrybenta za pomocą dostępnych pól](media/filter-subscribers.png)

   Lista subskrybentów jest zgodna z wybranymi kryteriami.  Możesz kliknąć nagłówki filtrowanych wyników, aby zmienić kolejność sortowania dla każdego pola.  
   > [!NOTE]
   > Pole **Nazwa** jest sortowane według imienia.

   Należy zauważyć, że ikona **filtru** jest wyróżniona, co oznacza, że filtr jest stosowany.  Wyświetlany jest również wybór **filtru Clear.** 

3. Aby wyczyścić filtr i przywrócić niefiltrowane listy, kliknij kartę **Wyczyść filtr** w prawym górnym rogu. 
   > [!div class="mx-imgBorder"]
   > ![Wyczyść filtr z listy subskrybentów](_img/search-filter/clear-filter.png)
