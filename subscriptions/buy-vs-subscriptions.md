### <a name="buy-subscriptions"></a>Kupować subskrypcje


1.  Zaloguj się do [programu Visual Studio Marketplace > Subskrypcje](https://marketplace.visualstudio.com/subscriptions).

2.  Wybierz subskrypcję, którą chcesz kupić, na przykład:

    <img alt="Buy a Visual Studio subscription" src="_img/buy-vs-subscriptions/buy-vs-sub-start.png" style="border: 1px solid #CCCCCC" />

3.  Wybierz subskrypcję platformy Azure na potrzeby rozliczeń.
Lub Utwórz nową subskrypcję, jeśli nie masz.

    <img alt="Select Azure subscription for biling" src="_img/buy-vs-subscriptions/buy-vs-sub-Azure-sub.png" style="border: 1px solid #CCCCCC" />

4.  Wybierz liczbę subskrypcji kupić.

    <img alt="Select number of subscriptions" src="_img/buy-vs-subscriptions/buy-vs-sub-users.png" style="border: 1px solid #CCCCCC" />

    Jeśli nie chcesz subskrypcje przypisane do użytkownika, wyczyść **Przypisz do mnie, ta subskrypcja**.

5.  Potwierdź i Zakończ proces zakupu. Aby wyświetlić swoje korzyści subskrypcję teraz, przejdź do programu Visual Studio [portal subskrypcje](https://my.visualstudio.com?wt.mc_id=o~msft~docs) lub kliknij przycisk **wyświetlić subskrypcję**.


### <a name="activating-subscriptions-purchased-through-the-microsoft-store"></a>Aktywowanie subskrypcje zakupione w ramach Microsoft Store

Jeśli zakupione za pośrednictwem Microsoft Store subskrypcji programu Visual Studio, kroki aktywacji można zainicjować z płatności stronę potwierdzenia lub wiadomość E-mail z potwierdzeniem. Uwaga: Jeśli nie otrzymasz wiadomość e-mail, sprawdź poprawność adresu e-mail i sprawdź folderów wiadomości-śmieci.

Aby aktywować subskrypcję: 

1. Polecenie **kliknij tutaj, aby zarejestrować się w**. Nastąpi przekierowanie do [https://my.visualstudio.com/subscriptions/activate](https://my.visualstudio.com/subscriptions/activate?wt.mc_id=o~msft~docs)

2. Wprowadź adres e-mail ma być używany do uzyskania dostępu do subskrypcji programu Visual Studio, a następnie kliknij przycisk **Kontynuuj**.

3. W zależności od adresu e-mail, użytkownik może zostać przekierowany do jednej **konta Microsoft (MSA)** strony logowania lub **pracy lub nauki** strony logowania na koncie. 

4. Wprowadź swoje hasło lub wykonania procesów logowania wymagania.
5. W przypadku pomyślnego logowania **aktywować swój subskrypcji Studio Visual** zostanie wyświetlone okno.
6. Wprowadź **klucz produktu** informacji dostępnych w płatności stronę potwierdzenia lub wiadomość E-mail z potwierdzeniem i kliknij przycisk **Aktywuj**.

7. Subskrypcję teraz powinny być widoczne na [https://my.visualstudio.com/subscriptions](https://my.visualstudio.com/subscriptions?wt.mc_id=o~msft~docs).
