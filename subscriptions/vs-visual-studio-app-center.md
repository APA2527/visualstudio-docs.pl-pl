# <a name="visual-studio-app-center-in-visual-studio-subscriptions"></a>Visual Studio aplikacji Centrum w subskrypcji programu Visual Studio

## <a name="overview"></a>Omówienie
Aplikacji Centrum jest kontrola misji dla aplikacji.

-  Automatycznie tworzenie aplikacji, przetestować go na urządzeniach rzeczywistych i rozpowszechnienie go do wersji beta testerów
-  Zbieraj raporty awarii i analizy użytkownika, w tym zdarzeń niestandardowych.
-  Dodawanie powiadomień wypychanych do aplikacji w minutach

## <a name="activation-steps"></a>Kroki aktywacji

Wprowadzenie do korzystania z programu Visual Studio aplikacji Centrum jest bardzo proste. 
1.  Zaloguj się do [https://my.visualstudio.com/benefits](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs).

2. Znajdź Kafelek Centrum aplikacji programu Visual Studio, a następnie kliknij przycisk **wprowadzenie**.

    ![Kafelek Centrum aplikacji VS](_img/vs-app-center/vs-app-center-tile.png)

3. Następnie należy utworzyć konto.  Możesz używać do logowania istniejące konta Microsoft, Google, GitHub lub Facebook lub utworzyć nowe konto, podając nazwę użytkownika, poczty e-mail i hasło.  Kliknij przycisk **Tworzenie konta** aby kontynuować. 

    ![Centrum aplikacji VS Tworzenie konta](_img/vs-app-center/vs-app-center-create-account.png)

4. To wszystko jest do niego.  Po prostu kliknij **Dodaj nową organizację** lub **Dodaj nową aplikację** rozpocząć pracę!

    ![Portal programu VS Center aplikacji](_img/vs-app-center/vs-app-center-portal.png)

## <a name="faq"></a>Najczęściej zadawane pytania
*Wyszukaj aktualizacje w tym miejscu*

### <a name="q--when-i-click-on-the-get-started-link-i-get-an-error-that-my-browser-is-unsupported--whats-wrong"></a>Pytanie: po kliknięciu łącza "Wprowadzenie" Błąd że przeglądarka jest nieobsługiwany.  Co jest nie tak? 
A: Centrum aplikacji wymaga nowszej możliwości, które nie mogą być udostępniane przez niektóre starsze przeglądarki.  Jeśli Twoja przeglądarka nie jest obsługiwana, wykonaj następujące kroki:
1.  Otwórz innej przeglądarki, takich jak Microsoft Edge lub najnowsze wersje programu Google Chrome, Firefox lub Safari.  
2.  Połącz do programu Visual Studio [https://my.visualstudio.com/benefits](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs) i zaloguj się. 
3.  Teraz kliknij **wprowadzenie** łącza na kafelku Centrum aplikacji programu Visual Studio. 
4.  Teraz można uzyskiwać dostęp do portalu Centrum aplikacji i utworzyć konto.
