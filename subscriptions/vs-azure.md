# <a name="use-microsoft-azure-in-visual-studio-subscriptions"></a>Dzięki platformie Microsoft Azure w ramach subskrypcji programu Visual Studio

## <a name="overview"></a>Omówienie

Jako subskrybent programu Visual Studio mogą używać programu Microsoft Azure, bez dodatkowych opłat.  Za pomocą usługi [miesięcznych środków platformy Azure](https://azure.microsoft.com/pricing/member-offers/msdn-benefits-details/), system Azure jest dostępny z osobistej "piaskownicy" do tworzenia i testowania.  Możesz aprowizować maszyny wirtualne, cloud services i innych zasobów platformy Azure.  Kwoty kredytu zależą od poziomu subskrypcji.

## <a name="activation-steps"></a>Procedurę aktywacji

1. Zaloguj się do witryny [https://my.visualstudio.com/benefits](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs).

2. Znajdź Kafelek Azure w sekcji narzędzia dotyczące zalet strony, a następnie kliknij przycisk **Aktywuj** link w dolnej części kafelka korzyści.
   > [!div class="mx-imgBorder"]
   > ![Kafelek Azure](_img/vs-azure/vs-azure-tile.png)

3. Wprowadź informacje wymagane do utworzenia subskrypcji usługi Azure, kliknij pole wyboru, aby zaakceptować warunki i postanowienia, a następnie kliknij przycisk **Zarejestruj**:
   > [!div class="mx-imgBorder"]
   > ![Utwórz konto platformy Azure](_img/vs-azure/vs-azure-sign-up-cropped.png)

4. Jeśli Twoje konto jest gotowe, będziesz mieć opcję, aby wybrać plan pomocy technicznej platformy Azure (sprzedawany oddzielnie) lub zacznij pracę ze swoją subskrypcją platformy Azure.
   > [!div class="mx-imgBorder"]
   > ![Trwa przygotowywanie platformy Azure](_img/vs-azure/vs-azure-getting-ready-cropped.png)

5. Zostanie załadowany pulpit nawigacyjny platformy Azure i przy pierwszym użyciu będziesz mieć możliwość skorzystaj z przewodnika.  Przewodnik składa się z szeregu etykietki ekranowe, wskazujących się, jak można wykonywać różnorodne zadania i znajdować odpowiednie informacje.  Kończenie lub zamyka samouczka wyświetli pulpit nawigacyjny platformy Azure.

6. Zakładki [witryny Azure Portal](https://portal.azure.com) łatwy dostęp w przyszłości.
   > [!div class="mx-imgBorder"]
   > ![Pulpit nawigacyjny platformy Azure](_img/vs-azure/vs-azure-dashboard-cropped.png)

## <a name="faq"></a>Najczęściej zadawane pytania

### <a name="q-how-do-i-submit-a-technical-support-incident-from-within-the-azure-portal"></a>PYT.: Jak przesłać zdarzenie pomocy technicznej w witrynie Azure Portal?
Odp.:  Przesyłanie żądania pomocy technicznej w witrynie Azure portal jest procesem trzech krokach.
1. Aktywuj korzyść pomocy technicznej i uzyskać swojego identyfikatora umowy identyfikator dostępu.
2. Połącz umowę na pomoc techniczną do subskrypcji platformy Azure.
3. Prześlij żądanie pomocy technicznej.

Odwiedź stronę [pomocy technicznej](vs-tech-support.md) dokumentacji, aby uzyskać szczegółowe informacje.

### <a name="q-ive-received-a-message-stating-that-my-access-to-azure-will-end-soon--what-can-i-do-to-continue-using-azure"></a>PYT.: Czy mogę otrzymał komunikat z informacją, że moje dostęp do platformy Azure wkrótce się zakończy.  Co mogę zrobić, aby kontynuować korzystanie z platformy Azure?

Odp.:  Otrzymujesz tę wiadomość e-mail, ponieważ używasz korzyści platformy Azure skojarzonych z subskrypcją programu Visual Studio, a subskrypcja wygasła lub została usunięta.  Aby kontynuować korzystanie z systemu Azure, masz kilka opcji:

**Zalecana akcja:**

Aby nadal korzystać z usługi Visual Studio na platformie Azure środków i rabatów skojarzonych, wykonaj jedną z następujących czynności:

Jeśli ktoś w Twój zespół lub organizacja zakupi subskrypcje w Twoim imieniu, skontaktuj się z administratorem programu Visual Studio w Twojej organizacji, a żądania subskrypcji programu Visual Studio, który umożliwia Ci dostęp do platformy Azure miesięczną ilość środków, które są potrzebne.   Poproś administratora, aby przypisać nową subskrypcję do subskrypcji poprzedniej została przypisana do tego samego Identyfikatora użytkownika, dzięki czemu nie trzeba już martwić się o przeniesienie subskrypcji platformy Azure.

Jeśli możesz zakupić subskrypcję bezpośrednio, wykonaj jedną z następujących czynności:
1. Zakupu rocznej programu Visual Studio [Enterprise](https://marketplace.visualstudio.com/items?itemName=ms.vs-enterprise-annual) lub [Professional](https://marketplace.visualstudio.com/items?itemName=ms.vs-professional-annual) subskrypcję za pośrednictwem witryny Marketplace programu Visual Studio w chmurze.
2. Skorzystaj z zalet 50% lub rabat w wysokości wyższej przez odnawiając bieżącą subskrypcję za pośrednictwem [Microsoft Store](https://www.microsoft.com/en-us/search/result.aspx?q=visual+studio+subscription).

Porównaj wszystkie [subskrypcje i korzyści](https://visualstudio.microsoft.com/vs/pricing/).

Jeśli masz aktywną inną subskrypcję programu Visual Studio na na **tego samego** subskrypcji poziomu przy użyciu konta Microsoft, można je połączyć w tym Azure korzyści, dodając [alternatywne konto](manage-vs-subscriptions.md#linking-my-subscription-to-existing-azure-devops-services-or-azure-subscriptions) w programie Visual Studio portal subskrypcji. Jeśli to nie rozwiąże problemu, możesz również spróbować [przesyłania korzyści platformy Azure](/azure/billing/billing-subscription-transfer/) na aktywną subskrypcję programu Visual Studio.

Jeśli masz aktywną inną subskrypcję programu Visual Studio na na **różnych** poziomie i w ramach konta Microsoft, skontaktuj się z [pomocy technicznej platformy Azure](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade) uzyskać pomoc.

Jeśli Twoja subskrypcja jest objęta limitem wydatków i podejmiesz żadnych działań przed jego wygaśnięcia, subskrypcje platformy Azure Twojej subskrypcji platformy Azure zostaną wyłączone. Aby przekonwertować ją do oferty płatności zgodnie z rzeczywistym i nadal z niego korzystać, Dodaj instrument płatniczy i [Usuń limit wydatków](/azure/billing/billing-spending-limit/).

Jeśli nie chcesz kontynuować tę subskrypcję programu Visual Studio na platformie Azure, możesz przetransferować swoje zasoby, takie jak maszyny wirtualne i bazy danych do innej subskrypcji platformy Azure i [anulować](https://azure.microsoft.com/support/cancel-account/) tę subskrypcję w dowolnym momencie.

Jeśli uważasz, że błąd otrzymujesz tę wiadomość e-mail, skontaktuj się z [dział obsługi klienta platformy Azure](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade).
