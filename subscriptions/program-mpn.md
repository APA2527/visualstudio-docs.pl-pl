# <a name="visual-studio-subscriptions-offered-to-partners-in-the-microsoft-partner-network-mpn"></a>Subskrypcje programu Visual Studio oferowane partnerom w sieci Microsoft Partner Network (MPN)

Partnerzy, którzy uzyskują kompetencje lub zakupią pakiet Action Pack w mpn, mogą korzystać z subskrypcji programu Visual Studio. Te subskrypcje są licencjonowane dla użytkownika i są prawidłowe tylko wtedy, gdy są przypisane do określonego nazwanego użytkownika, który pracuje dla firmy partnerskiej.

## <a name="subscriptions-for-partners-with-an-mpn-competency"></a>Subskrypcje dla partnerów z kompetencją MPN

Gdy partner zdobywa kompetencje — na poziomie silver lub gold — partnerzy zarabiają podstawową liczbę subskrypcji programu Visual Studio Enterprise. Ta podstawowa korzyść zmienia się, jeśli partner zdobędzie co najmniej jedną kompetencję gold. Subskrypcja programu Visual Studio Enterprise to oferta niestandardowa, która jest w przybliżeniu równoważna [subskrypcji Visual Studio Enterprise Standard,](https://visualstudio.microsoft.com/vs/pricing/)ale z pewnymi korzyściami usuniętymi (wiele z nich jest oferowanych oddzielnie partnerom jako ogólne korzyści z usługi MPN, a nie jako część subskrypcji programu Visual Studio Enterprise oferowanych za pośrednictwem usługi MPN).

### <a name="core-visual-studio-benefit-for-earning-at-least-one-competency-of-any-kind"></a>Podstawowa korzyść programu Visual Studio za zdobycie co najmniej jednej kompetencji dowolnego rodzaju

| Kompetencje zdobyte tylko na poziomie silver               | Co najmniej jedna kompetencja zdobyta na poziomie złota   |
|------------------------------------------------------------|----------------------------------------------------|
| 5 subskrypcji programu Visual Studio Enterprise                   | 10 subskrypcji programu Visual Studio Enterprise          |

### <a name="additional-visual-studio-benefit-for-select-competencies"></a>Dodatkowa korzyść programu Visual Studio dla wybranych kompetencji

| Kompetencji                                  | Dodatkowe subskrypcje uzyskane za kompetencję **Silver** | Dodatkowe subskrypcje uzyskane za kompetencję **gold** |
|---------------------------------------------|-----------------------------------------------------------|---------------------------------------------------------|
| Projektowanie aplikacji                     | 5 subskrypcji programu Visual Studio Enterprise                  | 25 subskrypcji programu Visual Studio Enterprise               |
| Zarządzanie relacjami z klientami w chmurze      | 5 subskrypcji programu Visual Studio Enterprise                  | 25 subskrypcji programu Visual Studio Enterprise               |
| Platforma chmurowa                              | 5 subskrypcji programu Visual Studio Enterprise                  | 25 subskrypcji programu Visual Studio Enterprise               |
| Produktywność chmury                          | 5 subskrypcji programu Visual Studio Enterprise                  | 25 subskrypcji programu Visual Studio Enterprise               |
| DevOps                                      | 5 subskrypcji programu Visual Studio Enterprise                  | 25 subskrypcji programu Visual Studio Enterprise                |

### <a name="limits-for-visual-studio-subscriptions-earned-through-competencies"></a>Limity subskrypcji programu Visual Studio uzyskane za pośrednictwem kompetencji

## <a name="subscriptions-for-partners-purchasing-the-action-pack"></a>Subskrypcje dla partnerów nabywających Pakiet Action Pack

Partnerzy posiadający kompetencje mogą zakupić pakiet Action Pack. Na jednego partnera można kupić tylko jeden pakiet akcji.

Dzięki pakietowi Action Pack partner otrzymuje **3 subskrypcje programu Visual Studio Professional (MPN).** Visual Studio Professional (MPN) to oferta niestandardowa bez dostępnego na rynku odpowiednika. Oto, co jest zawarte:

- Visual Studio Professional — tylko najnowsza wersja
- Windows Enterprise — tylko najnowsza wersja, do użycia tylko do tworzenia i testowania
- Windows Server — tylko najnowsza wersja, do użycia tylko do tworzenia i testowania

Subskrybenci programu Visual Studio Professional (MPN) mogą również pobierać i używać oprogramowania do ich tworzenia, w tym programu SQL Server Developer Edition.

Subskrypcja programu Visual Studio Professional (MPN) nie obejmuje korzyści z usług Azure DevOps Services, indywidualnego kredytu usługi Azure DevTest, pomocy technicznej, szkolenia ani innych korzyści dla subskrybentów dostępnych w subskrypcjach programu Visual Studio, które są oferowane Komercyjnie.

## <a name="internal-use-only-restriction"></a>Wewnętrzne ograniczenie dotyczące wyłącznie użytkowania

Reguły MPN przewidują, że całe oprogramowanie oferowane jako świadczenie z programu może być używane tylko wewnętrznie. Dotyczy to oprogramowania oferowanego do użytku poszczególnych subskrybentów programu Visual Studio oraz ogólnych korzyści wynikających z licencji na oprogramowanie MPN.

To wewnętrzne ograniczenie tylko do użytku zasadniczo oznacza, że partnerzy nie mogą korzystać z oprogramowania w ramach pracy, którą wykonują dla stron spoza własnej organizacji. Tworzenie aplikacji, która będzie używana tylko przez pracowników partnerskich jest w porządku, ale tworzenie aplikacji, która jest częścią projektu klienta nie jest dozwolone.

## <a name="sign-in-required-with-visual-studio"></a>Logowanie wymagane za pomocą programu Visual Studio

W przypadku subskrypcji programu Visual Studio oferowanych za pośrednictwem nazwy MPN licencjonowany użytkownik musi zalogować się do programu Visual Studio przy użyciu tej samej tożsamości/logowania, która została użyta do aktywowania subskrypcji programu Visual Studio. Klucze produktu nie są oferowane do odblokowania produktu; logowanie jest jedyną opcją uruchamiania programu Visual Studio. Subskrybent programu Visual Studio może działać w trybie offline przez maksymalnie 30 dni, zanim program Visual Studio przestanie działać. Podczas pracy w trybie online program Visual Studio dyskretnie odświeża stan licencji użytkownika okresowo, aby upewnić się, że subskrypcja jest nadal ważna.
