# <a name="how-do-i-know-what-software-is-available-for-download"></a>Jak sprawdzić, jakie oprogramowanie jest dostępne do pobrania?

[Listy programów do pobrania](http://download.microsoft.com/download/1/5/4/15454442-CF17-47B9-A65D-DF84EF88511B/Visual_Studio_by_Subscription_Level.xlsx) dostępne jest zależny od subskrypcji używasz.  Jeśli masz więcej niż jedną subskrypcję skojarzony z tym samym adresem logowania, zobaczysz dołączonego najwyższego poziomu subskrypcji, niezależnie od tego, które subskrypcji wyświetlasz pliki do pobrania.  Na przykład jeśli masz subskrypcję standardowe Visual Studio Enterprise i członkostwa programu Visual Studio Dev Essentials, zobaczysz pełną listę dostępnych składników do pobrania w ramach subskrypcji Enterprise.  Aby zapoznać się z listą subskrypcji, odwiedź stronę [https://my.visualstudio.com/subscriptions](https://my.visualstudio.com/subscriptions?wt.mc_id=o~msft~docs). 

Jeśli masz już subskrypcję programu Visual Studio: 
1. Przejdź do [https://my.visualstudio.com/downloads](https://my.visualstudio.com/downloads?wt.mc_id=o~msft~docs)
2. Zaloguj się do subskrypcji.  (Uzyskaj Pomoc [logowanie](/visualstudio/subscriptions/signing-in).)
3. Po pomyślnym logowania można wyszukać użytkownika odpowiednie pliki do pobrania.  Można także wyświetlić listę popularnych pliki do pobrania, które można sortować według dowolnej z kilku kryteriów. 

Aby szybko znaleźć produktów, użyj funkcji "Wyszukiwanie" w górnej części strony, lub kliknij różnych rodzajów po lewej stronie.

> [!TIP]
> Jeśli szukasz klucze produktu, kliknij przycisk "Klucze produktu" w górnej części strony.

Możesz również pobrać arkusz kalkulacyjny programu Excel zawierający obszernej listy [tytułów oprogramowania dostępne](http://download.microsoft.com/download/1/5/4/15454442-CF17-47B9-A65D-DF84EF88511B/Visual_Studio_by_Subscription_Level.xlsx) dla każdego typu subskrypcji.
