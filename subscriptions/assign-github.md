# <a name="manage-visual-studio-subscriptions-with-github-enterprise"></a>Zarządzanie subskrypcjami programu Visual Studio za pomocą usługi GitHub Enterprise
Klienci, którzy mają umowy Enterprise Agreement (EA) z firmą Microsoft, mogą zakupić nową ofertę subskrypcji, która łączy standardowe subskrypcje programu Visual Studio i gitHub Enterprise. Jest to łatwy i ekonomiczny sposób dla subskrybentów programu Visual Studio do nabycia GitHub Enterprise. 

Gdy organizacja kupuje subskrypcje programu Visual Studio w usłudze GitHub Enterprise, są one aprowizowane i zarządzane w dwóch częściach.

## <a name="manage-visual-studio-subscriptions"></a>Zarządzanie subskrypcjami programu Visual Studio
Gdy organizacja kupuje subskrypcje programu Visual Studio w usłudze GitHub Enterprise, część subskrypcji programu Visual Studio jest aprowizowana natychmiast, a subskrypcje są dostępne do przypisania i zarządzania w portalu [administracji subskrypcjami](https://manage.visualstudio.com) programu Visual Studio. 

Aby uzyskać więcej informacji na temat zarządzania subskrypcjami, zapoznaj się z tymi tematami:
- [Korzystanie z portalu administracyjnego](using-admin-portal.md)
- [Przypisywanie subskrypcji](assign-license.md)
- [Edytowanie subskrypcji](edit-license.md)
- [Usuwanie subskrypcji](delete-license.md)
- [Nadmierne alokacje](handle-overclaimed-license.md)

> [!Important]
> Jeśli subskrypcje programu Visual Studio z programem GitHub Enterprise są przypisywane przez administratorów subskrypcji programu Visual Studio i nigdy nie było zakupu tych subskrypcji, nie będą one widoczne dla administratorów usługi GitHub Enterprise w organizacji. Aby upewnić się, że subskrypcje GitHub Enterprise są widoczne, zakup obejmujący **co najmniej jeden** program Visual Studio Professional z pakietem GitHub Enterprise lub Visual Studio Enterprise z subskrypcją GitHub Enterprise powinien zostać dokonany przy pierwszym przypisaniu subskrypcji.  
>
> Obowiązkiem klienta jest upewnienie się, że dla każdej przypisanej subskrypcji Usługi GitHub istnieje odpowiedni program Visual Studio z subskrypcją Usługi GitHub przypisany w portalu administracji subskrypcji programu Visual Studio, aby zachować zgodność z wymagania licencyjne dla tej subskrypcji.

## <a name="manage-github-enterprise-subscriptions"></a>Zarządzanie subskrypcjami GitHub Enterprise
Po zakupie subskrypcji GitHub Enterprise gitHub współpracuje z klientami, aby pomóc w tworzeniu i konfigurowaniu organizacji, które będą uzyskiwać dostęp do usługi GitHub i identyfikować administratorów.  Administratorzy ci otrzymują powiadomienie, że zostali skonfigurowani jako administratorzy.  

Ponieważ ten proces jest bardziej złożony, może upłynąć kilka dni po zakupie subskrypcji dla organizacji i administratorów, aby w pełni skonfigurować.

GitHub jest dostępny jako GitHub.com oparte na chmurze lub lokalnego programu GitHub Enterprise Server.  Procesy zarządzania tymi dwiema wersjami różnią się.  GitHub udostępnia wiele tematów pomocy i przewodników administratorów ułatwiające zarządzanie subskrypcjami usługi GitHub Enterprise.  Poniżej podajemy linki do wybranych tematów.
