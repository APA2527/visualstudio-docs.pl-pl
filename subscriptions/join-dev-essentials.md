# <a name="how-to-join-visual-studio-dev-essentials"></a>Jak sprzęgać Essentials deweloperów programu Visual Studio

Visual Studio Dev Essentials zapewnia wszystkie elementy potrzebne do tworzenia i wdrażania aplikacji na dowolnej platformie. Stan graficznych narzędzi, mocy chmury, szkoleń i pomocy technicznej jest kiedykolwiek nasz najbardziej kompleksowy program wolnego developer.

Odwiedź stronę [https://www.visualstudio.com/dev-essentials/](https://www.visualstudio.com/dev-essentials/) Aby dowiedzieć się o nowościach, który znajduje się w Visual Studio Dev Essentials członkostwa. 

Gdy wszystko jest gotowe do rejestracji, wystarczy kliknąć [Join lub dostępu teraz](https://my.visualstudio.com/Benefits?wt.mc_id=o~msft~vscom~devessentials-hero~mt689&campaign=o~msft~vscom~devessentials-hero~mt689) przycisku. 

To wszystko jest do niego.
