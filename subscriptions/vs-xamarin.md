# <a name="xamarin-university-training-benefit-in-visual-studio-subscriptions"></a>Xamarin University szkolenia korzyści w ramach subskrypcji programu Visual Studio

Tworzenie natywnych iOS i aplikacje dla systemu Android w języku C# z szkoleń w zarejestrowany i wykładów gościa.  
-  Szkolenie Mobile University Xamarin dla Visual Studio Dev Essentials zapewnia dostęp do dwanaście kursów i trzy wykładów gościa. 
-  Szkolenia Mobile University Xamarin dla Visual Studio Professional i Visual Studio Enterprise subskrypcji zawiera szkoleń w tym samym i lectures jako oferty podstawowe informacje dotyczące programu Visual Studio Dev, oraz 18 więcej kursów i sześciu wykładów więcej.

## <a name="activation-steps"></a>Kroki aktywacji
1.  Zaloguj się do [ https://my.visualstudio.com/benefits ](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs). 
2.  Zlokalizuj kafelka Xamarin University szkolenia w sekcji programowanie Professional i kliknij pozycję **Pobierz** łącza w lewym dolnym kafelka korzyści.   Otrzymasz powiadomienie, że kod kuponu zostały pomyślnie pobrane.  Kliknij przycisk **Aktywuj** aby kontynuować.

    ![Kafelek korzyści szkolenia Xamarin University](_img\vs-xamarin\vs-xamarin-tile.png)

3.  Na stronie rejestracja zakończyć swoje informacje kontaktowe, kliknij pole wyboru, aby zaakceptować warunki instrukcji użycia i ochrona prywatności i kliknij przycisk **aktywować korzyści**. 

    ![Rejestracja korzyści szkolenia Xamarin University](_img\vs-xamarin\vs-xamarin-registration-resized.png)
4.  To wszystko jest do niego!  Skoro już aktywowana korzyści Xamarin University szkolenia, użytkownik zostanie przekierowany szkolenia Mobile University Xamarin dla strony programu Microsoft Visual Studio i będzie gotowy do uruchomienia learning.  

    ![Strona główna korzyści szkoleń University Xamarin](_img\vs-xamarin\vs-xamarin-home-resized.png)

## <a name="frequently-asked-questions"></a>Często zadawane pytania
### <a name="q--what-topics-are-covered-in-the-classes-and-lectures"></a>Pytanie: jakie tematy klas i wykładów?
A: pełną listę kursów i wykładów można znaleźć zarówno ofert:  https://www.xamarin.com/university/mobile-training 

### <a name="q-im-already-a-xamarin-university-subscriber-does-this-affect-my-xamarin-university-mobile-training-benefit"></a>Pytanie: Mam już subskrybenta Xamarin University. To wpływa na mój korzyści Xamarin University Mobile szkolenia?
A: nie istniejących Xamarin University subskrybenci będą mieli pełny dostęp do ich Xamarin University roczne subskrypcji, w tym kursów na żywo, materiałów i wykładów gościa przez rejestrowanie w na university.xamarin.com.  
Uwzględnione kursy jako korzyści w programie Visual Studio Dev Essentials i Visual Studio subskrypcje są oddzielone od subskrypcji Xamarin University i dostępne z programu Visual Studio Dev Essentials lub portalu Visual Studio, nie university.xamarin.com .  Nagrania plan szkolenia Mobile University Xamarin uwzględnione w programie Visual Studio Dev Essentials i subskrypcji programu Visual Studio są dołączone do subskrypcji Xamarin University roczne, są one nadal dostępne tak długo, jak program członkostwo lub Visual Studio subskrypcji są prawidłowe. Możesz uzyskać dostęp z portalem programu Visual Studio Dev Essentials lub program Visual Studio, niezależnie od stan subskrypcji Xamarin University.

### <a name="q--can-i-become-a-certified-xamarin-developer-with-the-xamarin-university-included-in-visual-studio-dev-essentials-what-about-through-visual-studio-subscriptions"></a>Pytanie: czy można zostać dewelopera Xamarin certyfikowane z University Xamarin uwzględnione w programie Visual Studio Dev Essentials? Informacje o za pośrednictwem subskrypcji programu Visual Studio?
A: nie kursy i materiały dołączonego University Xamarin w Visual Studio Dev Essentials i Visual Studio/MSDN subskrypcji nie są prawidłowe dla certyfikacji. Materiałów, szkoleń i nagrania potrzebne do certyfikacji Xamarin, tym znajdują się tylko w pełną subskrypcję Xamarin University.
