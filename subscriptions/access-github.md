# <a name="introducing-visual-studio-subscriptions-with-github-enterprise"></a>Wprowadzenie subskrypcji programu Visual Studio w usłudze GitHub Enterprise  

Klienci, którzy mają umowy Enterprise Agreement (EA) z firmą Microsoft, mogą zakupić nową ofertę subskrypcji, która łączy standardowe subskrypcje programu Visual Studio i gitHub Enterprise. Jest to łatwy i ekonomiczny sposób dla subskrybentów programu Visual Studio do nabycia GitHub Enterprise. 

- Wprowadzaj innowacje na dużą skalę dzięki programom Visual Studio i GitHub, bezpiecznie wprowadzając kod open source, społeczność i najlepsze rozwiązania do środowiska IDE i przepływów pracy.
- Użyj najlepszych w swojej klasie narzędzi i usług programistycznych z najbardziej wszechstronnym zestawem zasobów do tworzenia, wdrażania i zarządzania doskonałymi aplikacjami. 
- Współpracuj, zarządzaj żądaniami ściągania w usłudze GitHub Enterprise i recenzuj kod źródłowy, w którym został napisany - w edytorze. 

## <a name="whats-available"></a>Co jest dostępne? 

Jeśli twoja organizacja wprowadza nową umowę EA lub odnawia istniejącą, dostępne są następujące opcje:

- Visual Studio Enterprise z gitHub Enterprise
- Visual Studio Professional z gitHub Enterprise

Jeśli twoja organizacja ma EA, która nie jest jeszcze z powodu odnowienia, można zakupić "Step-up" SKU, aby dodać GitHub Enterprise do istniejących subskrypcji Visual Studio Enterprise lub Visual Studio Professional. Następnie, gdy umowa zostanie odnowiona, możesz kupić ofertę.

Istniejące jednostki SKU dla programów Visual Studio Enterprise, Visual Studio Professional i GitHub Enterprise będą nadal dostępne oddzielnie. 

Aby dowiedzieć się więcej https://visualstudio.microsoft.com/subscriptions/visual-studio-githubo korzyściach i opcjach, odwiedź stronę . 

## <a name="getting-your-subscriptions"></a>Uzyskiwanie subskrypcji

Subskrypcje programu Visual Studio + GitHub Enterprise są przypisywane w dwóch częściach:
1. Subskrypcja programu Visual Studio (Enterprise lub Professional)
2. Subskrypcja GitHub Enterprise

Gdy organizacja kupuje subskrypcje, są one przypisywane przez administratorów. Subskrypcje programu Visual Studio i subskrypcje Usługi GitHub Enterprise są przypisywane oddzielnie.  

Jeśli zostanie przypisana subskrypcja programu Visual Studio + GitHub, otrzymasz wiadomość e-mail z informacją, że administrator subskrypcji programu Visual Studio przypisał Ci subskrypcję.  W tym momencie, będzie można https://my.visualstudio.com/subscriptionszobaczyć go w .  

Ponieważ subskrypcje Usługi GitHub Enterprise są przypisywane oddzielnie, możesz otrzymać powiadomienie od innego administratora, w zależności od tego, jak firma zdecydowała się zorganizować swoje subskrypcje.  Może również wystąpić opóźnienie między przypisaniem subskrypcji programu Visual Studio a subskrypcją Usługi GitHub Enterprise, więc nie zainteresowanych, jeśli nie są one natychmiast dostępne.  Proces przypisywania subskrypcji Usługi GitHub jest bardziej złożony i może potrwać kilka dni roboczych.

## <a name="frequently-asked-questions"></a>Często zadawane pytania

### <a name="q--where-is-my-github-subscription"></a>P: Gdzie jest moja subskrypcja Usługi GitHub?
Odp.: Proces konfigurowania subskrypcji usługi GitHub może trochę potrwać.  Jeśli czekałeś i nic nie słyszałeś, możesz skontaktować się z administratorem Usługi GitHub.  Możliwe, że administrator nadal nad nim pracuje lub postanowił nie przypisywać Ci subskrypcji Usługi GitHub. 

### <a name="q--how-do-i-reach-my-github-administrator"></a>P: Jak mogę skontaktować się z administratorem usługi GitHub?
Odp.: Jeśli korzystasz z programu GitHub Enterprise Server, możesz kliknąć łącze **pomocy technicznej** widoczne u dołu klienta, aby połączyć się z administratorem.

### <a name="q-do-i-get-the-full-visual-studio-subscription-professional-or-enterprise-or-do-i-just-get-the-ide-with-this-offering"></a>Pyt.: Czy mogę uzyskać pełną subskrypcję programu Visual Studio (Professional lub Enterprise), czy po prostu uzyskać IDE z tej oferty?
Odp.: Wraz z subskrypcją GitHub Enterprise otrzymujesz pełną subskrypcję programu Visual Studio.  Otrzymujesz wszystkie korzyści, takie jak pobieranie oprogramowania, usługi w chmurze, pomoc techniczna i korzyści dla partnerów zawarte w tych subskrypcjach, tak jak w przypadku zakupu subskrypcji oddzielnie.
