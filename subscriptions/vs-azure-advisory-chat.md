# <a name="azure-advisory-chat"></a>Czat z doradcą Azure Advisor
Uzyskaj pomoc dotyczącą rozpoczynania pracy z platformą Azure w czasie rzeczywistym! Obejmuje poradnik dotyczący usług Azure PaaS Services.  Korzyść z rozmowy doradczej platformy Azure jest dostępna w Visual Studio Enterprise standardowej i rocznej subskrypcji chmury.

## <a name="activation-steps"></a>Kroki aktywacji
Korzystanie z usługi Azure Advisory Chat jest proste.
1. Zaloguj się do witryny [https://my.visualstudio.com/benefits](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs).
2. Znajdź kafelek Azure Advisory chat w sekcji Pomoc techniczna, a następnie kliknij pozycję **czat teraz**.
    > [!div class="mx-imgBorder"]
    > ![Kafelek rozmowy doradczej platformy Azure](_img/vs-azure-advisory/vs-azure-advisory-tile.png)

3. Zobaczysz powiadomienie, że usługa jest obecnie oferowana w języku angielskim tylko z wyjątkiem pomocy technicznej dotyczącej kont, rozliczeń i problemów z subskrypcją.  Kliknij przycisk **Kontynuuj**.
    > [!div class="mx-imgBorder"]
    > ![Oświadczenie dotyczące rozmowy doradczej platformy Azure](_img/vs-azure-advisory/vs-azure-advisory-disclaimer.png)

4. To wszystko.  Zostanie otwarte okno rozmowy i zostanie wyświetlony stan informujący o tym, gdzie znajduje się w kolejce.  Otrzymasz odpowiedź od następnego dostępnego eksperta.
    > [!div class="mx-imgBorder"]
    > ![Czat z doradcą Azure Advisor](_img/vs-azure-advisory/vs-azure-advisory-chat.png)
