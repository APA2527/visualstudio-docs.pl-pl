# <a name="large-teams-and-external-contractors"></a>Duże zespoły i zewnętrzni wykonawcy
Administratorzy subskrypcji programu Visual Studio są odpowiedzialni za zapewnienie, że każdy użytkownik, który wchodzi w interakcję z oprogramowaniem licencjonowanym przez program Visual Studio, jest odpowiednio licencjonowany z własną subskrypcją programu Visual Studio.

## <a name="internal-teams"></a>Zespoły wewnętrzne
Zazwyczaj nowoczesne organizacje oprogramowania obejmują interesariuszy z kilku grup. Zidentyfikuj kontakty z każdej grupy, które mogą pomóc w śledzeniu zasobów reklamowych użytkowników i zmian.
Każda organizacja jest inna, ale typowa lista zespołów zaangażowanych w rozwój może zawierać:
- Zespoły inżynierów oprogramowania.
- Zespoły biznesowe, w tym właściciele produktów i analitycy biznesowi.
- Zespoły zarządzania projektami.
- Zespoły wysokiej jakości, w tym pracownicy qa i testerzy ręczni.
- Operacje IT, w tym preprodukcji i kierowników infrastruktury laboratoryjnej.

## <a name="external-contractors-and-partners"></a>Zewnętrzni wykonawcy i partnerzy
Zewnętrzni wykonawcy mogą udzielać licencji na interakcję ze środowiskiem licencjonowanym przez program Visual Studio. Certyfikowani partnerzy firmy Microsoft mogą otrzymać kilka bezpłatnych subskrypcji programu Visual Studio do użytku wewnętrznego. Jednak te subskrypcje nie obejmują działań generujących przychody, takich jak tworzenie niestandardowego oprogramowania dla klienta. Poproś partnerów o wysłanie uwierzytelnionego listu wyjaśniającego udzielane przez nich licencje i te, które są im potrzebne do uzyskania.
