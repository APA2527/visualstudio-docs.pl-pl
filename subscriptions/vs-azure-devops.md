# <a name="azure-devops-benefits-for-visual-studio-subscribers"></a>Korzyści DevOps platformy Azure dla subskrybentów programu Visual Studio

Jako aktywnego subskrybenta programu Visual Studio można użyć różnych funkcji w DevOps platformy Azure, dostępnych w ramach subskrypcji, jako pakiety korzyści. Będzie można używać tych samych funkcji w każdej organizacji DevOps platformy Azure, w którym jesteś członkiem, czy utworzone organizacji, czy zostały dodane przez kogoś innego.

Po zalogowaniu się w DevOps platformy Azure przy użyciu tożsamości, która została użyta w przypadku aktywowania subskrypcji programu Visual Studio lub [alternatywnej tożsamości](vs-alternate-identity.md), możemy rozpoznać to automatycznie.

[Załóż bezpłatne organizacji DevOps platformy Azure](https://visualstudio.microsoft.com/team-services/)

| Platformy MSDN (standardowa)                                          | Licencjonowania zbiorowego, handlu detalicznego                                              | Tablice platformy Azure i repozytoriów (Basic), planów testowych platformy Azure                                              |  Tak          |
