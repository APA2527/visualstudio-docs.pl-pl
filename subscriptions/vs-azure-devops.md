# <a name="azure-devops-services-benefits-for-visual-studio-subscribers"></a>Korzyści usługom DevOps platformy Azure dla subskrybentów programu Visual Studio

Jako aktywnego subskrybenta programu Visual Studio można użyć różnych funkcji w usługach Azure DevOps Services są uwzględniane w ramach subskrypcji, jako pakiety korzyści. Będzie można używać tych samych funkcji w ramach każdego konta usługi DevOps platformy Azure, w którym jesteś członkiem, czy utworzone konto, czy zostały dodane przez kogoś innego.

Po zalogowaniu się usługom DevOps platformy Azure przy użyciu tożsamości, która została użyta w przypadku aktywowania subskrypcji programu Visual Studio lub [alternatywnej tożsamości](vs-alternate-identity.md), możemy rozpoznać to automatycznie.

[Załóż bezpłatne konto usługi DevOps platformy Azure](https://visualstudio.microsoft.com/team-services/)

## <a name="eligibility"></a>Uprawnienie

| Poziom subskrypcji                                                 |     Kanały                                            | Korzyść                                                          | Podlega odnowieniu?    |
|--------------------------------------------------------------------|---------------------------------------------------------|------------------------------------------------------------------|---------------|
| Program Visual Studio Enterprise (standardowy, roczna w chmurze, chmura miesięczne)   | Licencjonowania zbiorowego, Azure, w sprzedaży detalicznej, wybrane NFR<sup>1</sup>, ekwiwalentu pełnego wymiaru czasu  | Podstawowe, Test Manager, Zarządzanie pakietami, może być samodzielnie hostowane ciągłej integracji/ciągłego wdrażania       |  Tak          |
| Program Visual Studio Professional (standardowy, roczna w chmurze, chmura miesięczne) | Licencjonowania zbiorowego, Azure, w sprzedaży detalicznej                                        | Podstawowy                                                            |  Tak          |
| Visual Studio Test Professional (standardowa)                         | Licencjonowania zbiorowego, handlu detalicznego                                              | Basic, Test Manager                                              |  Tak          |
| Platformy MSDN (standardowa)                                          | Licencjonowania zbiorowego, handlu detalicznego                                              | Basic, Test Manager                                              |  Tak          |

<sup>1</sup>*obejmuje: nie for Resale (NFR) firmy Microsoft zwracającej partnera (MVP), dyrektor Region (usług pulpitu zdalnego), Visual Studio Industry Partner (VSIP), sieci Microsoft Partner Network (przedsiębiorstwo), BizSpark, MCT Software & usług dla deweloperów.   Nie obejmuje: MCT Software & Services, Wyobraź sobie.*

Nie masz pewności której subskrypcji używasz?  Połączyć się z [ https://my.visualstudio.com/subscriptions ](https://my.visualstudio.com/subscriptions?wt.mc_id=o~msft~docs) Aby wyświetlić wszystkie subskrypcje, które są przypisane do Ciebie.
Jeśli nie widzisz wszystkie swoje subskrypcje, mogą mieć co najmniej jeden przypisany do innego adresu e-mail.  Musisz zalogować się przy użyciu tego adresu e-mail, aby wyświetlić te subskrypcje.
