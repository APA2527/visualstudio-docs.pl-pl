# <a name="visual-studio-tools-for-unity"></a>Narzędzia Visual Studio Tools for Unity
![Narzędzia Visual Studio Tools for Unity](../media/hero.png)

## <a name="overview"></a>Omówienie
Visual Studio Tools for Unity to bezpłatne rozszerzenie programu Visual Studio, które umożliwia programowi Visual Studio zaawansowane narzędzie do tworzenia gier i aplikacji międzyplatformowych za pomocą aparatu Unity.

Chociaż Edytor Unity doskonale nadaje się do pracy ze sobą w świecie, nie można pisać w nim kodu. Za pomocą Visual Studio Tools for Unity można korzystać ze znanych funkcji edytowania kodu, debugowania i produktywności programu Visual Studio, aby tworzyć skrypty edytora i gier dla projektu Unity przy użyciu języka C#, a także debugować je przy użyciu zaawansowanych możliwości debugowania programu Visual Studio.

Ale Visual Studio Tools for Unity jest więcej niż to; ma także ścisłą integrację z edytorem Unity, dzięki czemu można poświęcać mniej czasu na przełączenie się z powrotem i w czasie w celu wykonywania prostych zadań, zapewnia ulepszenia produktywności specyficzne dla aparatu Unity i umieszcza dokumentację aparatu Unity na wyskakującym rogu.

### <a name="compatible-with-visual-studio-community-on-windows-and-macos-and-bundled-with-unity"></a>Zgodne z programem Visual Studio Community w systemie Windows i macOS oraz z pakietem Unity
Programy Visual Studio i Visual Studio dla komputerów Mac Community są dostępne bezpłatnie i są dołączone do instalacji aparatu Unity. Więcej informacji na temat instalacji i konfiguracji można znaleźć w dokumentacji umożliwiającej [wprowadzenie](getting-started-with-visual-studio-tools-for-unity.md) Visual Studio Tools for Unity.

## <a name="intellisense-for-unity-messages"></a>Funkcja IntelliSense dla komunikatów Unity
Uzupełnianie kodu IntelliSense umożliwia szybkie i łatwe [implementowanie komunikatów interfejsu API Unity](using-visual-studio-tools-for-unity.md#intellisense-for-unity-api-messages) , takich jak `OnCollisionEnter` parametry.

![Okno dialogowe IntelliSense z funkcja oncollisionenter](../media/vs/intellisense-example.png)

## <a name="superior-debugging"></a>Doskonałe debugowanie
Visual Studio Tools for Unity obsługuje niezawodne funkcje [debugowania](using-visual-studio-tools-for-unity.md#unity-debugging) , które są oczekiwane w programie Visual Studio:

* Ustaw punkty przerwania, w tym warunkowe punkty przerwania.
* Oceń złożone wyrażenia w okno wyrażeń kontrolnych.
* Sprawdzanie i modyfikowanie wartości zmiennych i argumentów.
* Przechodzenie do szczegółów obiektów złożonych i struktur danych.

![Zatrzymano w punkcie przerwania podczas sprawdzania zmiennych](../media/vs/debugging-inspecting.png)

## <a name="codelens-support-for-unity-scripts-and-messages"></a>CodeLens obsługa skryptów i komunikatów aparatu Unity
Skrypty i funkcje komunikatów aparatu Unity są uzupełnione wskazówkami, które ułatwiają rozpoznawanie informacji zapewnianych przez środowisko Unity i jaki jest Twój kod.

 ![Nowy skrypt przedstawiający wskazówki CodeLens dla skryptu aparatu Unity i komunikatu aparatu Unity](../media/vs/codelens-support.png)

> [!NOTE]
> Obsługa CodeLens jest dostępna w programie Visual Studio 2019.

## <a name="optimized-view-of-all-your-scripts-to-match-unity"></a>Zoptymalizowany widok wszystkich skryptów do dopasowania do aparatu Unity
Eksplorator projektów aparatu Unity (UPE) jest alternatywnym sposobem wyświetlania plików projektu za pośrednictwem standardowej Eksplorator rozwiązań. UPE filtruje wyświetlane pliki i prezentuje je w hierarchii zgodnej z Unity ( **widok > Eksplorator projektów aparatu Unity** w programie Visual Studio 2019).

![Eksplorator projektów aparatu Unity](../media/vs/unity-project-explorer.png)

> [!NOTE]
> Eksplorator projektów aparatu Unity jest dostępny w programie Visual Studio 2019. W Visual Studio dla komputerów Mac okienko rozwiązania ma podobne zachowanie domyślne dla projektów Unity — nie są wymagane żadne dodatkowe widoki.
