# <a name="debug-uwp-apps-in-visual-studio"></a>Debugowanie aplikacji platformy uniwersalnej systemu Windows w programie Visual Studio
Debuger programu Visual Studio umożliwia kontrolowanie wykonania programu i sprawdź, czy jego stanu. Znajdź przyczynę wady w aplikacjach platformy uniwersalnej systemu Windows i zrozumienie dokładnie, jak aplikacja działa, można użyć debugera. Po wstrzymaniu wykonywania (podział) w debugerze programu Visual Studio zawiera plik źródłowy, który zawiera wykonywanie kodu i zaznacza wykonywania instrukcji. Można przyjrzeć się wartości zmiennych, stos wywołań wykonywania funkcji i innych aspektów stanu użytkownika programu. Można kontynuować wykonywania (kroków) program jednej instrukcji w czasie, aby zobaczyć, jak instrukcje zmienić wartości programu. W aplikacji napisanych z użyciem języka JavaScript można badania i manipulowania modelu DOM strony.
