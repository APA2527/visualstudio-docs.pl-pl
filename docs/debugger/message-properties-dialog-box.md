# <a name="message-properties-dialog-box"></a>Właściwości komunikatu — Okno dialogowe
Użyj tego okna dialogowego, aby dowiedzieć się więcej na temat określonego komunikatu. Aby wyświetlić to okno dialogowe, Przenieś fokus do [widoku komunikatów](../debugger/messages-view.md) okna. Zaznacz dowolny węzeł komunikat w drzewie, a następnie wybierz pozycję **właściwości** z **widoku** menu.  
  
 **Ogólne** karta jest karcie tylko wyświetlić. Dostępne są następujące ustawienia:  
  
 **Uchwyt okna**  
 Unikatowy identyfikator tego okna. Numery uchwyt okna są używane ponownie; identyfikują one okna jedynie przez czas ich istnienia tego okna. Kliknij tę wartość, aby wyświetlić właściwości tego okna.  
  
 **Poziom zagnieżdżenia**  
 Głębokość zagnieżdżenia tego komunikatu, gdzie 0 oznacza nie zagnieżdżenia.  
  
 **Komunikat**  
 Liczba, stanu i nazwa komunikatu wybranego systemu windows.  
  
 **elementu lResult**  
 Wartość *elementu lResult* parametru, jeśli istnieje.  
  
 **wParam**  
 Wartość *wParam* parametru, jeśli istnieje.  
  
 **lParam**  
 Wartość *lParam* parametru, jeśli istnieje. Ta wartość jest dekodowane, jeśli jest wskaźnikiem do ciągu lub struktury.
