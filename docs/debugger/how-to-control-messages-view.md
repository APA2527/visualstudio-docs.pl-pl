# <a name="how-to-control-messages-view"></a>Porady: kontrolowanie widoku komunikatów
Z programu Spy ++, masz znaczną kontrolę nad zawartość [widoku komunikatów](../debugger/messages-view.md) okna. Możesz [uruchamianie i zatrzymywanie](../debugger/how-to-start-and-stop-the-message-log-display.md) wyświetlania wiadomości w dowolnym momencie, a można określić:  
  
-   Typy wiadomości, które chcesz wyświetlić.  
  
-   Windows, które chcesz monitorować.  
  
-   Format wyświetlania komunikatu wierszy.  
  
 Te ustawienia są dostępne z [okno dialogowe opcji komunikatów](../debugger/message-options-dialog-box.md), i dotyczą tylko aktywne [widoku komunikatów](../debugger/messages-view.md) okna. Aby otworzyć to okno dialogowe, wybierz **komunikaty dziennika** z **Spy** menu.  
  
 Dostępne są trzy okienka z kartami opcje wiadomości. Aby uzyskać więcej informacji, zobacz [karta okna, okno dialogowe opcji komunikatów](../debugger/windows-tab-message-options-dialog-box.md), [karta komunikaty, okno dialogowe opcji komunikatów](../debugger/messages-tab-message-options-dialog-box.md), i [karta danych wyjściowych, okno dialogowe opcji komunikatów](../debugger/output-tab-message-options-dialog-box.md).
