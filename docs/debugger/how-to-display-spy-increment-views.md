# <a name="how-to-display-spy-views"></a>Porady: wyświetlanie widoków Spy++
Gdy narzędzie Spy ++, zostanie otwarty w [widoku systemu Windows](../debugger/windows-view.md), widok drzewa wszystkie okna i kontrolek, które są dostępne na tym komputerze. Tego okna głównego jest podpisanych "Systemu Windows 1". Ponadto są dostępne trzy innych widoków Spy ++: [widoku komunikatów](../debugger/messages-view.md), [widok procesy](../debugger/processes-view.md), i [Widok wątków](../debugger/threads-view.md). Można wyświetlić, żądany widok z **Spy** menu. Aby uzyskać więcej informacji, zobacz [widoków Spy ++](../debugger/spy-increment-views.md).
