# <a name="how-to-specify-debugger-settings"></a>Porady: Określanie ustawień debugera
W [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)], można określić różne ustawienia dla zachowania debugera, w tym wyświetlania zmiennych, czy określone ostrzeżenia są prezentowane, jak ustawić punktów przerwania i wpływ podziału uruchomione programy. Określ ustawienia debugera w **opcje** okno dialogowe.  
  
### <a name="to-set-debugger-options"></a>Aby ustawić opcje debugera  
  
1.  Na **narzędzia** menu, kliknij przycisk **opcje**.  
  
2.  W **opcje** po otwarciu okna dialogowego **debugowanie** folderu.  
  
3.  W **debugowanie** folderu, wybierz kategorię opcje.  
  
     Najbardziej typowe opcje znajdują się w **ogólne** kategorii. Aby uzyskać więcej informacji, zobacz [ogólne, debugowanie, opcje — Okno dialogowe](../debugger/general-debugging-options-dialog-box.md).  
  
4.  Wybierz lub wyczyść odpowiednią opcję lub opcji. Naciśnij klawisz F1, aby uzyskać pomoc dotyczącą opcji.
