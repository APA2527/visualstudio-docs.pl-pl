# <a name="select-code-type-dialog-box"></a>Wybór typu kodu — Okno dialogowe
Aby otworzyć to okno dialogowe, otwórz **dołączyć do procesu** , a następnie kliknij przycisk **wybierz** przycisku.  
  
 **Automatycznie Określ typ kodu do debugowania**  
 Odpowiednie debugera zostanie wybrany oparte na rodzaj kodu, który jest uruchomiony.  
  
 **Debuguj te typy kodu:**  
 Z listy wybierz typ kodu do debugowania.
