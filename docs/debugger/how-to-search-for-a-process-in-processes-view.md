# <a name="how-to-search-for-a-process-in-processes-view"></a>Porady: wyszukiwanie procesu w widoku procesów
Możesz wyszukać określonego procesu w widoku procesów za pomocą jego Identyfikatora lub moduł ciąg procesu jako kryterium wyszukiwania. Można również określić początkowego kierunku wyszukiwania. Pola w oknie dialogowym zostaną wyświetlone atrybuty proces wybranego w drzewie procesu.  
  
### <a name="to-search-for-a-process-in-processes-view"></a>Aby wyszukać procesu w widoku procesów  
  
1.  Rozmieścić okna, tak że Spy ++ i aktywne [widok procesy](../debugger/processes-view.md) okna są widoczne.  
  
2.  Z **wyszukiwania** menu, wybierz **Znajdź proces**  
  
     [Proces wyszukiwania — okno dialogowe](../debugger/process-search-dialog-box.md) otwiera.  
  
3.  Wpisz identyfikator procesu lub ciąg modułu jako kryterium wyszukiwania.  
  
4.  Wyczyść wszystkie pola, dla których nie chcesz określać wartości.  
  
    > [!TIP]
    >  Aby znaleźć wszystkie procesy należących do modułu, wyczyść **procesu** polu, a następnie wpisz nazwę modułu w **modułu** pole. Następnie użyj **Znajdź następny** kontynuować wyszukiwanie dla procesów.  
  
5.  Wybierz **się** lub **dół** dla początkowego kierunku wyszukiwania.  
  
6.  Kliknij przycisk **OK**.  
  
 Jeśli zgodny proces zostanie znaleziony, zostanie wyróżniona w **widok procesu** okna.
