# <a name="message-search-dialog-box"></a>Wyszukiwanie komunikatów — Okno dialogowe
Umożliwia to okno dialogowe Znajdź i wybierz węzeł, aby uzyskać szczegółowy komunikat o błędzie w [widoku komunikatów](../debugger/messages-view.md). Aby wyświetlić to okno dialogowe, Przenieś fokus do okna widoku komunikatów. Następnie wybierz **Znajdź komunikat** z **wyszukiwania** menu.

 Dostępne są następujące ustawienia:

 **Narzędzia wyszukiwania** używany do wybierania okna Narzędzie przeciągania i upuszczania.

 **Ukryj narzędzie Spy ++** usuwa głównego okna programu Spy ++ i pozostawia **MessageSearch** okno dialogowe widoczne na podstawie innych aplikacji.

 **Obsługa** Unikatowy identyfikator wiadomości do wyszukania.

 **Typ** typ wiadomości do wyszukania.

 **Komunikat** identyfikator wiadomości do wyszukania.

 **Wyszukaj kierunek w górę lub w dół** początkowej kierunek wyszukiwania.
