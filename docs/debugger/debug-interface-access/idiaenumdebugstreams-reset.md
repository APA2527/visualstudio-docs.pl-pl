# <a name="idiaenumdebugstreamsreset"></a>IDiaEnumDebugStreams::Reset

## <a name="syntax"></a>Składnia  
  
```C++  
HRESULT Reset ();  
```  
  
## <a name="return-value"></a>Wartość zwracana  
 Zwraca `S_OK`.
