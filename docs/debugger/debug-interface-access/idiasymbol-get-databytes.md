# <a name="idiasymbolgetdatabytes"></a>IDiaSymbol::get_dataBytes
Pobiera z jaką bajty danych symbolu OEM.  
  
## <a name="syntax"></a>Składnia  
  
```C++  
HRESULT get_dataBytes (   
   DWORD  cbData,  
   DWORD* pcbData,  
   BYTE   data[]  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `cbData`  
 [in] Rozmiar buforu do przechowywania danych.  
  
 `pcbData`  
 [out] Zwraca liczbę bajtów zapisanych, lub jeśli `data` parametr jest `NULL`, zwraca liczba dostępnych bajtów.  
  
 `data[]`  
 [out] Buforu, który jest wypełniane bajtów danych.  
  
## <a name="return-value"></a>Wartość zwracana  
 Jeśli to się powiedzie, zwraca `S_OK`; w przeciwnym razie zwraca `S_FALSE` lub kod błędu.  
  
> [!NOTE]
>  Zwracana wartość `S_FALSE` oznacza, że właściwość nie jest dostępna symbolu.
