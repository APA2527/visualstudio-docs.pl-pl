# <a name="idiaenumdebugstreamdatareset"></a>IDiaEnumDebugStreamData::Reset

## <a name="syntax"></a>Składnia  
  
```C++  
HRESULT Reset ();  
```  
  
## <a name="return-value"></a>Wartość zwracana  
 Zwraca `S_OK`.
