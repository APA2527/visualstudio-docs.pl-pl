# <a name="idiaenumframedatareset"></a>IDiaEnumFrameData::Reset
Resetuje sekwencję wyliczenia na początek.

## <a name="syntax"></a>Składnia

```
HRESULT Reset();
```

## <a name="return-value"></a>Wartość zwracana
 Zwraca wartość `S_OK`.
