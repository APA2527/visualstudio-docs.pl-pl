# <a name="idiaenumsourcefilesreset"></a>IDiaEnumSourceFiles::Reset
Resetuje sekwencji wyliczenia na początku.  
  
## <a name="syntax"></a>Składnia  
  
```C++  
HRESULT Reset ();  
```  
  
## <a name="return-value"></a>Wartość zwracana  
 Zwraca `S_OK`.
