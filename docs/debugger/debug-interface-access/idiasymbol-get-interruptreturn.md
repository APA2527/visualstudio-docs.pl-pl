# <a name="idiasymbolgetinterruptreturn"></a>IDiaSymbol::get_interruptReturn
Pobiera flagę określającą, czy funkcja zawiera typ zwracany instrukcja przerwania (na przykład X86 kodu zestawu `iret`).  
  
## <a name="syntax"></a>Składnia  
  
```C++  
HRESULT get_interruptReturn(  
   BOOL *pFlag  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `pFlag`  
 [out] Zwraca `TRUE` Jeśli funkcja ma typ zwracany instrukcja przerwania; w przeciwnym razie zwraca `FALSE`.  
  
## <a name="return-value"></a>Wartość zwracana  
 Jeśli to się powiedzie, zwraca `S_OK`; w przeciwnym razie zwraca `S_FALSE` lub kod błędu.  
  
> [!NOTE]
>  Zwracana wartość `S_FALSE` oznacza, że właściwość nie jest dostępna symbolu.
