# <a name="idiaenumtablesreset"></a>IDiaEnumTables::Reset
Resetuje sekwencję wyliczenia na początku.  
  
## <a name="syntax"></a>Składnia  
  
```C++  
HRESULT Reset ();  
```  
  
## <a name="return-value"></a>Wartość zwracana  
 Zwraca `S_OK`.
