# <a name="idiaenuminjectedsourcesreset"></a>IDiaEnumInjectedSources::Reset

## <a name="syntax"></a>Składnia

```
HRESULT Reset();
```

## <a name="return-value"></a>Wartość zwracana
 Zwraca `S_OK`.
