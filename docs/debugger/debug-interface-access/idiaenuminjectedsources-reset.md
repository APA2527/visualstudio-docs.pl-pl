# <a name="idiaenuminjectedsourcesreset"></a>IDiaEnumInjectedSources::Reset
Resetuje sekwencję wyliczenia na początku.  
  
## <a name="syntax"></a>Składnia  
  
```  
HRESULT Reset();  
```  
  
## <a name="return-value"></a>Wartość zwracana  
 Zwraca `S_OK`.
