# <a name="debug-universal-windows-apps-uwp-in-visual-studio"></a>Debugowanie aplikacji uniwersalnych systemu Windows (UWP) w programie Visual Studio

Debuger programu Visual Studio zapewnia pełną obsługę debugowanie aplikacji platformy uniwersalnej systemu Windows. 
  
Aby debugować aplikacje Windows 8.1, zobacz [aplikacji ze sklepu debugowania w programie Visual Studio](https://msdn.microsoft.com/en-us/library/hh441472.aspx). Aby debugować aplikacje Windows Phone 8, zobacz [debugowanie aplikacji dla systemu Windows Phone 8](https://msdn.microsoft.com/library/windows/apps/ff402572(v=vs.105).aspx).
