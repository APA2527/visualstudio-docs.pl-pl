# <a name="error-mixed-mode-debugging-for-ia64-processes-is-unsupported"></a>Błąd: debugowanie w trybie mieszanym dla procesów IA64 nie jest obsługiwane
[!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] Debuger nie obsługuje debugowania mieszanych kodu natywnego i zarządzanego w procesie opartych na architekturze Itanium.  
  
### <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Tworzenie 32-bitowej wersji do debugowania aplikacji.
