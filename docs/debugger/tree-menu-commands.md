# <a name="tree-menu-commands"></a>Polecenia menu Drzewo
Menu **drzewa** zawiera polecenia służące do rozszerzania lub zwijania widoków.

 **Rozwiń jeden poziom** Rozwija obecnie wybrany element w widoku system, wyświetlając jeden poziom poniżej.

 **Rozwiń gałąź** Rozwija wszystkie poziomy poniżej aktualnie wybranego elementu.

 **Rozwiń wszystko** Rozwija i wyświetla wszystkie poziomy w widoku system.

 **Zwiń** Zwija wszystkie poziomy poniżej aktualnie wybranego elementu.
