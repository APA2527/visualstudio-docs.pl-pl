# <a name="com-debugging-tools"></a>Narzędzia debugowania COM
Ten temat zawiera narzędzia, które mogą pomóc w testowania i sprawdzić COM programy.  
  
 Przeglądarka obiektów OLE/COM  
 OLEVIEW Wyświetla obiekty ActiveX i OLE zainstalowane na komputerze i interfejsów, które obsługują. Umożliwia również edytować rejestr i przyjrzyj się biblioteki typów. Aby uzyskać więcej informacji, zobacz [przy użyciu przeglądarka obiektów OLE/COM](/cpp/data/ado-rdo/using-the-ole-com-object-viewer).  
  
 Klasa Microsoft Foundation — przykłady  
 Przykłady MFC ActiveX OCLIENT i SUPERPAD są także dostępne do testowania aplikacji przed. Dołączono pełny kod źródłowy dla tych przykładów.
