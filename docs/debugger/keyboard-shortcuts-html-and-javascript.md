# <a name="keyboard-shortcuts-html-and-javascript"></a>Skróty klawiaturowe (HTML i JavaScript)

Skróty klawiaturowe w programie Visual Studio do debugowania aplikacji platformy UWP przy użyciu języka HTML i języka JavaScript.

## <a name="general-shortcuts"></a>Skróty ogólne

 Te skróty działają we wszystkich narzędziach dla aplikacji platformy UWP przy użyciu języka HTML i języka JavaScript.

|Akcja|Skrót|
|------------|--------------|
|Odśwież aplikację systemu Windows|Ctrl+Shift+R|
|Uruchom ponownie aplikację systemu Windows|Ctrl+Shift+F5|
|Zatrzymaj debugowanie|Shift+F5|
|Otwórz Centrum wydajności i diagnostyki|Alt+F2|

## <a name="javascript-console-shortcuts"></a>Skróty konsoli JavaScript

 Użyj tych skrótów w oknie konsoli JavaScript.

|Akcja|Skrót|
|------------|--------------|
|Uruchom konsolę polecenia|Enter|
|Wyczyść konsolę|Esc|
|Przełączanie między trybem jednowierszowym i wielowierszowym|Ctrl + Alt + M|

## <a name="dom-explorer-shortcuts"></a>Skróty DOM Explorer

 Użyj tych skrótów w DOM Explorer.

|Akcja|Skrót|
|------------|--------------|
|Wybierz element|Ctrl+B|
|Przełączaj między wyświetlaniem i ukrywaniem pól wyróżnienia strony sieci Web podczas korzystania z **elementu Select**|Ctrl+Shift+I|
|Zatwierdź zmiany za pomocą polecenia **Edytuj jako HTML**|Ctrl+Enter|
|Odrzuć zmiany za pomocą polecenia **Edytuj jako HTML**|Esc|
|Zwiększ wybraną wartość CSS o 1|Strzałka w górę|
|Zmniejsz wybraną wartość CSS o 1|Strzałka w dół|
