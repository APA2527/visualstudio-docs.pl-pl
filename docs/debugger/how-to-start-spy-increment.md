# <a name="how-to-start-spy"></a>Porady: korzystanie z programu Spy++
Można uruchomić narzędzie Spy ++ z programu Visual Studio lub w wierszu polecenia.  
  
 Po uruchomieniu narzędzia Spy ++, jeśli zostanie wyświetlony komunikat, aby zadać uprawnień, aby wprowadzić zmiany w komputerze, kliknij przycisk **tak**.  
  
> [!NOTE]
>  Można uruchomić tylko jedno wystąpienie programu Spy ++. Jeśli użytkownik próbuje uruchomić inne wystąpienie, tylko powoduje aktualnie uruchomione wystąpienie uzyskać fokusu.  
  
### <a name="to-start-spy-from-visual-studio"></a>Aby uruchomić narzędzie Spy ++ z programu Visual Studio  
  
-   Na **narzędzia** menu, kliknij przycisk **Spy ++**.  
  
     Ponieważ Spy ++ działa niezależnie, po jej uruchomieniu, można zamknąć program Visual Studio.  
  
    > [!NOTE]
    >  Podczas rejestrowania komunikatów z programu Spy ++, może to spowodować wolniejsze systemu operacyjnego.  
  
### <a name="to-start-spy-at-a-command-prompt"></a>Aby uruchomić narzędzie Spy ++ w wierszu polecenia  
  
1.  W oknie wiersza polecenia przejdź do folderu, który zawiera spyxx.exe. Zazwyczaj jest ścieżka tego folderu. \\ *Folder instalacji programu visual Studio*\Common7\Tools\\.  
  
2.  Typ **spyxx.exe** , a następnie naciśnij klawisz ENTER.
