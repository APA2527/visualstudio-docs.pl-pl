> [!NOTE]
> Kodowany test interfejsu użytkownika dla zautomatyzowanych testów funkcjonalnych opartych na interfejsie użytkownika jest przestarzały. Visual Studio 2019 to Ostatnia wersja, w której będzie dostępny kodowany test interfejsu użytkownika. Zalecamy używanie [selen](https://docs.seleniumhq.org/) do testowania aplikacji sieci Web i [Appium z WinAppDriver](https://github.com/Microsoft/WinAppDriver) na potrzeby testowania aplikacji klasycznych i platformy UWP. Rozważmy platformę [Xamarin. UITest](/appcenter/test-cloud/uitest/) do testowania aplikacji dla systemów iOS i Android przy użyciu platformy testów NUnit.
