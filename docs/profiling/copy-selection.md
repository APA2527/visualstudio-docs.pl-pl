# <a name="copy-selection"></a>Kopiuj dokonany wybór
Aby skopiować cały stos wywołań kartę raportu, kliknij przycisk **kopiowania**. Następnie można wkleić stos wywołań w każdym programie, która obsługuje tę akcję.
