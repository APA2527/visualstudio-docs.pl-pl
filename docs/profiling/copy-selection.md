# <a name="copy-selection"></a>Kopiuj zaznaczenie
Aby skopiować cały stos wywołań z karty raport, kliknij przycisk **Kopiuj**. Następnie można wkleić stos wywołań w dowolnym programie, który obsługuje tę akcję.
