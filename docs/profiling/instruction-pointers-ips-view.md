# <a name="instruction-pointers-ips-view"></a>Widok wskaźników instrukcji (IP)
Widok wskaźników instrukcji temu możesz zobaczyć szczegółowe instrukcje, które są wykonywane podczas profilowania próbkowania.  
  
 Ten widok nie jest dostępna podczas profilowania instrumentacji.  
  
 Do pobierania próbek danych profilu, w widoku adresów IP identyfikuje instrukcje w funkcji, która powodowała bezpośrednio po próbki został zebrany.  
  
 W przypadku danych pamięci .NET widok linii identyfikuje instrukcje, które przydzielić pamięci.
