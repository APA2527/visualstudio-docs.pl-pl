# <a name="no-disk-activity-to-show-threads-view"></a>Brak aktywności dysku do wyświetlenia (widok wątków)
Nie można wyświetlić aktywności dysku dla bieżącego zakresu czasu w kanałach We/Wy dysku. Aktywność dysku może być wyświetlana po pomniejszanie lub przewijanie. Upewnij się również, że żadne kanały dyskowe nie są ukryte.
