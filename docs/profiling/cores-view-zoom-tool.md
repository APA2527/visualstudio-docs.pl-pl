# <a name="cores-view-zoom-tool"></a>Narzędzie Powiększenie widoku rdzeni
Kontrolka Powiększenie to suwak, który umożliwia powiększanie i pomniejszanie na osi czasu, dzięki czemu można skupić się na obszarach o szczególnym znaczeniu. Ten formant powiększa środek bieżącego widoku, więc dostosuj widok przed powiększeniem.

## <a name="other-ways-to-zoom"></a>Inne sposoby powiększania

- Powiększanie: przeciągnij, aby zaznaczyć obszar w widoku osi czasu. Po zwolnieniu przycisku myszy widok osi czasu powiększa wybrany zakres.

- Powiększanie i pomniejszanie: umieść wskaźnik myszy na osi czasu, a następnie naciśnij klawisz **Ctrl** podczas przesuwania kółka myszy.
