# <a name="tier-interaction-views"></a>Widok interakcji warstwowej

Profilowanie interakcji pomiędzy warstwami zawiera dodatkowe informacje na temat czasu wykonania funkcji w aplikacji wielowarstwowych, które komunikują się z bazami danych za pośrednictwem [!INCLUDE[vstecado](../data-tools/includes/vstecado_md.md)]. Dane są zbierane tylko w przypadku wywołania funkcji synchronicznej. Dane interakcji między warstwami można uwzględnić w wszystkich metod profilowania.

Widoki interakcji między warstwami są dostępne tylko w programie Visual Studio Enterprise.
