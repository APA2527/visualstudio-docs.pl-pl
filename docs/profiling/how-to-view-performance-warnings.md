# <a name="how-to-view-performance-warnings"></a>Porady: wyświetlanie ostrzeżeń dotyczących wydajności
Ostrzeżenia wydajności z [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] narzędzi profilowania narzędzi profilowania wskazują na problemy w profilowanych aplikacji, która może spowolnić działanie programu. Ostrzeżenia może również oznaczać, że można zmienić metody kolekcji do zbierania danych bardziej użyteczne. Ostrzeżenia wydajności są generowane automatycznie podczas sesji profilowania i są wyświetlane w **listy błędów** okna po otwarciu pliku danych profilowania w [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] IDE.  
  
 Jeśli **listy błędów** podczas otwierania pliku danych profilowania nie jest wyświetlane okno, kliknij przycisk **widoku** menu, a następnie kliknij przycisk **listy błędów**.
