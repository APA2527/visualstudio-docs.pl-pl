# <a name="functions-view"></a>Widok funkcji
Widok Funkcje wyświetla listę funkcji, które są wywoływane podczas profilowania.

 Domyślnie funkcje są wyświetlane według nazwy. Aby wyświetlić funkcje według modułu, kliknij prawym przyciskiem myszy okno Widok funkcji, a następnie wybierz polecenie **Pokaż w widoku modułu**. Spowoduje to wyświetlenie widoku Moduł.

 Aby dodać lub usunąć kolumny, kliknij prawym przyciskiem myszy okno raportu, a następnie wybierz polecenie **Dodaj/Usuń kolumny**. Ponadto można sortować dane, klikając nazwę kolumny. Aby uzyskać więcej informacji, zobacz [Jak: Dostosowywanie kolumn widoku raportu](../profiling/how-to-customize-report-view-columns.md).

 Kolumny, które są dostępne w widoku funkcje zależą od metody profilowania (próbkowania lub instrumentacji), który został użyty do zbierania danych i czy dane pamięci .NET zostały zebrane w przebiegu profilowania.
