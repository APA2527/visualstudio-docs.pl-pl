# <a name="getting-started-with-performance-tools"></a>Wprowadzenie do korzystania z narzędzia wydajności
Ta sekcja zawiera omówienie pojęć i zadań, które będą stosowane podczas zbierania, wyświetlania i analizowania wydajności kodu za pomocą narzędzi wydajności Visual Studio.  
  
 Te narzędzia umożliwiają kilka sposobów zbieranie i analizowanie danych dotyczących wydajności. Ale w wielu przypadkach jest najlepszy sposób, aby rozpocząć, aby użyć domyślnych ustawień kreatora wydajności, które umożliwia zbieranie statystyk aplikacji, które wskazują na problemy z wydajnością w kodzie.  
  
-   *Ostrzeżenia wydajności* powiadamia użytkownika o typowych problemów kodowania. Ostrzeżenia są wyświetlane w oknie programu Visual Studio błędy. Można nawigować do kodu źródłowego z ostrzeżenia i do szczegółowych tematów pomocy, które ułatwiają pisanie kodu większą wydajność.  
  
-   Raporty wydajności zawierają widoki na różnych poziomach struktury aplikacji ze źródła wierszy kodu do procesów i danych wykonywania programu z funkcji wywołującego i o nazwie określonych funkcji, aby drzewie wywołań w całej aplikacji.
