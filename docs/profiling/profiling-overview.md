# <a name="overview-of-profiling-in-visual-studio"></a>Omówienie profilowania w programie Visual Studio

Narzędzia profilowania i informacji diagnostycznych ułatwiają diagnozowanie pamięć i użycie procesora CPU i inne problemy z poziomu aplikacji. Z tych narzędzi gromadzone dane (takie jak wartości zmiennych, wywołania funkcji i zdarzeń) w danym okresie czasu, np. podczas uruchamiania aplikacji. Możesz wyświetlić stan aplikacji w różnych punktach w czasie wykonywania kodu lub analiza wydajności aplikacji w których post.
