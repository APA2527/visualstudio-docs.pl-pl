# <a name="how-to-pause-and-resume-performance-data-collection"></a>Instrukcje: wstrzymywanie i wznawianie zbierania danych o wydajności
W oknie strony sesji profilowania można interaktywnie kontrolować zbieranie danych profilowania.

 Kontrolowanie zbierania danych umożliwia zmniejszenie rozmiaru pliku danych profilu i zbieranie danych tylko dla tych operacji, które Cię interesują. W sesji wydajności można wstrzymać i wznowić profilowanie wiele razy.

 ![Strona sesji profilowania](../profiling/media/prof_profilingsessionpage.png "PROF_ProfilingSessionPage")

> [!NOTE]
> Można również rozpocząć sesję wydajności z profilowania wstrzymane, a następnie wznowić profilowanie w późniejszym momencie wykonywania programu. Aby rozpocząć sesję wydajności z wstrzymanym profilowaniem, wybierz polecenie **Rozpocznij analizę wydajności z poleceniem Profilowanie wstrzymane** w menu **Debugowania.**

### <a name="to-pause--resume-or-stop-profiling"></a>Aby wstrzymać, wznowić lub zatrzymać profilowanie

- Na stronie sesji profilowania:

  - Wybierz **pozycję Wstrzymaj zbieranie danych,** aby zawiesić zbieranie danych.

  - Wybierz **polecenie Wznów zbieranie** danych, aby ponownie uruchomić zbieranie danych po wstrzymaniu.

  - Wybierz **pozycję Zatrzymaj profilowanie,** aby zakończyć sesję profilowania i wygenerować raporty.
