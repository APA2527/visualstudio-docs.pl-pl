# <a name="command-line-profiling-tools-reference"></a>Narzędzia profilowania wiersza polecenia — dokumentacja
Narzędzia wydajności wiersza polecenia umożliwiają programistom profilowanie aplikacji w wierszu polecenia. Następujące narzędzia do profilowania wiersza polecenia umożliwiają profilowanie kodu w wierszu polecenia.

## <a name="in-this-section"></a>W tej sekcji
- [VSInstr](../profiling/vsinstr.md)

 Wyświetla listę opcji instrumentacji VSInstr.exe.

- [Ostrzeżenia VSInstr](../profiling/vsinstr-warnings.md)

 Wyświetla listę ostrzeżeń wydanych przez narzędzie VSInstr.

- [VSPerfCLREnv](../profiling/vsperfclrenv.md)

 Wyświetla listę opcji programu VSPerfCLREnv.exe w celu skonfigurowania środowiska do profilowania kodu zarządzanego.

- [VSPerfCmd](../profiling/vsperfcmd.md)

 Wyświetla listę opcji programu VSPerfCmd.exe do zbierania danych o wydajności.

- [VSPerfMon](../profiling/vsperfmon.md)

 Wyświetla listę opcji vsperfMon.exe do zbierania danych.

- [VSPerfReport](../profiling/vsperfreport.md)

 Wyświetla listę opcji wiersza polecenia VSPerfReport.exe służących do tworzenia raportów.
