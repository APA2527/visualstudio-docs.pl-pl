# <a name="how-to-filter-report-views-from-the-summary-timeline"></a>Jak: Filtrowanie widoków raportu z osi czasu podsumowania
Dane wyświetlane w raportach wydajności można filtrować, określając segment czasu na wykresie osi czasu widoku podsumowania. Podczas filtrowania raportów plik danych profilowania jest ponownie analizowany w celu uwzględnienia tylko określonego segmentu czasu.

### <a name="to-filter-a-profiling-report-from-the-summary-timeline-graph"></a>Aby filtrować raport profilowania z wykresu osi czasu podsumowania

1. Na wykresie osi czasu widoku podsumowania kliknij punkt na osi czasu, a następnie przeciągnij myszą do innego punktu na osi czasu.

     Określony region jest zacieniowany.

2. Kliknij **pozycję Filtruj według zaznaczenia**.
