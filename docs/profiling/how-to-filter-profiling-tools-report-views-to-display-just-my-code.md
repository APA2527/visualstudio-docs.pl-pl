# <a name="how-to-filter-profiling-tools-report-views-to-display-just-my-code"></a>Jak: Filtrowanie widoków z raportów narzędzi profilowania w celu wyświetlenia tylko mojego kodu

Visual Studio zapewnia filtrowanie raportów wydajności, dzięki czemu są wyświetlane tylko kod, który zostały zaimplementowane. Ta funkcja o nazwie Just My Code jest domyślnie włączona. Jest on skonfigurowany za pomocą opcji **Tylko mój kod** w oknie dialogowym **Opcje.**

## <a name="to-enable-just-my-code"></a>Aby włączyć tylko mój kod

1. W menu **Tools** (Narzędzia) kliknij pozycję **Options** (Opcje).

2. W oknie dialogowym **Opcje** rozwiń węzeł **Narzędzia wydajności,** a następnie wybierz węzeł **Ogólne.**

3. W **obszarze Ustawienia ogólne**zaznacz pole wyboru Włącz tylko mój **kod.**

## <a name="to-disable-just-my-code"></a>Aby wyłączyć tylko mój kod

1. W menu **Tools** (Narzędzia) kliknij pozycję **Options** (Opcje).

2. W oknie dialogowym **Opcje** rozwiń węzeł **Narzędzia wydajności,** a następnie wybierz węzeł **Ogólne.**

3. W **obszarze Ustawienia ogólne**wyczyść pole wyboru Włącz tylko mój **kod.**
