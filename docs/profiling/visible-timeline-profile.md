# <a name="visible-timeline-profile"></a>Widoczny profil osi czasu
Widoczny profil osi czasu dla widoku blokowania wątków zawiera informacje statystyczne i łącza do raportów. Podczas powiększania, powiększania, przewijania w poziomie, ukrywania kanałów lub pokazywalek kanałów numery w aktywnej legendzie zmieniają się, aby odzwierciedlić to, co jest obecnie widoczne. Aby wyświetlić raport o elemencie w legendzie, kliknij go.
