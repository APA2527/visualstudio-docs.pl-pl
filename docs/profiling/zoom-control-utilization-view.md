# <a name="zoom-control-utilization-view"></a>Kontrola powiększenia (widok wykorzystania)
Kontrolka powiększenia ułatwia powiększenie wykresu wykorzystania procesora CPU, aby skupić się na obszarach o szczególnym znaczeniu. Ten formant powiększa środek widoku. W związku z tym należy wyśrodkować obszar zainteresowania przed powiększeniem.

 Można przeciągnąć wskaźnik myszy na wykresie wykorzystania procesora CPU lub wykres aktywności GPU, aby utworzyć wyróżniony obszar. Po zwolnieniu przycisku myszy widok przybliża wybrany zakres.
