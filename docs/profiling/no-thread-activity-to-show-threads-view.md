# <a name="no-thread-activity-to-show-threads-view"></a>Żadnej aktywności wątku, aby wyświetlić (Widok wątków)
Ten obszar zawiera dane dotyczące odkrywane wątków w aktualnie widocznym zakresie czasu.  
  
 Jeśli informacje nie są widoczne, sprawdź następujące ustawienia:  
  
-   Poziom powiększenia jest wysoka? Spróbuj powiększanie out lub przewijania, aby wyświetlić więcej aktywności wątku w zakresie.  
  
-   Zbyt wiele wątków są ukryte? Jeśli tak, spróbuj przedstawiający wszystkie wątki  
  
-   Jeśli **tylko mój kod** jest wybrana, możesz jedynie wyświetlić dane o kodzie. Wyczyść ustawienie, aby ustalić, czy występuje aktywność wątku systemu.  
  
-   Upewnij się, że redukcji szumu ma ustawioną wartość progu dolnego.
