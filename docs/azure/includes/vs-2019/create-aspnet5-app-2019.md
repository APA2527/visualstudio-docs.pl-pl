1. W oknie uruchamiania programu Visual Studio wybierz **Utwórz nowy projekt**.
1. Wybierz **aplikacji sieci Web programu ASP.NET Core**, a następnie wybierz pozycję **dalej**.
1. Wprowadź nazwę dla nowej aplikacji (lub użyj domyślnej nazwy), a następnie wybierz pozycję **Utwórz**.
1. Wybierz **aplikacji sieci Web**.
1. Wybierz, czy obsługa protokołu SSL przez zaznaczenie lub wyczyszczenie **Konfigurowanie protokołu HTTPS** pole wyboru.
1. Wybierz **włączyć obsługę platformy Docker** pole wyboru.
1. Wybierz typ kontenera (Windows lub Linux), a następnie wybrać **Utwórz**.
