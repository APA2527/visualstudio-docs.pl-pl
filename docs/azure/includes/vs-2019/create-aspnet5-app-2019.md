1. W oknie startowym programu Visual Studio wybierz pozycję **Utwórz nowy projekt**.
1. Wybierz **ASP.NET podstawową aplikację sieci Web,** a następnie wybierz pozycję **Dalej**.
1. Wprowadź nazwę nowej aplikacji (lub użyj nazwy domyślnej), a następnie wybierz pozycję **Utwórz**.
1. Wybierz **aplikację sieci Web**.
1. Wybierz, czy chcesz obsługiwać protokół SSL, zaznaczając lub czyszcząc pole wyboru **Konfiguruj dla protokołu HTTPS.**
1. Zaznacz pole wyboru **Włącz obsługę platformy Docker.**
1. Wybierz odpowiedni typ kontenera (Windows lub Linux), a następnie wybierz pozycję **Utwórz**.
