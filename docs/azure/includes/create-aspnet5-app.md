1. W menu Programu Visual Studio wybierz polecenie **Plik** > **nowego** > **projektu**.
2. W sekcji **Szablony** w oknie dialogowym **Nowy projekt** wybierz pozycję **Visual C#** > **Web**.
3. Wybierz **ASP.NET podstawową aplikację sieci Web**.
4. Wprowadź nazwę nowej aplikacji (lub użyj nazwy domyślnej), a następnie wybierz **przycisk OK**.
5. Wybierz **aplikację sieci Web**.
6. Zaznacz pole wyboru **Włącz obsługę platformy Docker.**
7. Wybierz odpowiedni typ kontenera (Windows lub Linux), a następnie wybierz przycisk **OK**.
