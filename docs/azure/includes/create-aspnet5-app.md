1. W menu programu Visual Studio wybierz **pliku** > **New** > **projektu**.
2. W **szablony** części **nowy projekt** okno dialogowe, wybierz opcję **Visual C#**   >  **Web**.
3. Wybierz **aplikacji sieci Web platformy ASP.NET Core**.
4. Wprowadź nazwę dla nowej aplikacji (lub użyj domyślnej nazwy), a następnie wybierz pozycję **OK**.
5. Wybierz **aplikacji sieci Web**.
6. Wybierz **włączyć obsługę platformy Docker** pole wyboru.
7. Wybierz typ kontenera (Windows lub Linux), a następnie wybrać **OK**.
