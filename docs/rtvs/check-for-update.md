# <a name="rtvs-current-version"></a>RTVS bieżąca wersja

Możesz sprawdzić wersję, przechodząc do **pomocy** > **w** programie Visual Studio i przewijając w dół do wpisu **Narzędzia języka R dla programu Visual Studio:**

![Badanie wersji RTVS w programie Visual Studio (VS2017)](media/current-version.png)

Aby zaktualizować wersję rtvs w programie Visual Studio 2017, przejdź do instalatora programu Visual Studio i wybierz przycisk **Aktualizuj.**

W programie Visual Studio 2015 najnowszy instalator jest dostępny od [aka.ms/rtvs-current](https://rtvs.blob.core.windows.net/download/RTVS_2017-12-18.1.exe) (do pobrania bezpośredniego).

Zobacz [Wymagania dotyczące instalacji](installing-r-tools-for-visual-studio.md) dla programu Visual Studio i systemu Windows.

Aby uzyskać aktualizacje na microsoft R Open i Microsoft Machine Learning Server (dawniej Microsoft R Server), zobacz: [Produkty Microsoft R.](https://azure.microsoft.com/?ocid=cloudplat_hp)
