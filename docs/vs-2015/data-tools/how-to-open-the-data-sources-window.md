# <a name="how-to-open-the-data-sources-window"></a>Porady: otwieranie okna źródeł danych
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

**Źródeł danych** okna wyświetla źródła danych w projekcie. Możesz użyć **źródeł danych** okna, aby utworzyć interfejs użytkownika formantów powiązanych z danymi przez przeciąganie elementów z okna na formularze w projekcie. Zobacz [formanty powiązania formularzy Windows do danych w programie Visual Studio](../data-tools/bind-windows-forms-controls-to-data-in-visual-studio.md).  
  
 Możesz wypełnić **źródeł danych** okna, uruchamiając **konfiguracji źródła danych** kreatora. Aby uzyskać więcej informacji, zobacz [Kreatora konfiguracji źródła danych](http://msdn.microsoft.com/library/c4df7de5-5da0-4064-940c-761dd6d9e28f).  
  
### <a name="to-open-the-data-sources-window"></a>Aby otworzyć okno Źródła danych  
  
-   Na pasku menu wybierz **widoku**, **Windows inne**, **źródeł danych** (lub klawisze Shift + Alt + D).  
  
    > [!IMPORTANT]
    >  Możesz dodać źródło danych do projektu tylko wtedy, gdy obsługuje tworzenie źródeł danych i pracę z nimi. Na przykład nie można otworzyć **źródeł danych** okna w projekcie dla aplikacji Windows Store.
