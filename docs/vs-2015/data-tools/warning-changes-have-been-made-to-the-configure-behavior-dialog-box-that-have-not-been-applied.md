# <a name="warning-changes-have-been-made-to-the-configure-behavior-dialog-box-that-have-not-been-applied"></a>Ostrzeżenie. Wprowadzono zmiany w oknie dialogowym Konfigurowanie zachowania, które nie zostały zastosowane
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ostrzeżenie. Wprowadzono zmiany w oknie dialogowym Konfigurowanie zachowania, które nie zostały zastosowane. Czy chcesz zastosować zmiany?

 Okno dialogowe **Konfigurowanie zachowania** umożliwia skonfigurowanie zachowania INSERT, Update i DELETE dla wszystkich dostępnych klas. Ten komunikat jest wyświetlany po wybraniu nowej kombinacji **klas** i **zachowań** , a Poprzednia zmiana nie została jeszcze zastosowana.

### <a name="to-apply-the-change-and-continue"></a>Aby zastosować zmianę i kontynuować

- Kliknij przycisk **tak** , aby zastosować zmianę w wybranej **klasie** i **zachowaniu**.

### <a name="to-cancel-the-previous-change-and-continue"></a>Aby anulować poprzednią zmianę i kontynuować

- Kliknij przycisk **nie** , aby anulować zmianę wybranej **klasy** i **zachowania**.
