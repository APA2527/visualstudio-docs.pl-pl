# <a name="one-or-more-selected-database-objects-return-a-schema-that-does-not-match-the-schema-of-the-target-class"></a>Co najmniej jeden wybrany obiekt bazy danych zwraca schemat, który nie jest zgodny ze schematem klasy docelowej
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Co najmniej jeden wybrany obiekt bazy danych zwraca schemat, który nie jest zgodny ze schematem klasy docelowej. Nic nie zostało dodane do projektanta.

 Podczas przeciągania obiektów bazy danych do istniejących klas jednostek dane zwrócone przez obiekt bazy danych muszą być zgodne ze schematem klasy jednostki docelowej. Upewnij się, że wybrano poprawny obiekt bazy danych i że jest poprawna Klasa jednostki.

### <a name="to-correct-this-error"></a>Aby poprawić ten błąd

1. Kliknij przycisk **OK** , aby zamknąć okno dialogowe.

2. Wybierz obiekt bazy danych, który zwraca dane zgodne ze schematem klasy docelowej (Klasa, do której zostanie porzucony obiekt bazy danych w Projektancie O/R).
