# <a name="data-user-interface-elements"></a>Elementy interfejsu użytkownika danych
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ta sekcja zawiera informacje dotyczące wszystkich okien dialogowych i kreatorów, których używasz, aby podczas projektowania dostęp do danych w Twojej [!INCLUDE[vbprvb](../includes/vbprvb-md.md)] lub aplikacji programu Visual C#.  
  
 [!INCLUDE[note_settings_general](../includes/note-settings-general-md.md)]  
  
## <a name="in-this-section"></a>W tej sekcji  
 [Tagi inteligentne danych](http://msdn.microsoft.com/en-us/1e0a848f-c57b-47ab-b884-eaaa40726f43)  
 Zawiera informacje dotyczące niektórych poleceń tagów inteligentnych, które są dostępne dla danych.  
  
 [Dodaj okno dialogowe Zestaw danych](http://msdn.microsoft.com/en-us/0e03c0ff-212b-4bfa-ac51-3c2adb71ead0)  
 Umieszcza istniejące wpisany zestaw danych lub nowych nietypizowany zestaw danych (wystąpienie **System.Data.Dataset** klasy) na formularzu lub składnika.  
  
 [Zaawansowane SQL generowania okno dialogowe Opcje](http://msdn.microsoft.com/en-us/41420450-1ff4-4a1a-b85b-6f6901538fef)  
 Umożliwia kontrolowanie sposobu instrukcji SQL lub procedury składowane są tworzone dla adaptera danych, określając opcje aktualizacji, optymistycznej współbieżności i odświeżania zestawu danych.  
  
 [Wybierz okno dialogowe wystąpienia źródła danych](http://msdn.microsoft.com/en-us/51c47f06-fdc5-453e-9178-0a5a2c5c9f34)  
 Można wybrać żądany zestaw danych z listy zestawów danych w projekcie.  
  
 [Wybierz źródło danych do scalenia okna dialogowego](http://msdn.microsoft.com/en-us/accafff7-f6bd-481c-a121-fe8a76cd681d)  
 Służy do wybierania źródła danych do scalenia kiedy są dostępne różne źródła danych.  
  
 [Wybierz okno dialogowe klucza](http://msdn.microsoft.com/en-us/4ddbfbb7-a80a-412a-b80d-291d86376ca3)  
 Umożliwia wybierz klucz, gdy kolumna uczestniczy w wiele kluczowych ograniczeń.  
  
 [Edytory kolekcji](http://msdn.microsoft.com/library/030095bd-fb9a-4b21-b628-fc1cc5985bb7)  
 Zawiera łącza do tematów dotyczących edytory innej kolekcji, które pozwalają tworzyć i edytować poszczególne elementy członkowskie kolekcji.  
  
 [Brakujące połączenia](http://msdn.microsoft.com/en-us/bb9b2e12-7f76-4ee5-acbb-5d20116ee044)  
 Informuje użytkownika, gdy zestaw danych zawiera odwołanie do parametrów połączenia, który nie znajduje się w aplikacji ustawienia.  
  
 [Okno dialogowe błąd konfiguracji adaptera danych](http://msdn.microsoft.com/en-us/9ce65cd2-0c7d-4f51-8685-d68be5f3009b)  
 Wyświetla jeden lub więcej błędów, które wystąpiły w trakcie programu Visual Studio próbował utworzyć wystąpienie w obrębie adaptera danych i ustaw jego właściwości.  
  
 [Kreatora konfiguracji adaptera danych](http://msdn.microsoft.com/en-us/efff90cb-0e4c-4eb3-87dc-65dd9d418809)  
 Konfiguruje poleceń SQL lub procedur składowanych, używanych przez adapter danych do odczytu danych do zestawu danych z bazy danych i zapisu go z powrotem. Temacie opisano sposób uruchamiania kreatora i co należy zrobić po zakończeniu pracy kreatora.  
  
 [Okno dialogowe podglądu karty danych](http://msdn.microsoft.com/en-us/1f614cd3-4530-457e-84af-00ccbaea08cc)  
 Umożliwia zobaczenie, jak zostaną wypełnione danych do zestawu danych w obrębie adaptera danych, która jest przydatna przy testowaniu, że karta zwraca dane oczekujesz, mapowania tabel działać prawidłowo, a wpływ różne wartości parametrów.  
  
 [Okno dialogowe logowania źródła danych](http://msdn.microsoft.com/en-us/6f2d9a57-53c3-4841-bd37-a3643eb68d2e)  
 Umożliwia zażądanie dostępu do źródła danych (zwykle bazy danych), dla którego użytkownik nie już zostały uwierzytelnione.  
  
 [Karta zestawu danych, przybornik](http://msdn.microsoft.com/en-us/fa5f2d6f-924d-4262-ba1b-e9e7f90e7764)  
 Wyświetla obiekty, które można dodać do typizowany zestaw danych.  
  
 [Czy chcesz zawierały hasło w parametrach połączenia, okno dialogowe](http://msdn.microsoft.com/en-us/193696a7-5213-4396-8328-05ac2df6ee94)  
 Umożliwia kontrolowanie, czy hasło jest wbudowane w parametrach połączenia.  
  
 [Edytuj okno dialogowe klucza](http://msdn.microsoft.com/en-us/f5c80e39-3a42-4284-b222-6ca009fd9675)  
 Umożliwia definiowanie i edytowanie kluczy.  
  
 [Okno dialogowe ograniczenie klucza obcego](http://msdn.microsoft.com/en-us/45d15629-1f4d-40a7-8708-c9ddfebedc1e)  
 Pozwala na umieszczenie ograniczenie klucza obcego w co najmniej jedną kolumnę w tabeli zestawu danych, który jest powiązany z inną tabelą.  
  
 [Generowanie zestawu danych, okno dialogowe](http://msdn.microsoft.com/en-us/c0efdbaf-13b1-4ee8-ade6-f8a784126cdc)  
 Umożliwia generowanie nowego typizowany zestaw danych na podstawie informacji dostarczonych przez jeden lub więcej adapterów danych i dodać tabele do istniejącego zestawu danych.  
  
 [Wiele źródeł powiązania dialogowe](http://msdn.microsoft.com/en-us/db76f70c-4fb5-479d-9b64-a67158d48f97)  
 Umożliwia wybranie <xref:System.Windows.Forms.BindingSource> do użycia podczas więcej niż jeden <xref:System.Windows.Forms.BindingSource> jest dostępna.  
  
 [Okno dialogowe danych (wersja zapoznawcza)](http://msdn.microsoft.com/en-us/aa4f0d04-2695-4bb8-946d-54a97ae7287f)  
 Umożliwia przeglądanie danych zwracanych przez zapytania w projekcie.  
  
 [Okno dialogowe skryptu SQL (wersja zapoznawcza)](http://msdn.microsoft.com/en-us/e9571e8b-821c-492d-9bc8-b44eba898bdd)  
 Pojawia się jako część [Kreatora konfiguracji adaptera danych](http://msdn.microsoft.com/en-us/efff90cb-0e4c-4eb3-87dc-65dd9d418809) umożliwia wyświetlony skrypt SQL, w kreatorze zostaną użyte do utworzenia procedur składowanych do odczytu i zapisu danych.  
  
 [Okno dialogowe relacji](http://msdn.microsoft.com/en-us/ab8f4b0e-af4c-4725-a550-e2b2ebe43a02)  
 Służy do tworzenia relacji ( **DataRelation** obiektu), zachowuje informacje o rekordach nadrzędny podrzędny w dwóch tabelach w zestawie danych.  
  
 [Kompilator kryteriów wyszukiwania — okno dialogowe](http://msdn.microsoft.com/library/0b306b92-f35e-45ef-a4be-3f653cd00c3d)  
 Umożliwia tworzenie zapytania parametrycznego na formularzu Windows powiązane z danymi oraz automatycznie dodać formanty wymagane do uruchomienia zapytania.  
  
 [Okno dialogowe mapowania tabeli](http://msdn.microsoft.com/en-us/fb4cec1e-f3c8-4773-b409-c2de15293fea)  
 Pozwala określić, które kolumny w tabeli bazy danych lub innego źródła danych są równoważne do kolumn w tabeli zestawu danych.  
  
 [Okno dialogowe ograniczenie unikatowe](http://msdn.microsoft.com/en-us/e71a60d7-fae2-4bd0-a1e8-43aae351707d)  
 Pozwala na umieszczenie unikatowego ograniczenia na co najmniej jedną kolumnę w tabeli w nietypizowany zestaw danych.
