# <a name="debug-xslt"></a>Debugowanie XSLT
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aby debugować XSLT, można użyć debugera programu Visual Studio. Debuger obsługuje ustawianie punktów przerwania, wyświetlanie stanów wykonywania XSLT i tak dalej. Debugowanie XSLT jest dostępne w wersji Visual Studio Professional.
