# <a name="what39s-new-in-the-xslt-debugger"></a>Co&#39;nowego w debugerze XSLT
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Następujące funkcje są nowością w programie Visual Studio 2010.  
  
## <a name="xslt-profiler"></a>Profiler XSLT  
 Profiler XSLT jest narzędziem, które umożliwia deweloperom mierzyć, oceny i określania elementów docelowych problemy związane z wydajnością w kodzie XSLT, tworząc szczegółowe raporty dotyczące wydajności XSLT. Aby uzyskać więcej informacji, zobacz [Profiler XSLT](../xml-tools/xslt-profiler.md).
