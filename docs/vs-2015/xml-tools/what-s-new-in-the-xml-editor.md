# <a name="what39s-new-in-the-xml-editor"></a>Co&#39;s Nowość w edytorze XML
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

  
Następujące funkcje są nowością w programie Visual Studio 2010.  
  
## <a name="xslt-intellisense"></a>XSLT IntelliSense  
 Funkcja XSLT IntelliSense umożliwia nazwy zestawu atrybutów automatycznego uzupełniania; tryby szablonu i nazwy. Parametr nazwy i dla określonego trybu lub określonym szablonie o podanej nazwie. Aby uzyskać więcej informacji, zobacz [funkcje IntelliSense w edytorze XML](../xml-tools/xml-editor-intellisense-features.md).  
  
## <a name="xslt-profiler"></a>Profiler XSLT  
 Profiler XSLT tworzy szczegółowe wydajności XSLT, raporty, które ułatwiają mierzyć, szacować i wybierać problemy związane z wydajnością w kodzie XSLT. Profiler XSLT zawiera także przydatne wskazówki dotyczące optymalizacji o arkusza stylów XSL i XSLT. Aby uzyskać więcej informacji, zobacz [przewodnik: Profiler XSLT](../xml-tools/walkthrough-xslt-profiler.md).  
  
## <a name="xslt-hierarchy"></a>Hierarchii XSLT  
 Narzędzie hierarchii XSLT, pozwala dodać punkty przerwania w arkusze stylów dołączone i/lub wbudowany szablon reguły. Aby uzyskać więcej informacji, zobacz [wskazówki: Korzystanie z hierarchii XSLT](../xml-tools/walkthrough-using-xslt-hierarchy.md).
