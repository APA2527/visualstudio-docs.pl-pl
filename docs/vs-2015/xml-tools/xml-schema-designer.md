# <a name="xml-schema-designer"></a>Projektant schematu XML
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Projektant schematu XML (Projektant XSD) jest graficznym narzędziem, które pozwala na wizualizację zestawu schematów na różnych poziomach abstrakcji. Poniżej przedstawiono główne składniki projektanta XSD:

- [Eksplorator schematu XML](../xml-tools/xml-schema-explorer.md) umożliwia przeglądanie i nawigowanie po drzewie schematu XML oraz wykonywanie operacji wyszukiwania.

- [Widok startowy](../xml-tools/start-view.md) jest punktem uruchamiania projektanta XSD; w widoku Start można przechodzić do innych widoków projektanta XSD i przeglądać szczegóły zestawu schematów.

- [Widok grafu](../xml-tools/graph-view.md) umożliwia wyświetlenie przeglądu zestawu schematów i relacji między węzłami schematu.

- [Widok model zawartości](../xml-tools/content-model-view.md) zawiera graficzną reprezentację szczegółów lokalnych i globalnych węzłów schematu, w tym prostych i złożonych typów, elementów, grup, atrybutów i grup atrybutów.
