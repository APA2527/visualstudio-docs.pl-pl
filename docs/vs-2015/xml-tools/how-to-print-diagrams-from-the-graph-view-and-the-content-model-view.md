# <a name="how-to-print-diagrams-from-the-graph-view-and-the-content-model-view"></a>Instrukcje: Drukowanie diagramów z widoku wykresu i widoku modelu zawartości
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

W tym temacie opisano sposób drukowania diagramu z widoku grafu lub widoku modelu zawartości.

### <a name="to-print-diagrams-from-the-xml-schema-designer"></a>Aby wydrukować diagramy z projektanta schematu XML

1. Otwórz plik XSD w programie Visual Studio i dodaj węzły do [obszaru roboczego projektanta schematu XML](../xml-tools/xml-schema-designer-workspace.md).

2. Wyeksportuj diagram do pliku XPS przy użyciu **diagramu eksportu jako obrazu...** element menu kontekstowego na powierzchni projektowej widoku wykresu lub widoku modelu zawartości.

     Podczas eksportowania diagramu z widoku wykresu cała powierzchnia projektu zostanie wyeksportowana do pliku XPS. Po wyeksportowaniu diagramu z widoku modelu zawartości, gdy na powierzchni projektowej widoku modelu zawartości zostanie wyświetlony więcej niż jeden węzeł, tylko pierwszy węzeł zostanie wyeksportowany do pliku XPS.

3. Wydrukuj zapisany obraz w pliku XPS przy użyciu przeglądarki plików XPS.
