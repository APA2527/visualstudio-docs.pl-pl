# <a name="create-and-run-unit-tests-for-a-store-app-in-visual-studio"></a>Tworzenie i uruchamianie testów jednostkowych dla aplikacji ze sklepu w programie Visual Studio
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Narzędzia testów jednostkowych w programie Visual Studio 2012 Express for Windows 8 zostały zaprojektowane z myślą o wspieraniu deweloperów i zespołów, które obejmują testy jednostkowe w ramach praktyk rozwojowych oprogramowania. Testy jednostkowe umożliwiają deweloperom i testerom wydajny, obsługiwany sposób wyszukiwania błędów logicznych w metodach klas w wizualizacji C#, Visual Basic i Visual. C++

> [!NOTE]
> W tematach w tej sekcji opisano funkcje programu Visual Studio 2012 Express for Windows 8. Visual Studio Ultimate, VS Premium i VS Professional oferują dodatkowe funkcje do testowania jednostkowego.
>
> - W programie VS Ultimate, VS Premium i VS Professional można użyć dowolnej platformy testów jednostkowych innej firmy lub Open Source, która utworzyła adapter dodatku dla programu Microsoft Test Explorer. Możesz również analizować i wyświetlać informacje o pokryciu kodu dla testów za pomocą programu VS Ultimate i VS Premium.
>   - W programie VS Ultimate można uruchomić testy po każdej kompilacji. Można również użyć sztucznych produktów firmy Microsoft, struktury izolacji dla kodu zarządzanego, aby skoncentrować testy na własnym kodzie poprzez podstawianie kodu testowego dla systemu i funkcji innych firm.
>   - **Język JavaScript**. Testowanie jednostkowe języka JavaScript nie jest dostępne w środowisku IDE Visual Studio Express dla systemu Windows 8. W Visual Studio Professional, Premium i Ultimate struktury testów jednostkowych JavaScript są dostępne jako rozszerzenia w galerii programu Visual Studio. Możesz również testować kod JavaScript przy użyciu jednej z autonomicznych struktur testów jednostkowych języka JavaScript innych firm lub typu open source.
>
>   Aby uzyskać więcej informacji, zobacz [testowanie jednostkowe kodu](../test/unit-test-your-code.md) w bibliotece MSDN.
