# <a name="tier-interaction-views"></a>Widok interakcji warstwowej
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Profilowanie interakcji między warstwami zawiera dodatkowe informacje o czasach wykonywania funkcji w aplikacjach wielowarstwowych, które komunikują się z bazami danych za pomocą programu [!INCLUDE[vstecado](../includes/vstecado-md.md)] . Dane są zbierane tylko dla wywołań funkcji synchronicznych. Możesz uwzględnić dane interakcji warstwy we wszystkich metodach profilowania.
