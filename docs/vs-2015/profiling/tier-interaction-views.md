# <a name="tier-interaction-views"></a>Widok interakcji warstwowej
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Najnowszą wersję tego tematu znajduje się w temacie [widoki interakcji między warstwami](https://docs.microsoft.com/visualstudio/profiling/tier-interaction-views).  
  
Profilowanie interakcji pomiędzy warstwami zawiera dodatkowe informacje na temat czasu wykonania funkcji w aplikacji wielowarstwowych, które komunikują się z bazami danych za pośrednictwem [!INCLUDE[vstecado](../includes/vstecado-md.md)]. Dane są zbierane tylko w przypadku wywołania funkcji synchronicznej. Dane interakcji między warstwami można uwzględnić w wszystkich metod profilowania.
