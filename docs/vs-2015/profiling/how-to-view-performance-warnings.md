# <a name="how-to-view-performance-warnings"></a>Porady: wyświetlanie ostrzeżeń dotyczących wydajności
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ostrzeżenia dotyczące wydajności [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] Profiling Tools Profiling Tools wskazują na problemy w profilowanej aplikacji, która może spowolnić działanie programu. Ostrzeżenia mogą również wskazywać, że można zmienić metody kolekcji, aby zebrać więcej przydatnych danych. Ostrzeżenia wydajności są automatycznie generowane w sesji profilowania i są wyświetlane w **lista błędów** okna po otwarciu pliku danych profilowania w [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] IDE.  
  
 Jeśli **lista błędów** okna nie pojawia się po otwarciu pliku danych profilowania, kliknij przycisk **widoku** menu, a następnie kliknij przycisk **lista błędów**.
