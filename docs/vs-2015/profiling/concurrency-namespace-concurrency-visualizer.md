# <a name="concurrency-namespace-concurrency-visualizer"></a>Przestrzeń nazwy współbieżności (Concurrency Visualizer)
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

`Concurrency`Przestrzeń nazw zawiera klasy i funkcje, które zapewniają dostęp do środowisko uruchomieniowe współbieżności, współbieżne środowisko programistyczne dla języka C++. Aby uzyskać więcej informacji, zobacz [środowisko uruchomieniowe współbieżności](https://msdn.microsoft.com/library/874bc58f-8dce-483e-a3a1-4dcc9e52ed2c).  
  
## <a name="syntax"></a>Składnia  
  
```  
namespace Concurrency;  
```
