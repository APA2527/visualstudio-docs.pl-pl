# <a name="how-to-filter-profiling-tools-report-views-to-display-just-my-code"></a>Porady: filtrowanie widoków raportu odnośnie do narzędzi profilowania w celu wyświetlenia Tylko mojego kodu
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Program Visual Studio 2010 Premium zapewnia filtrowanie raportów wydajności, tak aby wyświetlały tylko kod, który został zaimplementowany. Ta funkcja, która nosi nazwę Tylko mój kod, jest domyślnie włączona. Jest ona konfigurowana za pomocą opcji **tylko mój kod** w oknie dialogowym **Opcje** .  
  
### <a name="to-enable-just-my-code"></a>Aby włączyć Tylko mój kod  
  
1. W menu **Tools** (Narzędzia) kliknij pozycję **Options** (Opcje).  
  
2. W oknie dialogowym **Opcje** rozwiń węzeł **Narzędzia wydajności** , a następnie wybierz węzeł **Ogólne** .  
  
3. W obszarze **Ustawienia ogólne**zaznacz pole wyboru **Włącz tylko mój kod** .  
  
### <a name="to-disable-just-my-code"></a>Aby wyłączyć Tylko mój kod  
  
1. W menu **Tools** (Narzędzia) kliknij pozycję **Options** (Opcje).  
  
2. W oknie dialogowym **Opcje** rozwiń węzeł **Narzędzia wydajności** , a następnie wybierz węzeł **Ogólne** .  
  
3. W obszarze **Ustawienia ogólne**Usuń zaznaczenie pola wyboru **Włącz tylko mój kod** .
