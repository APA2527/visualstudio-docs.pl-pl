# <a name="command-line-profiling-tools-reference"></a>Narzędzia profilowania wiersza polecenia — Odwołanie
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Narzędzia do oceny wydajności z wiersza polecenia umożliwiają deweloperom profile swoich aplikacji, w tym celu w wierszu polecenia. Następujące narzędzia profilowania wiersza polecenia umożliwiają kod profilu w wierszu polecenia.  
  
## <a name="in-this-section"></a>W tej sekcji  
 [VSInstr](../profiling/vsinstr.md)  
 Wyświetla listę opcji Instrumentacji VSInstr.exe.  
  
 [Ostrzeżenia VSInstr](../profiling/vsinstr-warnings.md)  
 Wyświetla listę ostrzeżeń, wystawiony przez narzędzie VSInstr.  
  
 [VSPerfCLREnv](../profiling/vsperfclrenv.md)  
 Wyświetla listę opcji VSPerfCLREnv.exe, aby skonfigurować środowisko pod kątem profilowania kodu zarządzanego.  
  
 [VSPerfCmd](../profiling/vsperfcmd.md)  
 Wyświetla listę opcji VSPerfCmd.exe zbierania danych dotyczących wydajności.  
  
 [VSPerfMon](../profiling/vsperfmon.md)  
 Wyświetla listę opcji VSPerfMon.exe zbierania zbierania danych.  
  
 [VSPerfReport](../profiling/vsperfreport.md)  
 Wyświetla listę opcji wiersza polecenia VSPerfReport.exe w celu tworzenia raportów.
