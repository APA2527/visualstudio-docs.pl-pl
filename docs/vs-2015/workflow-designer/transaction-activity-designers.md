# <a name="transaction-activity-designers"></a>Projektanci działań Transaction
Projektanci działań transakcji są używani do tworzenia i konfigurowania działań transakcyjnych, takich jak kompensacja i potwierdzenie. Tematy zawarte w tej sekcji i wymienione w poniższej tabeli zawierają wskazówki dotyczące korzystania z [!INCLUDE[wfd2](../includes/wfd2-md.md)] projektantów działań i szablonów.

## <a name="in-this-section"></a>W tej sekcji
 [CancellationScope](../workflow-designer/cancellationscope-activity-designer.md) Służy do tworzenia i konfigurowania <xref:System.Activities.Statements.CancellationScope> działania.

 [Działanie CompensableActivity](../workflow-designer/compensableactivity-activity-designer.md) Służy do tworzenia i konfigurowania <xref:System.Activities.Statements.CompensableActivity> działania.

 [Kompensacja](../workflow-designer/compensate-activity-designer.md) Służy do tworzenia i konfigurowania <xref:System.Activities.Statements.Compensate> działania.

 [Potwierdź](../workflow-designer/confirm-activity-designer.md) Służy do tworzenia i konfigurowania <xref:System.Activities.Statements.Confirm> działania.

 Element [TransactionScope](../workflow-designer/transactionscope-activity-designer.md) Służy do tworzenia i konfigurowania <xref:System.Activities.Statements.TransactionScope> działania.
