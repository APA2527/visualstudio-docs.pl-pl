# <a name="collection-activity-designers"></a>Projektanci działań Collection
Projektanci działań Collection są używane do tworzenia i konfigurowania działań, które umożliwia zarządzanie kolekcjami, tematy zawarte w tej sekcji i wymienione w poniższej tabeli zawierają wskazówki dotyczące sposobu używania [!INCLUDE[wfd2](../includes/wfd2-md.md)]projektantów działań i szablonu.  
  
## <a name="in-this-section"></a>W tej sekcji  
 [AddToCollection\<T>](../workflow-designer/addtocollection-t-activity-designer.md)  
 Umożliwia tworzenie i konfigurowanie <xref:System.Activities.Statements.AddToCollection%601> działania.  
  
 [ClearCollection\<T>](../workflow-designer/clearcollection-t-activity-designer.md)  
 Umożliwia tworzenie i konfigurowanie <xref:System.Activities.Statements.ClearCollection%601> działania.  
  
 [ExistsInCollection\<T>](../workflow-designer/existsincollection-t-activity-designer.md)  
 Umożliwia tworzenie i konfigurowanie <xref:System.Activities.Statements.ExistsInCollection%601> działania.  
  
 [RemoveFromCollection\<T>](../workflow-designer/removefromcollection-t-activity-designer.md)  
 Umożliwia tworzenie i konfigurowanie <xref:System.Activities.Statements.RemoveFromCollection%601> działania.
