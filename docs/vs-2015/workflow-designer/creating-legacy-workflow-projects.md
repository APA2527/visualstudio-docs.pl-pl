# <a name="creating-legacy-workflow-projects"></a>Tworzenie starszej wersji projektów przepływu pracy
W tej sekcji opisano sposób tworzenia projektów przepływu pracy za pomocą starszego [!INCLUDE[wfd1](../includes/wfd1-md.md)] dostarczone przez vs2010. Starszego [!INCLUDE[wfd2](../includes/wfd2-md.md)] cele [!INCLUDE[netfx35_long](../includes/netfx35-long-md.md)] lub [!INCLUDE[vstecwinfx](../includes/vstecwinfx-md.md)]. Zawiera informacje dotyczące sposobu korzystania z szablonów projektów przepływu pracy, które są wymienione w poniższej tabeli.  
  
|Szablon projektu|Opis|  
|----------------------|-----------------|  
|Aplikacja Konsolowa sekwencyjnego przepływu pracy|Projekt służący do tworzenia aplikacji konsolowej sekwencyjnego przepływu pracy.|  
|Biblioteka sekwencyjnego przepływu pracy|Projekt służący do tworzenia biblioteki sekwencyjnego przepływu pracy.|  
|Biblioteka działań przepływu pracy|Projekt służący do tworzenia biblioteki działań, które później można ponownie użyć jako bloków konstrukcyjnych w przepływach pracy.|  
|Aplikacja konsoli przepływu pracy automatu stanów|Projekt służący do tworzenia aplikacji konsolowej przepływu pracy stanu komputera.|  
|Biblioteka przepływu pracy automatu stanów|Projekt służący do tworzenia biblioteki przepływu pracy stanu komputera.|  
|Pusty projekt przepływu pracy|Pusty projekt do tworzenia aplikacji przepływu pracy.|
