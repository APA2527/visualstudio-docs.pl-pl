# <a name="visual-studio-tools-for-unity"></a>Narzędzia Visual Studio Tools for Unity
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

W tej sekcji znajdziesz informacje o Visual Studio Tools for Unity, bezpłatnym rozszerzeniu programu Visual Studio, które umożliwia programowi Visual Studio zaawansowane narzędzie do tworzenia gier z użyciem aparatu Unity.  
  
## <a name="visual-studio-tools-for-unity"></a>Narzędzia Visual Studio Tools for Unity  
 **Unity** to środowisko gier i środowiska programistyczne do tworzenia gier dla wielu platform. Aparat Unity działa na wielu platformach obejmujących zaawansowane komputery, do konsol gier, do tabletów z obsługą dotykową i telefonów komórkowych — nawet w sieci Web. Edytor Unity zapewnia przyjazny interfejs do tworzenia rozbudowanych światów gier. Możliwości, łatwość użycia i zasięg środowiska Unity sprawiają, że jest to popularny wybór wśród współczesnych deweloperów gier.  
  
 Chociaż Edytor Unity doskonale nadaje się do pracy ze sobą w świecie, nie można pisać w nim kodu. Za pomocą **Visual Studio Tools for Unity**można korzystać ze znanych funkcji edytowania kodu, debugowania i produktywności programu Visual Studio, aby tworzyć skrypty edytora i gier dla projektu Unity przy użyciu języka C#, a także debugować je przy użyciu zaawansowanych możliwości debugowania programu Visual Studio.  
  
 Ale Visual Studio Tools for Unity jest więcej niż to; ma także ścisłą integrację z edytorem Unity, dzięki czemu można poświęcać mniej czasu na przełączenie się z powrotem i w czasie w celu wykonywania prostych zadań, zapewnia ulepszenia produktywności specyficzne dla aparatu Unity i umieszcza dokumentację aparatu Unity na wyskakującym rogu.  
  
 Przejdź do [omówienia](../cross-platform/overview-of-visual-studio-tools-for-unity.md) , aby dowiedzieć się więcej na temat ofert Visual Studio Tools for Unity lub [wprowadzenie](../cross-platform/getting-started-with-visual-studio-tools-for-unity.md) , aby dowiedzieć się, jak zainstalować Visual Studio Tools for Unity i rozpocząć korzystanie z niego w projekcie aparatu Unity.
