 Niektóre pakiety nigdy nie powinny zostać usunięte, ponieważ wiele innych funkcji jest zależnych od nich. Na przykład, wymienione w obszarze "podstawowe środowisko IDE" nigdy nie powinno być usuwane.

 Niektóre funkcje nie mogą zostać całkowicie usunięte. Na przykład nie istnieje pakiet, który można wyrejestrować w celu usunięcia **Widok klasy** i skojarzonych z nim menu, opcji i usług. Okno **Widok klasy** jest dostarczane przez pakiet środowiska programu Visual Studio, który udostępnia również inne kluczowe funkcje środowiska IDE. Jeśli chcesz usunąć **Widok klasy**, musisz również usunąć pozycję **Znajdź i Zamień**, strony **Opcje** środowiska, **okno polecenia**i okno **dane wyjściowe** .
