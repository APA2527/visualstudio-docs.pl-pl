# <a name="extending-projects"></a>Rozszerzanie projektów
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Projekty i rozwiązania są sposobem, w jaki program Visual Studio organizuje kod i pliki zasobów w ramach kompilacji i jednostek wdrożenia. Więcej informacji na temat projektów można znaleźć w [projektach (Visual Studio SDK)](../extensibility/extending-projects.md).  
  
 Możesz tworzyć własne typy projektów za pomocą zestawu Visual Studio SDK i zarządzanej struktury pakietów dla projektów, które można pobrać w [zarządzanej strukturze pakietów dla projektów](https://archive.codeplex.com/?p=mpfproj12). Aby zrozumieć, w jaki sposób są implementowane projekty niestandardowe, zobacz [Nowa generacja projektu: pod okapem, część pierwszej](../extensibility/internals/new-project-generation-under-the-hood-part-one.md) i [generowanie nowego projektu: pod okapem, druga część](../extensibility/internals/new-project-generation-under-the-hood-part-two.md).  
  
 W tematach w tej sekcji opisano sposób tworzenia projektów niestandardowych i zarządzania różnymi typami rozwiązań programu Visual Studio.
