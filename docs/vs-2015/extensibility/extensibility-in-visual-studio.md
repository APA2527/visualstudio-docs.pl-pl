# <a name="extensibility-in-visual-studio"></a>Rozszerzalność w programie Visual Studio
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Program Visual Studio oferuje wiele opcji rozszerzalności. Możesz tworzyć własne zestawy SDK, korzystać z systemu Windows, Windows Phone i zestawów Azure SDK (które są instalowane w ramach programu Visual Studio) i poszerzać sam program Visual Studio.

## <a name="extend-visual-studio"></a>Rozszerzanie funkcjonalności programu Visual Studio
 Możesz użyć zestawu Visual Studio SDK, aby dowiedzieć się więcej o każdej części programu Visual Studio: polecenia, menu, okna, Edytory i projekty. Aby dowiedzieć się więcej, zobacz [Visual Studio SDK](../extensibility/visual-studio-sdk.md).

## <a name="create-your-own-sdks"></a>Tworzenie własnych zestawów SDK
 Dowiedz się, jak utworzyć, spakować i wdrożyć własne zestawy SDK dla platformy i rozszerzeń: [Tworzenie zestawu Software Development Kit](../extensibility/creating-a-software-development-kit.md).

## <a name="roslyn-extensibility"></a>Rozszerzalność Roslyn
 Dowiedz się, jak używać Roslyn do rozszerzalności: [.NET compiler platform ("Roslyn")](../extensibility/dotnet-compiler-platform-roslyn-extensibility.md).

## <a name="azure-sdk"></a>Zestaw Azure SDK
 Dowiedz się, jak włączyć zestaw Azure SDK: [Włączanie zestawu Azure SDK](../extensibility/enabling-the-azure-sdk.md).
