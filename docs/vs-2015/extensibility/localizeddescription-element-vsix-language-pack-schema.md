# <a name="localizeddescription-element-vsix-language-pack-schema"></a>LocalizedDescription — element (schemat pakietu językowego VSIX)
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Wymagany. Zawiera zlokalizowany opis rozszerzenia.  
  
## <a name="syntax"></a>Składnia  
  
```  
<LocalizedDescription>Localized description of the extension</LocalizedDescription>  
```
