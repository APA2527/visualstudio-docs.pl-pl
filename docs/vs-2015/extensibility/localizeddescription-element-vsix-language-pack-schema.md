# <a name="localizeddescription-element-vsix-language-pack-schema"></a>Element LocalizedDescription (schemat VSIX Language Pack)
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Najnowszą wersję tego tematu znajduje się w temacie [LocalizedDescription — Element (VSIX Language Pack schemat)](https://docs.microsoft.com/visualstudio/extensibility/localizeddescription-element-vsix-language-pack-schema).  
  
Wymagane. Zapewnia zlokalizowany opis rozszerzenia.  
  
## <a name="syntax"></a>Składnia  
  
```  
<LocalizedDescription>Localized description of the extension</LocalizedDescription>  
```
