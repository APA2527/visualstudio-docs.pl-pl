# <a name="color-theming-tools"></a>Narzędzia do tworzenia motywów kolorów
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Zestaw SDK programu VS udostępnia kilka narzędzi, które ułatwiają tworzenie motywów kolorów i zarządzanie nimi.  
  
- Narzędzie [edytora kolorów VSIX](../../extensibility/internals/vsix-color-editor.md) umożliwia tworzenie i edytowanie niestandardowych kolorów dla programu Visual Studio.  
  
- Narzędzie [kompilatora kolorów VSIX](../../extensibility/internals/vsix-color-compiler.md) Konwertuje istniejące pliki Visual Studio Theme. XML na. pkgdef dla rozszerzenia.
