# <a name="how-to-set-multiple-startup-projects"></a>Porady: ustawianie wielu projektów startowych
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Program Visual Studio umożliwia określenie, jak w momencie uruchamiania debugera ma być uruchamiany więcej niż jeden projekt.

### <a name="to-set-multiple-startup-projects"></a>Aby ustawić wiele projektów startowych

1. W **Eksplorator rozwiązań**wybierz rozwiązanie (bardzo najpopularniejszy węzeł). Kliknij prawym przyciskiem myszy węzeł, aby wyświetlić menu kontekstowe.

2. Wybierz pozycję **Właściwości**. Zostanie otwarte okno dialogowe **strony właściwości rozwiązania** .

3. Rozwiń węzeł **wspólne właściwości** , a następnie kliknij pozycję **projekt startowy**.

4. Kliknij przycisk **wiele projektów startowych** i ustaw odpowiednie akcje. Aby uzyskać więcej informacji, zobacz [NIB How to: Modify Project Properties and Settings Configuration](https://msdn.microsoft.com/e7184bc5-2f2b-4b4f-aa9a-3ecfcbc48b67).
