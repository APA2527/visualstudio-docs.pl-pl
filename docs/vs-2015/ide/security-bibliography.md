# <a name="security-bibliography"></a>Bibliografia dotyczące zabezpieczeń
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Poniżej przedstawiono wybrane zasoby dostępne w trybie online i w temacie Tworzenie bezpiecznych aplikacji oraz Konfigurowanie bezpiecznych środowisk:

## <a name="microsoft-web-sites"></a>Witryny sieci Web firmy Microsoft
 [Zabezpieczenia firmy Microsoft](https://www.microsoft.com/security) Dowiedz się, w jaki sposób firma Microsoft działa, aby zapewnić bezpieczeństwo aplikacji i systemów, od pulpitu do systemów na poziomie sieci, z linkami do zasobów zabezpieczeń dla informatyków, deweloperów i użytkowników domowych.

 [Zasoby zabezpieczeń w witrynie MSDN](https://www.microsoft.com/msrc?rtc=1) Prowadzi użytkownika do dokumentacji zorientowanej na dewelopera, przykładów kodu, artykułów technicznych i innych zasobów na potrzeby tworzenia bezpiecznych aplikacji.

 [Zabezpieczenia w witrynie Microsoft TechNet](https://www.microsoft.com/securityengineering) Zawiera wskaźniki do najważniejszych informacji technicznych dotyczących zabezpieczeń dla specjalistów ds. technologii informatycznych i "użytkowników zaawansowanych".
