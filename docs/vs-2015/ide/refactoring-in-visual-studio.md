# <a name="refactoring-in-visual-studio"></a>Refaktoryzacja w programie Visual Studio
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Refaktoryzacja to proces modyfikacji kodu w celu ułatwienia utrzymania, zrozumienia i rozbudowania, ale bez zmiany jego działania. Niektóre typowe *refaktoryzacje* obejmują:

1. dzielenie długich funkcji na mniejsze

2. zmiana nazw zmiennych, typów, funkcji lub parametrów

3. Zmienianie kolejności parametrów

4. Usuwanie lub zmiana kolejności przy użyciu dyrektyw

   Ponieważ logika refaktoryzacji jest ściśle związana ze składnią języka, każda usługa językowa w programie Visual Studio zapewnia własne możliwości refaktoryzacji. Aby uzyskać więcej informacji, zobacz następujące strony:
