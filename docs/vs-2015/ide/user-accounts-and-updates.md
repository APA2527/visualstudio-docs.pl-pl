# <a name="user-accounts-and-updates"></a>Konta użytkowników i aktualizacje
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Visual Studio jest połączone środowisko. Logowanie umożliwia dostęp do wszystkich usług, które są skojarzone z kontem użytkownika, a także wszystkie konta, które są połączone z tym kontem. Ustawienia konta są przenoszone z Tobą, wszędzie tam, gdzie możesz użyć programu Visual Studio. Poziomy uprawnień można skonfigurować na określonym wystąpieniu programu Visual Studio dla poszczególnych użytkowników. Ponadto program Visual Studio informuje użytkownika, kiedy dowolny składnik instalacji, takich jak zestaw SDK, był dostępna jest aktualizacja.
