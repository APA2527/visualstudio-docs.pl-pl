# <a name="application-properties"></a>Właściwości aplikacji
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Można ustawić ogólne właściwości aplikacji C#dla Visual Basic,, i F# aplikacji, takich jak typ aplikacji, obiekt startowy i informacje o zestawie, w okienku **aplikacji** **projektanta projektu**.

 W okienku **aplikacji** można wykonać następujące zadania:

- Określ informacje o zestawie

- Określ ekran powitalny dla aplikacji

- Określanie zdarzeń na poziomie aplikacji

- Zmień nazwę zestawu

- Zmiana przestrzeni nazw dla aplikacji

- Zmień typ aplikacji

- Zmiana obiektu uruchomieniowego aplikacji

- Określ ikonę aplikacji

- Włącz style wizualne systemu Windows XP

- Określ zachowanie dotyczące wystąpienia aplikacji

- Włączanie niestandardowego uwierzytelniania użytkownika

- Określanie zachowania zamykania dla aplikacji

  Zadania dostępne dla aplikacji zależą od używanego języka deweloperskiego.
