# <a name="talk-to-us"></a>Porozmawiaj z nami
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aby uzyskać najnowszą dokumentację programu Visual Studio 2017, zobacz [Porozmawiaj z nami](https://docs.microsoft.com/visualstudio/ide/talk-to-us) w witrynie docs.microsoft.com.  

  

Jesteśmy zainteresowani pomysłami dotyczącymi poprawy naszej dokumentacji. Nie możemy osobiście, ale możemy upewnij się, że Twoja opinia pobiera do odpowiednich osób.  
  
## <a name="i-likedislike-something-in-the-visual-studio-documentation"></a>Jak/podoba mi się coś, co w dokumentacji programu Visual Studio  
 Użyj naszego przycisku wysyłania opinii (**ta strona była pomocna?**) w dolnej części strony. Ten przycisk, można znaleźć na wszystkich stronach w witrynie MSDN. Jeśli szukasz odpowiedzi, nie możesz znaleźć, powiedz nam, jakich informacji szukasz.  
  
## <a name="i-would-like-to-report-a-problem-with-visual-studio"></a>Chcę zgłosić problem z programem Visual Studio  
 Jeśli zostały przekroczone problemów za pomocą programu Visual Studio, takich jak awarie, wolna wydajności, nieoczekiwane zachowanie i tak dalej, zgłoś do nas przez kliknięcie prawym przyciskiem myszy ikonę opinii obok **szybkiego uruchamiania**, lub wybierając  **Pomoc &#124; Wyślij opinię &#124; Zgłoś Problem** z menu głównego. Aby uzyskać więcej informacji, zobacz [jak zgłosić Problem z programem Visual Studio](../ide/how-to-report-a-problem-with-visual-studio-2015.md).  
  
## <a name="i-want-to-make-a-suggestion-about-visual-studio-features"></a>Czy chcesz sugestię o funkcje programu Visual Studio  
 Jeśli jest coś, co firma Microsoft może lepiej Daj nam znać! Wybierz **sugestię** zasugerować funkcję lub zmienić do zespołu usługi Visual Studio. Twoja sugestia zostanie automatycznie opublikowane w usłudze [UserVoice](https://visualstudio.uservoice.com) której inne osoby mogą głosować na.  
  
## <a name="rate-this-product"></a>Oceń ten produkt  
 Wersje wstępne programu Visual Studio mają **Oceń ten produkt** elementu menu, który umożliwia użytkownikowi na przesyłanie opinii na temat jakości kompilacji, są uruchomione. Nie widzisz to w wydanej wersji produktu.  
  
## <a name="i-need-help"></a>Potrzebuję pomocy!  
 Zobacz [Microsoft Support](http://go.microsoft.com/fwlink/?LinkID=99019) pomocy online.  
  
 Pomoc techniczna poza terenem Stanów Zjednoczonych i Kanady może się różnić. Aby uzyskać listę kontaktów regionalnych, zobacz [witryny Microsoft na świecie](http://www.microsoft.com/worldwide/).  
  
 Aby uzyskać więcej opcji, zobacz [pomoc techniczna — omówienie](http://www.visualstudio.com/support/support-overview-vs).  
  
 W przypadku większych organizacji, które wymagają zarządzanej pomocy technicznej udzielanej bezpośrednio przez firmę Microsoft, dostępne są umowy w ramach różnych ofert pomocy technicznej klasy Premier. Aby uzyskać więcej informacji, zobacz [Microsoft Services Premier Support](http://go.microsoft.com/fwlink/?LinkId=258223).  
  
 Jeżeli produkt jest preinstalowany na nowym komputerze lub urządzeniu, producent sprzętu zapewnia pomoc i obsługę techniczną dla tego oprogramowania. Skontaktuj się bezpośrednio z producentem, aby uzyskać wsparcie.  
  
 Usługi pomocy technicznej firmy Microsoft podlegają aktualnej cenie i warunkom. Ceny i warunki mogą ulec zmianie bez powiadomienia.  
  
## <a name="i-want-to-get-involved-in-the-developer-community"></a>Chcę angażowanie się w społeczność deweloperów  
 Jeśli chcesz udostępnić swoje pytania i odpowiedzi z innymi deweloperami, możesz użyć [fora MSDN](http://social.msdn.microsoft.com/Forums/home) lub [StackOverflow](http://stackoverflow.com/). Możesz przeglądać kod z innymi deweloperami i udostępniać własne przykłady w [próbki kodu deweloperskiego](http://code.msdn.microsoft.com/) i [CodePlex](http://www.codeplex.com/) witryn sieci Web.
