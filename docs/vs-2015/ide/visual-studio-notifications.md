# <a name="visual-studio-notifications"></a>Powiadomienia programu Visual Studio
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Program Visual Studio ma system powiadomień, który ostrzega o różnych zdarzeniach:

1. Dostępna jest aktualizacja składnika w programie Visual Studio

2. W programie Visual Studio jest dostępna aktualizacja

3. Istnieje zdarzenie środowiskowe, które może Cię zainteresować

   Program Visual Studio powiadamia o dostępności aktualizacji dla samego programu Visual Studio lub w przypadku któregokolwiek z zainstalowanych składników. Zawiera również powiadomienia dotyczące niektórych rodzajów zdarzeń, które występują w środowisku programu Visual Studio. Ikona powiadomienia jest flagą w prawym górnym rogu paska tytułu programu Visual Studio. Gdy ikona ma kolor czerwony, oznacza to, że występuje problem z licencją. Gdy jest to kolor żółty, można zainstalować aktualizację produktu Visual Studio. Gdy jest to "normalne" lub czarny, zobaczysz komunikaty, które są informacyjne lub opcjonalne, takie jak awarie, dla których warto znaleźć obejście. Możesz kliknąć ikonę, aby otworzyć **centrum powiadomień** i wybrać, które z nich mają działać. Możesz zdecydować się na jednorazowe odrzucanie powiadomienia lub zawsze zignorować powiadomienie (przez umieszczenie nad nim), a program Visual Studio nigdy nie będzie ponownie wyświetlał tego komunikatu.

   Jeśli użytkownik zdecyduje się zawsze zignorować powiadomienie, program Visual Studio przestanie go pokazywać. Możesz zresetować listę ignorowanych powiadomień w [oknie dialogowym powiadomienia, środowisko, opcje](../ide/reference/notifications-environment-options-dialog-box.md).
