# <a name="supplying-xml-code-comments"></a>Stosowanie komentarzy kodu XML
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Komentarze można wprowadzać w plikach kodu źródłowego jako IntelliSense. Po dodaniu przełącznika/doc do wiersza polecenia Visual Basic, C#lub C++ kompilatora, generowany jest plik. XML, który służy jako podstawa dokumentacji IntelliSense.

 Aby uzyskać więcej informacji na temat dokumentowania kodu w Visual Basic, zobacz [How to: Create XML Documentation](https://msdn.microsoft.com/library/27b5b06c-09b9-496a-8245-f9542d846230).

 Aby uzyskać więcej informacji na temat dokumentowania C#kodu w, zobacz [Komentarze w dokumentacji XML](https://msdn.microsoft.com/library/803b7f7b-7428-4725-b5db-9a6cff273199).

 Aby uzyskać więcej informacji na temat dokumentowania C++kodu w, zobacz [Komentarze](https://msdn.microsoft.com/library/6fcb906c-c264-4083-84bc-373800b2e514) i [dokumentacja XML](https://msdn.microsoft.com/library/a1aec1c5-b2d1-4c74-83ae-1dbbbb76b506).
