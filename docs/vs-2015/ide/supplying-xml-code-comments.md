# <a name="supplying-xml-code-comments"></a>Stosowanie komentarzy kodu XML
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Najnowszą wersję tego tematu znajduje się w temacie [podawania komentarzy kodu XML](https://docs.microsoft.com/visualstudio/ide/supplying-xml-code-comments).  
  
Możesz wprowadzać komentarze w plikach kodu źródłowego dostępna jako funkcja IntelliSense. Po dodaniu przełącznika/doc do języka Visual Basic, C# lub wiersza polecenia kompilatora C++ generowany jest plik XML, służący jako podstawa dokumentacji funkcji IntelliSense.  
  
 Aby uzyskać więcej informacji na temat dokumentowanie kodu w Visual Basic, zobacz zobacz [porady: tworzenie dokumentacji XML](http://msdn.microsoft.com/library/27b5b06c-09b9-496a-8245-f9542d846230).  
  
 Aby uzyskać więcej informacji na temat dokumentowanie kodu w języku C#, zobacz [komentarze dokumentacji XML](http://msdn.microsoft.com/library/803b7f7b-7428-4725-b5db-9a6cff273199).  
  
 Aby uzyskać więcej informacji na temat dokumentowanie kodu w języku C++, zobacz [komentarze](http://msdn.microsoft.com/library/6fcb906c-c264-4083-84bc-373800b2e514) i [dokumentacji XML](http://msdn.microsoft.com/library/a1aec1c5-b2d1-4c74-83ae-1dbbbb76b506).
