|Tytuł|Opis|  
|-----------|-----------------|

|[Strona Zdarzenia kompilacji, Projektant projektu (C#)](../../ide/reference/build-events-page-project-designer-csharp.md)|Użyj w tym okienku, aby określić instrukcje dotyczące konfiguracji kompilacji.|  
|[Strona kompilacji, Projektant projektu (C#)](../../ide/reference/build-page-project-designer-csharp.md)|Użyj w tym okienku, aby określić właściwości konfiguracji kompilacji dla [!INCLUDE[csprcs](../../includes/csprcs-md.md)] projektu.|
