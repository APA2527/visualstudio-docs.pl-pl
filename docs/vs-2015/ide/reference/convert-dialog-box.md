# <a name="convert-dialog-box"></a>Konwertowanie — Okno dialogowe
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Okno dialogowe **Konwertuj** umożliwia wybranie narzędzia konwersji w celu utworzenia nowego projektu na podstawie starego, takiego jak tworzenie [!INCLUDE[vbprvblong](../../includes/vbprvblong-md.md)] projektu na podstawie projektu Visual Basic 6. Możesz również użyć konwertera [projektów programu Visual Studio](http://go.microsoft.com/fwlink/?LinkID=246465)dostępnego w witrynie sieci Web CodePlex, aby uaktualnić projekty utworzone w poprzednich wersjach programu Visual Studio.  
  
 [!INCLUDE[note_settings_general](../../includes/note-settings-general-md.md)]

## <a name="opening-the-convert-dialog-box"></a>Otwieranie okna dialogowego Konwertuj  
  
#### <a name="to-access-the-convert-dialog-box"></a>Aby uzyskać dostęp do okna dialogowego Konwertuj  
  
1. Na pasku menu wybierz **plik**, **Otwórz**, **Konwertuj**.  
  
     Zostanie wyświetlona lista narzędzi do konwersji kodu, które są obecnie dostępne.  
  
2. Wybierz narzędzie, które jest odpowiednie dla Twojej aplikacji.
