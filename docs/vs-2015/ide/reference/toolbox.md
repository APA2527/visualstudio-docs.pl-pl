# <a name="toolbox"></a>Przybornik
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

W **przyborniku** są wyświetlane ikony formantów i innych elementów, które można dodać do [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] projektów. Aby otworzyć **Przybornik**, kliknij polecenie **Przybornik** w menu **Widok** . Można zadokować **Przybornik**i można go przypiąć lub ustawić do **autoukrywania**.

 Ikony **przybornika** można przeciągać do widoku projektu lub wkleić w edytorze kodu. Każda akcja dodaje podstawowy kod, aby utworzyć wystąpienie elementu **przybornika** w aktywnym pliku projektu.

 W **przyborniku** są wyświetlane tylko te elementy, które są odpowiednie dla typu pliku, w którym pracujesz. Możesz wyszukać w przyborniku, aby bardziej odfiltrować elementy, które są wyświetlane. Jeśli projekt wymaga formantu, który nie jest obsługiwany przez profil klienta, można ustawić projekt jako docelowy dla całej struktury, edytując właściwości projektu.

> [!NOTE]
> Na komputerze w poniższych instrukcjach mogą być wyświetlane inne nazwy i lokalizacje niektórych elementów interfejsu użytkownika programu Visual Studio. Te elementy są określane przez numer wersji Visual Studio oraz twoje ustawienia. Aby uzyskać więcej informacji, zobacz [Dostosowywanie ustawień deweloperskich w programie Visual Studio](https://msdn.microsoft.com/22c4debb-4e31-47a8-8f19-16f328d7dcd3).
