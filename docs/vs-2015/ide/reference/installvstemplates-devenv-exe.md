# <a name="installvstemplates-devenvexe"></a>/InstallVSTemplates (devenv.exe)
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

  
Rejestruje szablony projektu lub elementu, które znajdują się w  *\<ścieżka instalacji programu Visual Studio >* \Common7\IDE\ProjectTemplates\ lub  *\<ścieżka instalacji programu Visual Studio >* \Common7\IDE\ItemTemplates\, dzięki czemu są one dostępne za pośrednictwem **nowy projekt** i **Dodaj nowy element** okien dialogowych.  
  
> [!WARNING]
>  Ten przełącznik jest obsługiwana tylko w celach deweloperskich partnerów programu Visual Studio i nie jest dostępny w wersji Express. Devenv musi działać jako administrator, aby można było używać [/Setup (devenv.exe)](../../ide/reference/setup-devenv-exe.md) i [/installvstemplates (devenv.exe)](../../ide/reference/installvstemplates-devenv-exe.md) przełączników. Aby uzyskać więcej informacji, zobacz [uprawnienia użytkownika](../../ide/user-permissions-and-visual-studio.md).  
  
## <a name="syntax"></a>Składnia  
  
```  
devenv.exe /InstallVSTemplates  
```
