# <a name="diff"></a>/Diff
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

  
Porównuje dwa pliki. Różnice są wyświetlane w specjalne okno programu Visual Studio.  
  
## <a name="syntax"></a>Składnia  
  
```  
devenv /Diff SourceFile, TargetFile, [SourceDisplayName],[TargetDisplayName]  
```  
  
## <a name="arguments"></a>Argumenty  
 `SourceFile`  
 Wymagane. Pełna ścieżka i nazwa pierwszego pliku do porównania.  
  
 `TargetFile`  
 Wymagane. Pełna ścieżka i nazwa drugiego pliku do porównania  
  
 `SourceDisplayName`  
 Opcjonalna. Wyświetlana nazwa pierwszego pliku.  
  
 `TargetDisplayName`  
 Opcjonalna. Wyświetlana nazwa drugiego pliku.
