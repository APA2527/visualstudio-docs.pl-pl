# <a name="accessibility-tips-and-tricks"></a>Porady i wskazówki związane z ułatwieniami dostępu
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

[!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] obejmuje wbudowane funkcje ułatwień dostępu, dzięki czemu można łatwo korzystać z klawiatury i korzystać z czytników ekranu lub innych urządzeń z technologią pomocniczą. Temat zawiera kilka sugestii dotyczących optymalizacji [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] pod kątem ułatwień dostępu, a także kilka przydatnych kombinacji klawiszy skrótów.

> [!NOTE]
> Okna dialogowe i polecenia menu mogą się różnić od tych opisanych w Pomocy, w zależności od ustawień aktywnych lub wydania. Aby zmienić ustawienia, wybierz **Importuj i Eksportuj ustawienia** w menu **Narzędzia** . Aby uzyskać więcej informacji, zobacz [Dostosowywanie ustawień deweloperskich w programie Visual Studio](https://msdn.microsoft.com/22c4debb-4e31-47a8-8f19-16f328d7dcd3).

## <a name="save-your-ide-settings"></a>Zapisz ustawienia środowiska IDE
 Środowisko IDE można dostosować przez zapisanie układu okna, schematu mapowania klawiatury i innych preferencji. Aby uzyskać więcej informacji, zobacz [Dostosowywanie ustawień deweloperskich w programie Visual Studio](https://msdn.microsoft.com/22c4debb-4e31-47a8-8f19-16f328d7dcd3).

## <a name="common-windows-shortcut-key-combinations"></a>Typowe kombinacje klawiszy skrótów systemu Windows
 Do wykonywania zadań w ramach [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] można użyć wielu standardowych kombinacji klawiszy skrótów systemu Windows. Niektóre z tych kombinacji klawiszy skrótów obejmują:

|Opis|Kombinacja klawiszy|
|-----------------|---------------------|
|duży kontrast włączania i wyłączania|Lewy **Alt + Strzałka w lewo + Print Screen**|
|Zaznacz lub wyczyść opcję pola wyboru w oknie dialogowym|**SPACJA**|
|Wyświetl właściwości wybranego elementu. Na przykład umieszcza fokus w oknie właściwości, jeśli formularz jest zaznaczony lub otwiera okno dialogowe strony właściwości, jeśli wybrano projekt.|**ALT + ENTER**|
|Wyświetlanie elementów na aktywnej liście, takich jak lista rozwijana, w oknie dialogowym|**Alt** + Strzałka w dół|
|Wyświetlanie elementów na aktywnej liście rozwijanej w siatce|**SHIFT**  + **Alt**  + **Strzałka w dół**|

## <a name="hidden-visual-studio-shortcut-key-combinations"></a>Ukryte kombinacje klawiszy skrótów programu Visual Studio
 Niektóre funkcje mają dostępne klawisze skrótów, które nie są wyświetlane w oknie dialogowym Opcje w okienku Environment\Keyboard. Te kombinacje klawiszy skrótów obejmują następujące elementy:

|Funkcja|Opis|Kombinacja klawiszy|
|-------------|-----------------|---------------------|
|Okno przybornika|Przechodzenie między kartami przybornika|**CTRL**  + **Strzałka w dół**<br /><br /> and<br /><br /> **CTRL**  + **strzałka**|
|Okno przybornika|Dodawanie kontrolki z przybornika do formularza lub projektanta|**WEJŚĆ**|
|Klawiatura, środowisko, Opcje — okno dialogowe|Usuń kombinację klawiszy wprowadzoną w opcji **naciśnij klawisze skrótów**|**BACKSPACE**|
|Wszystkie okna narzędzi|Wybierz pierwszy przycisk na pasku narzędzi okna|**SHIFT**  + **Alt**|
|Paski narzędzi IDE|Wybierz pierwszy przycisk na pasku narzędzi Standardowy|**Alt**, **Ctrl**  + **karta** **Uwaga:** ponownie naciśnij klawisz **Ctrl**  + **Tab** , aby wybrać pierwszy przycisk na następnym pasku narzędzi IDE.|
