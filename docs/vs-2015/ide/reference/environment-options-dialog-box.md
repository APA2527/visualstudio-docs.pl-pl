# <a name="environment-options-dialog-box"></a>Środowisko, opcje — Okno dialogowe
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Możesz użyć stron w folderze **Environment** w oknie dialogowym **Opcje** , aby określić, jak są wyświetlane i stosowane są pewne elementy zintegrowanego środowiska programistycznego (IDE). Dostęp do stron środowiska można uzyskać, na pasku menu, wybierając **Narzędzia**, **Opcje**, a następnie wybierając **środowisko**.

> [!NOTE]
> Okna dialogowe i polecenia menu, które są wyświetlane na komputerze, mogą się różnić od tych, które są opisane w pomocy, w zależności od aktywnych ustawień i używanej wersji Visual Studio. Aby zmienić ustawienia, wybierz **Importuj i Eksportuj ustawienia** w menu **Narzędzia** . Aby uzyskać więcej informacji, zobacz [Dostosowywanie ustawień deweloperskich w programie Visual Studio](https://msdn.microsoft.com/22c4debb-4e31-47a8-8f19-16f328d7dcd3).
