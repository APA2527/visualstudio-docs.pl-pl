# <a name="set-current-thread-command"></a>Ustaw bieżący wątek — Polecenie
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Najnowszą wersję tego tematu znajduje się w temacie [Ustaw bieżący wątek — polecenie](https://docs.microsoft.com/visualstudio/ide/reference/set-current-thread-command).  
  
  
Ustawia określony wątek jako bieżącego wątku.  
  
## <a name="syntax"></a>Składnia  
  
```  
Debug.SetCurrentThread index  
```  
  
## <a name="arguments"></a>Argumenty  
 `index`  
 Wymagane. Wybiera wątku według indeksu.  
  
## <a name="example"></a>Przykład  
  
```  
>Debug.SetCurrentThread 1  
```
