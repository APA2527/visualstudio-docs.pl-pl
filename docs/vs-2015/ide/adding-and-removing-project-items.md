# <a name="adding-and-removing-project-items"></a>Dodawanie i usuwanie elementów projektu
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aby dodać lub usunąć projekty i elementy projektu, kliknij prawym przyciskiem myszy węzeł projekt lub rozwiązanie w **Eksplorator rozwiązań**i wybierz polecenie **Dodaj** lub **Usuń**. Usunięcie projektu z rozwiązania lub elementu z projektu nie powoduje usunięcia go z dysku lub kontroli źródła, chyba że zostanie wybrana opcja trwałego usunięcia.
