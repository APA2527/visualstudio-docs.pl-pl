# <a name="adding-and-removing-project-items"></a>Dodawanie i usuwanie elementów projektu
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Możesz można dodawać i usuwać projektów i elementów projektu, klikając prawym przyciskiem myszy węzeł projektu lub rozwiązania w **Eksploratora rozwiązań**i wybierając pozycję **Dodaj** lub **Usuń**. Usuwanie projektu rozwiązanie lub element z projektu, nie powoduje usunięcia go z dysku lub kontroli źródła dopiero po wybraniu opcji, aby trwale usunąć.
