# <a name="globalizing-and-localizing-applications"></a>Globalizacja i lokalizacja aplikacji
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Jeśli planujesz dystrybucję aplikacji do odbiorców międzynarodowych, musisz pamiętać o kilku kwestiach w fazie projektowania i opracowywania. Nawet jeśli nie masz takich planów, niewielki nakład pracy może znacznie ułatwić pracę w przyszłości, jeśli plany zmieniają się w przyszłych wersjach aplikacji. Usługi wbudowane w ułatwią [!INCLUDE[dnprdnshort](../includes/dnprdnshort-md.md)] opracowywanie pojedynczej aplikacji, która może zostać przystosowana do różnych ustawień regionalnych przy użyciu zarządzania zarządzanego w programie Visual Studio.

 Program Visual Studio został zaprojektowany od początku, aby ułatwić rozwój dla użytkowników międzynarodowych, wykorzystując usługi wbudowane w [!INCLUDE[dnprdnshort](../includes/dnprdnshort-md.md)] . Poniższe strony ułatwią wprowadzenie do funkcji wielojęzycznych wbudowanych w program Visual Studio.
