# <a name="xml-documentation-comments-javascript"></a>Komentarze dokumentacji XML (JavaScript)
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

W języku JavaScript można utworzyć dokumentację dla kodu, dołączając tagi XML w specjalnych polach komentarzy w kodzie źródłowym. Te tagi XML są używane przez funkcję IntelliSense do dostarczania informacji o obiektach JavaScript.

 Położenie komentarzy do kodu różni się w zależności od konkretnego tagu.

 Obsługiwane są następujące Tagi:

- [\<deprecated>](../ide/deprecated-javascript.md)

- [\<field>](../ide/field-javascript.md)

- [\<loc>](../ide/loc-javascript.md)

- [\<param>](../ide/param-javascript.md)

- [\<returns>](../ide/returns-javascript.md)

- [\<signature>](../ide/signature-javascript.md)

- [\<summary>](../ide/summary-javascript.md)

- [\<value>](../ide/value-javascript.md)

- [\<var>](../ide/var-javascript.md)
