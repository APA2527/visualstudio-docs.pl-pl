# <a name="getting-started-with-ptvs-setting-up-visual-studio"></a>Pierwsze kroki z narzędziami PTVS: konfigurowanie programu Visual Studio

[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Instalowanie PTVS i powiązanych bibliotek jest szybkie, jeśli masz program Visual Studio. Jeśli nie masz programu Visual Studio, możesz uzyskać bezpłatną kopię wersji profesjonalnej jakości.

Możesz obejrzeć te instrukcje w bardzo krótkim [filmie wideo](https://www.youtube.com/watch?v=_okUV47eM5c&list=PLReL099Y5nRdLgGAdrb_YeTdEnd23s6Ff&index=1)w serwisie YouTube.

Poniżej przedstawiono następujące czynności: Instalowanie programu Visual Studio, instalowanie PTVS, instalowanie bibliotek Python i danych (Anaconda, koroner), a następnie sprawdzanie instalacji.

Pierwszą rzeczą, której potrzebujesz, jest program Visual Studio. Jeśli jesteś deweloperem Open Source lub indywidualnym, możesz korzystać z programu Visual Studio [Community Edition](https://www.visualstudio.com/products/visual-studio-community-vs), który jest bezpłatny i funkcjonalny dla specjalistów. Możesz również korzystać z wersji Community w organizacji, o ile jest to przeznaczone do uczenia się, badań akademickich lub współtworzenia projektów open source. Studenci i uruchomienia powinny odszukać programy DreamSpark i BizSpark, aby sprawdzić, czy masz prawo do bezpłatnego dostępu lub polecić pracodawcę, czy masz subskrypcję MSDN.

Po zainstalowaniu programu Visual Studio należy [zainstalować PTVS](https://archive.codeplex.com/?p=pytools). Jest to bezpłatne, autonomiczne rozszerzenie, które jest w pełni obsługiwane przez firmę Microsoft i opracowane z myślą o składkach ze społeczności.

Teraz musisz zainstalować język [Python](https://www.python.org/download/). Język Python jest obsługiwany przez społeczność, a jej Strona główna to python.org. Program Continuum Analytics tworzy bezpłatny pakiet o nazwie Anaconda, który ma Język Python i wiele przydatnych bibliotek (szczególnie w przypadku analizy i przetwarzania danych), a Enthought tworzy podobny pakiet o nazwie koroner. Wystarczy zainstalować tylko jeden z tych produktów. Jeśli nie masz pewności, który z nich należy zacząć od [Anaconda](https://www.anaconda.com/products/individual), który zapewnia najbardziej aktualne środowisko Python i większość trudnych do zainstalowania pakietów.

Uruchom program Visual Studio i upewnij się, że wszystko działa. W menu Widok wybierz inne okna. Zobaczysz element o nazwie środowiska Python. To okno zawiera wszystkie wykryte instalacje języka Python PTVS i wszystkie zainstalowane pakiety. Okno kontroluje również odświeżanie bazy danych w celu wyświetlania popełnień podczas edytowania kodu. Ten proces odświeżania trwa trochę czasu, ale po zakończeniu PTVS może zawierać bardziej przydatne informacje o pakietach.

Jeśli chcesz użyć IPython z PTVS, postępuj zgodnie z tymi [instrukcjami](https://archive.codeplex.com/?p=pytools).

Te instrukcje można obejrzeć w krótkim [filmie wideo](https://www.youtube.com/watch?v=_okUV47eM5c&list=PLReL099Y5nRdLgGAdrb_YeTdEnd23s6Ff&index=1)w serwisie YouTube.
