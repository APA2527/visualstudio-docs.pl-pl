# <a name="getting-started-with-ptvs-setting-up-visual-studio"></a>Pierwsze kroki z narzędziami PTVS: konfigurowanie programu Visual Studio
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Instalowanie narzędzi PTVS i powiązane biblioteki odbywa się szybko, jeśli masz program Visual Studio.  Jeśli nie masz programu Visual Studio, możesz uzyskać bezpłatną kopię wersji profesjonalnej jakości.  
  
 Możesz obserwować te instrukcje w bardzo krótkim [wideo usługi youtube](https://www.youtube.com/watch?v=_okUV47eM5c&list=PLReL099Y5nRdLgGAdrb_YeTdEnd23s6Ff&index=1).  
  
 Kroki highlevel są instalacji programu Visual Studio, instalowanie narzędzi PTVS, instalowanie bibliotek danych i języka Python (Anaconda, korony), a następnie na koniec można sprawdzić instalację.  
  
 Pierwszą rzeczą, musisz mieć program Visual Studio.  Jeśli opracowujesz typu open source lub poszczególnych, możesz użyć programu Visual Studio [Community Edition](https://www.visualstudio.com/products/visual-studio-community-vs)wersji Community jest programem bezpłatnym i funkcjonalności dla profesjonalistów z zakresu.  Można również użyć Community Edition w organizacji, tak długo, jak jest on przeznaczony dla klasy, nauki i badań akademickich lub współtworzenia projektów typu open source.  Uczniowie i startupy powinien wyglądać w programie DreamSpark i programy programu BizSpark, aby sprawdzić, czy masz uprawnienia do swobodnego dostępu lub poproś pracodawcy, czy masz subskrypcję MSDN.  
  
 Po zainstalowaniu programu Visual Studio, konieczne będzie [Instalowanie narzędzi PTVS](http://pytools.codeplex.com/wikipage?title=PTVS%20Installation).  Jest to bezpłatne, autonomiczne rozszerzenie, które w pełni obsługiwane przez firmę Microsoft i jawnie opracowane z uwzględnieniem materiałów przekazanych przez społeczność.  
  
 Teraz należy [zainstaluj język Python](http://python.org/download/).  Python jest całkowicie obsługiwany przez społeczność, a jego strony głównej jest python.org.  Continuum Analytics daje bezpłatny pakiet o nazwie Anaconda Python i wielu przydatne bibliotek (szczególnie w przypadku analizy i przetwarzania danych), a Enthought zwalnia korony, która jest podobna pakietu.  Musisz zainstalować jedną z nich.  Jeśli nie wiesz, który z nich, skorzystaj z [Anaconda](https://www.continuum.io/downloads), które zapewnia najbardziej aktualne języka Python i większość pakietów trudne do zainstalowania.  
  
 Uruchom program Visual Studio i upewnij się, że wszystko działa.  W menu Widok wybierz inne Windows.  Zostanie wyświetlony element o nazwie środowiska Python.  To okno zawiera wszystkie instalacje języka Python, narzędzi PTVS wykryte i wszystkie zainstalowane pakiety.  Okno kontroluje również Odświeżanie bazy danych do wyświetlania zakończeń podczas edycji kodu.  Ten proces odświeżania zajmuje trochę czasu i dużo zasobów Procesora i pamięci, ale po zakończeniu PTVS można wyświetlić więcej pomocnych informacji o pakietach.  
  
 Jeśli chcesz użyć IPython przy użyciu narzędzi PTVS, postępuj zgodnie z tymi [instrukcje](http://pytools.codeplex.com/wikipage?title=Using%20IPython%20with%20PTVS).  
  
 Możesz obserwować te instrukcje w bardzo krótkim [wideo usługi youtube](https://www.youtube.com/watch?v=_okUV47eM5c&list=PLReL099Y5nRdLgGAdrb_YeTdEnd23s6Ff&index=1).
