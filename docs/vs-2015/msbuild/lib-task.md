# <a name="lib-task"></a>LIB — Zadanie
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Opakowuje narzędzia Microsoft 32-bitowy Library Manager, lib.exe. Menedżer biblioteki tworzy i zarządza Biblioteka plików obiektów Common Object File Format (COFF). Menedżer biblioteki można tworzyć pliki eksportu i Importuj biblioteki do definicji odwołanie, eksportowane. Aby uzyskać więcej informacji, zobacz [odwołanie do biblioteki LIB](https://msdn.microsoft.com/library/ecc7f643-bbd4-47a3-8dc6-b360f880db91) i [uruchamianie LIB](https://msdn.microsoft.com/library/d54f5c81-7147-4b2c-a8db-68ce6eb1eabd).
