# <a name="idiaenumlinenumbersreset"></a>IDiaEnumLineNumbers::Reset

## <a name="syntax"></a>Składnia  
  
```cpp#  
HRESULT Reset();  
```  
  
## <a name="return-value"></a>Wartość zwracana  
 Zwraca `S_OK`.
