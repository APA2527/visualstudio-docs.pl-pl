# <a name="idiasymbolgetundecoratedname"></a>IDiaSymbol::get_undecoratedName
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Najnowszą wersję tego tematu znajduje się w temacie [idiasymbol::get_undecoratedname —](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiasymbol-get-undecoratedname).  
  
Pobiera nieozdobioną nazwę zawartej dekorowane, C++ lub połączenia, nazwa.  
  
## <a name="syntax"></a>Składnia  
  
```cpp#  
HRESULT get_undecoratedName (   
   BSTR* pRetVal  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `pRetVal`  
 [out] Zwraca nazwy ozdobionej nieozdobioną nazwę zawartej w języku C++.  
  
## <a name="return-value"></a>Wartość zwracana  
 Jeśli operacja się powiedzie, zwraca `S_OK`; w przeciwnym razie zwraca `S_FALSE` albo kod błędu.  
  
> [!NOTE]
>  Zwracana wartość wynosząca `S_FALSE` oznacza, że właściwość nie jest dostępna dla symbolu.
