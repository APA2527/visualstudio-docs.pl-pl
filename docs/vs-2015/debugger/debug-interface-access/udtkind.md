# <a name="udtkind"></a>UdtKind
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Najnowszą wersję tego tematu znajduje się w temacie [udtkind —](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/udtkind).  
  
W tym artykule opisano różne typy zdefiniowane przez użytkownika (UDT).  
  
## <a name="syntax"></a>Składnia  
  
```cpp#  
enum UdtKind {   
   UdtStruct,  
   UdtClass,  
   UdtUnion,  
   UdtInterface  
};  
```  
  
## <a name="elements"></a>Elementy  
 UdtStruct  
 UDT to struktura.  
  
 UdtClass  
 UDT jest klasą.  
  
 UdtUnion  
 UDT to Unii.  
  
 UdtInterface  
 UDT jest interfejsem.  
  
## <a name="remarks"></a>Uwagi  
 Wartości w tym wyliczeniu są zwracane przez [idiasymbol::get_udtkind —](../../debugger/debug-interface-access/idiasymbol-get-udtkind.md) metody.

 Nagłówek: cvconst.h
