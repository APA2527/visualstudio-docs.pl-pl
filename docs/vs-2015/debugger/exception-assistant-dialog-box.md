# <a name="exception-assistant-dialog-box"></a>Asystent wyjątków — Okno dialogowe
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Okno dialogowe **Asystent wyjątków** pojawia się, gdy zostanie zgłoszony wyjątek czasu wykonywania. **Asystent wyjątków** wyświetla typ wyjątku, zawiera dodatkowe informacje i linki do wskazówek dotyczących rozwiązywania problemów, umożliwia wyszukiwanie dodatkowej pomocy online i umożliwia użytkownikowi wykonywanie określonych akcji, takich jak wyświetlanie szczegółów wyjątku.  
  
## <a name="task-list"></a>Lista zadań  
 Aby wyświetlić temat rozwiązywania problemów z napotkanym typem wyjątku, kliknij jeden z komunikatów TIP wyświetlanych w okienku **wskazówki dotyczące rozwiązywania problemów** .  
  
 Aby wykonać akcje skojarzone z wyjątkiem, kliknij jedną z akcji wyświetlanych w okienku akcji.  
  
 Aby uzyskać informacje na temat włączania lub wyłączania asystenta wyjątków, zobacz [Ogólne, debugowanie, opcje](../debugger/general-debugging-options-dialog-box.md)— okno dialogowe.  
  
## <a name="user-interface-element-list"></a>Lista elementów interfejsu użytkownika  
 **Typ wyjątku**  
 Wyświetla typ zgłoszonego wyjątku.  
  
 **Dodatkowe informacje**  
 Wyświetla dodatkowe informacje o wyjątku.  
  
 **Wskazówki dotyczące rozwiązywania problemów**  
 Wyświetla linki do wskazówek dotyczących rozwiązywania problemów, które mogą pomóc w znalezieniu źródła wyjątku.  
  
 **Akcje**  
 Wyświetla listę akcji, które można wykonać, takich jak wyświetlanie dodatkowych informacji o obiekcie wyjątku.  
  
 **Uzyskaj pomoc online**  
 Umożliwia wyszukiwanie dodatkowej pomocy w trybie online.
