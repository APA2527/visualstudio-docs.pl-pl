# <a name="debugger-settings-and-preparation"></a>Ustawienia debugera i przygotowanie
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

W tej sekcji opisano ustawienia właściwości powiązane z debugerem oraz przygotowania, które należy wykonać w celu debugowania programu przy użyciu [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] debugera. W przypadku tworzenia programu [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] przy użyciu szablonów projektu te ustawienia są ustawiane poprawnie dla Ciebie w konfiguracji debugowania.  
  
 Aby uzyskać więcej informacji, zobacz [How to: Set Debug and Release Configurations](../debugger/how-to-set-debug-and-release-configurations.md).
