# <a name="edit-and-continue-not-supported-for-f"></a>Opcje edytuj i kontynuuj nie są obsługiwane w F# #
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Najnowszą wersję tego tematu znajduje się w temacie [Edytuj i Kontynuuj nie obsługiwane w F #](https://docs.microsoft.com/visualstudio/debugger/edit-and-continue-not-supported-for-f-hash).  
  
Edytuj i Kontynuuj nie jest obsługiwane podczas debugowania kodu F #. Zmiany w kodzie języka F # są możliwe podczas sesji debugowania, ale należy unikać. Zmiany w kodzie nie są stosowane podczas sesji debugowania. W związku z tym wszelkie zmiany wprowadzone w kodzie języka F # podczas debugowania w wyniku kodu źródłowego, który nie jest zgodny z debugowany kod.
