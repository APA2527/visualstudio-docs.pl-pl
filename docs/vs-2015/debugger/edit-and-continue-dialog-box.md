# <a name="edit-and-continue-dialog-box"></a>Edytuj i kontynuuj — okno dialogowe
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Tylko natywne) to okno dialogowe pojawia się, gdy edytowano kod podczas debugowania, a **Edycja i kontynuacja** uznaje, że można zastosować zmiany kodu. Aby uzyskać więcej informacji, zobacz [Edytuj i Kontynuuj](../debugger/edit-and-continue.md).  
  
 **Nigdy nie pokazuj tego okna dialogowego ponownie**  
 Jeśli zaznaczysz to pole wyboru, polecenie **Edytuj i Kontynuuj** zastosuje zmiany kodu bez pytania o zgodę w przyszłości. To ostrzeżenie można ponownie włączyć, przechodząc do okna dialogowego **Opcje** , otwierając folder **debugowanie** , klikając stronę **Edytuj i Kontynuuj** , a następnie wybierając polecenie **wywoływane przez polecenia debugowania** i **pytaj mnie jako pierwsze**.
