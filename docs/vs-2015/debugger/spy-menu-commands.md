# <a name="spy-menu-commands"></a>Polecenia w menu Spy
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Menu **Spy** zawiera polecenia do pracy z oknami i widokami programu Spy + +.  
  
 **Windows**  
 Otwiera nowy widok systemu Windows.  
  
 **Procesy**  
 Otwiera nowy widok procesów.  
  
 **Wątki**  
 Otwiera widok nowych wątków.  
  
 **Komunikaty dziennika**  
 Otwiera nowy widok wiadomości i otwiera okno [dialogowe Opcje wiadomości](../debugger/message-options-dialog-box.md) dla tego widoku.  
  
 **Znajdź okno**  
 Wyświetla okno dialogowe **FindWindow** , aby wybrać okno z myszą.  
  
 **Zakończ**  
 Zamyka narzędzie Spy + +.
