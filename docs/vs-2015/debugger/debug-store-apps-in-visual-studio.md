# <a name="debug-store-apps-in-visual-studio"></a>Debugowanie aplikacji ze Sklepu w programie Visual Studio
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Debuger programu Visual Studio umożliwia kontrolowanie wykonywania programu i sprawdzanie jego stanu. Możesz użyć debugera, aby znaleźć przyczyny wad w aplikacji ze sklepu Windows i dokładnie zrozumieć, jak działa aplikacja. W przypadku wstrzymania (przerwania) wykonywania w debugerze program Visual Studio wyświetla plik źródłowy, który zawiera kod wykonywany i podświetla instrukcję wykonującą. Można przyjrzeć się wartościom zmiennych, stosie wywołań wykonywania funkcji i innym aspektom stanu programu. Można kontynuować wykonywanie (krok po kroku) programu po jednej instrukcji, aby zobaczyć, jak instrukcje zmieniają wartości programu. W aplikacjach pisanych przy użyciu języka JavaScript można przeanalizować i manipulować modelem DOM strony.
