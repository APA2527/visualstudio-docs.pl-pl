# <a name="debug-store-apps-in-visual-studio"></a>Debugowanie aplikacji Store w programie Visual Studio
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Najnowszą wersję tego tematu znajduje się w temacie [Store debugowania aplikacji w programie Visual Studio](https://docs.microsoft.com/visualstudio/debugger/debug-store-apps-in-visual-studio).  
  
Debuger programu Visual Studio pozwala kontrolować wykonywanie program i sprawdź jego stan. Można użyć debugera, aby znaleźć przyczynę usterki w aplikacji Windows Store i zrozumieć, dokładnie tak, jak aplikacja działa. Po wstrzymaniu wykonywania (podziału) w debugerze programu Visual Studio Wyświetla plik źródłowy, który zawiera wykonywanie kodu i wyróżnienie wykonywania instrukcji. Można sprawdzić wartości zmiennych, stos wywołań wykonywania funkcji i innych aspektów stanu programu. Można kontynuować wykonywania (krok za pośrednictwem) program jedną instrukcję, w czasie, aby zobaczyć, jak zmienić wartości program w instrukcji. W aplikacjach napisanych w JavaScript możesz badania i manipulowania modelu DOM strony.
