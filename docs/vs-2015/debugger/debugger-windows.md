[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Najnowszą wersję tego tematu znajduje się w temacie [debuger Windows w programie Visual Studio](https://docs.microsoft.com/visualstudio/debugger/debugger-windows).
