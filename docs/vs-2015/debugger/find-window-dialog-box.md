# <a name="find-window-dialog-box"></a>Okno dialogowe Znajdź okno
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

To okno dialogowe służy do wybierania okna i wyświetlania jego właściwości lub komunikatów. Aby wyświetlić to okno dialogowe, wybierz pozycję **Znajdź okno** z menu **Spy** .  
  
 Dostępne są następujące ustawienia:  
  
 **Narzędzie wyszukiwania**  
 Narzędzie do przeciągania i upuszczania używane do wybierania okna.  
  
 **Ukryj Spy + +**  
 Usuwa główne okno programu Spy + + i pozostawia okno dialogowe **Znajdowanie okna** widoczne na wierzchu innych aplikacji.  
  
 **Uchwyty**  
 Unikatowy identyfikator okna do odnalezienia.  
  
 **Caption**  
 Tekst napisu okna.  
  
 **Klasa**  
 Klasa okna.  
  
 **Stylów**  
 Styl okna.  
  
 **Rect**  
 Prostokąt ograniczający dla okna.  
  
 **Pokaż właściwości lub komunikaty**  
 Określa, czy naciśnięcie przycisku OK wyświetli okno [dialogowe Właściwości okna](../debugger/window-properties-dialog-box.md) lub Otwórz [Widok komunikatów](../debugger/messages-view.md).
