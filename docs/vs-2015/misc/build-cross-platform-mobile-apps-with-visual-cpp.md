# <a name="build-cross-platform-mobile-apps-with-visual-c"></a>Twórz aplikacje mobilne dla wielu platform za pomocą Visual C++
Możesz utworzyć Międzyplatformowy kod dla urządzeń z systemami Android, iOS i Windows za pomocą Visual C++ dla opracowywania aplikacji mobilnych na wiele platform. Jest to opcjonalna funkcja dostępna w programie Visual Studio 2015, która umożliwia Międzyplatformowe Programowanie kodu dla systemów iOS, Android i Windows przy użyciu Visual C++.  
  
 Instrukcje dotyczące instalacji i przykłady kodu znajdują się w temacie [Visual C++ dla opracowywania aplikacji mobilnych na wiele platform](../cross-platform/visual-cpp-for-cross-platform-mobile-development.md).
