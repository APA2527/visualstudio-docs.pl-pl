# <a name="vssdk-samples"></a>Przykłady VSSDK
Oficjalne narzędzia do rozszerzania dla programu Visual Studio to witryna sieci Web [galerii kodu MSDN](https://go.microsoft.com/fwlink/?LinkID=127810) . Lokacja zawiera próbki wstępne, przykłady ilustrujące nowe funkcje w programie Visual Studio, szczegółowej dokumentacji szczegółowe, linki do udostępnionych projektów źródłowych i innych zasobów ułatwiających rozbudowanie programu Visual Studio. Aby wyświetlić listę tych zasobów, wpisz **VSX** w polu **Wyszukaj przykłady** na stronie głównej.  
  
 Przykłady zestawu [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] SDK pokazują, jak zintegrować rozszerzenie w zintegrowanym środowisku programistycznym (IDE) programu Visual Studio. Katalog kodu źródłowego każdego przykładu zawiera dokument o nazwie Overview. XML, który zawiera informacje o implementacji dla tego przykładu.  
  
 Dostępny jest również przykład IronPython. Jest ona zintegrowana z [projektem IronPython Studio](https://go.microsoft.com/fwlink/?LinkID=183554) w witrynie sieci Web CodePlex.
