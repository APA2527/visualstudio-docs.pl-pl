# <a name="vssdk-samples"></a>Przykłady VSSDK
To miejsce, w którym oficjalne przykłady Visual Studio Extensibility [galerii kodu MSDN](http://go.microsoft.com/fwlink/?LinkID=127810) witryny sieci Web. Lokacja zawiera wprowadzające przykłady, przykłady, które ilustrują nowe funkcje w Visual Studio, dokumentacji szczegółowe informacje, łącza do projektów udostępnionych source i innych zasobów, ułatwiające Rozszerz program Visual Studio. Aby wyświetlić listę tych zasobów, wpisz **VSX** w **Wyszukiwanie przykładów** pola na stronie głównej.  
  
 [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] Przykładowych zestawach SDK pokazują, jak zintegrować rozszerzenia programu Visual Studio zintegrowane środowisko programistyczne (IDE). Katalog kodu źródłowego, co przykładu zawiera dokument o nazwie Overview.xml, który zawiera uwagi dotyczące implementacji dla tego przykładu.  
  
 Dostępna jest również przykładowy Ironpythonu. Jest zintegrowana z usługą [projektu IronPython Studio](http://go.microsoft.com/fwlink/?LinkID=183554) w witrynie CodePlex.
