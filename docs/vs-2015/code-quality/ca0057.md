# <a name="ca0057"></a>CA0057

Nie można zamapować naruszeń w bieżącym zestawie elementów docelowych i reguł.

Ten błąd występuje, jeśli FxCopCmd.exe jest wywołać jej ręcznie przy użyciu pliku projektu lub / zaimportowany Raport XML, który zawiera naruszeń, które nie mogą być mapowane w bieżącym zestawie elementów docelowych i reguł.

Występuje tylko wtedy, gdy analiza odbywa się przy użyciu narzędzia w wersji wiersza polecenia.
