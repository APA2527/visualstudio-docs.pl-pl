# <a name="c28216"></a>C28216
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ostrzeżenie C28216: adnotacja \_Check_return\_ dotyczy tylko warunków post dla określonego parametru funkcji.  
  
 Adnotacja `_Check_return_` została zastosowana w kontekście innym niż `__post`; może być potrzebny modyfikator `__post` (lub `__drv_out`) lub może być niepoprawnie umieszczony.
