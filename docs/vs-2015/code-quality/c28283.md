# <a name="c28283"></a>C28283
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ostrzeżenie C28283: Dla symbolu określona specyfikacja rozmiaru nie jest jeszcze obsługiwana  
  
 To ostrzeżenie wskazuje, czy adnotacja rozmiar specyfikacji "wartownik" Odebrano wartość różną od zera. Zasadniczo obiekt wywołujący próbował Załóżmy, że ciąg znaków jest zakończony przez znak inny niż binarne zero.
