# <a name="c28137"></a>C28137
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ostrzeżenie C28137: Argument w postaci zmiennej musi być zamiast tego stałą (literałem)  
  
 To ostrzeżenie jest zgłaszane, gdy w wywołaniu funkcji brakuje wymagane stałą (literałem). Zajrzyj do dokumentacji dla tej funkcji.  
  
## <a name="example"></a>Przykład  
 Na przykład [ExAcquireResourceExclusiveLite](https://msdn.microsoft.com/library/windows/hardware/ff544351.aspx) procedura wymaga podania wartości TRUE lub FALSE dla `Wait` parametru. Poniższy przykładowy kod generuje to ostrzeżenie:  
  
```  
ExAcquireResourceExclusiveLite(Resource, Wait);  
```  
  
 Poniższy kod pozwala uniknąć tego ostrzeżenia:  
  
```  
ExAcquireResourceExclusiveLite(Resource, TRUE);  
```
