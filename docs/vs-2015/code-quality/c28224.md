# <a name="c28224"></a>C28224
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ostrzeżenie C28224: adnotacja wymaga parametrów  
  
 To ostrzeżenie wskazuje, że nazwana adnotacja jest używana bez parametrów, a co najmniej jeden parametr jest wymagany. Nie powinno to być możliwe, jeśli standardowe makra są używane do adnotacji; to ostrzeżenie nie jest oczekiwane podczas normalnego użytkowania.
