# <a name="c1253"></a>C1253
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ostrzeżenie C1253: nie można załadować pliku modelu.  
  
 Narzędzie analizy kodu raportuje to ostrzeżenie, gdy w pliku modelu wystąpi błąd wewnętrzny, a nie w analizowanym kodzie.
