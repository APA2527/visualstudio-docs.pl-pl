# <a name="ca0501"></a>CA0501

Nie można odczytać raportu wyjściowego.

Dzieje się tak, gdy nie można zapisać danych wyjściowych raportu z jednego z następujących powodów:

-   Dysk zapełniony

-   Brak uprawnień do zapisu do katalogu

-   Katalog nie istnieje
