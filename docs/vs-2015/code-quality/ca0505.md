# <a name="ca0505"></a>CA0505

Przestarzałe właściwość zostanie zignorowana, ponieważ jest definiowana przez właściwość.

Ostrzeżenie CA505 jest generowany, gdy określony przestarzałe i przez właściwości analizy kodu zawiera wartości w projekcie lub tworzenia pliku konfiguracji. Aby rozwiązać ten problem, Usuń właściwość przestarzałe.
