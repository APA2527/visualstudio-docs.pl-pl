# <a name="ca0505"></a>CA0505

Właściwość przestarzała zostanie zignorowana, ponieważ zdefiniowano Właściwość Superceding.

Ostrzeżenie CA505 jest generowane, gdy zarówno określone przestarzałe i Superceding właściwości analizy kodu zawierają wartości w projekcie lub pliku konfiguracyjnym kompilacji. Aby rozwiązać ten problem, Usuń przestarzałą właściwość.
