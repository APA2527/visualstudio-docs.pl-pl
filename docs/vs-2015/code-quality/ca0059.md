# <a name="ca0059"></a>CA0059

Błąd przełącznik wiersza polecenia.

Ten błąd występuje, jeśli przełącznik wiersza polecenia wymagane są spełnione, na przykład

/ out lub nie określono przełącznika/Console. Ten błąd występuje także w przypadku nieprawidłowy przełącznik, na przykład wartość nieprawidłowy limit czasu.
