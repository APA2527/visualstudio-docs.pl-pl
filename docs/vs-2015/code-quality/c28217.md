# <a name="c28217"></a>C28217
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Najnowszą wersję tego tematu znajduje się w temacie [C28217](https://docs.microsoft.com/visualstudio/code-quality/c28217).  
  
Ostrzeżenie C28217: dla funkcji, liczba parametrów dla adnotacji odpowiada znalezionej w pliku  
  
 Adnotacje w bieżącym wierszu i wiersz w wiadomości są niespójne. To nie powinna być może wystąpić, jeśli są używane standardowe makra dla adnotacji; to ostrzeżenie nie powinien być widoczny w czasie normalnego użycia.
