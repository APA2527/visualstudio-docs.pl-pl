# <a name="c28215"></a>C28215
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ostrzeżenie C28215: Nie można zastosować elementu typefix parametr, który ma już typefix  
  
 Stosowanie `__typefix` adnotację parametru, który już ma adnotacji tego błędu. `__typefix` Adnotacje są używane tylko w niektórych przypadkach specjalnych i to ostrzeżenie nie oczekuje się, aby była widoczna podczas normalnego użytkowania.
