# <a name="how-to-view-managed-code-defects"></a>Porady: wyświetlanie defektów kodu zarządzanego
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

W analizie kodu, błędy, ostrzeżenia i komunikaty są raportowane w oknie Analiza kodu. Okno Analiza kodu pojawia się automatycznie podczas przeprowadzania analizy kodu w projekcie lub rozwiązaniu.

 W oknie Analiza kodu można:

- **Wybierz tytuł ostrzegawczy.**

     Dodatkowe informacje o ostrzeżeniu są wyświetlane w oknie Analiza kodu. Wiersz w pliku źródłowym, który uruchomił ostrzeżenie, jest wyświetlany i wyróżniony w edytorze kodu źródłowego.

- **Wybierz identyfikator ostrzeżenia.** Identyfikatory ostrzeżeń są poprzedzone przez **urząd certyfikacji** i zawierają liczbę czterocyfrową. Na przykład CA1234.

     Zostanie wyświetlony temat pomocy biblioteki MSDN dla tego ostrzeżenia. Temat pomocy zawiera szczegółowe informacje o przyczynie ostrzeżenia i zawiera przykłady, które pokazują, jak występuje problem z ostrzeżeniem i został rozwiązany.
