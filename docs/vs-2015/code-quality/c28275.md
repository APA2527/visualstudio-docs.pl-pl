# <a name="c28275"></a>C28275
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Najnowszą wersję tego tematu znajduje się w temacie [C28275](https://docs.microsoft.com/visualstudio/code-quality/c28275).  
  
Ostrzeżenie C28275: parametr _Macro_value\_ ma wartość null  
  
 To ostrzeżenie wskazuje, że jest błąd wewnętrzny w pliku modelu, a nie w kodzie, analizowane. *MacroValue* funkcja została wywołana bez parametrów.
