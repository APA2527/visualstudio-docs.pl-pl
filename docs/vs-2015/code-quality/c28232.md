# <a name="c28232"></a>C28232
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Najnowszą wersję tego tematu znajduje się w temacie [C28232](https://docs.microsoft.com/visualstudio/code-quality/c28232).  
  
Ostrzeżenie C28232: _Pre\_, cząstkę _Post\_, lub _deref —\_ nie są stosowane do jakichkolwiek adnotacji  
  
 To ostrzeżenie wskazuje, że `_Pre_`, `_Post_`, lub `_Deref_` operator, który znajduje się w wyrażeniu adnotacji nie kolejnych adnotacji funkcjonalności; został zignorowany, modyfikator, ale oznacza to, kodowane niepoprawnie adnotacji.
