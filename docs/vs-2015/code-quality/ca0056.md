# <a name="ca0056"></a>CA0056

Plik projektu ma nieprawidłową wersję narzędzia do analizy.

To ostrzeżenie występuje, jeśli FxCopCmd.exe jest wywołać jej ręcznie przy użyciu pliku projektu, który ma wersję narzędzia analizy nieprawidłowy kod.
