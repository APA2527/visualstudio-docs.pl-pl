# <a name="c28288"></a>C28288
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ostrzeżenie C28288: dla funkcji, błąd składni w \_ elemencie \_ adnotacji at () (Nieprawidłowa nazwa parametru)  
  
 Narzędzie analizy kodu raportuje to ostrzeżenie, gdy `SAL_at` `__drv_at` jest używana adnotacja () i wyrażenie parametru nie może być interpretowane w bieżącym kontekście. Może to obejmować użycie błędnej pisowni parametru lub nazwy elementu członkowskiego, lub błędnego pisowni słów kluczowych "return" lub "This".
