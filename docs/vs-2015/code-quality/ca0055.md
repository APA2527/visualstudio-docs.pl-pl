# <a name="ca0055"></a>CA0055

Nie można załadować pliku: \<path >.

Narzędzie do analizy kodu nie może załadować pliku docelowego, który został określony na potrzeby analizy. Dzieje się tak, jeśli plik nie zostanie znaleziony lub nastąpiła odmowa dostępu do pliku. Zwykle zdarza się to, gdy użytkownik zdecydował się przesłonić Właściwość zestawu wejściowego w programie Visual Studio lub ręcznie wywołać plik FxCopCmd. exe.
