# <a name="c28021"></a>C28021
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ostrzeżenie C28021: parametr \<param > z adnotacją \<anno > musi być wskaźnikiem  
  
 To ostrzeżenie jest zgłaszane, gdy obiekt, którego dodawana jest adnotacja nie jest typem wskaźnika. Ta adnotacja nie można używać z `void` lub typów całkowitych.
