# <a name="c28213"></a>C28213
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ostrzeżenie C28213: adnotacja `_Use_decl_annotations_` musi być używana do odwoływania się, bez modyfikacji, wcześniejszej deklaracji. \<dlaczego >  
  
 To ostrzeżenie jest zgłaszane w przypadku odwołania do wcześniejszej deklaracji bez wymaganej adnotacji `_Use_decl_annotations_`.
