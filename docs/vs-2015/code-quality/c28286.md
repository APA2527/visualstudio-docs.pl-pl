Ostrzeżenie C28286: dla funkcji, błąd składni w pobliżu końca  
  
 Zgłaszanej przez narzędzie analizy kodu, to ostrzeżenie, w przypadku błędu prawdopodobne w pliku modelu. Kilka błędów pliku źródłowego może spowodować również takie błędy.
