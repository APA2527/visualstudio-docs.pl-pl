# <a name="c28223"></a>C28223
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ostrzeżenie C28223: nie znaleziono oczekiwanego tokenu/identyfikatora dla adnotacji, parametr  
  
 To ostrzeżenie wskazuje, że parametr do adnotacji powinien być identyfikatorem i został napotkany inny symbol. Zwykle oznacza to nieprawidłowe kodowane makro adnotacji. Użycie języka C lub C++ słów kluczowych w tym miejscu spowoduje wystąpienie tego błędu.
