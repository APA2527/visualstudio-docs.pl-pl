# <a name="c28219"></a>C28219
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Najnowszą wersję tego tematu znajduje się w temacie [C28219](https://docs.microsoft.com/visualstudio/code-quality/c28219).  
  
Ostrzeżenie C28219: element członkowski wyliczenia oczekiwał na adnotację parametru w adnotacji  
  
 Parametr adnotacji powinien należeć nazwany `enum` napotkano typu, a inny symbol; Użyj elementu członkowskiego, który `enum` typu. Wskazuje to zazwyczaj makro adnotacji kodowane niepoprawnie.
