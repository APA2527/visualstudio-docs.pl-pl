# <a name="c6705"></a>C6705
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ostrzeżenie C6705: Oczekiwano błędu adnotacji < expected_number > argumentów dla \<parametru adnotacji > znaleziono < actual_number >.  
  
 To ostrzeżenie jest zgłaszane w przypadku błędu w adnotacjach.
