# <a name="c28304"></a>C28304
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ostrzeżenie C28304: dla C++ < parametru odwołania parameter_name > znaleziono nieprawidłowo umieszczony operator `_Notref_` został zastosowany do \<tokenu >.  
  
 Operatora `_Notref_` należy używać tylko w szczególnych okolicznościach obejmujących C++ parametry odwołania i tylko w makrach dostarczonych przez system. Bezpośrednio następuje operator `_Deref_` lub Adnotacja funkcjonalna.
