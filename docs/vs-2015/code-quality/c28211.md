# <a name="c28211"></a>C28211
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ostrzeżenie C28211: Oczekiwano nazwy kontekstu statycznego dla SAL_context  
  
 To ostrzeżenie wskazuje, że argument przekazywany do `_Static_context_` adnotacja musi zawierać nazwę kontekstu zdefiniowanych przez narzędzie. To ostrzeżenie nigdy nie powinny wystąpić w czasie normalnego użycia.
