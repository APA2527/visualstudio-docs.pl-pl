# <a name="c28312"></a>C28312
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Najnowszą wersję tego tematu znajduje się w temacie [C28312](https://docs.microsoft.com/visualstudio/code-quality/c28312).  
  
Ostrzeżenie 28312: adnotacja < annotation_name > dla powtórzonej deklaracji \<funkcja > \<parametr > jest przestarzałą wersją SAL.  
  
 Adnotacja jest w starej wersji i powinna zostać uaktualniona do równowartości [SAL2](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md). To ostrzeżenie nie jest emitowane, jeśli emitowane było uprzednie ostrzeżenie o niespójnej adnotacji i jest ono zgłaszane tylko raz na deklarację. Sprawdź pozostałą część deklaracji pod kątem przestarzałego SAL.
