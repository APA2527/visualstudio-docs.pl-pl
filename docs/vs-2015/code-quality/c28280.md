# <a name="c28280"></a>C28280
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Najnowszą wersję tego tematu znajduje się w temacie [C28280](https://docs.microsoft.com/visualstudio/code-quality/c28280).  
  
Ostrzeżenie C28280: dla symbolu, element "end" znaleziono bez pasującego "begin"  
  
 Język adnotacji obsługuje begin i end (`{` i `}` w języku C) konstrukcję i parowanie uzyskał niezrównoważone. Użycie makra można uniknąć tej sytuacji.
