# <a name="c28227"></a>C28227
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Najnowszą wersję tego tematu znajduje się w temacie [C28227](https://docs.microsoft.com/visualstudio/code-quality/c28227).  
  
Ostrzeżenie C28227: Adnotacja także nie może być PrimOp (patrz Poprzednia deklaracja)  
  
 To ostrzeżenie wskazuje, że adnotacja o nazwie jest został zadeklarowany jako zwykłe adnotacji i również został wcześniej zadeklarowany jako PrimOp. To nie powinna być może wystąpić, jeśli są używane standardowe makra dla adnotacji; to ostrzeżenie nie powinien być widoczny w typowym zastosowaniem.
