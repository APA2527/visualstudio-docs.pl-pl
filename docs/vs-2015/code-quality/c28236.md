# <a name="c28236"></a>C28236
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Najnowszą wersję tego tematu znajduje się w temacie [C28236](https://docs.microsoft.com/visualstudio/code-quality/c28236).  
  
Ostrzeżenie C28236: nie można użyć adnotacji w wyrażeniu  
  
 To ostrzeżenie wskazuje, że podjęto próbę użycia funkcja zadeklarowana jako adnotacji w kontekście wyrażenia. To nie powinna być może wystąpić, jeśli są używane standardowe makra dla adnotacji; to ostrzeżenie nie powinien być widoczny w typowym zastosowaniem.
