# <a name="c28235"></a>C28235
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ostrzeżenie C28235: Funkcja nie może występować samodzielnie jako adnotacja  
  
 To ostrzeżenie wskazuje, że podjęto próbę użycia funkcji, która nie została zadeklarowana jako adnotacji w kontekście adnotacji. W tym przy użyciu operacji pierwotnych (PrimOp) w kontekście autonomicznych. To nie powinien być może wystąpić, jeśli są używane standardowe makra dla adnotacji. To ostrzeżenie nie powinien być widoczny w typowym zastosowaniem.
