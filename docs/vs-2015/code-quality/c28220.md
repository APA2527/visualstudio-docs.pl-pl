# <a name="c28220"></a>C28220
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Najnowszą wersję tego tematu znajduje się w temacie [C28220](https://docs.microsoft.com/visualstudio/code-quality/c28220).  
  
Ostrzeżenie C28220: wyrażenie całkowite oczekiwana dla adnotacji parametru w adnotacji  
  
 To ostrzeżenie wskazuje, że parametr adnotacji powinien być wyrażeniem liczby całkowitej i napotkano innego typu. Wskazuje to zazwyczaj makro adnotacji kodowane niepoprawnie.
