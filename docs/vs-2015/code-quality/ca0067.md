# <a name="ca0067"></a>CA0067

Nie znaleziono katalogu.

Wartość **/katalog** opcji w wierszu polecenia FxCopCmd nie został znaleziony.

To ostrzeżenie może wystąpić, jeśli używasz **CodeAnalysisDependentAssemblyPaths** właściwości programu MSBuild, aby określić dodatkowe ścieżki zestawów odwołania wyszukiwania i jeden z tych ścieżek nie istnieje.

- Jeśli zostanie wyświetlone ostrzeżenie CA0067 z [CA0058](ca0058.md) błędy lub [CA0060](ca0060.md) CA0067 jest rozpoznawana jako ostrzeżenia, zwykle rozwiązywaniu inne błędy.

- Jeśli bez ostrzeżeń ani błędów, zostanie wyświetlone ostrzeżenie CA0067, zazwyczaj można zignorować ostrzeżenie, ponieważ katalog nie jest wymagane.
