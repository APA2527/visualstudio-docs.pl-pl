# <a name="c28302"></a>C28302
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ostrzeżenie C28302: w C++ przypadku < parametru odwołania parameter_name > znaleziono dodatkowy operator `_Deref_` na \<adnotacji.  
  
 To ostrzeżenie jest zgłaszane, gdy na parametrze jest używany dodatkowy poziom `_Deref_`.  
  
 ADNOTACJĄ SAL2 nie wymaga użycia dodatkowego poziomu `_Deref_` podczas rozpatrywania parametrów referencyjnych. Ta określona adnotacja jest niejednoznaczna i jest interpretowana prawidłowo, ale należy ją skorygować.  
  
 Często można to naprawić przez usunięcie starszej adnotacji `__deref` i użycie składni adnotacją SAL2. Czasami może być konieczne użycie `_At_`, aby odwołać się do określonego obiektu, aby można było dodać adnotację.
