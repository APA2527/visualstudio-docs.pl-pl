# <a name="c6506"></a>C6506
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ostrzeżenie C6506: nieprawidłowa adnotacja: \<name > Właściwość może być używana tylko na wartościach typu wskaźnikowego lub tablicowego  
  
 To ostrzeżenie wskazuje, że właściwość jest używana na typu innego niż wskaźnik lub tablica. Dla wszystkich typów danych można dostępu, Tainted i prawidłowe właściwości. Inne właściwości, takie jak ValidBytesConst "," ValidElementsConst "," ElementSize "i" NullTerminted obsługuje wskaźnika, wskaźnik do elementów członkowskich i typy tablic. Aby uzyskać pełną listę właściwości i obsługiwanych typów danych, zobacz [właściwości adnotacji](http://msdn.microsoft.com/f77b4370-6bda-4294-bd2a-e7d0df182a3d).  
  
## <a name="example"></a>Przykład  
 Poniższy kod generuje to ostrzeżenie:  
  
```cpp  
#include<sal.h>  
void f(_Out_ char c)  
{  
    c = 'd';  
}  
```  
  
 Aby poprawić to ostrzeżenie, użyj wskaźnika lub typu tablicowego, jak pokazano w poniższym przykładowym kodzie:  
  
```cpp  
#include<sal.h>  
void f(_Out_ char *c)  
{  
    *c = 'd';  
}  
```
