# <a name="c28224"></a>C28224
Ostrzeżenie C28224: adnotacja wymaga parametrów  
  
 To ostrzeżenie wskazuje, że nazwanego adnotacji jest używane bez parametrów, i co najmniej jeden parametr jest wymagany. To nie powinna być możliwa, jeśli są używane standardowe makra dla adnotacji; to ostrzeżenie nie powinien być widoczny w normalnych warunkach użytkowania.
