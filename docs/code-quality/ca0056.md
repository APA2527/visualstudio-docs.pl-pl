# <a name="ca0056"></a>CA0056
Plik projektu ma niepoprawną wersję narzędzia do analizy.

To ostrzeżenie występuje, gdy plik FxCopCmd. exe jest wywoływany ręcznie z plikiem projektu, który ma nieprawidłową wersję narzędzia do analizy kodu.
