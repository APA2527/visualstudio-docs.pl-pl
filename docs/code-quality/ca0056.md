# <a name="ca0056"></a>CA0056
Plik projektu ma nieprawidłową wersję narzędzia do analizy.  
  
 To ostrzeżenie występuje, gdy FxCopCmd.exe została wywołana ręcznie z pliku projektu, który ma niepoprawny kod wersję narzędzia do analizy.
