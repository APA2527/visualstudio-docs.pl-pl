# <a name="c28216"></a>C28216
Ostrzeżenie C28216: _Check_return\_ adnotacji dotyczy tylko po warunki dla parametru określonych funkcji.  
  
 `_Check_return_` Adnotacji była stosowana w kontekście innych niż `__post`; może być konieczne `__post` (lub `__drv_out`) modyfikator lub może zostać wykonane nieprawidłowo.
