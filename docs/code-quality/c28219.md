# <a name="c28219"></a>C28219
Ostrzeżenie C28219: Oczekiwano elementu członkowskiego dla adnotacji parametr w adnotacji

 Parametr adnotacji powinien być elementem członkowskim nazwany `enum` napotkano typu i inny symbol; Użyj członkiem tej `enum` typu. Zwykle oznacza to makro niepoprawnie zakodowany adnotacji.
