# <a name="c28219"></a>C28219
Ostrzeżenie C28219: oczekiwano elementu członkowskiego wyliczenia dla adnotacji, parametr w adnotacji

 Oczekiwanym parametrem adnotacji będzie element członkowski nazwanego typu `enum` i został napotkany inny symbol; Użyj elementu członkowskiego tego typu `enum`. Zwykle oznacza to nieprawidłowe kodowane makro adnotacji.
