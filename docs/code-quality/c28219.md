# <a name="c28219"></a>C28219
Ostrzeżenie C28219: Element członkowski wyliczenia oczekiwał na adnotację parametru w adnotacji

 Oczekiwanym parametrem adnotacji będzie element członkowski o nazwie `enum` i napotkano jakiś inny symbol; Użyj elementu członkowskiego tego typu `enum`. Zwykle oznacza to nieprawidłowe kodowane makro adnotacji.
