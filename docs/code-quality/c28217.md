# <a name="c28217"></a>C28217
Ostrzeżenie C28217: dla funkcji, liczbę parametrów dla adnotacji nie odpowiada znalezionej w pliku

 Adnotacje w bieżącym wierszu i wierszu w komunikacie nie są zgodne. To nie powinna być możliwa, jeśli są używane standardowe makra dla adnotacji; to ostrzeżenie nie powinien być widoczny w normalnych warunkach użytkowania.
