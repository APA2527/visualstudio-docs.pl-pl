# <a name="c28215"></a>C28215
Ostrzeżenie C28215: nie można zastosować typefix do parametru, który ma już typefix

 Zastosowanie adnotacji `__typefix` do parametru, który ma już tę adnotację, jest błąd. Adnotacje `__typefix` są używane tylko w kilku szczególnych przypadkach, a to ostrzeżenie nie powinno być widoczne w normalnych warunkach użytkowania.
