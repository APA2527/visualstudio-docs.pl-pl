# <a name="c28215"></a>C28215
Ostrzeżenie C28215: nie można zastosować typefix do parametru, który ma już typefix

 Wystąpił błąd podczas stosowania adnotacji `__typefix` do parametru, który ma już tę adnotację. Adnotacje `__typefix` są używane tylko w kilku szczególnych przypadkach, a to ostrzeżenie nie powinno być widoczne w normalnych warunkach użytkowania.
