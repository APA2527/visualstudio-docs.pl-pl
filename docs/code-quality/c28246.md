# <a name="c28246"></a>C28246
Ostrzeżenie C28246: adnotacja dla funkcji "\<name >"-parametr "\<parametru >" nie jest zgodny z typem parametru  
  
 Operator __deref — została zastosowana do typu wskaźnika z systemem innym niż podczas tworzenia adnotacji.
