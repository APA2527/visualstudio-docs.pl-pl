# <a name="creating-custom-code-analysis-rule-sets"></a>Tworzenie niestandardowych zestawów reguł analizy kodu
Należy zastosować reguł analizy kodu do projektu kodu zarządzanego przez określenie zestawu reguł, który zawiera co najmniej jedną regułę. Zestawy reguł standardowe firmy Microsoft Podaj grupy reguł, które są zorganizowane według funkcji i głębokość. Na przykład podstawowe reguły wskazówek projektowania Microsoft i rozszerzone reguły Microsoft wskazówki dotyczące projektowania zawierają skupić się na użyteczność i łatwości zagadnienia, z dodano nacisk położono na nazewnictwa reguły w zestawie reguł rozszerzone reguły.  
  
 Można również utworzyć niestandardowego zestawu reguł łączącą dwa lub więcej standardowe zestawy reguł lub który określa zasady w co najmniej jeden zbiór standardowych reguł. Można zapisać reguły niestandardowej ustawia plik .ruleset, i określ je w wielu projektach kodu.
