# <a name="arithmetic-overflow-left-shift-count-is-negative-or-greater-than-or-equal-to-the-operand-size-which-is-undefined-behavior"></a>Przepełnienie arytmetyczne: Liczba przesunięcia w lewo jest ujemna lub większa lub równa rozmiarowi operandu, który jest niezdefiniowanym zachowaniem

To ostrzeżenie oznacza, że licznik przesunięć jest ujemny lub większy lub równy liczbie bitów operandu, co powoduje niezdefiniowane zachowanie.
Uwaga: C4293 jest podobnym sprawdzaniem w kompilatorze firmy Microsoft C++ .

## <a name="example"></a>Przykład

```cpp
unsigned __int64 combine(unsigned lo, unsigned hi)
{
  return (hi << 32) | lo; // C26252 here
}
```

Aby poprawić to ostrzeżenie, należy użyć następującego kodu:

```cpp
unsigned __int64 combine(unsigned lo, unsigned hi)
{
  return ((unsigned __int64)hi << 32) | lo; // OK
}
```
