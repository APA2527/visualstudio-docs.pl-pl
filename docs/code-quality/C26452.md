# <a name="arithmetic-overflow-left-shift-count-is-negative-or-greater-than-or-equal-to-the-operand-size-which-is-undefined-behavior"></a>Przepełnienie arytmetyczne: Przesunięcia w lewo jest ujemna lub większy lub równy rozmiarowi operandu, co jest niezdefiniowanym zachowaniem

 To ostrzeżenie wskazuje, licznik przesunięć jest ujemny lub większa niż lub równa liczbie bitów jest operand przesunięte, co spowoduje niezdefiniowane zachowanie.
Uwaga: C4293 jest podobne wyboru w kompilatorze Microsoft C++.

## <a name="example"></a>Przykład

```cpp
unsigned __int64 combine(unsigned lo, unsigned hi)
{
  return (hi << 32) | lo; // C26252 here
}
```
 Aby poprawić to ostrzeżenie, użyj następującego kodu:

```cpp
unsigned __int64 combine(unsigned lo, unsigned hi)
{
  return ((unsigned __int64)hi << 32) | lo; // OK
}
```
