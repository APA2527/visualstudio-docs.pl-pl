# <a name="c28226"></a>C28226
Ostrzeżenie C28226: Adnotacja nie może być PrimOp (w bieżącej deklaracji)

 To ostrzeżenie wskazuje, że nazwana adnotacja jest zadeklarowana jako PrimOp, a także poprzednio zadeklarowana jako normalna adnotacja. Nie powinno to być możliwe, jeśli standardowe makra są używane do adnotacji; to ostrzeżenie nie jest oczekiwane podczas normalnego użytkowania.
