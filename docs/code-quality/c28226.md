# <a name="c28226"></a>C28226
Ostrzeżenie C28226: Adnotacji nie może być PrimOp (w bieżącej deklaracji)  
  
 To ostrzeżenie wskazuje, że jest on zadeklarowany jako PrimOp adnotacji o nazwie i również został wcześniej zadeklarowany jako normalne adnotacji. To nie powinna być możliwa, jeśli są używane standardowe makra dla adnotacji; to ostrzeżenie nie powinien być widoczny w normalnych warunkach użytkowania.
