# <a name="c1253"></a>C1253
Ostrzeżenie C1253: Nie można załadować pliku modelu.

 Narzędzie do analizy kodu raporty tego ostrzeżenia, gdy występuje błąd wewnętrzny w pliku modelu, a nie w kodzie, analizowane.
