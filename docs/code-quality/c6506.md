# <a name="c6506"></a>C6506
Ostrzeżenie C6506: nieprawidłowa adnotacja: \<name > Właściwość może być używana tylko na wartościach typu wskaźnikowego lub tablicowego

 To ostrzeżenie wskazuje, że właściwość jest używana na typu innego niż wskaźnik lub tablica. Dla wszystkich typów danych można dostępu, Tainted i prawidłowe właściwości. Inne właściwości, takie jak ValidBytesConst "," ValidElementsConst "," ElementSize "i" NullTerminted obsługuje wskaźnika, wskaźnik do elementów członkowskich i typy tablic. Aby uzyskać pełną listę właściwości i obsługiwanych typów danych, zobacz [przy użyciu adnotacji SAL w celu zmniejszenia liczby błędów kodu](using-sal-annotations-to-reduce-c-cpp-code-defects.md).

## <a name="example"></a>Przykład
 Poniższy kod generuje to ostrzeżenie:

```cpp
#include<sal.h>
void f(_Out_ char c)
{
    c = 'd';
}
```

 Aby poprawić to ostrzeżenie, użyj wskaźnika lub typu tablicowego, jak pokazano w poniższym przykładowym kodzie:

```cpp
#include<sal.h>
void f(_Out_ char *c)
{
    *c = 'd';
}
```
