# <a name="c26461-use_const_pointer_arguments"></a>C26461 USE_CONST_POINTER_ARGUMENTS:
  Argument wskaźnika "% argument%" dla funkcji "% Function%" może być oznaczony jako wskaźnik do `const`. Zobacz [ C++ podstawowe wytyczne con. 3](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rconst-ref).
