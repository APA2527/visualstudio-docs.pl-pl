# <a name="warning-c26814-use-constexpr-for-constants-whose-value-is-known-at-compile-time-con5"></a>Ostrzeżenie C26814 używać elementu constexpr dla stałych, których wartość jest znana w czasie kompilacji. (CON. 5)
Zmienna const "% variable%" może być obliczana w czasie kompilacji. Rozważ użycie constexpr (CON. 5)

## <a name="example"></a>Przykład

```cpp
const int foo = 1234;  // C26814 reported here.
constexpr int getMagicNumber()
{
    return 42;
}

void bar()
{
    const int myval = 3; // C26814 reported here
    const int magicNumber = getMagicNumber(); // C26814 reported here.
}
```
