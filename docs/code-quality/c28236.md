# <a name="c28236"></a>C28236
Ostrzeżenie C28236: nie można użyć adnotacji w wyrażeniu

 To ostrzeżenie wskazuje, że podjęto próbę użycia funkcji zadeklarowanej jako adnotacja w kontekście wyrażenia. Nie powinno to być możliwe, jeśli standardowe makra są używane do adnotacji; to ostrzeżenie nie jest oczekiwane podczas typowego użycia.
