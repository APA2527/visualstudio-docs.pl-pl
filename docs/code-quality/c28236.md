# <a name="c28236"></a>C28236
Ostrzeżenie C28236: nie można użyć adnotacji w wyrażeniu

 To ostrzeżenie oznacza, że nastąpiła próba funkcja zadeklarowana jako adnotacji w kontekście wyrażenia. To nie powinna być możliwa, jeśli są używane standardowe makra dla adnotacji; to ostrzeżenie nie powinien być widoczny w typowy sposób użycia.
