# <a name="c28244"></a>C28244
Ostrzeżenie C28244: adnotacja dla funkcji jest niemożliwy do przeanalizowania parametr/zewnętrzna adnotacja  
  
 Obecnie nie powinna być możliwa, jeśli są używane standardowe makra dla adnotacji; to ostrzeżenie nie powinien być widoczny w typowy sposób użycia.
