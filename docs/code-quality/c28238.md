# <a name="c28238"></a>C28238
Ostrzeżenie C28238: adnotacji w parametrze ma więcej niż jedną wartość, stringValue i longValue. Użyj zapisu paramn = xxx

 Wystąpił wewnętrzny błąd w pliku modelu PREfast. To ostrzeżenie nie powinien wystąpić w typowy sposób użycia.
