# <a name="c28238"></a>C28238
Ostrzeżenie C28238: Adnotacja dla parametru ma więcej niż jedną wartość, stringValue i longValue. Użyj paramn = xxx

 Wystąpił błąd wewnętrzny w pliku modelu PREfast. To ostrzeżenie nie powinien wystąpić w typowym zastosowaniem.
