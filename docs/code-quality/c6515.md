# <a name="c6515"></a>C6515
Ostrzeżenie C6515 - nieprawidłowa adnotacja: \<name > Właściwość może być używana tylko dla wartości typu wskaźnik

 To ostrzeżenie wskazuje, że właściwość do użycia na wskaźnikach została zastosowana do typu innego niż wskaźnik. Aby uzyskać listę właściwości adnotacji, zobacz [przy użyciu adnotacji SAL w celu zmniejszenia liczby błędów kodu](using-sal-annotations-to-reduce-c-cpp-code-defects.md).

## <a name="example"></a>Przykład
 Poniższy kod generuje to ostrzeżenie:

```cpp
#include <sal.h>

void f(_Readable_bytes_(c) char pc,  size_t c)
{
    // code ...
}
```

 Aby poprawić to ostrzeżenie, użyj następującego kodu:

```
#include <sal.h>

void f(_Readable_bytes_(c) char * pc,  size_t c)
{
    // code ...
}
```
