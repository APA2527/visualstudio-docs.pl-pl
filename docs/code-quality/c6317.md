# <a name="c6317"></a>C6317
Ostrzeżenie C6317: nieprawidłowy operator: logiczny-not (!) nie jest zamienny z uzupełnieniem (~)

 To ostrzeżenie wskazuje, że wartość logiczna nie (`!`) jest stosowana do stałej, która prawdopodobnie jest flagą bitową. Wynik logiczny nie jest wartością logiczną; zastosowanie operatora bitowe-i (`&`) do wartości logicznej jest nieprawidłowe. Aby przerzucić flagi, użyj operatora z uzupełnieniem (`~`).

## <a name="example"></a>Przykład
 Poniższy kod generuje to ostrzeżenie:

```
#define FLAGS   0x4004

void f(int i)
{
  if (i & !FLAGS) // warning
  {
    // code
  }
}
```

 Aby poprawić to ostrzeżenie, należy użyć następującego kodu:

```
#define FLAGS   0x4004

void f(int i)
{
  if (i & ~FLAGS)
  {
    // code
  }
}
```
