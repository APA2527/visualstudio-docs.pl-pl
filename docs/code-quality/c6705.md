# <a name="c6705"></a>C6705
Ostrzeżenie C6705: błąd adnotacji, oczekiwana < expected_number > argumenty dla adnotacji \<parametru > znaleziono < actual_number >.  
  
 To ostrzeżenie jest zgłaszany, gdy występuje błąd w adnotacjach.
