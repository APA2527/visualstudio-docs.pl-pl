# <a name="c6705"></a>C6705
Ostrzeżenie C6705: Błąd adnotacji oczekiwał < expected_number > argumentów dla adnotacji \<parametr > znaleziono < actual_number >.

 To ostrzeżenie jest zgłaszane, gdy występuje błąd w adnotacjach.
