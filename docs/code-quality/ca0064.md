# <a name="ca0064"></a>CA0064

Analiza nie została wykonana, ponieważ określonego zestawu reguł nie zawiera żadnych reguł programu FxCop.

To ostrzeżenie może wystąpić w następujących sytuacjach:

- Jeśli wystąpią to ostrzeżenie w połączeniu z ostrzeżeniami CA0063 wystąpił problem podczas ładowania pliku zestaw reguł. Aby uzyskać więcej informacji, zobacz [CA0063](ca0063.md) dokumentacji.

- W razie wystąpienia tego błędu w połączeniu z ostrzeżeniami CA0062, prawdopodobnie jaki analiza kodu nie można znaleźć zestawu lub zestawów zawierających z regułami określonymi przez zestaw reguł. Aby uzyskać więcej informacji, zobacz [CA0062](ca0062.md) dokumentacji.

- W przeciwnym razie to ostrzeżenie zazwyczaj występuje, gdy Twoje zestaw reguł jest pusta lub wszystkie reguły włączone w zestawie reguł podrzędne zostały wyłączone. Za pomocą programu Visual Studio [edytorze zestawu reguł](../code-quality/working-in-the-code-analysis-rule-set-editor.md) umożliwiające niektórych reguł w Twojej zestaw reguł.
