# <a name="ca0064"></a>CA0064

Analiza nie została wykonana, ponieważ określonego zestawu reguł nie zawiera żadnych reguł programu FxCop.

To ostrzeżenie może wystąpić w następujących sytuacjach:

- Jeśli wystąpi tego ostrzeżenia w połączeniu z ostrzeżeniami CA0063 wystąpił problem podczas ładowania pliku zestawu reguł. Aby uzyskać więcej informacji, zobacz [CA0063](ca0063.md) artykułu.

- Jeśli wystąpi ten błąd wystąpił w połączeniu z ostrzeżeniami CA0062 prawdopodobnie analizy kodu nie można odnaleźć zestawu lub zestawów zawierających zasady określone przez zestaw reguł. Aby uzyskać więcej informacji, zobacz [CA0062](ca0062.md) artykułu.

- W przeciwnym razie to ostrzeżenie, zazwyczaj występuje, gdy zestaw reguł jest pusta lub wszystkie reguły włączone w zestawie reguł podrzędne są wyłączone. Za pomocą programu Visual Studio [edytorze zestawu reguł](../code-quality/working-in-the-code-analysis-rule-set-editor.md) umożliwiające niektóre reguły w zestawie reguł.
