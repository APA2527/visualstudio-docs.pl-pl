# <a name="ca0504"></a>CA0504
Katalog reguł zostało zignorowane, ponieważ nie istnieje

 Ostrzeżenie CA504 jest generowany, gdy katalogu określonego w **CodeAnalysisRuleDirectories** nie można odnaleźć właściwości w pliku konfiguracji projektu lub kompilacji. Aby rozwiązać ten problem, należy wprowadzić prawidłową ścieżką katalogu.
