# <a name="ca0504"></a>CA0504
Katalog reguł został zignorowany, ponieważ nie istnieje  
  
 Ostrzeżenie CA504 jest generowany, gdy katalog określony w **CodeAnalysisRuleDirectories** nie można odnaleźć właściwości w pliku konfiguracji projektu lub kompilacji. Aby rozwiązać ten problem, wprowadź prawidłową ścieżkę katalogu.
