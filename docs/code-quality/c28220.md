# <a name="c28220"></a>C28220
Ostrzeżenie C28220: wyrażenie całkowite oczekiwany dla adnotacji, parametr w adnotacji  
  
 To ostrzeżenie wskazuje, że parametr do adnotacji powinien być wyrażeniem liczby całkowitej i napotkano innego typu. Zwykle oznacza to makro niepoprawnie zakodowany adnotacji.
