# <a name="warning-c26253-arithmetic-overflow-left-shift-of-a-negative-signed-number-is-undefined-behavior"></a>Ostrzeżenie C26253: Przepełnienie arytmetyczne: Przesunięcie w lewo ujemna liczba ze znakiem jest niezdefiniowanym zachowaniem

To ostrzeżenie wskazuje, że są pozostawiane, przesunięcie podpisem całkowitą wartość ujemną, który to zły pomysł i wywoła zachowanie definiowany przez implementację.

## <a name="example"></a>Przykład

```cpp
void leftshift(int shiftCount)
{
  const auto result = -1 << shiftCount;  // C26453 reported here

  // code
}
```

 Aby poprawić to ostrzeżenie, użyj następującego kodu:

```cpp
void leftshift(int shiftCount)
{
  const auto result = ((unsigned)-1) << shiftCount; // OK

  // code
}
```
