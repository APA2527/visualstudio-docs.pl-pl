# <a name="warning-c26453-arithmetic-overflow-left-shift-of-a-negative-signed-number-is-undefined-behavior"></a>Ostrzeżenie C26453: Przepełnienie arytmetyczne: Przesunięcie w lewo z ujemną liczbą cyfr jest zachowaniem niezdefiniowanym

To ostrzeżenie wskazuje, że pozostawiamy przesuniętą ujemną, nieujemną wartość całkowitą, która jest niewłaściwym pomysłem i wyzwalaczem zachowanie zdefiniowane w implementacji.

## <a name="example"></a>Przykład

```cpp
void leftshift(int shiftCount)
{
  const auto result = -1 << shiftCount;  // C26453 reported here

  // code
}
```

Aby poprawić to ostrzeżenie, należy użyć następującego kodu:

```cpp
void leftshift(int shiftCount)
{
  const auto result = ((unsigned)-1) << shiftCount; // OK

  // code
}
```
