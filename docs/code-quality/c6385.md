# <a name="c6385"></a>C6385
Ostrzeżenie C6385: nieprawidłowe dane: uzyskiwanie dostępu do \<buforu nazwa >, czytelny rozmiar wynosi \<size1 > bajtów, ale \<size2 > bajtów może być odczytanych: Linie: x, y

 To ostrzeżenie wskazuje, że czytelny zakres określony bufor może być mniejszy niż indeks umożliwia odczytanie z niego. Próby odczytu danych poza prawidłowym zakresem prowadzi do przepełnienia buforu.

## <a name="example"></a>Przykład
 Poniższy kod generuje to ostrzeżenie:

```
void f(int i)
{
   char a[20];
   char j;
   if (i <= 20)
   {
      j = a[i];
   }
}
```

 Aby poprawić to ostrzeżenie, użyj następującego kodu:

```
void f(int i)
{
   char a[20];
   char j;
   if (i < 20)
   {
      j = a[i];
   }
}
```
