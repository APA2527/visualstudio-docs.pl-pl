# <a name="c6385"></a>C6385
Ostrzeżenie C6385: nieprawidłowe dane: uzyskiwanie dostępu do > nazwy \<buffer, rozmiar możliwy do odczytu to \<size1 > bajty, ale \<size2 > bajty: Linie: x, y

 To ostrzeżenie wskazuje, że czytelny zakres określonego buforu może być mniejszy niż indeks użyty do odczytu. Próby odczytu danych poza prawidłowym zakresem powodują przepełnienie buforu.

## <a name="example"></a>Przykład
 Poniższy kod generuje to ostrzeżenie:

```
void f(int i)
{
   char a[20];
   char j;
   if (i <= 20)
   {
      j = a[i];
   }
}
```

 Aby poprawić to ostrzeżenie, należy użyć następującego kodu:

```
void f(int i)
{
   char a[20];
   char j;
   if (i < 20)
   {
      j = a[i];
   }
}
```
