# <a name="c28283"></a>C28283
Ostrzeżenie C28283: dla symbolu, określona specyfikacja rozmiaru nie jest jeszcze obsługiwana  
  
 Ostrzeżenie wskazuje, że adnotacji Specyfikacja rozmiaru "wartownik" odebrała wartość inną niż zero. Zasadniczo obiekt wywołujący próbował powiedzieć, że ciąg kończy się znakiem niezerowym binarnego.
