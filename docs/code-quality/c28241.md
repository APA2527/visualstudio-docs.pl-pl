# <a name="c28241"></a>C28241
Ostrzeżenie C28241: adnotacja dla funkcji w parametrze nie została rozpoznana

 Użyto nierozpoznanej nazwy adnotacji. Nie powinno to być możliwe, jeśli standardowe makra są używane do adnotacji; to ostrzeżenie nie jest oczekiwane podczas typowego użycia.
