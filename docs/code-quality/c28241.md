# <a name="c28241"></a>C28241
Ostrzeżenie C28241: Adnotacja dla funkcji na parametrze nie została rozpoznana.

 Użyto nazwy nierozpoznany adnotacji. To nie powinna być może wystąpić, jeśli są używane standardowe makra dla adnotacji; to ostrzeżenie nie powinien być widoczny w typowym zastosowaniem.
