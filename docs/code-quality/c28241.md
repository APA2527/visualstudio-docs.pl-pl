# <a name="c28241"></a>C28241
Ostrzeżenie C28241: adnotacja dla funkcji na parametr nie został rozpoznany.

 Nazwa adnotacji nierozpoznany została użyta. To nie powinna być możliwa, jeśli są używane standardowe makra dla adnotacji; to ostrzeżenie nie powinien być widoczny w typowy sposób użycia.
