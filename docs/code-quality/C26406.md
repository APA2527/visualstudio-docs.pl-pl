# <a name="c26406--dontassignrawtoowner"></a>C26406  DONT_ASSIGN_RAW_TO_OWNER
Właściciele są inicjowane z przydziałów lub innych właścicieli. Przypisywanie wartości z surowy wskaźnik do wskaźnika właściciel nie jest dozwolone. Surowe wskaźniki nie gwarantuje przenoszenie własności; jest nadal może być pierwotnego właściciela przechowuje zasobu, która będzie podejmować próby jej wersji. Należy zauważyć, że przypisanie wartości od właściciela do surowy wskaźnik jest w dobrym stanie; surowe wskaźniki są prawidłowe klientom dostęp do zasobów, ale nie do zarządzania nimi.

## <a name="example-1--using-address-of-object"></a>Przykład 1:  Przy użyciu adresu obiektu
```cpp
gsl::owner<Socket*> socket = defaultSocket ? &defaultSocket : new Socket(); // C26406
```
