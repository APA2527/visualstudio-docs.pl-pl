# <a name="how-to-enable-and-disable-automatic-code-analysis-for-managed-code"></a>Porady: Włączanie i wyłączanie automatycznej analizy kodu dla zarządzanego kodu

Można skonfigurować analizy kodu do uruchomienia po każdej kompilacji projektu zarządzanego kodu. Można ustawić inny kod analizy właściwości dla każdej konfiguracji kompilacji.

## <a name="to-enable-or-disable-automatic-code-analysis"></a>Aby włączyć lub wyłączyć automatycznej analizy kodu

1. W **Eksploratora rozwiązań**, kliknij prawym przyciskiem myszy projekt, a następnie wybierz pozycję **właściwości**.

1. W oknie dialogowym właściwości projektu, wybierz opcję **analizy kodu**.

1. Określ typ kompilacji w **konfiguracji** i platformy docelowej w **platformy**.

1. Aby włączyć lub wyłączyć automatycznej analizy kodu, zaznacz lub wyczyść **Włącz analizę kodu podczas kompilacji** pole wyboru.
