# <a name="measuring-complexity-and-maintainability-of-managed-code"></a>Mierzenie złożoności i łatwości konserwacji zarządzanego kodu
Zwiększenie stopnia złożoności nowoczesne aplikacje zwiększa także trudności dokonywania kod, niezawodny i łatwy w obsłudze. W ostatnich latach wiele miar oprogramowania, znany jako metryki kodu zostały opracowane który ułatwiają deweloperom zrozumieć, gdy ich kod musi przeróbek lub zwiększona testowania.  
  
 Deweloperzy mogą używać [!INCLUDE[vsprvsts](../code-quality/includes/vsprvsts_md.md)] do generowania kodu danych metryki pomiaru złożoności i łatwości konserwacji ich kodu zarządzanego. Można wygenerować danych metryki kodu dla całego rozwiązania lub projektu jednego.  
  
 W tej sekcji opisano sposób generowania i użytkowania metryki kodu w [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)].
