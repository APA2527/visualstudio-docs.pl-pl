# <a name="ca0503"></a>CA0503
Właściwość jest przestarzała. Użyj właściwości zastępującej

 CA0503 ostrzeżenie jest generowane, gdy właściwość analizy kodu przestarzałe zawiera wartość w pliku konfiguracji kompilacji lub projektu. Aby rozwiązać tego ostrzeżenia, należy użyć określonej właściwości, która zastępuje go.
