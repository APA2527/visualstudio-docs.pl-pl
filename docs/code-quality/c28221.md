# <a name="c28221"></a>C28221
Ostrzeżenie C28221: Oczekiwano wyrażenia ciągu dla parametrów w adnotacji  
  
 To ostrzeżenie wskazuje, że parametr do adnotacji powinien być ciągiem i napotkano innego typu. To zwykle wskazuje makro niepoprawnie zakodowany adnotacji i nie powinien być widoczny w normalnych warunkach użytkowania.
