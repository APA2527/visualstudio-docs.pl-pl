# <a name="ca0057"></a>CA0057
Nie można zamapować naruszeń w bieżącym zestawie elementów docelowych i reguł

 Ten błąd występuje, gdy FxCopCmd.exe została wywołana ręcznie z pliku projektu lub / zaimportowane raportu XML, który posiada naruszenia, których nie można zamapować do bieżącego zestawu elementów docelowych i reguł.

 Występuje tylko wtedy, gdy analiza jest wykonywane przy użyciu wiersza polecenia wersji narzędzia.
