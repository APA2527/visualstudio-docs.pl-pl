# <a name="c28210"></a>C28210
Ostrzeżenie 28210: adnotacje dla kontekstu `_On_failure_` nie mogą być w jawnym kontekście wstępnym

 Adnotacje `_On_failure_` muszą być jawnie lub niejawnie wskazywane w kontekście `__post`, który ma zostać zastosowany po zwracaniu funkcji.  Użyj `_drv_out`, aby to zapewnić.
