# <a name="c28208"></a>C28208
Ostrzeżenie C28208: Funkcja \<function > była wcześniej zdefiniowana z inną listą parametrów w \<File > (@no__t 2line >). Niektóre narzędzia analityczne spowodują nieprawidłowe wyniki

 To ostrzeżenie jest zgłaszane, gdy znana definicja funkcji nie jest zgodna z innym wystąpieniem.
