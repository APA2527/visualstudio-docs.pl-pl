# <a name="c28208"></a>C28208
Ostrzeżenie C28208: Funkcja \<funkcja > została wcześniej zdefiniowana z inną listą parametrów w \<Plik > (\<wiersza >). Niektóre narzędzia analizy dadzą niepoprawne wyniki

 To ostrzeżenie jest zgłaszane, gdy definicja znanych funkcji nie pasuje inne wystąpienie.
