# <a name="c28208"></a>C28208
Ostrzeżenie C28208: funkcja \<funkcja > została wcześniej zdefiniowana z inną listą parametrów w \<pliku > (\<wiersz >). Niektóre narzędzia do analizy będą dawać błędne wyniki

 To ostrzeżenie jest zgłaszany, gdy funkcja znane definicji nie pasuje inne wystąpienie.
