# <a name="c28227"></a>C28227
Ostrzeżenie C28227: Adnotacji nie może być PrimOp (patrz Poprzednia deklaracja)

 To ostrzeżenie wskazuje, że jest on zadeklarowany jako zwykłej adnotacji adnotacji o nazwie i również został wcześniej zadeklarowany jako PrimOp. To nie powinna być możliwa, jeśli są używane standardowe makra dla adnotacji; to ostrzeżenie nie powinien być widoczny w typowy sposób użycia.
