# <a name="how-to-view-managed-code-defects"></a>Porady: wyświetlanie defektów kodu zarządzanego
Analiza kodu w oknie analizy kodu są zgłaszane błędy, ostrzeżenia i komunikaty. Okno Analiza kodu jest wyświetlany automatycznie podczas uruchamiania analizy kodu na projekt lub rozwiązanie.  
  
 W oknie analizy kodu można:  
  
-   **Wybierz tytuł ostrzeżenie.**  
  
     Dodatkowe informacje na temat tego ostrzeżenia, zostanie wyświetlony w oknie analizy kodu. Wiersz w pliku źródłowym, która wywołała to ostrzeżenie jest wyświetlane i podświetlane w edytorze kodu źródłowego.  
  
-   **Wybierz identyfikator ostrzeżenia.** Ostrzeżenie nazwy są poprzedzone **urzędu certyfikacji** i obejmują czterech cyfr. Na przykład CA1234.  
  
     Temat pomocy biblioteki MSDN Library ostrzeżenia jest wyświetlany. Temat Pomocy zawiera szczegółowe informacje o przyczynie ostrzeżenia i przykłady, które przedstawiają sposób ostrzeżenie problem występuje i zostanie rozwiązany.
