# <a name="c28291"></a>C28291
Ostrzeżenie C28291: Post null/notnull na poziomie deref 0 jest bez znaczenia dla funkcji \<x > w param \<numer >

 Narzędzie do analizy kodu raporty tego ostrzeżenia, gdy jest określony warunek post, wyłuskania parametru poziomie od zera, mieć właściwość o wartości null/inną niż null. Ten błąd występuje, ponieważ wartość o wyłuskania nie można zmienić poziomu zero.
