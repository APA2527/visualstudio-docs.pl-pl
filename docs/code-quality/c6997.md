# <a name="c6997"></a>C6997
Ostrzeżenie 6997: adnotacje w tej lokalizacji są bezsensowne i zostaną zignorowane.

 Nie można zastosować adnotacji do `extern "C" {...}`. Zastosuj adnotacje do określonego obiektu.
