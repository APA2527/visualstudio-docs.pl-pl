# <a name="c28351"></a>C28351
Ostrzeżenie C28351: adnotacja \<adnotacji > w tym artykule opisano, gdzie wartość dynamiczna (zmienna) nie można użyć w tym stanie.  
  
 To ostrzeżenie jest zgłaszany, gdy adnotacji jest stosowana, gdy jest sprawdzana typu C/C++.  
  
 Wyrażenie w `_When_` powinna być stała. `_When_` Jest ignorowana.
