# <a name="c28351"></a>C28351
Ostrzeżenie C28351: Adnotacja \<annotation > opisuje, gdzie wartość dynamiczna (zmienna) nie może być użyta w warunku.

 To ostrzeżenie jest zgłaszane, gdy jest stosowana Adnotacja, w którejC++ jest przeprowadzana inspekcja C/typ.

 Wyrażenie w `_When_` powinno być oceniane jako stała. @No__t-0 jest ignorowana.
