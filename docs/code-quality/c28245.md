# <a name="c28245"></a>C28245
Ostrzeżenie C28245: adnotacja dla funkcji oznacza "this" w innej niż — funkcja członkowska

 Wystąpił wewnętrzny błąd w pliku modelu PREfast. To ostrzeżenie nie powinien wystąpić w typowy sposób użycia.
