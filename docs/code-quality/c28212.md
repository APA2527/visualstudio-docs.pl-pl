# <a name="c28212"></a>C28212
Ostrzeżenie C28212: oczekiwano wyrażenia wskaźnika dla adnotacji

 To ostrzeżenie wskazuje, że parametr numerowany do adnotacji (nie jest adnotacją funkcji) powinien być wskaźnikiem lub typem tablicy, ale napotkano jakiś inny typ. Należy poprawić adnotację.
