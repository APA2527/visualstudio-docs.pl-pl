# <a name="ca2328-ensure-that-jsonserializersettings-are-secure"></a>CA2328: Upewnij się, że ustawienia JsonSerializerSettings zostały zabezpieczone

|||
|-|-|
|TypeName|DoNotUseInsecureSettingsForJsonNet|
|CheckId|CA2328|
|Kategoria|Microsoft.Security|
|Zmiana podziału|Nieprzerwanie|

## <a name="cause"></a>Przyczyna

Ta zasada jest wyzwalana, gdy dla wystąpienia [Newtonsoft. JSON. JsonSerializerSettings](https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_JsonSerializerSettings.htm) spełnione są obie poniższe warunki:
- Właściwość [TypeNameHandling](https://www.newtonsoft.com/json/help/html/P_Newtonsoft_Json_JsonSerializerSettings_TypeNameHandling.htm) jest wartością inną niż `None`.
- Właściwość [pomocą elementu SerializationBinder](https://www.newtonsoft.com/json/help/html/P_Newtonsoft_Json_JsonSerializerSettings_SerializationBinder.htm) ma wartość null.

Gdy wystąpienie [JsonSerializerSettings](https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_JsonSerializerSettings.htm) jest używane w następujących okolicznościach:
- Zainicjowany jako pole klasy lub właściwość.
- Jako wartość zwracana metody.
- Używane jako ustawienia w metodzie [Newtonsoft. JSON. JsonConvert](https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_JsonConvert.htm) .
- Używane jako ustawienia w metodzie [Newtonsoft. JSON. JsonSerializer](https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_JsonSerializer.htm) .

## <a name="rule-description"></a>Opis reguły

[!INCLUDE[insecure-deserializers-description](includes/insecure-deserializers-description-md.md)]

Ta reguła umożliwia znalezienie wystąpień [Newtonsoft. JSON. JsonSerializerSettings](https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_JsonSerializerSettings.htm) , które mogą być skonfigurowane do deserializacji typów określonych na podstawie danych wejściowych, ale nie można ich skonfigurować do ograniczania typów deserializowanych za pomocą [ Newtonsoft. JSON. Serialization. ISerializationBinder](https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_Serialization_ISerializationBinder.htm). Jeśli chcesz nie zezwalać na deserializacji typów określonych na podstawie danych wejściowych, wyłącz reguły [CA2327](ca2327.md), CA2328, [CA2329](ca2329.md)i [CA2330](ca2330.md), a następnie Włącz regułę [CA2326](ca2326.md) .

## <a name="how-to-fix-violations"></a>Jak naprawić naruszenia

- Jeśli [](https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_TypeNameHandling.htm)to możliwe `None` , użyj wartości TypeNameHandling.
- Przekształć dane serializowane. Po serializacji należy kryptograficznie podpisywać serializowane dane. Przed deserializacji Sprawdź poprawność podpisu kryptograficznego. Ochrona klucza kryptograficznego przed ujawnieniem i projektowaniem na potrzeby rotacji kluczy.
- Ogranicz typy deserializowane. Zaimplementuj niestandardowy [Newtonsoft. JSON. Serialization. ISerializationBinder](https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_Serialization_ISerializationBinder.htm). Przed deserializacji z Json.NET upewnij się, że niestandardowe [ISerializationBinder](https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_Serialization_ISerializationBinder.htm) są określone we właściwości [Newtonsoft. JSON. JsonSerializerSettings. pomocą elementu SerializationBinder](https://www.newtonsoft.com/json/help/html/P_Newtonsoft_Json_JsonSerializerSettings_SerializationBinder.htm) . W zastąpionej metodzie [Newtonsoft. JSON. Serialization. ISerializationBinder. BindToType](https://www.newtonsoft.com/json/help/html/M_Newtonsoft_Json_Serialization_ISerializationBinder_BindToType.htm) , jeśli typ jest nieoczekiwany `null` , zwróć lub Zgłoś wyjątek, aby zatrzymać deserializacji.

## <a name="when-to-suppress-warnings"></a>Kiedy pominąć ostrzeżenia

Można bezpiecznie pominąć ostrzeżenie z tej reguły, jeśli:
- Wiadomo, że dane wejściowe są zaufane. Należy wziąć pod uwagę, że granica zaufania aplikacji i przepływy danych mogą ulec zmianie z upływem czasu.
- Wykonano jedną z tych środków ostrożności w zakresie [rozwiązywania naruszeń](#how-to-fix-violations).
- Wiadomo, że właściwość [Newtonsoft. JSON. JsonSerializerSettings. pomocą elementu SerializationBinder](https://www.newtonsoft.com/json/help/html/P_Newtonsoft_Json_JsonSerializerSettings_SerializationBinder.htm) jest zawsze ustawiana, gdy właściwość [TypeNameHandling](https://www.newtonsoft.com/json/help/html/P_Newtonsoft_Json_JsonSerializerSettings_TypeNameHandling.htm) ma wartość inną niż `None`.

## <a name="pseudo-code-examples"></a>Przykłady pseudo kodu

### <a name="violation"></a>Krocz

```csharp
using Newtonsoft.Json;

public class BookRecord
{
    public string Title { get; set; }
    public string Author { get; set; }
    public object Location { get; set; }
}

public abstract class Location
{
    public string StoreId { get; set; }
}

public class AisleLocation : Location
{
    public char Aisle { get; set; }
    public byte Shelf { get; set; }
}

public class WarehouseLocation : Location
{
    public string Bay { get; set; }
    public byte Shelf { get; set; }
}

public class ExampleClass
{
    public ISerializationBinder SerializationBinder { get; set; }

    public BookRecord DeserializeBookRecord(string s)
    {
        JsonSerializerSettings settings = new JsonSerializerSettings();
        settings.TypeNameHandling = TypeNameHandling.Auto;
        settings.SerializationBinder = this.SerializationBinder;
        return JsonConvert.DeserializeObject<BookRecord>(s, settings);    // CA2328 -- settings might be null
    }
}
```

### <a name="solution"></a>Rozwiązanie

```csharp
using System;
using Newtonsoft.Json;
using Newtonsoft.Json.Serialization;

public class BookRecordSerializationBinder : ISerializationBinder
{
    // To maintain backwards compatibility with serialized data before using an ISerializationBinder.
    private static readonly DefaultSerializationBinder Binder = new DefaultSerializationBinder();

    public void BindToName(Type serializedType, out string assemblyName, out string typeName)
    {
        Binder.BindToName(serializedType, out assemblyName, out typeName);
    }

    public Type BindToType(string assemblyName, string typeName)
    {
        // If the type isn't expected, then stop deserialization.
        if (typeName != "BookRecord" && typeName != "AisleLocation" && typeName != "WarehouseLocation")
        {
            return null;
        }

        return Binder.BindToType(assemblyName, typeName);
    }
}

public class BookRecord
{
    public string Title { get; set; }
    public string Author { get; set; }
    public object Location { get; set; }
}

public abstract class Location
{
    public string StoreId { get; set; }
}

public class AisleLocation : Location
{
    public char Aisle { get; set; }
    public byte Shelf { get; set; }
}

public class WarehouseLocation : Location
{
    public string Bay { get; set; }
    public byte Shelf { get; set; }
}

public class ExampleClass
{
    public BookRecord DeserializeBookRecord(string s)
    {
        JsonSerializerSettings settings = new JsonSerializerSettings();
        settings.TypeNameHandling = TypeNameHandling.Auto;
        settings.SerializationBinder = new BookRecordSerializationBinder();
        return JsonConvert.DeserializeObject<BookRecord>(s, settings);
    }
}
```

```vb
Imports System
Imports Newtonsoft.Json
Imports Newtonsoft.Json.Serialization

Public Class BookRecordSerializationBinder
    Implements ISerializationBinder

    ' To maintain backwards compatibility with serialized data before using an ISerializationBinder.
    Private Shared ReadOnly Property Binder As New DefaultSerializationBinder()

    Public Sub BindToName(serializedType As Type, ByRef assemblyName As String, ByRef typeName As String) Implements ISerializationBinder.BindToName
        Binder.BindToName(serializedType, assemblyName, typeName)
    End Sub

    Public Function BindToType(assemblyName As String, typeName As String) As Type Implements ISerializationBinder.BindToType
        ' If the type isn't expected, then stop deserialization.
        If typeName <> "BookRecord" AndAlso typeName <> "AisleLocation" AndAlso typeName <> "WarehouseLocation" Then
            Return Nothing
        End If

        Return Binder.BindToType(assemblyName, typeName)
    End Function
End Class

End Class

Public MustInherit Class Location

End Class

Public Class AisleLocation
    Inherits Location

End Class

Public Class WarehouseLocation
    Inherits Location

    Public Property Bay As String

End Class
```
