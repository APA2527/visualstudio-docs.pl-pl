# <a name="c28022"></a>C28022
Ostrzeżenie C28022: klasy funkcji \<classlist1 > w tej funkcji są niezgodne z klasami funkcji \<classlist2 > dla elementu typedef użytego do jego zdefiniowania.

 To ostrzeżenie jest zgłaszane w przypadku błędu w adnotacjach. Zarówno element typedef, jak i sama funkcja mają adnotacje `_Function_class_`, ale nie pasują do siebie. Jeśli oba są używane, muszą być zgodne.
