# <a name="c28022"></a>C28022
Ostrzeżenie C28022: klasach funkcji \<classlist1 > w tej funkcji nie są zgodne z klasami funkcji \<classlist2 > w definicji typu użytej definiować go.  
  
 To ostrzeżenie jest zgłaszany, gdy występuje błąd w adnotacjach. Zarówno typedef i ta funkcja ma `_Function_class_` adnotacje, ale nie są zgodne. Jeśli używane są oba muszą one odpowiadać.
