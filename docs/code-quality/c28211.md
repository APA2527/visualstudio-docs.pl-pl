# <a name="c28211"></a>C28211
Ostrzeżenie C28211: nazwy kontekstu statycznego dla SAL_context  
  
 To ostrzeżenie oznacza to, że argument operacji do `_Static_context_` adnotacji musi być nazwą kontekstu zdefiniowane przez narzędzie. To ostrzeżenie nigdy nie powinna wystąpić w normalnych warunkach użytkowania.
