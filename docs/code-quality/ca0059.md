# <a name="ca0059"></a>CA0059
Błąd przełącznika wiersza polecenia.  
  
 Ten błąd występuje, jeśli przełącznik wiersza polecenia wymagane brakuje, na przykład  
  
 / out lub nie został określony przełącznik/Console. Ten błąd występuje także w przypadku nieprawidłowy przełącznik, na przykład value nieprawidłowy limit czasu.
