# <a name="c28160"></a>C28160
Ostrzeżenie C28160: błąd adnotacji  
  
 To ostrzeżenie jest zgłaszany, gdy `__drv_error` napotkał adnotacji. Ta adnotacja jest używany do praktyk kodowania, należy ustalić, które mogą być używane z flagą `__drv_when` adnotacji, aby wskazać określone kombinacje parametrów.
