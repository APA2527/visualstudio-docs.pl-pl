# <a name="c28160"></a>C28160
Ostrzeżenie C28160: błąd adnotacji

 To ostrzeżenie jest zgłaszane, gdy napotkano adnotację `__drv_error`. Ta adnotacja służy do flagi, które powinny być naprawione, i może być używana z adnotacją `__drv_when` w celu wskazania określonych kombinacji parametrów.
