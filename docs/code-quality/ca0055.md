# <a name="ca0055"></a>CA0055
Nie można załadować pliku:\<ścieżka >

 Narzędzie do analizy kodu nie może załadować pliku docelowego określono dla analizy. Dzieje się tak, jeśli plik nie zostanie znaleziony, lub nastąpiła odmowa dostępu do pliku. Zwykle dzieje się tak Jeśli użytkownik wybrał opcję zastąpienia właściwości zestawu danych wejściowych w programie Visual Studio lub fxcopcmd.exe wywołać jej ręcznie.
