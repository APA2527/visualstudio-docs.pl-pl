# <a name="ca0055"></a>CA0055
Nie można załadować pliku:\<ścieżkę >  
  
 Narzędzie do analizy kodu nie może załadować pliku docelowego określono do analizy. Dzieje się tak, jeśli plik nie został znaleziony lub nastąpiła odmowa dostępu do pliku. Zazwyczaj dzieje się tak, jeśli użytkownik wybrał opcję zastępować właściwość zestawu wejściowego w [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)], lub wywołać fxcopcmd.exe ręcznie.
