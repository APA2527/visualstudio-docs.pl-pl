# <a name="c28304"></a>C28304
Ostrzeżenie C28304: w C++ przypadku parametru odwołania < parameter_name >, znaleziono nieprawidłowo umieszczony operator `_Notref_` do @no__t > 2token.

 Operator `_Notref_` powinien być używany tylko w szczególnych okolicznościach obejmujących C++ parametry odwołania i tylko w makrach dostarczonych przez system. Bezpośrednio następuje operator `_Deref_` lub Adnotacja funkcjonalna.
