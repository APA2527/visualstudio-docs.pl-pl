# <a name="c28304"></a>C28304
Ostrzeżenie C28304: Aby uzyskać C++ parametru odwołania < parameter_name >, nieprawidłowo umieszczony `_Notref_` znaleziono operator zastosowany do \<token >.

 `_Notref_` Operator można stosować tylko w szczególnych okolicznościach obejmujące parametrów odwołania C++ i tylko w makrach dostarczane przez system. Jego musi bezpośrednio poprzedzać `_Deref_` operatora lub funkcjonalności adnotacji.
