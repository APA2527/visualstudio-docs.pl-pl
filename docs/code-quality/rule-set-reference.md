# <a name="code-analysis-rule-set-reference"></a>Odwołanie zestawu reguł analizy kodu

Po skonfigurowaniu statycznej analizy kodu dla kodu zarządzanego, projekty w programie Visual Studio możesz korzystać z listy wbudowanych *zestawów reguł*. Można użyć jednego z tych zestawów reguł wbudowanych lub możesz [Dostosuj zestaw reguł](../code-quality/how-to-create-a-custom-rule-set.md) zgodnie z wymaganiami projektu.

W tematach w tej sekcji opisano wbudowane reguły zestawów i zasady (lub ostrzeżenia) zawierają.

> [!NOTE]
> Zestawy reguł w tej sekcji odnoszą się do statycznej analizy kodu. Aby uzyskać informacji na temat zestawów reguł dostępnych dla platformy Roslyn analizatora pakietów, zobacz [używanie zestawów reguł analizatorów Roslyn](analyzer-rule-sets.md).
