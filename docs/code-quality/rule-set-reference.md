# <a name="code-analysis-rule-set-reference"></a>Odwołanie zestawu reguł analizy kodu

Podczas konfigurowania analizy kodu dla kodu zarządzanego projektów programu Visual Studio, można wybrać z listy wbudowanych *zestawów reguł*. Można użyć jednego z tych zestawów reguł wbudowanych lub możesz [dostosować zestaw reguł](../code-quality/how-to-create-a-custom-rule-set.md) zgodnie z własnymi wymaganiami projektu.

W tematach w tej sekcji opisano wbudowane reguły zestawów i zasady (lub ostrzeżenia) zawierają.
