# <a name="c28239"></a>C28239
Ostrzeżenie C28239: adnotacja dla parametru ma wartość Value, stringValue lub longValue; i paramn = xxx. Użyj tylko paramn = xxx

 Wystąpił błąd wewnętrzny w pliku modelu PREfast. To ostrzeżenie nie powinno występować w typowym użyciu.
