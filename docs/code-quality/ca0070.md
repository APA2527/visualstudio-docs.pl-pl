# <a name="ca0070"></a>CA0070

Ten błąd występuje, gdy analiza kodu analizuje zestawu i napotka atrybuty uprawnień i co najmniej jeden z następujących warunków jest spełniony:

- Analiza kodu nie można odnaleźć zestawu, który zawiera atrybuty.

- Analiza kodu nie ma uprawnień do załadowania zestawu.

- Zestaw, który jest ładowany podczas analizy kodu nie zawiera atrybutu.

W większości przypadków można rozwiązać ten problem, upewniając się, że masz najnowszą wersję narzędzi analizy kodu.
