# <a name="ca0070"></a>CA0070

Ten błąd występuje, gdy analiza kodu analizuje zestaw i napotyka atrybuty uprawnień i co najmniej jeden z następujących warunków jest spełniony:

- Analiza kodu nie może znaleźć zestawu, który zawiera atrybuty.

- Analiza kodu nie ma uprawnień do załadowania zestawu.

- Zestaw, który jest ładowany przez analizę kodu, nie zawiera atrybutu.

W większości przypadków problem można rozwiązać, upewniając się, że masz najnowszą wersję narzędzi do analizy kodu.
