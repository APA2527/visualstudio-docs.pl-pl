# <a name="ca0070"></a>CA0070
Ten błąd występuje, gdy kod — analiza analizuje zestawu i napotka atrybuty uprawnień i co najmniej jeden z następujących warunków jest spełniony:  
  
-   Kod — analiza nie można odnaleźć zestawu, który zawiera atrybuty.  
  
-   Kod — analiza nie ma uprawnienie do ładowania zestawu.  
  
-   Zestaw, który jest ładowany podczas analizy kodu nie zawiera atrybutu.  
  
 W większości przypadków problem można rozwiązać, upewniając się, że masz najnowszą wersję narzędzi analizy kodu.
