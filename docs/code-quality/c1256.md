# <a name="c1256"></a>C1256
Ostrzeżenie C1256: PCH musi być zarówno zapisywane i do odczytu.  
  
 Narzędzie do analizy kodu raporty to ostrzeżenie, po wystąpił błąd wewnętrzny w narzędziu, a nie w kodzie analizowany.
