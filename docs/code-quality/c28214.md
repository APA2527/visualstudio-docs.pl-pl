# <a name="c28214"></a>C28214
Ostrzeżenie C28214: nazwy parametrów atrybutu muszą być: p1... p9  
  
 To ostrzeżenie wskazuje, że podczas tworzenia deklaracji adnotacji, nazwy parametrów są ograniczone do p1... p9. To ostrzeżenie nigdy nie powinna wystąpić w normalnych warunkach użytkowania.
