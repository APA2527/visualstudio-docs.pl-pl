# <a name="c28228"></a>C28228
Ostrzeżenie C28228: parametr adnotacji: nie można użyć typu w adnotacjach  
  
 To ostrzeżenie oznacza, że parametr jest typu, który nie jest obsługiwany. Adnotacje można używać tylko określonych typów jako parametry. To nie powinna być możliwa, jeśli są używane standardowe makra dla adnotacji; to ostrzeżenie nie powinien być widoczny w typowy sposób użycia.
