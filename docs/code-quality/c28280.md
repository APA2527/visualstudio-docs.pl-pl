# <a name="c28280"></a>C28280
Ostrzeżenie C28280: Dla symbolu element "end" znaleziono bez pasującego "begin"

 Język adnotacji obsługuje konstrukcję BEGIN i End (`{` i `}` w języku C), a parowanie nie jest zrównoważone. Tę sytuację można uniknąć, jeśli są używane makra.
