# <a name="c28209"></a>C28209
Ostrzeżenie C28209: deklaracja symbolu ma sprzeczną deklarację  
  
 To ostrzeżenie oznacza deklaracji niepoprawnie zbudowane adnotacji. To ostrzeżenie nigdy nie powinna wystąpić w normalnych warunkach użytkowania.
