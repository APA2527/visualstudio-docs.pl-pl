# <a name="c28284"></a>C28284
Ostrzeżenie C28284: W przypadku symbolu predykaty nie są obecnie obsługiwane dla symboli niebędących funkcjami

 To ostrzeżenie wskazuje, że znaleziono adnotację warunkową (predykat, `__drv_when`) dla czegoś innego niż funkcja.
