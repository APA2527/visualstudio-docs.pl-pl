# <a name="c26465-noconstcastunnecessary"></a>C26465 NO_CONST_CAST_UNNECESSARY

Nie używaj `const_cast` do oddać `const`. `const_cast` nie jest wymagana. Ta konwersja nie powoduje usunięcia stałości ani zmienności. Zobacz [C++ podstawowe wytyczne Type.3](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Pro-type-constcast).
