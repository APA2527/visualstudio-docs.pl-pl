# <a name="c28229"></a>C28229
Ostrzeżenie C28229: adnotacji nie obsługuje parametrów  
  
 To ostrzeżenie wskazuje, czy adnotacja została użyta z parametrem podczas adnotacja została zadeklarowana bez parametrów. To nie powinna być możliwa, jeśli są używane standardowe makra dla adnotacji; to ostrzeżenie nie powinien być widoczny w typowy sposób użycia.
