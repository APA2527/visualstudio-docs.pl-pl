# <a name="c28230"></a>C28230
Ostrzeżenie C28230: typ parametru nie ma elementu członkowskiego.  
  
 To ostrzeżenie oznacza, że argument adnotacji próbuje uzyskać dostęp do `struct`, `class`, lub `union` i element członkowski o nazwie nie istnieje.
