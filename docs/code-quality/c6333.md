# <a name="c6333"></a>C6333
Ostrzeżenie C6333: nieprawidłowy parametr: przekazanie MEM_RELEASE i niezerowego parametru dwSize do \<function > jest niedozwolone. Powoduje to niepowodzenie tego wywołania

 To ostrzeżenie wskazuje, że nieprawidłowy parametr jest przesyłany do VirtualFree lub VirtualFreeEx. Obie te funkcje odrzucają dwFreeType MEM_RELEASE o wartości innej niż zero dla dwSize. Podczas przekazywania MEM_RELEASE parametr dwSize musi mieć wartość zero. Upewnij się również, że wartość zwracana przez tę funkcję nie jest ignorowana.

## <a name="example"></a>Przykład
 Następujący przykładowy kod generuje to ostrzeżenie:

```
#include <windows.h>
#define PAGELIMIT 80

DWORD dwPages = 0;  // count of pages
DWORD dwPageSize;   // page size

VOID f( VOID )
{
  LPVOID lpvBase;            // base address of the test memory
  BOOL bSuccess;
  SYSTEM_INFO sSysInfo;      // system information

  GetSystemInfo( &sSysInfo );
  dwPageSize = sSysInfo.dwPageSize;

  // Reserve pages in the process's virtual address space
  lpvBase = VirtualAlloc(
                         NULL,                // system selects address
                         PAGELIMIT*dwPageSize,// size of allocation
                         MEM_RESERVE,
                         PAGE_NOACCESS );
  if (lpvBase)
  {
    // code to access memory
  }
  else
  {
    return;
  }

  bSuccess = VirtualFree(lpvBase, PAGELIMIT * dwPageSize, MEM_RELEASE);
  //code...
}
```

 Aby poprawić to ostrzeżenie, użyj następującego przykładowego kodu:

```
#include <windows.h>
#define PAGELIMIT 80

DWORD dwPages = 0;  // count of pages
DWORD dwPageSize;   // page size

VOID f( VOID )
{
  LPVOID lpvBase;            // base address of the test memory
  BOOL bSuccess;
  SYSTEM_INFO sSysInfo;      // system information

  GetSystemInfo( &sSysInfo );
  dwPageSize = sSysInfo.dwPageSize;

  // Reserve pages in the process's virtual address space
  lpvBase = VirtualAlloc(
                         NULL,                // system selects address
                         PAGELIMIT*dwPageSize,// size of allocation
                         MEM_RESERVE,
                         PAGE_NOACCESS );
  if (lpvBase)
  {
    // code to access memory
  }
  else
  {
    return;
  }
  bSuccess = VirtualFree(lpvBase, 0, MEM_RELEASE );

  //  VirtualFree(lpvBase, PAGELIMIT * dwPageSize, MEM_DECOMMIT);
  // code...
}
```

 Możesz również użyć VirtualFree (lpvBase, PAGELIMIT * dwPageSize, MEM_DECOMMIT); Wywołaj, aby wycofać zatwierdzenie stron i później wydać je przy użyciu flagi MEM_RELEASE.
