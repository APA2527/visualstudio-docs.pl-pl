# <a name="c26485-noarraytopointerdecay"></a>C26485 NO_ARRAY_TO_POINTER_DECAY
Podobnie jak C26481 to sprawdzenie pomaga wymusić regułę, I.13: *Nie przekazuj tablicę jako pojedynczy wskaźnik* został określony poprzez wykrycie miejsca, w którym utracone z powodu zanikania do wskaźnika pierwotnego pobiera informacje o typie tablicy statycznej. typy zstring i czstring nie są wyłączone.

## <a name="remarks"></a>Uwagi
Pochodzą C26481 i C26485 [profil bezpieczeństwa granic](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md) reguł wdrożonych w pierwszej wersji wytycznych podstawowe sprawdzanie języka C++. Są one zastosowanie do surowych wskaźników, kategorii, ponieważ mogą pomóc uniknąć niebezpieczne użycie surowych wskaźników.
