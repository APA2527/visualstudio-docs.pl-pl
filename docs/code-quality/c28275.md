# <a name="c28275"></a>C28275
Ostrzeżenie C28275: parametr _Macro_value\_ ma wartość null

 To ostrzeżenie oznacza, że jest błąd wewnętrzny w pliku modelu, a nie w kodzie analizowany. *MacroValue* funkcja została wywołana bez parametrów.
