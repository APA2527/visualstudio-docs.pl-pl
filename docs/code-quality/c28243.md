# <a name="c28243"></a>C28243
Ostrzeżenie C28243: adnotacja dla funkcji w parametrze wymaga więcej wyłuskań niż pozwala właściwy adnotowany typ

 Liczba `__deref` operatory w adnotacji jest większa niż liczba poziomów wskaźnika zdefiniowane przez typ parametru. Popraw to ostrzeżenie, zmieniając albo numer w adnotacji lub poziomów wskaźnika odwołuje się do parametru.
