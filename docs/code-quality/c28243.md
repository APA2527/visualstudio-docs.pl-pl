# <a name="c28243"></a>C28243
Ostrzeżenie C28243: Adnotacja dla funkcji na parametrze, wymaga więcej wyłuskań niż pozwala właściwy adnotowany typ

 Liczba operatorów `__deref` w adnotacji jest większa niż liczba poziomów wskaźnika zdefiniowanych przez typ parametru. Popraw to ostrzeżenie, zmieniając liczbę w adnotacji lub na poziomach wskaźnika parametru, do którego istnieje odwołanie.
