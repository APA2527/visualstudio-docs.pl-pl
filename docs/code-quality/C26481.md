# <a name="c26481-nopointerarithmetic"></a>C26481 NO_POINTER_ARITHMETIC
To sprawdzenie obsługuje reguły *I.13: Nie przekazuj tablicę jako pojedynczy wskaźnik*. Zawsze, gdy surowe wskaźniki są używane w operacjach arytmetycznych powinny zostać zastąpione przy użyciu bezpieczniejszego rodzaje buforów, takich jak `span<T>` lub `vector<T>`.

## <a name="remarks"></a>Uwagi
- Ten test jest nieco bardziej restrykcyjny niż I.13: nie pomija, zstring lub czstring typów.
- Pochodzą C26481 i C26485 [profil bezpieczeństwa granic](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md) reguł wdrożonych w pierwszej wersji wytycznych podstawowe sprawdzanie języka C++. Są one zastosowanie do surowych wskaźników, kategorii, ponieważ mogą pomóc uniknąć niebezpieczne użycie surowych wskaźników.
