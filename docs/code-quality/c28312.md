# <a name="c28312"></a>C28312
Ostrzeżenie 28312: < annotation_name > Adnotacja w powtórzonej deklaracji \<function > \<parameter > jest przestarzałaą wersją SAL.

 Adnotacja jest stara wersja i należy ją zaktualizować do równoważnej [adnotacją sal2](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md). To ostrzeżenie nie jest emitowane, jeśli emitowane było uprzednie ostrzeżenie o niespójnej adnotacji i jest ono zgłaszane tylko raz na deklarację. Sprawdź pozostałą część deklaracji pod kątem przestarzałego SAL.
