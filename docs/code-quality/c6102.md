# <a name="c6102"></a>C6102
Ostrzeżenie C6102: Używanie \<variable > z wywołania funkcji zakończonej niepowodzeniem w wierszu \<location >.

 To ostrzeżenie jest zgłaszane zamiast [C6001](../code-quality/c6001.md) , gdy zmienna nie jest ustawiona, ponieważ została oznaczona jako parametr `_Out_` w przypadku wcześniejszego wywołania funkcji, które zakończyło się niepowodzeniem.

 Problem może być spowodowany tym, że wcześniej wywoływana funkcja nie ma kompletnych adnotacji. Może to wymagać `_Always_`, `_Outptr_result_nullonfailure_` (`_COM_Outptr_` dla kodu COM) lub powiązanej adnotacji.
