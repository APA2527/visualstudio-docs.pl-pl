# <a name="c28289"></a>C28289
Ostrzeżenie C28289: dla funkcji: ReadableTo lub WritableTo nie ma elementu limit-spec jako parametru

 Narzędzie do analizy kodu raporty to ostrzeżenie, gdy adnotacji funkcji i parametru jest miscoded wspomnianego.
