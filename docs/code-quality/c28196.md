# <a name="c28196"></a>C28196
Ostrzeżenie C28196: Wymaganie nie jest spełnione. (Wyrażenie nie jest szacowane na wartość true).

 To ostrzeżenie wskazuje, że prototyp funkcji dla analizowanej funkcji ma `__notnull`, `__null` lub `__drv_valueIs` dla parametru `_Out_` lub wartości zwracanej, ale zwrócona wartość jest niespójna z tą adnotacją.
