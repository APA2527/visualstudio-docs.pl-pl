# <a name="c28196"></a>C28196
Ostrzeżenie C28196: nie zostały spełnione wymaganie. (Wyrażenie nie zwraca wartość true.)  
  
 To ostrzeżenie oznacza, że ma prototypu funkcji dla analizowanej funkcji `__notnull`, `__null` lub `__drv_valueIs` na `_Out_` parametrów lub wartości zwracanej, ale wartość zwracana jest niezgodne z tym adnotacji.
