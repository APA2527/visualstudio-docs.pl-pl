# <a name="c26466-nostaticdowncastpolymorphic"></a>C26466 NO_STATIC_DOWNCAST_POLYMORPHIC
  Nie używaj `static_cast` rzutowań. Rzutowanie z poziomu typu polimorficznego powinno używać elementu dynamic_cast. Zobacz [C++ podstawowe wytyczne Type.2](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Pro-type-downcast).
