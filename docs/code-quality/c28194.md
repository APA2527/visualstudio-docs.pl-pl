# <a name="c28194"></a>C28194
Ostrzeżenie C28194: Funkcja została zadeklarowana jako aliasowanie wartości w zmiennej i zakończyła się bez wykonania tej czynności

 To ostrzeżenie wskazuje, że prototyp funkcji dla analizowanej funkcji ma adnotację `__drv_isAliased`, która wskazuje, że będzie *aliasować* określony argument (to znaczy, przypisując wartość w sposób, w jaki będzie ona przewracana z funkcji). Jednak funkcja nie aliasuje argumentu na ścieżce wskazanej przez adnotację. Większość funkcji aliasuje zmienną Zapisz jej wartość w globalnej strukturze danych.
