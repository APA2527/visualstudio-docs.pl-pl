# <a name="c6259"></a>C6259
Ostrzeżenie C6259: kod z etykietą jest nieosiągalny: (\<expression > & \<stała >) w wyrażeniu Switch nie można obliczyć \<etykiety case >

 To ostrzeżenie wskazuje nieosiągalny kod spowodowany wynikiem porównania bitowe-i (`&`) w wyrażeniu Switch. Instrukcja Case, która pasuje do stałej w wyrażeniu Switch, jest tylko osiągalna; wszystkie inne instrukcje Case są nieosiągalne.

## <a name="example"></a>Przykład
 Następujący przykładowy kod generuje to ostrzeżenie, ponieważ wyrażenie `switch``(rand() & 3)` nie może oszacować do etykiety case (`case` `4`):

```
#include <stdlib.h>

void f()
{
     switch (rand () & 3) {
        case 3:
            /* Reachable */
            break;
        case 4:
            /* Not reachable */
            break;
        default:
            break;
    }
}
```

 Aby poprawić to ostrzeżenie, Usuń nieosiągalny kod lub sprawdź, czy stała użyta w instrukcji case jest prawidłowa. Poniższy kod usuwa nieosiągalną instrukcję CASE:

```
#include <stdlib.h>

void f()
{
     switch (rand () & 3) {
        case 3:
            /* Reachable */
            break;
        default:
            break;
    }
}
```
