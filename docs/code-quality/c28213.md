# <a name="c28213"></a>C28213
Ostrzeżenie C28213: Adnotacja `_Use_decl_annotations_` musi być używana do odwoływania się, bez modyfikacji, wcześniejszej deklaracji. @no__t — 0why >

 To ostrzeżenie jest zgłaszane, gdy następuje odwołanie do wcześniejszej deklaracji bez wymaganej adnotacji `_Use_decl_annotations_`.
