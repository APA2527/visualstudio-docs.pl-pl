# <a name="ca0053"></a>CA0053
Nie można załadować zestawu reguł.

 Oznacza to, że nie można załadować zestawu reguł przez odbicie. Mogło się tak zdarzyć, jeśli użytkownik ma reguły niestandardowe, które nie zostały utworzone prawidłowo lub narzędzia analizy nie można załadować zestawu z innych przyczyn, takich jak nastąpiła odmowa dostępu do systemu plików.
