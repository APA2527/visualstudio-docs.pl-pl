# <a name="ca0053"></a>CA0053
Nie można załadować zestawu reguł.  
  
 Oznacza to, że nie można załadować zestawu reguł przez odbicie. Może to wystąpić, jeśli użytkownik ma reguły niestandardowe, które nie zostały poprawnie zbudowany lub narzędzie do analizy nie można załadować zestawu z innych powodów, takich jak odmowa dostępu do systemu plików.
