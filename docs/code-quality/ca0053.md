# <a name="ca0053"></a>CA0053
Nie można załadować zestawu reguł.

Oznacza to, że nie można załadować zestawu reguł przez odbicie. Taka sytuacja może wystąpić, jeśli użytkownik ma niestandardowe reguły, które nie zostały prawidłowo skompilowane lub narzędzie do analizy nie może załadować zestawu z innych powodów, takich jak odmowa dostępu do systemu plików.
