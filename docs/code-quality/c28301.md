# <a name="c28301"></a>C28301
Ostrzeżenie C28301: Brak adnotacji dla pierwszej deklaracji \<funkcja >.\< note1 > zobacz \<nazwa pliku > (\<wiersz >). \<Uwaga2 >  
  
 To ostrzeżenie jest zgłaszany, gdy adnotacji nie zostały odnalezione w pierwszej deklaracji danej funkcji.
