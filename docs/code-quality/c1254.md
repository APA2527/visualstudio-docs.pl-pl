# <a name="c1254"></a>C1254
Ostrzeżenie C1254: niezgodność wersji wtyczek: wersja nie pasuje do wersji sterownika PREfast

 Narzędzie do analizy kodu raporty to ostrzeżenie, gdy wystąpił błąd wewnętrzny przy użyciu wersji dodatku nie w kodzie analizowanej.
