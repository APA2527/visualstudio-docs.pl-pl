# <a name="c1254"></a>C1254
Ostrzeżenie C1254: niezgodność wersji wtyczki: wersja nie jest zgodna z wersją sterownika PREfast

 Narzędzie do analizy kodu zgłasza to ostrzeżenie w przypadku błędu wewnętrznego w wersji wtyczki, a nie w analizowanym kodzie.
