# <a name="c6306"></a>C6306
Ostrzeżenie C6306: nieprawidłowe wywołanie > \<function: Rozważ użycie > \<function, który akceptuje va_list jako argument

 To ostrzeżenie wskazuje nieprawidłowe wywołanie funkcji. Rodzina `printf` zawiera kilka funkcji, które przyjmują zmienną listę argumentów; Jednak te funkcje nie mogą być wywoływane przy użyciu argumentu `va_list`. Istnieje odpowiednia rodzina `vprintf` funkcji, które mogą być używane dla takich wywołań. Wywołanie nieprawidłowej funkcji drukowania spowoduje wystąpienie niepoprawnych danych wyjściowych.

## <a name="example"></a>Przykład
 Poniższy kod generuje to ostrzeżenie:

```
#include <stdio.h>
#include <stdarg.h>

void f(int i, ...)
{
   va_list v;

   va_start(v, i);
   //code...
   printf("%s", v); // warning 6306
   va_end(v);
}
```

 Aby poprawić to ostrzeżenie, należy użyć następującego kodu:

```
#include <stdio.h>
#include <stdarg.h>

void f(int i, ...)
{
   va_list v;

   va_start(v, i);
   //code...
   vprintf_s("%d",v);
   va_end(v);
}
```
