# <a name="ca0502"></a>CA0502
Nieobsługiwany język

Ten błąd występuje, gdy użytkownik próbuje analizować projekt ASP.NET, który zawiera co najmniej jedną stronę lub klasę zapisaną w języku innym niż C# lub Visual Basic. Aby naprawić ten błąd, usuń plik z projektu lub Wyłącz analizę kodu.
