# <a name="ca0502"></a>CA0502
Nieobsługiwany język

 Ten błąd występuje, gdy użytkownik próbuje analizowanie projektu ASP.NET, która zawiera jedną lub więcej stron lub klasy napisany w języku innym niż C# lub Visual Basic. Aby naprawić ten błąd, usuń go z projektu lub wyłączanie analizy kodu.
