# <a name="ca0502"></a>CA0502
Nieobsługiwany język

 Ten błąd występuje, gdy użytkownik próbuje przeanalizować projektu ASP.NET, która zawiera jedną lub więcej stron lub klas w języku innym niż C# lub Visual Basic. Aby naprawić ten błąd, usuń plik z projektu lub wyłączanie analizy kodu.
