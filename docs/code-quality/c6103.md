# <a name="c6103"></a>C6103
Ostrzeżenie C6103: zwracanie > \<variable z wywołania funkcji zakończonej niepowodzeniem w wierszu \<location >.

 Pomyślna ścieżka przez funkcję zwraca zmienną, która została użyta jako parametr `_Out_` do wewnętrznego wywołania funkcji, które zakończyło się niepowodzeniem.

 Problem może być spowodowany tym, że funkcje wywołana i wywoływana nie mają kompletnych adnotacji. Wywoływana funkcja może wymagać `_Always_`, `_Outptr_result_nullonfailure_` (`_COM_Outptr_` dla kodu COM) lub powiązanej adnotacji, a funkcja wywołująca może wymagać adnotacji `_Success_`. Inną możliwą przyczyną tego ostrzeżenia jest to, że adnotacja `_Success_` w wywołanej funkcji jest niepoprawna.
