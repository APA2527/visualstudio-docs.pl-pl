# <a name="c28195"></a>C28195
Ostrzeżenie C28195: funkcja została zadeklarowana jako nabywająca pamięć w zmiennej i zakończyła się bez wykonania tej czynności

 To ostrzeżenie wskazuje, że prototyp funkcji dla analizowanej funkcji ma adnotację `__drv_acquiresMemory`. Adnotacja `__drv_acquiresMemory` wskazuje, że funkcja uzyskuje pamięć w określonej lokalizacji wyników, ale w co najmniej jednej ścieżce funkcja nie uzyskała pamięci. Należy zauważyć, że Narzędzie analizy kodu nie rozpoznaje rzeczywistej implementacji alokatora pamięci (obejmującego arytmetyczne adresy) i nie rozpoznaje przydzielenia pamięci (mimo że wiele otok zostanie rozpoznany). W takim przypadku Narzędzie analizy kodu nie rozpoznaje, że pamięć została przypisana i wygeneruje to ostrzeżenie. Aby pominąć fałszywą wartość dodatnią, użyj ostrzeżenia `#pragma` w wierszu, który poprzedza nawias otwierający `{` treści funkcji.
