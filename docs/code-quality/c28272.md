# <a name="c28272"></a>C28272
Ostrzeżenie C28272: adnotacja dla funkcji, parametr Rozpatrując jest niespójna z deklaracją funkcji

 To ostrzeżenie wskazuje na błąd w adnotacjach, a nie w kodzie analizowany. Adnotacje znajdujące się w definicji funkcji są niespójne z tymi znajdujących się w deklaracji. Dwa adnotacje mają zostać rozwiązane do dopasowania.
