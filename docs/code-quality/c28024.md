# <a name="c28024"></a>C28024
Ostrzeżenie C28024: wskaźnik funkcji, do której jest przypisany, ma adnotację z klasą funkcji \<class >, która nie jest zawarta w klasach funkcji \<classlist >.

 To ostrzeżenie jest zgłaszane, gdy obie funkcje mają adnotację z klasą funkcji, ale klasy nie są zgodne.

 to ostrzeżenie jest wydawane, gdy wskaźnik funkcji ma adnotację `_Function_class_`, która określa, że tylko funkcje konkretnej klasy funkcjonalnej powinny być przypisane do niej. W przypisaniu lub implikowanym przypisaniu w wywołaniu funkcji element źródłowy i docelowy muszą być tej samej klasy funkcji, ale klasy funkcji nie są zgodne.
