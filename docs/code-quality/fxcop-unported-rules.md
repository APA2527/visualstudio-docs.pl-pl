# <a name="unported-rules"></a>Nieprzeniesione reguły

## <a name="overview"></a>Omówienie

Zestaw reguł FxCop starszej analizy, które nie zostały zaimplementowane jako analizatory Roslyn, składa się z następujących dwóch przedziałów:
- Reguły, które nie zostały jeszcze przewoźny, ale nadal [mogą być przeliczane](fxcop-unported-rules-may-get-ported.md).
- Reguły, które są przestarzałe i [nie zostaną](fxcop-unported-deprecated-rules.md)przełączone.
