# <a name="c28223"></a>C28223
Ostrzeżenie C28223: Nie znaleziono oczekiwanego tokenu/identyfikatora dla adnotacji, parametr  
  
 To ostrzeżenie wskazuje, że parametr do adnotacji powinien być identyfikatorem i napotkano inny symbol. Zwykle oznacza to makro niepoprawnie zakodowany adnotacji. Użycie słowa kluczowe języka C lub C++ w tej pozycji spowoduje, że ten błąd.
