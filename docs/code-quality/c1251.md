# <a name="c1251"></a>C1251
Ostrzeżenie C1251: nie można załadować modeli.

 Narzędzie do analizy kodu raporty to ostrzeżenie, po wystąpił błąd wewnętrzny w pliku modelu, a nie w kodzie analizowany.
