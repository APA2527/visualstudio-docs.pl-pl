# <a name="ca0051"></a>CA0051

Nie wybrano żadnych reguł.

Ten błąd wskazuje, że jest włączone narzędzie do analizy kodu zarządzanego, ale nie wybrano żadnych reguł. Aby naprawić ten błąd, wybierz odpowiednie reguły.
