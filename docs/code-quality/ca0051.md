# <a name="ca0051"></a>CA0051
Nie wybrano żadnych reguł.

Ten błąd wskazuje, że narzędzie do analizy kodu zarządzanego jest włączona, ale nie wybrano żadnych reguł. Aby rozwiązać ten problem, wybierz odpowiednie zasady.
