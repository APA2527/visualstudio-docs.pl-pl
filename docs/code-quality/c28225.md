# <a name="c28225"></a>C28225
Ostrzeżenie C28225: nie znaleziono nieprawidłową liczbę wymaganych parametrów w adnotacji  
  
 To ostrzeżenie oznacza, że adnotacji o nazwie jest używany z niepoprawną liczbą parametrów. To nie powinna być możliwa, jeśli są używane standardowe makra dla adnotacji; to ostrzeżenie nie powinien być widoczny w typowy sposób użycia.
