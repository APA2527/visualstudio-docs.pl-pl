# <a name="c28282"></a>C28282
Ostrzeżenie C28282: Ciągi formatu muszą znajdować się w warunkach wstępnych

 To ostrzeżenie wskazuje, że zostanie znaleziona adnotacja `__drv_formatString`, która nie znajduje się w adnotacji `_Pre_` (`__drv_in`) (parametry funkcji są domyślnie `_Pre_`). Sprawdź, czy adnotacja użyta w bloku jawnym z adnotacją `_Post_` (`__drv_out`). W takim przypadku Usuń adnotację z dowolnego bloku otaczającego, który umieścił go w kontekście `_Post_`.
