# <a name="c28234"></a>C28234
Ostrzeżenie C28234: _At\_ wyrażenie nie ma zastosowania do bieżącej funkcji  
  
 To ostrzeżenie oznacza to, że wartość `_At_` wyrażenia nie będzie rozpoznawał dostępny obiekt.
