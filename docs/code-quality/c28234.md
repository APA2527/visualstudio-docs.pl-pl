# <a name="c28234"></a>C28234
Ostrzeżenie C28234: \_w wyrażeniu\_ nie ma zastosowania do bieżącej funkcji

 To ostrzeżenie wskazuje, że wartość wyrażenia `_At_` nie identyfikuje dostępnego obiektu.
