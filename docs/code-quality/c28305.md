# <a name="c28305"></a>C28305
Ostrzeżenie C28305: Wystąpił błąd podczas analizowania \<token > został wykryty.  
  
 To ostrzeżenie jest zgłaszany, gdy wyrażenie zawierających określony token jest źle sformułowany.
