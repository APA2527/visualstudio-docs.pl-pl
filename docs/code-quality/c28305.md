# <a name="c28305"></a>C28305
Ostrzeżenie C28305: Błąd podczas analizowania \<tokenu > zostało odnalezione.

 To ostrzeżenie jest zgłaszane, gdy wyrażenie zawierające określony token jest nieprawidłowo sformułowany.
