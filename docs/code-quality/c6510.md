# <a name="c6510"></a>C6510
Ostrzeżenie C6510: Nieprawidłowa Adnotacja: Właściwość "NullTerminated" może być używana tylko w przypadku buforów, których elementy są typu całkowitego lub wskaźnika: Funkcja "\<function >" \<parameter >.

 To ostrzeżenie wskazuje nieprawidłowe użycie właściwości **NullTerminated** (te kończące się na "`_z`"). Tego typu właściwości można używać tylko w przypadku typu wskaźnika lub tablicy.

## <a name="example"></a>Przykład
 Poniższy kod generuje to ostrzeżenie:

```
#include <sal.h>

void f(_In_z_ char x)
{
    // code ...
}
```

 Aby poprawić to ostrzeżenie, należy użyć następującego kodu:

```
#include <sal.h>

void f(_In_z_ char * x)
{
    // code ...
}
```
