# <a name="c6302"></a>C6302
Ostrzeżenie C6302: niezgodność formatu ciągu: ciąg znaków został przesłany jako parametr \<Number >, gdy w wywołaniu funkcji \<jest wymagany ciąg znaków dwubajtowych >

 To ostrzeżenie wskazuje, że ciąg formatu określa, że wymagany jest ciąg znaków dwubajtowych. Jednak ciąg znaków jest przesyłany. Usterka może spowodować awarię lub uszkodzenie jakiejś formy.

## <a name="example"></a>Przykład
 Następujący przykładowy kod generuje to ostrzeżenie, ponieważ ciąg znaków jest przesyłany do funkcji `wprintf`:

```cpp
#include <stdio.h>

void f()
{
  char buff[5] = "hi";

  wprintf(L"%s", buff);
}
```

 Następujący przykładowy kod używa `%hs`, aby określić jednobajtowy ciąg znaków z funkcją `wprintf`:

```cpp
#include <stdio.h>

void f()
{
  char buff[5] = "hi";

  wprintf(L"%hs", buff);
}
```

 Następujący przykładowy kod używa bezpiecznego `wprintf_s` funkcji manipulowania ciągami, aby poprawić to ostrzeżenie:

```cpp
#include <stdio.h>

void f()
{
  char buff[5] = "hi";

  wprintf_s(L"%hs", buff);
}
```
