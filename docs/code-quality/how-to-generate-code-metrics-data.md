# <a name="how-to-generate-code-metrics-data"></a>Porady: generowanie metryk kodów

Można wygenerować wyniki metryki kodu dla całego rozwiązania lub wybranego projektu.

## <a name="to-generate-code-metrics-results-for-an-entire-solution"></a>Aby wygenerować wyniki metryki kodu dla całego rozwiązania

- Na pasku menu wybierz **Analizuj** > **obliczanie metryki kodu** > **dla rozwiązania**.

   \-lub -

- W **Eksploratora rozwiązań**, kliknij prawym przyciskiem myszy rozwiązanie, a następnie wybierz pozycję **obliczanie metryki kodu**.

   \-lub -

- W **wyników metryk kodów** okna, wybierz **obliczanie metryki kodu dla rozwiązania** przycisku.

   Wyniki są generowane i **wyników metryk kodów** okno jest wyświetlane.

## <a name="to-generate-code-metrics-results-for-one-or-more-selected-projects"></a>Aby Generowanie wyników metryk kodów dla jednego lub więcej wybranych projektów

1. W **Eksploratora rozwiązań**, wybierz jeden lub więcej projektów.

1. Na pasku menu wybierz **Analizuj** > **obliczanie metryki kodu** > **dla < project\>**.

   Wyniki są generowane i **wyników metryk kodów** okno jest wyświetlane.

## <a name="to-view-the-results-details"></a>Aby wyświetlić szczegóły wyników

- W **wyników metryk kodów** okna, rozwiń drzewo w **hierarchii** kolumny.
