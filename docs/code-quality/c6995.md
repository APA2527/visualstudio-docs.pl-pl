# <a name="c6995"></a>C6995
Ostrzeżenie C6995: nie można zapisać pliku dziennika XML  
  
 To ostrzeżenie oznacza, że narzędzie do analizy kodu nie można utworzyć dziennika usterką, który jest plikiem wyjściowym analizy kodu.  
  
 Ten błąd może wskazywać na błąd dysku lub wskazać, że nie masz uprawnień do tworzenia pliku w określonym katalogu.
