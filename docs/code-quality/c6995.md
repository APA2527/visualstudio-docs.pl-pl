# <a name="c6995"></a>C6995
Ostrzeżenie C6995: nie można zapisać pliku dziennika XML

 To ostrzeżenie wskazuje, że Narzędzie analizy kodu nie może utworzyć dziennika wad, który jest danymi wyjściowymi analizy kodu.

 Ten błąd może wskazywać na błąd dysku lub wskazuje, że nie masz uprawnień do utworzenia pliku w określonym katalogu.
