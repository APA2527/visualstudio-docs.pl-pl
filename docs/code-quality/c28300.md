# <a name="c28300"></a>C28300
Ostrzeżenie C28300: < nazwa_parametru >: operandy wyrażenia niezgodnych typów dla operatora < operator_name >  
  
 To ostrzeżenie jest zgłaszany, gdy typy argumentów operacji w parametrze nie są zgodne z operatorem.
