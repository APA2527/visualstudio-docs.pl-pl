# <a name="c28350"></a>C28350
Ostrzeżenie C28350: adnotacja \<adnotacji > opisuje sytuację, której nie można zastosować warunkowo.

 Zwykle to ostrzeżenie jest generowany, gdy adnotacji jest stosowana, gdy jest sprawdzana typu C/C++.
