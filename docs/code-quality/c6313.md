# <a name="c6313"></a>C6313
Ostrzeżenie C6313: Nieprawidłowy operator: Nie można przetestować flagi o wartości zerowej przy użyciu koniunkcji bitowej i. Użyj testu równości, aby sprawdzić flagi o wartości zerowej

 To ostrzeżenie wskazuje, że wartość stała zero została podana jako argument w operatorze bitowym i (`&`) w kontekście testu. Wynikowe wyrażenie jest stałe i ma wartość false; wynik jest różny od przeznaczenia.

 Jest to zazwyczaj spowodowane użyciem funkcji bitowe i do testowania dla flagi, która ma wartość zero. Aby przetestować flagi o zerowej wartości, należy wykonać test pod kątem równości, na przykład przy użyciu `==` lub `!=`.

## <a name="example"></a>Przykład
 Poniższy kod generuje to ostrzeżenie:

```
#define FLAG 0

void f(int Flags )
{
  if (Flags & FLAG)
  {
    // code
  }
}
```

 Aby poprawić to ostrzeżenie, należy użyć następującego kodu:

```
#define FLAG 0

void f(int Flags )
{
  if (Flags == FLAG)
  {
    // code
  }
}
```
