# <a name="c28159"></a>C28159
Ostrzeżenie C28159: Zamiast tego Rozważ użycie innej funkcji.

 To ostrzeżenie jest zgłaszane w przypadku sterowników sugerujących użycie preferowanego wywołania funkcji, które jest semantycznie równoważne funkcji, którą wywołuje sterownik. Jest to ogólny komunikat ostrzegawczy. Adnotacja `__drv_preferredFunction` została użyta (prawdopodobnie z warunkową adnotacją `__drv_when` (), aby oflagować nieprawidłową technikę kodowania.

## <a name="example"></a>Przykład
 Poniższy przykład kodu generuje to ostrzeżenie:

```
char buff[MAX_PATH];

OemToChar(buff, input);

    // if strlen(input) > MAX_PATH
....// leads to buffer overrun
```

 Poniższy przykład kodu pozwala uniknąć tego ostrzeżenia:

```
char buff[MAX_PATH];

OemToCharBuff(buff, input, MAX_PATH);
```
