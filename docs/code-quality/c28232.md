# <a name="c28232"></a>C28232
Ostrzeżenie C28232: _Pre\_, cząstkę _Post\_, lub _deref —\_ nie są stosowane do jakichkolwiek adnotacji

 To ostrzeżenie oznacza to, że `_Pre_`, `_Post_`, lub `_Deref_` operator pojawia się w wyrażeniu adnotacji nie kolejnych adnotacji funkcjonalności; modyfikator została zignorowana, ale oznacza niepoprawnie zakodowany adnotacji.
