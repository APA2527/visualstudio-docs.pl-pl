# <a name="c28204"></a>C28204
Ostrzeżenie C28204: \<funkcji >: tylko jeden z tych przeciążeń i \<filename > (\<line >) są adnotacjami dla \<paramName >: oba lub nie muszą mieć adnotacji.

 To ostrzeżenie jest zgłaszane w przypadku błędu w adnotacjach.
