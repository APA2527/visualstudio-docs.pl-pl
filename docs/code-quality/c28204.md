# <a name="c28204"></a>C28204
Ostrzeżenie C28204: \<funkcja >: tylko jeden z tego przeciążenia i jedną \<nazwa pliku > (\<wiersza >) są przypisane do \<paramname >: obie lub nie musi być adnotowany.  
  
 To ostrzeżenie jest zgłaszany, gdy występuje błąd w adnotacjach.
