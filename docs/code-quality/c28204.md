# <a name="c28204"></a>C28204
Ostrzeżenie C28204: \<function >: Tylko jeden z tych przeciążeń i jeden na \<filename > (\<line >) są adnotacjami dla @no__t 2paramname >: oba lub nie muszą mieć adnotacji.

 To ostrzeżenie jest zgłaszane w przypadku błędu w adnotacjach.
