# <a name="c28309"></a>C28309
Ostrzeżenie 28309: < parameter_name >: operandy adnotacji muszą być typami całkowitymi/wyliczeniami/wskaźnikami. Operandy typu void C++ i przeciążone operatory nie są obsługiwane. Wartości zmiennoprzecinkowe są przybliżone jako liczby całkowite. Typy: \<typelist >.

 Próbowano użyć elementu void lub funkcji w wyrażeniu adnotacji, a analiza kodu nie może go obsłużyć.  Ten błąd występuje zazwyczaj, gdy jest używana `operator==`, która jest zaimplementowana jako funkcja, ale mogą również wystąpić inne przypadki. Sprawdź typy w @no__t 0typelist >, aby uzyskać wskazówki dotyczące nieprawidłowych informacji.
