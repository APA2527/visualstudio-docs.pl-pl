# <a name="c28021"></a>C28021
Ostrzeżenie C28021: parametr \<param > z adnotacją \<roku > musi być wskaźnikiem  
  
 To ostrzeżenie jest zgłaszany, gdy obiekt adnotacją nie jest typem wskaźnika. Ta adnotacja nie można używać z `void` lub typów całkowitych.
