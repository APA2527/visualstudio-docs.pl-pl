# <a name="c28137"></a>C28137
Ostrzeżenie C28137: Zmienna argument powinien zamiast niego być (literałem)

 To ostrzeżenie jest zgłaszane, gdy w wywołaniu funkcji brakuje wymaganej stałej (literału). Zapoznaj się z dokumentacją funkcji.

## <a name="example"></a>Przykład
 Na przykład procedura [ExAcquireResourceExclusiveLite](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-exacquireresourceexclusivelite) wymaga wartości true lub false dla parametru `Wait`. Poniższy przykładowy kod generuje to ostrzeżenie:

```
ExAcquireResourceExclusiveLite(Resource, Wait);
```

 Poniższy przykład kodu pozwala uniknąć tego ostrzeżenia:

```
ExAcquireResourceExclusiveLite(Resource, TRUE);
```
