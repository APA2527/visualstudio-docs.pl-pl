# <a name="c6282"></a>C6282
Ostrzeżenie C6282: Nieprawidłowy operator: przypisanie stałej w kontekście Boolean. Rozważ użycie zamiast tego "= ="

 To ostrzeżenie wskazuje, że w kontekście testu wykryto przypisanie stałej do zmiennej. Przypisanie stałej do zmiennej w kontekście testu jest niemal zawsze nieprawidłowe. Zastąp wartość `=` opcją `==` lub usuń przypisanie z kontekstu testu, aby rozwiązać to ostrzeżenie.

## <a name="example"></a>Przykład
 Poniższy kod generuje to ostrzeżenie:

```
void f( int i )
{
   while (i = 5)
   {
   // code
   }
}
```

 Aby poprawić to ostrzeżenie, należy użyć następującego kodu:

```
void f( int i )
{
   while (i == 5)
   {
   // code
   }
}
```
