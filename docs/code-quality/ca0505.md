# <a name="ca0505"></a>CA0505
Przestarzałe właściwość zostanie zignorowana, ponieważ przez właściwość jest zdefiniowana.

 Ostrzeżenie CA505 jest generowany, gdy zarówno określonego przestarzałe i przez właściwości analizy kodu zawiera wartości w projekcie, lub utworzyć pliku konfiguracji. Aby rozwiązać ten problem, Usuń właściwość przestarzałe.
