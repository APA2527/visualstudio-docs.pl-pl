# <a name="c28222"></a>C28222
Ostrzeżenie 28222: _tak\_, _NIE\_, lub _Maybe\_ oczekiwany dla adnotacji  
  
 To ostrzeżenie oznacza, że parametr adnotacji powinien być jeden z symboli `_Yes_`, `_No_`, lub `_Maybe_`, i napotkano inny symbol. Zwykle oznacza to makro niepoprawnie zakodowany adnotacji.
