# <a name="quickstart-create-a-python-project-from-existing-code"></a>Szybki start: tworzenie projektu języka Python na podstawie istniejącego kodu

Po [zainstalowaniu obsługi języka Python w programie Visual Studio](installing-python-support-in-visual-studio.md)można łatwo wprowadzić istniejący kod języka Python do projektu programu Visual Studio.

> [!Important]
> Proces opisany w tym miejscu nie przenosi ani nie kopiuje oryginalnych plików źródłowych. Jeśli chcesz pracować z kopią, najpierw zduplikuj folder.

[!INCLUDE[project-from-existing](includes/project-from-existing.md)]

W razie potrzeby uruchom program, naciskając klawisz **Ctrl**+**F5** lub wybierając **debugowanie** > **start bez debugowania**.
