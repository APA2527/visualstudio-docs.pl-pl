Aby można było korzystać z repozytoriów usługi GitHub z poziomu programu VS, należy zainstalować rozszerzenie GitHub dla programu Visual Studio. Aby to zrobić, uruchom Instalatora programu Visual Studio, wybierz pozycję **Modyfikuj**i wybierz kartę **poszczególne składniki** . Przewiń w dół do sekcji **Narzędzia kodu** , wybierz pozycję **rozszerzenie GitHub dla programu Visual Studio**, a następnie wybierz pozycję **Modyfikuj**.
