# <a name="update-visual-studio-2017-to-the-most-recent-release"></a>Aktualizacja do najnowszej wersji programu Visual Studio 2017 r.

Firma Microsoft zachęca do aktualizacji do maksymalnie [najnowszej wersji](/visualstudio/releasenotes/vs2017-relnotes) programu Visual Studio 2017 tak, aby zawsze uzyskać najnowsze funkcje, poprawki i ulepszenia.

A jeśli chcesz wypróbować niczego przed jego wydania, należy wziąć pod uwagę pobieranie [wersja preview](/visualstudio/releasenotes/vs2017-preview-relnotes) kolejnej wersji zbyt.

> [!IMPORTANT]
> Musisz zalogować się przy użyciu konta z uprawnieniami administracyjnymi, aby zainstalować, zaktualizować lub zmodyfikować Visual Studio. Aby uzyskać więcej informacji, zobacz [uprawnienia użytkownika i programu Visual Studio](../ide/user-permissions-and-visual-studio.md).

## <a name="update-visual-studio-2017-version-156-or-later"></a>Aktualizacja programu Visual Studio 2017 wersji 15.6 lub nowszej

Firma Microsoft, zostały uproszczone instalacji i zaktualizuj środowiska, aby ułatwić do użycia bezpośrednio w środowisku IDE. Poniżej przedstawiono sposób aktualizacji z wersji 15,6 i nowszych do nowszej wersji programu Visual Studio.

### <a name="use-the-notifications-hub"></a>Użyj Centrum powiadomień

Jeśli jest dostępna aktualizacja, brak odpowiednich flagę powiadomienia w programie Visual Studio.

1. Zapisz swoją pracę.

2. Wybierz flagę powiadomienia, aby otworzyć **powiadomienia** koncentratora, a następnie wybierz aktualizację, którą chcesz zainstalować.

  ![Aktualizacja programu Visual Studio 2017 przy użyciu Centrum powiadomień](media/vs-install-notifications-hub-15dot6.png "Centrum powiadomień w programie Visual Studio 2017 r.")

3. Gdy **aktualizacji** zostanie otwarte okno dialogowe, wybierz **Aktualizuj**.

    ![Aktualizacja programu Visual Studio 2017 za pomocą okna dialogowego aktualizacja Centrum powiadomień](media/vs-update-now-from-notifications-hub.png "okno dialogowe aktualizacji w Centrum powiadomień w programie Visual Studio")

     Otwiera okno dialogowe kontroli dostępu użytkownika, wybierz polecenie **tak**. Następnie okno dialogowe "Czekaj" może zostać otwarta przez chwilę, a następnie Instalator programu Visual Studio otworzy rozpoczęcia aktualizacji.

     ![Nowe środowisko Instalator programu Visual Studio w wersji 15,6](media/visual-studio-15dot6-installer.png "nowe środowisko Instalator programu Visual Studio w wersji 15,6")

     Nadal aktualizacji. Po zakończeniu ponownego uruchomienia programu Visual Studio.

     > [!NOTE]
     > Po uruchomieniu programu Visual Studio w trybie administratora, należy ręcznie ponownie program Visual Studio po aktualizacji.

### <a name="use-the-ide"></a>Za pomocą środowiska IDE

Możesz sprawdzić dostępność aktualizacji i zainstaluj aktualizację na pasku menu programu Visual Studio.

1. Zapisz swoją pracę.

2. Wybierz **pomocy** > **Sprawdź aktualizacje**.

     ![Menu Pomoc w programie Visual Studio wersja 15,6](media/vs-help-menu-check-for-updates.png "menu Pomoc w programie Visual Studio wersja 15,6")

3. Gdy **aktualizacji** zostanie otwarte okno dialogowe, wybierz **Aktualizuj**.

   Aktualizacja będzie kontynuowana zgodnie z opisem w poprzedniej sekcji, a następnie Visual Studio ponownego uruchomienia po pomyślnym ukończeniu aktualizacji.

   > [!NOTE]
   > Po uruchomieniu programu Visual Studio w trybie administratora, należy ręcznie ponownie program Visual Studio po aktualizacji.

### <a name="use-the-visual-studio-installer"></a>Użyj Instalatora programu Visual Studio

Jak w starszych wersjach programu Visual Studio 2017 Instalator programu Visual Studio służy do instalowania aktualizacji.

1. Zapisz swoją pracę.

2. Otwórz Instalator. Instalator programu Visual Studio może wymagać aktualizacji przed kontynuowaniem.

  > [!NOTE]
  > Na komputerze z systemem Windows 10, możesz znaleźć Instalatora w obszarze literę **V** jako **Instalator programu Visual Studio**, lub na podstawie litery **M** jako  **Instalator programu Microsoft Visual Studio**.

2. Na **produktu** strony w Instalatorze, poszukać dla wersji programu Visual Studio, który został zainstalowany.

3. Jeśli jest dostępna aktualizacja, zobacz **aktualizacji** przycisku. (Może potrwać kilka sekund Instalatora, aby ustalić, czy dostępna jest aktualizacja.)

  Wybierz **aktualizacji** przycisk, aby zainstalować te aktualizacje.

     ![Aktualizacja programu Visual Studio 2017 przy użyciu Instalator programu Visual Studio](media/update-visual-studio.png "aktualizacji programu Visual Studio 2017 przy użyciu Instalator programu Visual Studio")

## <a name="update-visual-studio-2017-version-155-or-earlier"></a>Aktualizacja programu Visual Studio 2017 wersji 15.5 lub starszej

Jeśli używasz starszej wersji, poniżej przedstawiono sposób zastosowania aktualizacji z programu Visual Studio 2017 wersji 15.0 za pomocą wersji 15,5 cala.

### <a name="update-by-using-the-notifications-hub"></a>Aktualizowanie za pomocą Centrum powiadomień

1. Gdy są dostępne aktualizacje, brak odpowiednich flagę powiadomienia w programie Visual Studio.

  ![Aktualizacja programu Visual Studio 2017 przy użyciu Centrum powiadomień](media/notification-flag.png "aktualizacji flagę powiadomienia w programie Visual Studio")

  Wybierz flagę powiadomienia, aby otworzyć **powiadomienia** koncentratora.

  ![Aktualizacja programu Visual Studio 2017 przy użyciu Centrum powiadomień](media/notifications-hub.png "Centrum powiadomień w programie Visual Studio")

2. Wybierz **"Programu Visual Studio Update" jest dostępny**, co spowoduje otwarcie **rozszerzenia i aktualizacje** okno dialogowe.

  ![Aktualizacja programu Visual Studio 2017 przy użyciu Centrum powiadomień](media/notifications-hub-select.png "Centrum powiadomień w programie Visual Studio")

3. W **rozszerzenia i aktualizacje** oknie dialogowym wybierz **aktualizacji** przycisku.

  ![Aktualizacja programu Visual Studio 2017 przy użyciu Centrum powiadomień](media/notifications-extensions-and-updates.png "dialogowym rozszerzenia i aktualizacje w programie Visual Studio")

#### <a name="more-about-visual-studio-notifications"></a>Więcej informacji na temat powiadomienia programu Visual Studio

Visual Studio umożliwia powiadomienie, gdy aktualizacja jest dostępna dla programu Visual Studio, sam lub wszystkie elementy, a także przypadku wystąpienia określonych zdarzeń w środowisku Visual Studio.

* Jeśli flaga powiadomienia jest żółty, Visual Studio produktu dostępna jest aktualizacja do zainstalowania.
* Flaga powiadomienia jest czerwony, ma problem z licencją.
* Po czarny flagę powiadomienia są opcjonalne i informacyjną wiadomości do przeglądania.

Wybierz flagę powiadomienia, aby otworzyć **powiadomienia** koncentratora, a następnie wybierz powiadomienia, które mają wpływ na. Lub wybrać opcję Ignoruj lub odrzucać powiadomienia.

 ![Wyświetl komunikat opcjonalne lub informacyjny w Centrum powiadomień](media/notification-flag-optional.png "opcjonalne lub informacyjne wiadomości flagę powiadomienia w programie Visual Studio")

Jeśli zostanie wybrane ignorowanie powiadomienie, Visual Studio zatrzymuje wyświetleniem. Jeśli chcesz zresetować listę ignorowanych powiadomień, wybierz **ustawienia** przycisk w Centrum powiadomień.

   ![Kliknij przycisk Ustawienia w Centrum powiadomień, aby wyświetlić opcje powiadamiania](media/vs-notifications-hub-settings-button.png "kliknij przycisk Ustawienia w Centrum powiadomień, aby wyświetlić opcje powiadamiania")

### <a name="update-by-using-the-visual-studio-installer"></a>Aktualizowanie za pomocą Instalator programu Visual Studio

1. Otwórz Instalator. Może być konieczne zaktualizowanie Instalator przed kontynuowaniem. Jeśli jest to możliwe, zostanie wyświetlony monit zrobić.

  > [!NOTE]
  > Na komputerze z systemem Windows 10, możesz znaleźć Instalatora w obszarze literę **V** jako **Instalator programu Visual Studio**, lub na podstawie litery **M** jako  **Instalator programu Microsoft Visual Studio**.

2. Na **produktu** strony w Instalatorze, poszukać dla wersji programu Visual Studio, który został zainstalowany.

3. Jeśli jest dostępna aktualizacja, zobacz **aktualizacji** przycisku. (Może potrwać kilka sekund Instalatora, aby ustalić, czy dostępna jest aktualizacja.)

  Wybierz **aktualizacji** przycisk, aby zainstalować te aktualizacje.

     ![Aktualizacja programu Visual Studio 2017 przy użyciu Instalator programu Visual Studio](media/update-visual-studio.png "aktualizacji programu Visual Studio 2017 przy użyciu Instalator programu Visual Studio")

## <a name="get-support"></a>Uzyskaj pomoc techniczną

Czasami może wystąpienia problemów. W przypadku niepowodzenia instalacji programu Visual Studio, zobacz [problemy dotyczące instalacji i uaktualniania Rozwiązywanie problemów z programu Visual Studio 2017](troubleshooting-installation-issues.md) strony. Jeśli żaden z kroki rozwiązywania problemów, można skontaktować się nam przez rozmów na żywo, aby uzyskać pomoc przy instalacji (tylko w języku angielskim). Aby uzyskać więcej informacji, zobacz [strony pomocy technicznej programu Visual Studio](https://visualstudio.microsoft.com/vs/support/#talktous).

Poniżej przedstawiono kilka więcej opcji pomocy technicznej:

* Problemy z produktu może raportować do nas za pomocą [zgłosić Problem](../ide/how-to-report-a-problem-with-visual-studio-2017.md) narzędzia, która pojawia się zarówno w Instalatorze programu Visual Studio, jak i w środowisku IDE programu Visual Studio.
* Można udostępniać sugestię produktu z nami na [UserVoice](https://visualstudio.uservoice.com/forums/121579).
* Można śledzić problemy z produktu i odpowiedzi w [Visual Studio Developer Community](https://developercommunity.visualstudio.com/).
* Można również kontaktowaniu się z nami i innymi deweloperami Visual Studio za pomocą [konwersacji programu Visual Studio w społeczności Gitter](https://gitter.im/Microsoft/VisualStudio). (Ta opcja wymaga [GitHub](https://github.com/) konta.)
