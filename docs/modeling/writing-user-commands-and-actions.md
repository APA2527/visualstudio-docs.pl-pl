# <a name="writing-user-commands-and-actions"></a>Pisanie poleceń i akcji użytkownika
Dodawanie poleceń do diagramu i odpowiedzi na gesty, takich jak przeciągania i upuszczania, kliknij przycisk, a następnie kliknij dwukrotnie zdarzenia można dostosować języka specyficznego dla domeny.

## <a name="user-actions"></a>Akcje użytkownika

-   [Porady: modyfikowanie standardowe polecenia Menu](../modeling/how-to-modify-a-standard-menu-command-in-a-domain-specific-language.md) — można dostosować zachowanie poleceń, takich jak kopiowanie i drukowanie.

-   [Porady: Dodawanie polecenia do Menu skrótów](../modeling/how-to-add-a-command-to-the-shortcut-menu.md) — tworzenie własnych poleceń, które można zaktualizować z modelu lub zasobów zewnętrznych.

-   [Porady: Dodawanie obsługi przeciągania i upuszczania](../modeling/how-to-add-a-drag-and-drop-handler.md) — Zezwalaj użytkownikom na przenoszenie elementów z jeden diagram do drugiego, w tym diagramów UML. Odwołania do modelu co można przechowywać w innym.

-   [Porady: przechwycić kliknięcie kształtu lub Dekoratora](../modeling/how-to-intercept-a-click-on-a-shape-or-decorator.md) — umożliwiają użytkownikom klikać z elementu do innego elementu, do której się odwołuje.
