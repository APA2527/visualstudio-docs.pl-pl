# <a name="writing-user-commands-and-actions"></a>Pisanie poleceń i akcji użytkownika

## <a name="user-actions"></a>Akcje użytkownika

-   [Porady: Dodawanie polecenia do Menu skrótów](../modeling/how-to-add-a-command-to-the-shortcut-menu.md) — tworzenie własnych poleceń, które można zaktualizować z modelu lub zasobów zewnętrznych.
