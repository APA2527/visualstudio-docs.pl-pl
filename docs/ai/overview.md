# <a name="overview-of-visual-studio-tools-for-ai"></a>Omówienie narzędzi programu Visual Studio dla aplikacji AI

Visual Studio Tools for AI to zintegrowane środowisko programistyczne (IDE), którego można używać do tworzenia, testowania i wdrażania rozwiązań uczenia głębokiego. Dowiedz się więcej o korzystaniu z narzędzi AI, korzystając z poniższych samouczków i przykładów.

## <a name="install-the-extension"></a>Instalowanie rozszerzenia

Zobacz [stronę Instalowanie narzędzi programu Visual Studio dla aplikacji AI,](installation.md) aby dowiedzieć się, jak pobrać i zainstalować rozszerzenie.
