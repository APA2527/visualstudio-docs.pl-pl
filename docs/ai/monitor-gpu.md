# <a name="monitoring-gpu-utilization"></a>Monitorowanie użycia procesora GPU

Aby monitorować użycie procesora GPU przez zdalne maszyny z systemem Linux:

1. W **Eksplorator serwera** rozwiń węzeł **maszyny zdalne**.
2. **Kliknij prawym przyciskiem myszy** maszynę zdalną, którą chcesz monitorować.

    ![Zrzut ekranu przedstawiający Eksplorator serwera pokazujący drzewo folderów dla narzędzi AI. Wybrana jest wersja demonstracyjna platformy Azure, a menu kontekstowe z poleceniem Pokaż mapę ciepła jest otwarte.](media/monitor-gpu/gpu-heatmap-0.png)

3. Kliknij pozycję **Pokaż mapę cieplną**.

    ![Zrzut ekranu okna Mapa cieplna narzędzi AI w Eksplorator serwera przedstawiający podsumowania, szczegóły i dane mapy cieplnej dla procesorów GPU na maszynie wirtualnej DLVM platformy Azure.](media/monitor-gpu/heatmap.png)
