# <a name="visual-studio-tools-for-ai"></a>Visual Studio Tools for AI

Visual Studio Tools for AI to rozszerzenie do tworzenia, testowania i wdrażania rozwiązań deep learning/ AI. Bezproblemowo integruje się z usługą Azure Machine Learning w celu uzyskania niezawodnych funkcji eksperymentowania, w tym między innymi przesyłania zadań przygotowywania danych i szkoleń modelu w sposób przejrzysty do różnych celów obliczeniowych. Ponadto zapewnia obsługę metryk niestandardowych i śledzenie historii uruchamiania, umożliwiając odtwarzalność i inspekcję nauki o danych. Współpraca w przedsiębiorstwie umożliwia bezpieczną pracę nad projektem z innymi osobami.

Zacznij korzystać z uczenia głębokiego za pomocą [zestawu narzędzi Microsoft Cognitive Toolkit (CNTK),](/cognitive-toolkit/)Google [TensorFlow](https://www.tensorflow.org)lub innych struktur głębokiego uczenia już dziś.

## <a name="develop-debug-and-deploy-deep-learning-models-and-ai-solutions"></a>Opracowywanie, debugowanie i wdrażanie modeli uczenia głębokiego i rozwiązań SI
Skorzystaj z funkcji zwiększających produktywność programu Visual Studio, aby przyspieszyć innowacje w zakresie si. Użyj wbudowanych funkcji edytora kodu, takich jak wyróżnianie składni, IntelliSense i automatyczne formatowanie tekstu. Aplikację uczenia głębokiego można interaktywnie przetestować w środowisku lokalnym, korzystając z debugowania krok po kroku na zmiennych lokalnych i modelach.

## <a name="scale-out-deep-learning-model-training-andor-inferencing-to-the-cloud"></a>Skalowanie w poziomie szkolenia modelu uczenia głębokiego i/lub wnioskowanie do chmury
To rozszerzenie ułatwia szkolenie modeli na komputerze lokalnym lub można przesyłać zadania do chmury przy użyciu naszej integracji z usługą Azure Machine Learning. Zadania można przesyłać do różnych celów obliczeniowych, takich jak klastry platformy Spark, maszyny wirtualne procesora GPU platformy Azure i inne.
