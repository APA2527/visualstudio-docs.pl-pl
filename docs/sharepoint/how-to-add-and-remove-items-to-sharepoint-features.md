# <a name="how-to-add-and-remove-items-to-sharepoint-features"></a>Porady: Dodawanie i usuwanie elementów do funkcji SharePoint
  Po utworzeniu rozwiązania SharePoint Visual Studio dodaje domyślne SharePoint — elementy projektu do funkcji programu. Przed wdrożeniem można dodawać i usuwać SharePoint — elementy projektu do modyfikowania funkcji programu SharePoint.  
  
## <a name="add-sharepoint-project-items-to-a-feature"></a>Dodaj do funkcji SharePoint — elementy projektu  
  
#### <a name="to-add-sharepoint-project-items-with-the-feature-designer"></a>Aby dodać przy użyciu projektanta funkcji SharePoint — elementy projektu  
  
1.  Otwórz projektanta funkcji.  
  
     Aby uzyskać więcej informacji, zobacz [porady: dostosowywanie funkcji SharePoint](../sharepoint/how-to-customize-a-sharepoint-feature.md).  
  
2.  Dodaj jeden lub więcej elementów z **elementów w rozwiązaniu** listy, aby **elementów w funkcji** listę, wykonując jedną lub więcej z następujących czynności:  
  
    -   Kliknij dwukrotnie każdy element, który chcesz dodać.  
  
    -   Wybierz element, który chcesz dodać, a następnie wybierz pozycję **Dodaj** przycisk (>).  
  
    -   Wybierz **Dodaj wszystkie** przycisk (>>).  
  
     SharePoint — elementy projektu są wyświetlane w **elementów w funkcji** listy.  
  
## <a name="remove-sharepoint-project-items-from-a-feature"></a>Usuń z funkcją SharePoint — elementy projektu  
  
#### <a name="to-remove-sharepoint-items-with-the-feature-designer"></a>Aby usunąć elementy programu SharePoint przy użyciu projektanta funkcji
  
1.  Wybierz co najmniej jeden element w **elementów w funkcji** listy.  
  
2.  Wybierz **Usuń** przycisk (<), aby usunąć jeden element w czasie, lub kliknij **Usuń wszystkie** przycisk (<<), aby usunąć wszystkie elementy.  
  
     SharePoint — elementy projektu są wyświetlane w **elementów w rozwiązaniu** listy.
