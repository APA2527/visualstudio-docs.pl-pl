# <a name="how-to-add-a-feature-to-sharepoint-solutions"></a>Porady: dodawanie funkcji do rozwiązań SharePoint
  Po utworzeniu rozwiązania SharePoint jedną funkcję i jeden pakiet zostaną dodane do rozwiązania. Możesz dodać więcej elementów funkcji do organizowania różnych typów plików programu SharePoint.  
  
### <a name="to-add-a-feature-through-solution-explorer"></a>Aby dodać funkcję za pomocą Eksploratora rozwiązań  
  
1.  W **Eksploratora rozwiązań**, otwórz menu skrótów **funkcji**, a następnie wybierz pozycję **dodawania funkcji**.
