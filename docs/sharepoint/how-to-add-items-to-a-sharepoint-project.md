# <a name="how-to-add-items-to-a-sharepoint-project"></a>Porady: dodawanie elementów do projektu SharePoint
  Rozwiązania programu SharePoint zawiera jeden lub więcej projektów, z których każdy zawiera wiele elementów projektu SharePoint. Po otwarciu lub tworzenie rozwiązań programu SharePoint, można dodać nowe lub istniejące elementy do tych projektów. Na przykład domyślnego formularza, o nazwie default.aspx wyposażone w nowych projektów przepływu pracy, ale zastąp formularz nowe lub inne formularza lub dodać innej formy ASPX.  
  
### <a name="to-add-a-new-project-item-to-a-sharepoint-solution"></a>Aby dodać nowy element projektu rozwiązania programu SharePoint  
  
1.  W [!INCLUDE[vsprvs](../sharepoint/includes/vsprvs-md.md)], Otwórz lub tworzenie rozwiązań programu SharePoint.  
  
2.  W **Eksploratora rozwiązań**, wybierz węzeł projektu.  
  
3.  Na pasku menu wybierz **projektu**, **Dodaj nowy element** do wyświetlenia **Dodaj nowy element** okno dialogowe.  
  
4.  W **zainstalowane szablony** listy, rozwiń węzeł **SharePoint** węzeł, a następnie wybierz pozycję **2010** węzła.  
  
5.  Na liście szablony elementów projektu wybierz szablon.  
  
6.  W **nazwa** polu tekstowym wprowadź nazwę, a następnie wybierz **OK** przycisku.  
  
### <a name="to-add-an-existing-project-item-to-a-sharepoint-solution"></a>Aby dodać istniejący element projektu rozwiązania programu SharePoint  
  
1.  W [!INCLUDE[vsprvs](../sharepoint/includes/vsprvs-md.md)], Otwórz lub tworzenie rozwiązań programu SharePoint.  
  
2.  W **Eksploratora rozwiązań**, wybierz węzeł projektu.  
  
3.  Na pasku menu wybierz **projektu**, **Dodaj istniejący element** do wyświetlenia **Dodaj istniejący element** okno dialogowe.  
  
4.  Przejdź do folderu, który zawiera element, który chcesz dodać, wybierz go, a następnie wybierz **Dodaj** przycisku.
