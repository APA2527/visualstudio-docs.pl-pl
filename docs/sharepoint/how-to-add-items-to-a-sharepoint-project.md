# <a name="how-to-add-items-to-a-sharepoint-project"></a>Instrukcje: Dodawanie elementów do projektu programu SharePoint
  Rozwiązania programu SharePoint zawiera jeden lub więcej projektów, z których każdy zawiera wiele elementów projektu programu SharePoint. Po otwarciu lub tworzyć rozwiązania programu SharePoint, można dodać nowych lub istniejących elementów, do tych projektów. Na przykład nowe projekty przepływu pracy dostarczane z domyślnego formularza, który nosi nazwę default.aspx, ale można zastąpić tworzące formularz nowego ani innego lub dodać innej formy ASPX.  
  
### <a name="to-add-a-new-project-item-to-a-sharepoint-solution"></a>Aby dodać nowy element projektu do rozwiązania programu SharePoint  
  
1.  W [!INCLUDE[vsprvs](../sharepoint/includes/vsprvs-md.md)]Otwórz lub Utwórz rozwiązania programu SharePoint.  
  
2.  W **Eksploratora rozwiązań**, wybierz węzeł projektu.  
  
3.  Na pasku menu wybierz **projektu** > **Dodaj nowy element** do wyświetlenia **Dodaj nowy element** okno dialogowe.  
  
4.  W **zainstalowane szablony** listy, rozwiń **SharePoint** węzła, a następnie wybierz **2010** węzła.  
  
5.  Na liście szablonów elementów projektu wybierz szablon.  
  
6.  W **nazwa** polu tekstowym wprowadź nazwę, a następnie wybierz **OK** przycisku.  
  
### <a name="to-add-an-existing-project-item-to-a-sharepoint-solution"></a>Aby dodać istniejący element projektu do rozwiązania programu SharePoint  
  
1.  W [!INCLUDE[vsprvs](../sharepoint/includes/vsprvs-md.md)]Otwórz lub Utwórz rozwiązania programu SharePoint.  
  
2.  W **Eksploratora rozwiązań**, wybierz węzeł projektu.  
  
3.  Na pasku menu wybierz **projektu** > **Dodaj istniejący element** do wyświetlenia **Dodaj istniejący element** okno dialogowe.  
  
4.  Przejdź do folderu, który zawiera element, który chcesz dodać, wybierz go, a następnie wybierz **Dodaj** przycisku.
