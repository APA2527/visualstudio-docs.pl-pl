# <a name="visual-studio-tools-for-apache-cordova"></a>Visual Studio Tools for Apache Cordova
Tworzenie aplikacji używających uruchomienia w systemach iOS, Android i Windows urządzeń i udostępniać prawie 100% kodu. Twórz ich w technologii web ulubionych. Apache Cordova umożliwia macierzysty dostęp do funkcji urządzenia, takich jak aparatu na telefonie, a najlepsze, jest wolne. Visual Studio dodaje te dodatki do jeszcze łatwiejsze do utworzenia aplikacji:  
  
-   Łatwe instalacji i konfiguracji.  
  
-   Zaawansowane debugera, aby ułatwić znajdowanie i rozwiązywanie problemów.  
  
-   Łatwy dostęp do emulatorów urządzeń.  
  
-   Projektant do zarządzania urządzeniami wtyczki.  
  
-   Projekt, który jest całkowicie zgodny z interfejsu wiersza polecenia.  
  
 Jeśli rozważasz tych narzędzi, ale nie jest gotowa do ich jeszcze, można znaleźć w naszych [funkcje](https://www.visualstudio.com/explore/cordova-vs) strony i firma Microsoft będzie więcej należy o nich.  
  
 Jeśli wszystko jest gotowe do uruchomienia, odwiedź stronę naszych [witryny sieci Web](http://taco.visualstudio.com/en-us/docs/get-started-vs-tools-apache-cordova/).
