# <a name="visual-studio-emulator-for-android"></a>Emulator programu Visual Studio dla systemu Android

Visual Studio Emulator dla systemu Android jest aplikacją klasyczną do emulacji urządzenia z systemem Android. Jest przestarzały w programie Visual Studio 2015 i nie jest obsługiwany w programie Visual Studio 2017 ani programie Visual Studio 2019. Zamiast tego zalecamy korzystanie z emulatorów Google Android. Aby wyświetlić emulator programu Visual Studio dla systemu Android instrukcje dla programu Visual Studio 2015, wybierz Visual Studio 2015 w selektorze wersji dokumentacji.

> [!IMPORTANT]
> W większości scenariuszy emulator Systemu Google Android jest zalecany do użycia zamiast emulatora programu Visual Studio dla systemu Android:
> - Visual Studio Emulator dla systemu Android nie jest obsługiwany po programie Visual Studio 2015.
> - Obrazy emulatora później niż android w wersji 6.0 nie są dostępne dla visual studio emulator dla systemu Android.
> - Google Android Emulator obsługuje teraz [Hyper-V](/xamarin/android/get-started/installation/android-emulator/hardware-acceleration#accelerating-with-hyper-v).
> - Zarówno C++ for Mobile Device Development i Visual Studio Tools for Apache Cordova współpracują z emulatorem systemu Google Android. Aby uzyskać więcej informacji, zobacz [Uruchamianie aplikacji Apache Cordova na Androida](/visualstudio/cross-platform/tools-for-cordova/run-your-app/run-app-android#google-android-emulator). (Nie trzeba już wyłączać funkcji Hyper-V, jak wyjaśniono w tym artykule).
>
> Aby uzyskać więcej informacji na temat konfigurowania i używania emulatora Google Android, zobacz [Konfiguracja emulatora systemu Android](/xamarin/android/get-started/installation/android-emulator/).
