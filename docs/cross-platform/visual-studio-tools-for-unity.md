# <a name="visual-studio-tools-for-unity"></a>Narzędzia Visual Studio Tools for Unity

![Narzędzia Visual Studio Tools for Unity](media/vstu_header.png)

Visual Studio Tools for Unity to bezpłatne rozszerzenie programu Visual Studio, które umożliwia programowi Visual Studio zaawansowane narzędzie do tworzenia gier i aplikacji międzyplatformowych za pomocą aparatu Unity.

Chociaż Edytor Unity doskonale nadaje się do pracy ze sobą w świecie, nie można pisać w nim kodu. Za pomocą Visual Studio Tools for Unity można korzystać ze znanych funkcji edytowania kodu, debugowania i produktywności programu Visual Studio, aby tworzyć skrypty edytora i gier dla projektu Unity przy użyciu języka C#, a także debugować je przy użyciu zaawansowanych możliwości debugowania programu Visual Studio.

Ale Visual Studio Tools for Unity jest więcej niż to; ma także ścisłą integrację z edytorem Unity, dzięki czemu można poświęcać mniej czasu na przełączenie się z powrotem i w czasie w celu wykonywania prostych zadań, zapewnia ulepszenia produktywności specyficzne dla aparatu Unity i umieszcza dokumentację aparatu Unity na wyskakującym rogu.

## <a name="overview"></a>Omówienie

Odwiedź stronę [Build Games for Visual Studio](https://visualstudio.microsoft.com/vs/unity-tools/) , aby zapoznać się z zaletami korzystania z programu Visual Studio do tworzenia aplikacji dla środowiska Unity, lub Kontynuuj poniżej, aby uzyskać więcej informacji na temat kluczowych funkcji Visual Studio Tools for Unity:

### <a name="compatible-with-visual-studio-community-and-bundled-with-unity"></a>Zgodne z programem Visual Studio Community i dołączone do aparatu Unity

[Program Visual Studio Community](https://visualstudio.microsoft.com/) jest dostępny bezpłatnie i jest instalowany z instalacją aparatu Unity, począwszy od aparatu Unity 2018,1. Aby uzyskać więcej informacji, zobacz [dokumentację wprowadzającą](getting-started-with-visual-studio-tools-for-unity.md) Visual Studio Tools for Unity.

### <a name="intellisense-for-unity-messages"></a>Funkcja IntelliSense dla komunikatów Unity

Uzupełnianie kodu IntelliSense umożliwia szybkie i łatwe [implementowanie komunikatów interfejsu API Unity](using-visual-studio-tools-for-unity.md#intellisense-for-unity-api-messages) , takich jak `OnCollisionEnter` parametry.

### <a name="superior-debugging"></a>Doskonałe debugowanie

Visual Studio Tools for Unity obsługuje niezawodne funkcje [debugowania](using-visual-studio-tools-for-unity.md#unity-debugging) , które są oczekiwane w programie Visual Studio:

* Ustaw punkty przerwania, w tym warunkowe punkty przerwania.
* Oceń złożone wyrażenia w okno wyrażeń kontrolnych.
* Sprawdzanie i modyfikowanie wartości zmiennych i argumentów.
* Przechodzenie do szczegółów obiektów złożonych i struktur danych.
