# <a name="work-with-a-conceptual-model-wcf-data-services"></a>Praca z modelu koncepcyjnego (usługi danych WCF)

 Modele koncepcyjne można użyć z aplikacjami usługi danych WCF. W następujących tematach opisano, jak wykonać zapytanie dotyczące danych za pośrednictwem modelu koncepcyjnego.
