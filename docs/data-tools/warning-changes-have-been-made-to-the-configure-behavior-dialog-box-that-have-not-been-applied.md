# <a name="warning-changes-have-been-made-to-the-configure-behavior-dialog-box-that-have-not-been-applied"></a>Ostrzeżenie: wprowadzono zmiany w oknie dialogowym Konfigurowanie zachowania, które nie zostały zastosowane

Ostrzeżenie. Wprowadzono zmiany w oknie dialogowym Konfigurowanie zachowania, które nie zostały zastosowane. Czy chcesz zastosować zmiany?

Okno dialogowe **Konfigurowanie zachowania** pozwala skonfigurować zachowanie `Insert`, `Update` i `Delete` dla wszystkich dostępnych klas. Ten komunikat jest wyświetlany po wybraniu nowej kombinacji **klas** i **zachowań** , a Poprzednia zmiana nie została jeszcze zastosowana.

## <a name="change-options"></a>Zmień opcje

- Aby zastosować zmianę i kontynuować, kliknij przycisk **tak**. Zmiana zostanie zastosowana do wybranej **klasy** i **zachowania**.

- Aby anulować poprzednią zmianę i kontynuować, kliknij przycisk **nie**.
