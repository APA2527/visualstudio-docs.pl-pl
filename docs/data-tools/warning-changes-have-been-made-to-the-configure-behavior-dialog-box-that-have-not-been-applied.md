# <a name="warning-changes-have-been-made-to-the-configure-behavior-dialog-box-that-have-not-been-applied"></a>Ostrzeżenie. Wprowadzono zmiany do okna dialogowego Konfigurowanie zachowania, które nie zostały zastosowane
Ostrzeżenie. Zmiany zostały wprowadzone do okna dialogowego Konfigurowanie zachowania, które nie zostały zastosowane. Czy chcesz zastosować zmiany?  
  
 **Konfigurowanie zachowania** okno dialogowe umożliwia konfigurowanie zachowania Insert, Update i Delete dla wszystkich klas dostępne. Ten komunikat jest wyświetlany po wybraniu nowej **klasy** i **zachowanie** kombinacja i poprzedniej zmiany nie jeszcze została zastosowana.  
  
### <a name="to-apply-the-change-and-continue"></a>Aby zastosować zmiany i Kontynuuj  
  
-   Kliknij przycisk **tak** można zastosować zmianę na wybranym **klasy** i **zachowanie**.  
  
### <a name="to-cancel-the-previous-change-and-continue"></a>Aby anulować poprzedniej zmiany i kontynuować  
  
-   Kliknij przycisk **nr** by anulować zmianę do wybranego **klasy** i **zachowanie**.
