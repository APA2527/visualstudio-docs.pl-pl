# <a name="the-connection-property-in-the-application-settings-file-is-missing-or-incorrect"></a>Właściwości połączenia w pliku ustawień aplikacji jest niewystarczające lub niepoprawne
Właściwości połączenia w pliku ustawień aplikacji jest niewystarczające lub niepoprawne. Parametry połączenia z pliku .dbml został użyty w tym miejscu.  
  
 Plik .dbml zawiera odwołanie do ciągu połączenia w pliku ustawień aplikacji, którego nie można odnaleźć. To jest komunikat informacyjny; Ustawienie parametrów połączenia zostanie utworzony podczas **OK** zostanie kliknięty.  
  
### <a name="to-respond-to-this-message"></a>Aby odpowiedzieć na ten komunikat  
  
-   Kliknij przycisk **OK**. Informacje o połączeniu, który jest zawarty w pliku .dbml dodaje się do ustawień aplikacji.
