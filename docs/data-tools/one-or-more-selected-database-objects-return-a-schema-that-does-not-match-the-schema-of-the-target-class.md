# <a name="one-or-more-selected-database-objects-return-a-schema-that-does-not-match-the-schema-of-the-target-class"></a>Co najmniej jeden obiekt wybranej bazy danych zwraca schemat, który nie pasuje do schematu klasy docelowej

Co najmniej jeden obiekt wybranej bazy danych zwraca schemat, który nie pasuje do schematu klasy docelowej. Nic nie zostały dodane do projektanta.

Przeciągnięcie obiektów bazy danych na istniejące klasy jednostki, danych zwróconych przez obiekt bazy danych musi być zgodna z schematu klasy docelowej jednostki. Sprawdź, czy wybrano prawidłowe obiekty i że klasa prawidłowe jednostki docelowej.

## <a name="to-correct-this-error"></a>Aby poprawić ten błąd

1. Kliknij przycisk **OK** aby odrzucić okno dialogowe.

2. Wybierz obiekt bazy danych, która zwraca dane zgodne ze schematem klasy docelowej (klasy obiektów bazy danych jest porzucana na w Projektancie obiektów relacyjnych).
