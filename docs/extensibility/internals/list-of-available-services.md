# <a name="list-of-available-services"></a>Lista dostępnych usług

[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]i Visual Studio SDK obsługują następujące usługi. Niektóre pakiety oferują własne usługi, które nie są wymienione w tym miejscu — na przykład usługi językowe nie mają identyfikatora GUID jednej usługi. Aby znaleźć identyfikator GUID usługi językowej w rejestrze, należy użyć nazwy języka.

Użyj identyfikatorów GUID usługi wymienionych w tym miejscu lub uzyskanych z innego źródła (na przykład usług językowych), aby uzyskać interfejs podstawowy lub interfejsy wyświetlane dla każdej usługi.
