# <a name="extend-solutions"></a>Rozszerzanie rozwiązań
Rozwiązanie to grupowanie jednego lub więcej projektów, które współpracują ze sobą w celu utworzenia aplikacji.

## <a name="in-this-section"></a>W tej sekcji
 Poniższe wskazówki pokazują kilka sposobów, w jaki można rozszerzyć rozwiązania.

1. [Rozszerzenie filtru Eksploratora rozwiązań](../extensibility/extending-the-solution-explorer-filter.md) wyjaśnia, jak filtrować wyświetlane przez Eksploratora rozwiązań.

2. [Zarządzanie ładowaniem projektu w rozwiązaniu](../extensibility/managing-project-loading-in-a-solution.md) wyjaśnia, jak zarządzać ładowaniem projektu, aby zwiększyć wydajność, gdy użytkownik ładuje rozwiązanie.
