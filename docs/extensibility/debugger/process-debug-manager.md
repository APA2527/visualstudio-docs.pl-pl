# <a name="process-debug-manager"></a>Menedżer debugowania procesów
Menedżer debugowania procesów (PDM) jest składnikiem programu Visual Studio, który zarządza programami i procesami, udostępniając je menedżerowi debugowania sesji i aparatom debugowania.

 Moduł PDM zarządza wszystkimi procesami, które można debugować. Aby można było debugować, program musi być zarejestrowany w pliku PDM. Ta rejestracja jest wykonywana w momencie uruchomienia programu przez port lub aparat debugowania.
