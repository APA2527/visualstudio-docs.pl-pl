# <a name="idebugportsupplier2enumports"></a>IDebugPortSupplier2::EnumPorts
Pobiera listę wszystkich portów dostarczonych przez dostawcę portu.

## <a name="syntax"></a>Składnia

```cpp
HRESULT EnumPorts( 
   IEnumDebugPorts2** ppEnum
);
```

```csharp
int EnumPorts( 
   out IEnumDebugPorts2 ppEnum
);
```

## <a name="parameters"></a>Parametry
`ppEnum`\
[na zewnątrz] Zwraca obiekt [IEnumDebugPorts2](../../../extensibility/debugger/reference/ienumdebugports2.md) zawierający listę dostarczonych portów.

## <a name="return-value"></a>Wartość zwracana
 Jeśli się `S_OK`powiedzie, zwraca ; w przeciwnym razie zwraca kod błędu.
