# <a name="adprocessid"></a>AD_PROCESS_ID
Określa identyfikator procesu, który może być identyfikator systemu lub identyfikatora GUID.  
  
## <a name="syntax"></a>Składnia  
  
```cpp  
typedef struct _AD_PROCESS_ID {  
   AD_PROCESS_ID_TYPE ProcessIdType;  
   union {  
      DWORD dwProcessId;   
      GUID  guidProcessId;   
      DWORD dwUnused;   
   } ProcessId;  
} AD_PROCESS_ID;  
```  
  
```csharp  
public struct AD_PROCESS_ID {  
   AD_PROCESS_ID_TYPE ProcessIdType;  
   DWORD              dwProcessId;   
   GUID               guidProcessId;   
   DWORD              dwUnused;   
};  
```  
  
## <a name="members"></a>Elementy członkowskie  
 `ProcessIdType`  
 Wartość z zakresu od [AD_PROCESS_ID_TYPE](../../../extensibility/debugger/reference/ad-process-id-type.md) wyliczenie opisujące jak interpretować `ProcessId` Unii (lub dla zarządzanego kodu, który element członkowski struktury, aby uzyskać dostęp do).  
  
 dwProcessId  
 Identyfikator procesu jako wartość z systemu.  
  
 guidProcessId  
 Identyfikator procesu jako identyfikatora GUID.  
  
 dwUnused  
 Dopełnienie.
