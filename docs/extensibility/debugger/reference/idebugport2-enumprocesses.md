# <a name="idebugport2enumprocesses"></a>IDebugPort2::EnumProcesses
Zwraca listę wszystkich procesów uruchomionych na porcie.  
  
## <a name="syntax"></a>Składnia  
  
```cpp  
HRESULT EnumProcesses(   
   IEnumDebugProcesses2** ppEnum  
);  
```  
  
```csharp  
int EnumProcesses(   
   out IEnumDebugProcesses2 ppEnum  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `ppEnum`  
 [out] Zwraca [IEnumDebugProcesses2](../../../extensibility/debugger/reference/ienumdebugprocesses2.md) obiekt, który zawiera listę wszystkich procesów uruchomionych na porcie.  
  
## <a name="return-value"></a>Wartość zwracana  
 Jeśli to się powiedzie, zwraca `S_OK`; w przeciwnym razie zwraca kod błędu.
