# <a name="idebuggenericfieldinstance"></a>IDebugGenericFieldInstance
Reprezentuje wystąpienie pola dla typu ogólnego kodu zarządzanego.

## <a name="syntax"></a>Składnia

```
IDebugGenericFieldInstance : IUnknown
```

## <a name="methods"></a>Metody
 Ten interfejs implementuje następujące metody:

|Metoda|Opis|
|------------|-----------------|
|[GetTypeArguments](../../../extensibility/debugger/reference/idebuggenericfieldinstance-gettypearguments.md)|Pobiera argumenty parametru typu dla tego wystąpienia.|
|[TypeArgumentCount](../../../extensibility/debugger/reference/idebuggenericfieldinstance-typeargumentcount.md)|Zwraca liczbę argumentów parametru typu dla tego wystąpienia.|
