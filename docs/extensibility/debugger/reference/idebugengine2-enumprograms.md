# <a name="idebugengine2enumprograms"></a>IDebugEngine2::EnumPrograms
Pobiera listę wszystkich programów debugowanych przez aparat debugowania (DE).

## <a name="syntax"></a>Składnia

```cpp
HRESULT EnumPrograms( 
   IEnumDebugPrograms2** ppEnum
);
```

```csharp
int EnumPrograms( 
   out IEnumDebugPrograms2 ppEnum
);
```

## <a name="parameters"></a>Parametry
`ppEnum`\
[na zewnątrz] Zwraca obiekt [IEnumDebugPrograms2,](../../../extensibility/debugger/reference/ienumdebugprograms2.md) który zawiera listę wszystkich programów debugowanych przez DE.

## <a name="return-value"></a>Wartość zwracana
 Jeśli się `S_OK`powiedzie, zwraca ; w przeciwnym razie zwraca kod błędu.
