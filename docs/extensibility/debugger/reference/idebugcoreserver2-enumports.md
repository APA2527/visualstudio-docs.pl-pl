# <a name="idebugcoreserver2enumports"></a>IDebugCoreServer2::EnumPorts
Pobiera listę wszystkich dostępnych portów.

## <a name="syntax"></a>Składnia

```cpp
HRESULT EnumPorts( 
   IEnumDebugPorts2** ppEnum
);
```

```csharp
int EnumPorts( 
   out IEnumDebugPorts2 ppEnum
);
```

## <a name="parameters"></a>Parametry
`ppEnum`\
[na zewnątrz] Zwraca obiekt [IEnumDebugPorts2](../../../extensibility/debugger/reference/ienumdebugports2.md) zawierający listę wszystkich portów od wszystkich dostawców portów.

## <a name="return-value"></a>Wartość zwracana
 Jeśli się `S_OK`powiedzie, zwraca ; w przeciwnym razie zwraca kod błędu.
