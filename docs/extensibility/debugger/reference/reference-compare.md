# <a name="referencecompare"></a>REFERENCE_COMPARE
Określa typ porównanie dla odwołań.

## <a name="syntax"></a>Składnia

```cpp
enum enum_REFERENCE_COMPARE { 
   REF_COMPARE_EQUAL        = 0x0001,
   REF_COMPARE_LESS_THAN    = 0x0002,
   REF_COMPARE_GREATER_THAN = 0x0003
};
typedef DWORD REFERENCE_COMPARE;
```

```csharp
public enum enum_REFERENCE_COMPARE { 
   REF_COMPARE_EQUAL        = 0x0001,
   REF_COMPARE_LESS_THAN    = 0x0002,
   REF_COMPARE_GREATER_THAN = 0x0003
};
```

## <a name="members"></a>Elementy członkowskie
 REF_COMPARE_EQUAL określa porównanie równości.

 REF_COMPARE_LESS_THAN określa mniej-niż porównania.

 REF_COMPARE_GREATER_THAN określa większą-niż porównania.

## <a name="remarks"></a>Uwagi
 Przekazywany jako argument do [porównania](../../../extensibility/debugger/reference/idebugreference2-compare.md) metody.

 Przestrzeń nazw: Microsoft.VisualStudio.Debugger.Interop
