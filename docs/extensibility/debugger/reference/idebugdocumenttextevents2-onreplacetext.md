# <a name="idebugdocumenttextevents2onreplacetext"></a>IDebugDocumentTextEvents2::onReplaceText
Powiadamia pakiet debugowania, że tekst został zastąpiony w dokumencie.

## <a name="syntax"></a>Składnia

```cpp
HRESULT onReplaceText( 
   TEXT_POSITION pos,
   DWORD         dwNumToReplace
);
```

```csharp
int onReplaceText( 
   enum_TEXT_POSITION pos,
   uint               dwNumToReplace
);
```

## <a name="parameters"></a>Parametry
`pos`\
[w] [TEXT_POSITION](../../../extensibility/debugger/reference/text-position.md) wskazuje miejsce, w którym tekst został zastąpiony.

`dwNumToReplace`\
[w] Określa liczbę znaków tekstu, które zostały zastąpione.

## <a name="return-value"></a>Wartość zwracana
 Jeśli się `S_OK`powiedzie, zwraca ; w przeciwnym razie zwraca kod błędu.
