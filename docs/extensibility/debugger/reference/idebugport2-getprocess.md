# <a name="idebugport2getprocess"></a>IDebugPort2::GetProcess
Pobiera określony proces uruchomiony na porcie.

## <a name="syntax"></a>Składnia

```cpp
HRESULT GetProcess( 
   AD_PROCESS_ID    ProcessId,
   IDebugProcess2** ppProcess
);
```

```csharp
int GetProcess( 
   AD_PROCESS_ID      ProcessId,
   out IDebugProcess2 ppProcess
);
```

## <a name="parameters"></a>Parametry
`ProcessId`\
[w] Struktura [AD_PROCESS_ID](../../../extensibility/debugger/reference/ad-process-id.md) określająca identyfikator procesu.

`ppProcess`\
[na zewnątrz] Zwraca obiekt [IDebugProcess2](../../../extensibility/debugger/reference/idebugprocess2.md) reprezentujący proces.

## <a name="return-value"></a>Wartość zwracana
 Jeśli się `S_OK`powiedzie, zwraca ; w przeciwnym razie zwraca kod błędu.
