# <a name="visual-studio-debugging-samples"></a>Przykłady debugowania programu Visual Studio
Dwa przykłady pokazują, jak rozszerzyć debuger programu Visual Studio przy użyciu [!INCLUDE[vsipsdk](../../extensibility/includes/vsipsdk_md.md)].  
  
-   Przykładowe ewaluatora wyrażenia debugera, zarządzane języków znajduje się w [!INCLUDE[vsipsdk](../../extensibility/includes/vsipsdk_md.md)]. W [!INCLUDE[vsipsdk](../../extensibility/includes/vsipsdk_md.md)] okna, kliknij przycisk **przykłady** , kliknij pozycję **wszystkie** lub **C#** na liście języków, a następnie kliknij przycisk **debugera**.  
  
-   Visual Studio debugowania aparatu próbki, natywnego języków, jest dostępna w galerii kodu MSDN. Aby uzyskać więcej informacji, zobacz tę stronę w witrynie sieci Web firmy Microsoft: [próbki z aparatu debugowania programu Visual Studio](http://go.microsoft.com/fwlink/?LinkId=150236).
