# <a name="debug-session"></a>Sesji debugowania
Pod względem architektury debugera **sesji**:  
  
-   Jest zestaw wszystkich procesów powiązanych lub niepowiązanych, który uruchomienia lub podłączone do pojedynczego wystąpienia IDE.  
  
-   Jest implementowany przez menedżera sesji debugowania (SDM). Aby uzyskać więcej informacji, zobacz [sesji debugowania Menedżera](../../extensibility/debugger/session-debug-manager.md).
