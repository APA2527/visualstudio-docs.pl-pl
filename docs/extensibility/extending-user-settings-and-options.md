# <a name="extend-user-settings-and-options"></a>Rozszerzanie ustawień i opcji użytkownika
W programie Visual Studio istnieją dwa główne sposoby zachowania preferencji użytkownika. Strony**Opcje** **narzędzi** > umożliwiają użytkownikom ustawianie własnych wartości dla wielu różnych rodzajów ustawień, zarówno ogólnych, jak i specyficznych dla języka. Ustawienia użytkownika umożliwiają użytkownikom określenie ogólnych właściwości konfiguracji programu Visual Studio. Aby uzyskać więcej informacji na temat stron opcji, zobacz [Opcje i opcje stron](../extensibility/internals/options-and-options-pages.md). Aby uzyskać więcej informacji o ustawieniach użytkownika, zobacz [Obsługa ustawień użytkownika](../extensibility/internals/support-for-user-settings.md).

- [Tworzenie strony z opcjami](../extensibility/creating-an-options-page.md) W tym artykule wyjaśniono, jak utworzyć stronę opcji.

- [Tworzenie kategorii ustawień](../extensibility/creating-a-settings-category.md) W tym artykule wyjaśniono, jak utworzyć kategorię ustawień.

- [Korzystanie ze sklepu z ustawieniami](../extensibility/using-the-settings-store.md) W tym artykule wyjaśniono, jak korzystać z magazynu ustawień.

- [Pobierz informacje o usłudze z magazynu ustawień](../extensibility/getting-service-information-from-the-settings-store.md) W tym artykule wyjaśniono, jak uzyskać dostępne usługi z magazynu ustawień.

- [Zapis do magazynu ustawień użytkownika](../extensibility/writing-to-the-user-settings-store.md) W tym artykule wyjaśniono, jak zapisywać w magazynie ustawień użytkownika.
