# <a name="visual-studio-user-experience-guidelines"></a>Wskazówki dotyczące czynności użytkownika dla programu Visual Studio
Wskazówki dotyczące programu Visual Studio użytkownika czynności są przeznaczone dla osób projektowanie nowych funkcji dla programu Visual Studio. Niniejsze wytyczne zawierają informacje o typowych modeli użytkownika i wzorce interakcji, aby tworzyć środowiska użytkownika, które są bezproblemową i spójną środowiska programistycznego projektantów nowe interfejsy użytkownika (UI).  
  
 Tworzenie oprogramowania dla średnich produktów Microsoft zapoznanie się ze wskazówkami dostarczonymi przez system Windows. Istnieją trzy ważnych zasobów pod uwagę:  
  
-   [Windows User Experience Interaction Guidelines](https://msdn.microsoft.com/en-us/library/aa511258.aspx). Te wytyczne mają podstawowej wiedzy na temat działania pulpitu systemu Windows i wyglądu.  
  
-   Te wytyczne programu Visual Studio, które są specyficzne dla platformy wskazówki, które zastępują ogólne wytyczne Windows wszędzie tam, gdzie należy w naszym środowisku.  
  
-   [Wzorców i przewodniki dotyczące aplikacji ze Sklepu Windows](https://dev.windows.com/en-us/design/interaction-ux) odwołanie pojawiających się wzorców, które mogą być używane w niektórych przypadkach w programie Visual Studio.  
  
 Niniejsze wytyczne zapewniają informacje o tym, co należy zrobić podczas projektowania dla programu Visual Studio, a w niektórych przypadkach informacje dotyczące sposobów jest również uwzględniony. Aby uzyskać szczegółowe informacje o implementacji i Dowiedz się więcej o określonych interfejsów programowania aplikacji (API), które umożliwiają pisanie kodu dla rozwiązań, które integrują Visual Studio zintegrowane środowisko programistyczne (IDE), należy użyć [Zestaw Software Development Kit (SDK) z programu visual Studio](https://msdn.microsoft.com/en-us/library/bb166441.aspx).
