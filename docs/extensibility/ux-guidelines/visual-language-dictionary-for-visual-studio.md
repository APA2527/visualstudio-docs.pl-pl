# <a name="visual-language-dictionary-for-visual-studio"></a>Słownik języka Visual dla programu Visual Studio
Visual Studio będzie korzystać w celu zilustrowania koncepcji, które są blokami konstrukcyjnymi jego nadruków i innych obrazów wspólnego języka visual. Podczas projektowania nowego ikony, umożliwia zidentyfikowanie pojęcia, które chcesz przekazać własny obraz tego słownika.  
  
## <a name="general-guidelines"></a>Ogólne wskazówki
  
-   Te pojęcia visual należy użyć spójnie z ich definicją słownika. Istnieje ryzyko w przeciwnym razie skomplikowana użytkowników, którzy oczekują, że typowe elementy mają znaczenie spójne.  
  
-   Wybierz tylko najbardziej odpowiednie lub ważne pojęcia do użycia w ikony, dzięki czemu obrazów jest rzeczowy i łatwe rozróżnienie.  
  
-   Użyj tylko jednego lub dwóch pojęcia w każdego obrazu.  
  
-   Aby uzyskać więcej strategii dla projektowania ikon, zobacz [projektu ikona](../../extensibility/ux-guidelines/images-and-icons-for-visual-studio.md#BKMK_IconDesign).
