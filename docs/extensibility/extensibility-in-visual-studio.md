# <a name="extensibility-in-visual-studio"></a>Rozszerzalność w programie Visual Studio
Visual Studio zapewnia wiele opcji rozszerzalności. Można tworzyć własne zestawy SDK, skorzystaj z systemu Windows, Windows Phone i zestawy SDK platformy Azure, (który jest instalowany jako część programu Visual Studio) i rozszerzenia programu Visual Studio, sam.  
  
## <a name="extend-visual-studio"></a>Rozszerzanie funkcjonalności programu Visual Studio  
 Visual Studio SDK umożliwia rozszerzanie o niemal każdej części programu Visual Studio: polecenia, menu systemu windows, edytory i projektów. Aby dowiedzieć się więcej, zobacz [programu Visual Studio SDK](../extensibility/visual-studio-sdk.md).  
  
## <a name="create-your-own-sdks"></a>Tworzenie własnych zestawów SDK  
 Dowiedz się, jak utworzyć, pakowanie i wdrażanie własnych zestawów SDK platformy i rozszerzenia: [tworzenie zestaw Software Development Kit](../extensibility/creating-a-software-development-kit.md).  
  
## <a name="roslyn-extensibility"></a>Rozszerzalność Roslyn  
 Dowiedz się, jak używać Roslyn dla rozszerzeń: [rozszerzalności platformy kompilatora .NET ("Roslyn")](../extensibility/dotnet-compiler-platform-roslyn-extensibility.md).
