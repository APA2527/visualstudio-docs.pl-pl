 Niektóre funkcje nie może całkowicie usunięte. Na przykład nie będzie żadnych pakietu, który można wyrejestrować usunąć **widoku klasy** i jego skojarzony menu, opcje i usług. **Widoku klasy** okna są udostępniane przez pakiet programu Visual Studio środowiska, które są także inne najważniejsze funkcje IDE. Jeśli chcesz usunąć **widoku klasy**, może także wymagać usunięcia **Znajdź i Zamień**, środowisko **opcje** stron, **okno polecenia**i **dane wyjściowe** okna.
