# <a name="objects"></a>Obiekty
Poniższe informacje referencyjne wyjaśniają obiekty, które są używane do tworzenia wystąpienia edytora tekstu podstawowego.

## <a name="in-this-section"></a>W tej sekcji
- [Obiekt vscodewindow](../extensibility/vscodewindow-object.md) W tym artykule opisano obiekt okna kodu, okno dokumentu specjalistycznego, które można ująć jeden lub więcej widoków tekstu.

- [Obiekt vscodewindowmanager](../extensibility/vscodewindowmanager-object.md) W tym artykule opisano obiekt menedżera okien kodu, który jest implementowany przez usługę języka i jest odpowiedzialny za zarządzanie ozdobami.

- [OBIEKT VSTextBuffer](../extensibility/vstextbuffer-object.md) W tym artykule opisano obiekt buforu tekstu, który reprezentuje strumień tekstu Unicode, dane dla pliku.

- [Obiekt VSTextView](../extensibility/vstextview-object.md) W tym artykule opisano widok tekstu, okno, które pozwala użytkownikom wyświetlać i edytować tekst Unicode buforu tekstu.
