# <a name="installshield-limited-edition"></a>InstallShield Limited Edition
Za pomocą InstallShield Limited Edition, można utworzyć pliku konfiguracji i rozpowszechnienie go do użytkowników, dzięki czemu mogą zainstalować aplikację lub składnik bez połączenia z siecią. InstallShield Limited Edition jest bezpłatna dla użytkowników wersji Visual Studio Professional i Enterprise. Zastępuje ona technologii Instalatora systemu Windows, która nie obsługuje programu Visual Studio. Alternatywnie można rozpowszechniać aplikacje i składniki przy użyciu technologii ClickOnce, która wymaga łączności sieciowej. Zobacz [zabezpieczenia ClickOnce i wdrażania](../deployment/clickonce-security-and-deployment.md).  
  
> [!NOTE]
>  Możesz kontynuować korzystanie z Instalatora Windows projektów utworzonych w starszych wersjach programu Visual Studio, instalując rozszerzenia projektów Instalator programu Visual Studio. Zobacz [Instalator Visual Studio projektów rozszerzenia](http://blogs.msdn.com/b/visualstudio/archive/2014/04/17/visual-studio-installer-projects-extension.aspx).  
  
### <a name="to-enable-installshield-limited-edition"></a>Aby włączyć InstallShield Limited Edition  
  
1.  Na pasku menu wybierz **pliku**, **nowy**, **projektu**.  
  
2.  W **nowy projekt** okna dialogowego rozwiń **inne typy projektów** węzeł, a następnie wybierz pozycję **instalacji i wdrażania** węzła.  
  
3.  Na liście szablonów wybierz **Włącz InstallShield Limited Edition**, a następnie wybierz pozycję **OK** przycisku.  
  
4.  W oknie przeglądarki zostanie otwarty, zapoznaj się z instrukcjami, a następnie wybierz **przejdź do witryny sieci web pobierania** łącza.
