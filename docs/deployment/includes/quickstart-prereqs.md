## <a name="prerequisites"></a>Wymagania wstępne

::: moniker range=">=vs-2019"

* [Visual Studio 2019](https://visualstudio.microsoft.com/downloads) zainstalowany z odpowiednimi obciążeniami dla wybranego języka:
  * ASP.NET: ASP.NET **i tworzenie stron internetowych**
  * Python: **Rozwój Pythona**
  * Node.js: **Tworzenie node.js**
::: moniker-end
::: moniker range="vs-2017"
* [Visual Studio 2017](https://visualstudio.microsoft.com/vs/older-downloads/?utm_medium=microsoft&utm_source=docs.microsoft.com&utm_campaign=vs+2017+download) zainstalowany z odpowiednimi obciążeniami dla wybranego języka:
  * ASP.NET: ASP.NET **i tworzenie stron internetowych**
  * Python: **Rozwój Pythona**
  * Node.js: **Tworzenie node.js**
::: moniker-end

* Projekt ASP.NET, ASP.NET Core, Python lub Node.js. Jeśli nie masz jeszcze projektu, wybierz opcję poniżej:
  * ASP.NET Core: Follow [Quickstart: Użyj programu Visual Studio, aby utworzyć pierwszą ASP.NET core aplikacji sieci web](../../ide/quickstart-aspnet-core.md), lub użyj **pliku** > **nowego projektu**, wybierz Visual **C#** > **.NET Core**, a następnie wybierz ASP.NET Core Web **Application**. Po wyświetleniu monitu wybierz szablon **Aplikacji sieci Web (Model-View-Controller),** upewnij się, że nie jest **zaznaczone żadne uwierzytelnianie,** a następnie wybierz przycisk **OK**.
  * Python: Śledź [Szybki start: Utwórz pierwszą aplikację internetową Języka Python za pomocą programu Visual Studio](../../ide/quickstart-python.md)lub użyj programu **File** > **New Project**, wybierz **python**, a następnie wybierz **flask Web Project**.
  * Node.js: Follow [Szybki start: Użyj programu Visual Studio, aby utworzyć pierwszą aplikację Node.js](../../ide/quickstart-nodejs.md), lub użyj **pliku** > **nowego projektu**, wybierz **JavaScript**, a następnie wybierz **pozycję Pusta aplikacja sieci Web Node.js**.

* Upewnij się, że projekt jest skompilowy przy użyciu polecenia menu **kompilacja > rozwiązania kompilacji** przed podjęciem kroków wdrażania.
