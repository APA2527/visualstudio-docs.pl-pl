## <a name="prerequisites"></a>Wymagania wstępne

::: moniker range=">=vs-2019"

* [Visual Studio 2019](https://visualstudio.microsoft.com/downloads) zainstalowany z odpowiednimi obciążeniami dla wybranego języka:
  * ASP.NET: ASP.NET **i tworzenie stron internetowych**
::: moniker-end
::: moniker range="vs-2017"
* [Visual Studio 2017](https://visualstudio.microsoft.com/vs/older-downloads/?utm_medium=microsoft&utm_source=docs.microsoft.com&utm_campaign=vs+2017+download) zainstalowany z odpowiednimi obciążeniami dla wybranego języka:
  * ASP.NET: ASP.NET **i tworzenie stron internetowych**
::: moniker-end

* Subskrypcja platformy Azure. Jeśli nie masz jeszcze subskrypcji, [zarejestruj się za darmo,](https://azure.microsoft.com/free/dotnet/)co obejmuje 200 dolarów w kredyt na 30 dni i 12 miesięcy popularnych bezpłatnych usług.

* Projekt ASP.NET Core. Jeśli jeszcze go nie masz, postępuj zgodnie z [przewodnikiem Szybki start: użyj programu Visual Studio, aby utworzyć pierwszą ASP.NET podstawową aplikację sieci Web](../../ide/quickstart-aspnet-core.md)lub użyj **aplikacji File** > **New Project**, wybierz visual **C#** > **.NET Core**, a następnie wybierz ASP.NET Core Web **Application**. Po wyświetleniu monitu wybierz szablon **Aplikacji sieci Web (Model-View-Controller),** upewnij się, że nie jest **zaznaczone żadne uwierzytelnianie,** a następnie wybierz przycisk **OK**.

* Upewnij się, że projekt jest skompilowy przy użyciu polecenia menu **kompilacja > rozwiązania kompilacji** przed podjęciem kroków wdrażania.
