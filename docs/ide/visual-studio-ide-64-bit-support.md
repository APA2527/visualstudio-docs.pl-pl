# <a name="visual-studio-ide-64-bit-support"></a>Obsługa programu Visual Studio IDE 64-bit

Program Visual Studio umożliwia konfigurowanie aplikacji przeznaczonych dla różnych platform, w tym na platformach 64-bitowych. Aby uzyskać więcej informacji o obsłudze platformy 64-bitowego w programie Visual Studio, zobacz [64-bitowe aplikacje](/dotnet/framework/64-bit-apps).

## <a name="deploy-a-64-bit-application"></a>Wdrażanie aplikacji 64-bitowej

[Wdrożenie wymagań wstępnych dla aplikacji 64-bitowych](../deployment/deploying-prerequisites-for-64-bit-applications.md) zawiera listę redystrybucyjnych, których można użyć jako wymagań wstępnych w przypadku instalacji aplikacji 64-bitowej.

## <a name="configure-projects-as-64-bit-applications"></a>Skonfiguruj projekty jako aplikacje 64-bitowe

[Instrukcje: Konfigurowanie projektów na platformach docelowych](../ide/how-to-configure-projects-to-target-platforms.md) omawia Konfigurowanie projektów do skompilowania jako aplikacje 64-bitowe.
