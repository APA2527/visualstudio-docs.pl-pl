# <a name="customize-the-editor"></a>Dostosowywanie edytora

Można dostosować formatowania, wszystkich kartach, czcionki i kolory, IntelliSense i wiele innych funkcji edytora tekstu, we wszystkich językach albo w określonym języku programowania. Tematy w tej sekcji opisano, jak wykonać niektóre z tych dostosowań.
