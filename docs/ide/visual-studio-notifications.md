# <a name="visual-studio-notifications"></a>Powiadomienia usługi Visual Studio

Visual Studio ma system powiadomień, który powiadamia o różnych zdarzeń, takie jak następujące:

- Aktualizacja jest dostępna dla programu Visual Studio lub składnik programu Visual Studio.

- Zdarzenie środowiska, które mogą być zainteresowane wystąpił.

Visual Studio umożliwia powiadomienie, gdy aktualizacja jest dostępna dla programu Visual Studio, sam lub któregokolwiek ze składników, które zostały zainstalowane. Umożliwia także powiadomienia dotyczące niektórych rodzajów zdarzeń, które występują w środowisku Visual Studio. Ikona powiadomienia jest flagę w prawym górnym rogu paska tytułu programu Visual Studio; w przypadku czerwoną ikonę, oznacza to, że istnieje problem z licencją. Jeśli jest żółty Brak programu Visual Studio Update produktu dostępne do zainstalowania. Po "normal" lub czarnym zostanie wyświetlone komunikaty informacyjne lub opcjonalne, takie jak awarie, dla których możesz znaleźć obejście tego problemu. Możesz kliknąć ikonę, aby otworzyć **Centrum powiadomień** i wybrać wiersze do działania na. Można wybrać w celu jednorazowego odrzucić na powiadomienie lub zawsze Ignoruj powiadomienie (przez umieszczenie na nim wskaźnika) i Visual Studio nigdy nie wyświetli tej wiadomości ponownie.

Jeśli wybierzesz zawsze Ignoruj powiadomienie, Visual Studio będzie przestanie być wyświetlana go. Można zresetować listę ignorowanych powiadomień w [powiadomienia, środowisko, opcje — Okno dialogowe](../ide/reference/notifications-environment-options-dialog-box.md).
