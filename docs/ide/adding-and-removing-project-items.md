# <a name="adding-and-removing-project-items"></a>Dodawanie i usuwanie elementów projektu
Można dodawać i usuwać projektów i elementów projektu przez kliknięcie prawym przyciskiem myszy węzeł projektu lub rozwiązania w **Eksploratora rozwiązań**i wybierając polecenie **Dodaj** lub **Usuń**. Usuwanie projektu z rozwiązania lub elementu projektu nie powoduje usunięcia go z dysku lub źródło kontroli dopiero po wybraniu opcji trwale usunąć.
