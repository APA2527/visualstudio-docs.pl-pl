# <a name="retain-topics-for-later-use"></a>Zaznaczanie tematów do późniejszego użycia
Po zlokalizowaniu przydatne tematu można do niego dostęp później go kopiować i wklejając je w pliku tekstowym, wydrukować ją lub zapisania go jako ulubione. Na przykład można zachować zawartość w celu późniejszego użycia przez otwarcie menu skrótów, tematy i wybierając **kopiowania**, **drukowania**, **Podgląd wydruku**, i **Dodaj do ulubionych** opcje.  
  
## <a name="using-the-favorites-tab"></a>Na karcie Ulubione  
 Na **ulubione** kartę, można otworzyć menu skrótów dla wejścia, a następnie zmień nazwę, usunąć lub zorganizować wpisów w folderach.
