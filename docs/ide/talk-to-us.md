# <a name="talk-to-us"></a>Porozmawiaj z nami
Interesuje nas pomysłów poprawy naszych produktów i naszej dokumentacji. Nie zawsze możemy osobiście, ale firma Microsoft będzie upewnij się, że Twoja opinia pobiera do prawej osobie lub zespołowi.  

## <a name="i-want-to-report-a-problem-with-visual-studio"></a>Chcę zgłosić problem z programem Visual Studio  
 Jeśli używasz do problemów przy użyciu programu Visual Studio, na przykład awarii (Crash), wydajność pogarsza nieoczekiwanego zachowania i tak dalej, zgłoś problem do nas przy użyciu **zgłosić Problem** narzędzia. Po prostu kliknij ikonę opinii obok pozycji **szybkiego uruchamiania**, lub wybierz **pomocy &#124; Wyślij opinię &#124; Zgłoś Problem** z poziomu menu głównego. Aby uzyskać więcej informacji, zobacz [jak zgłosić Problem z programu Visual Studio 2017](how-to-report-a-problem-with-visual-studio-2017.md).  

## <a name="i-want-to-make-a-suggestion-about-visual-studio-features"></a>Chcę sugestię o funkcje programu Visual Studio  
 Czy jest coś, co możemy ulepszyć, prosimy o kontakt! Wybierz **Podaj sugestię** Sugeruj funkcji lub zmień program Visual Studio team. Automatycznie zostaną opublikowane swoją sugestię w [UserVoice](https://visualstudio.uservoice.com) gdzie innym głosowanie dla niego.  

### <a name="rate-this-product"></a>Oceń ten produkt  
 Wersje wstępne programu Visual Studio ma **Oceń ten produkt** element menu, który pozwala na wysyłanie informacji zwrotnych jakości kompilacji są uruchomione. Nie będzie to widoczna w wydanej wersji produktu.  

## <a name="i-want-to-contact-microsoft-support"></a>Chcę skontaktuj się z Microsoft Support  
Dla programu Visual Studio opcje pomocy technicznej, zobacz nasze [obsługuje omówienie](https://www.visualstudio.com/vs/support/) strony.

Dla innych produktów firmy Microsoft i usług, zobacz [Microsoft Support](http://go.microsoft.com/fwlink/?LinkID=99019) pomocy online.

> [!NOTE]
> Pomoc techniczna poza terenem Stanów Zjednoczonych i Kanady może się różnić. Lista kontaktów regionalnych, zobacz [na całym świecie witryn firmy Microsoft](http://www.microsoft.com/worldwide/).  

W przypadku większych organizacji, które wymagają zarządzanej pomocy technicznej udzielanej bezpośrednio przez firmę Microsoft, dostępne są umowy w ramach różnych ofert pomocy technicznej klasy Premier. Aby uzyskać więcej informacji, zobacz [Premier pomocy technicznej firmy Microsoft usługi](http://go.microsoft.com/fwlink/?LinkId=258223).  

Jeżeli produkt jest preinstalowany na nowym komputerze lub urządzeniu, producent sprzętu zapewnia pomoc i obsługę techniczną dla tego oprogramowania. Skontaktuj się bezpośrednio z producentem, aby uzyskać wsparcie.  

Usługi pomocy technicznej firmy Microsoft podlegają aktualnej cenie i warunkom. Ceny i warunki mogą ulec zmianie bez powiadomienia.  

## <a name="i-want-to-get-involved-in-the-developer-community"></a>Chcę angażowanie się w społeczność deweloperów  
 Jeśli chcesz udostępnić pytania i odpowiedzi z innymi deweloperami, możesz użyć [Visual Studio Developer Community](https://developercommunity.visualstudio.com/index.html) lokacji, [fora MSDN](http://social.msdn.microsoft.com/Forums/home), lub [StackOverflow](http://stackoverflow.com/). Możesz również wyświetlić kod z innymi deweloperami i udostępnianie własnych przykłady na [przykłady kodu dewelopera](http://code.msdn.microsoft.com/) i [CodePlex](http://www.codeplex.com/) witryn sieci Web. Odpowiedzi na opinie klientów w sekcji komentarze na każdej stronie Visual Studio na może Podziel się swoją wiedzą, koniec [docs.microsoft.com](https://docs.microsoft.com/visualstudio/).

## <a name="i-want-to-help-improve-the-visual-studio-documentation"></a>Chcę, aby pomóc udoskonalić dokumentację programu Visual Studio  
  Użyj przycisku naszych opinii (**był pomocny tę stronę?**) w dolnej części strony. Ten przycisk, można znaleźć na wszystkich stronach dokumentacji. Jeśli szukasz odpowiedzi i nie można go znaleźć, prosimy o informacje, jakie informacje, których szukasz. (Można również dodać opinię do dowolnej strony Visual Studio na [docs.microsoft.com](https://docs.microsoft.com/visualstudio/) przy użyciu sekcji komentarzy.)
