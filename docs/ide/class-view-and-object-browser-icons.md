## <a name="signal-icons"></a>Ikony sygnałowe

Następujące ikony sygnałowe dotyczą wszystkich poprzednich ikon i wskazują ich dostępność.

|Ikona|Opis|
|----------|-----------------|
|\<Brak ikony sygnału >|Public. Dostępne z dowolnego miejsca, w tym składniku i z dowolnego składnika, który odwołuje się do niej.|
|![Sygnał chronione symboli](../ide/media/vxsignal_icon_key.gif)|Chronione. Dostępne z zawierającego klasy lub typu lub te pochodzące z klasy lub typu zawierającego.|
|![Sygnał prywatnych symboli](../ide/media/vxsignal_icon_lock.gif)|Prywatny. Dostępne tylko w zawierającej klasy lub typu.|
|![Sygnał zapieczętowany symboli](../ide/media/vxsignal_icon_envelope.gif)|Zapieczętowany.|
|![Sygnał Friend&#47;wewnętrznego symboli](../ide/media/vxsignal_icon_diamond.gif)|Zaprzyjaźnione/błąd wewnętrzny. Dostępne tylko z projektu.|
|![Sygnał ikona strzałki](../ide/media/vxsignal_icon_arrow.gif)|Skrót. Skrót do obiektu.|

> [!NOTE]
> Jeśli projekt jest uwzględniona w bazie danych kontroli źródła, ikony sygnałowe dodatkowe może być wyświetlany wskazują stan kontroli źródła, takich jak zaewidencjonowany lub wyewidencjonowany.
