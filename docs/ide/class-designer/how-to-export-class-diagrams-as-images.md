# <a name="how-to-export-class-diagrams-as-images"></a>Jak: Eksportowanie diagramów klas jako obrazów

Aby wyeksportować diagram klasy utworzony na podstawie kodu w projekcie, zapisz diagram jako obraz. Jeśli zamiast tego chcesz wyeksportować diagramy klasy UML, zobacz [Eksport diagramy jako obrazy](../../modeling/export-diagrams-as-images.md).

## <a name="export-a-diagram"></a>Eksportowanie diagramu

1. Otwórz plik diagramu klas (*cd).*

2. Z menu **Diagram klasy** lub menu skrótów powierzchni diagramu wybierz polecenie **Eksportuj diagram jako obraz**.

3. Wybierz diagram.

4. Wybierz odpowiedni format.

5. Wybierz **pozycję Eksportuj,** aby zakończyć eksportowanie.

Aby automatycznie zaktualizować eksportowane obrazy połączone z innych dokumentów, należy ponownie wyeksportować diagram w programie Visual Studio.
