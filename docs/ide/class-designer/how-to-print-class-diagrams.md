# <a name="how-to-print-class-diagrams"></a>Jak: Drukowanie diagramów klas

Diagram klas można wydrukować przy użyciu funkcji drukowania programu Visual Studio.

## <a name="to-print-a-class-diagram"></a>Aby wydrukować diagram klas

1. Otwórz diagram klas.

2. Kliknij polecenie **Drukuj** w menu **Plik.**

Drukuje się cały diagram klasy. Aby wydrukować w odpowiednim rozmiarze, może być konieczne dostosowanie ustawień w oknie dialogowym **Ustawienia strony.**
