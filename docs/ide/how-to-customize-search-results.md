# <a name="how-to-customize-search-results"></a>Porady: dostosowywanie wyników wyszukiwania
Można dostosować sposób wyświetlania wyników wyszukiwania w **wyszukiwania** kartę na kilka sposobów.  
  
## <a name="change-the-number-of-topics-that-appear-on-a-search-results-page"></a>Zmień liczbę tematów, które są wyświetlane na stronie wyników wyszukiwania  
 Można określić liczbę tematów, które pojawiają się na stronę wyników wyszukiwania pojedynczego.  
  
#### <a name="to-change-the-number-of-topics-displayed-at-a-time"></a>Aby zmienić liczbę tematów wyświetlane w czasie  
  
1.  Na **narzędzi**, wybierz **Opcje aplikacji Viewer**.  
  
     **Opcje aplikacji Viewer** zostanie wyświetlone okno dialogowe.  
  
2.  W **maksymalna liczba wyników wyszukiwania** wybierz 25, 50, 100 lub 200.  
  
## <a name="search-for-content-by-language"></a>Wyszukiwanie zawartości przez język  
 Dokumentacja może zawierać zawartości w języku innym niż angielski. Można ograniczyć wyniki wyszukiwania do zawartości w określonym języku. Możesz również uwzględnić żadnej zawartości w języku angielskim, który pasuje do wyszukania.  
  
> [!WARNING]
>  Ta funkcja jest włączona tylko wtedy, gdy podstawowy język zainstalowany zawartość nie jest angielski.  
  
#### <a name="to-include-english-content-alongside-content-in-another-language"></a>Aby uwzględnić zawartości w języku angielskim obok zawartości w innym języku  
  
1.  Na **narzędzi**, wybierz **Opcje aplikacji Viewer**.  
  
     **Opcje aplikacji Viewer** zostanie wyświetlone okno dialogowe.  
  
2.  Wybierz **obejmują angielskim zawartości we wszystkich kartach nawigacji i żądaniach F1** pole wyboru.
