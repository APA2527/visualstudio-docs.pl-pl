# <a name="how-to-manage-editor-windows"></a>Jak: Zarządzanie oknami edytora

Możesz pracować nad kodem w kilku lokalizacjach jednocześnie. W tym celu należy podzielić okno edytora lub otworzyć kilka wystąpień okien edytora.

> [!NOTE]
> Nie wszystkie okna edytora obsługują wiele wystąpień.

## <a name="split-an-editor-window"></a>Dzielenie okna edytora

Wystąpienie okna edytora można podzielić na dwa oddzielne widoki w celu łatwiejszej edycji.

### <a name="to-split-a-pane"></a>Aby podzielić okienko

1. Kliknij w oknie edytora, aby nadać mu fokus.

2. Z menu **Okno** wybierz polecenie **Podziel**.

     Obszar edycji dzieli się na dwa okienka oddzielone paskiem rozdzielacza. Można przewijać te okienka niezależnie, aby wyświetlać i edytować różne części aktywnego dokumentu w tym samym czasie. Wszelkie zmiany wprowadzone w jednym okienku są odzwierciedlane w drugim.

    > [!TIP]
    > Aby jedno okienko było większe od drugiego, przeciągnij pasek rozdzielacza w górę lub w dół.

### <a name="to-return-to-single-pane-view"></a>Aby powrócić do widoku z jednym okienka

- Z menu **Okno** wybierz polecenie **Usuń podział**.

## <a name="create-new-windows"></a>Tworzenie nowych okien

Można również utworzyć wiele wystąpień okna edytora. Ta funkcja umożliwia otwieranie długiego dokumentu w więcej niż jednym wystąpieniu edytora, dzięki czemu można wyświetlać i edytować różne sekcje jednocześnie w oddzielnych, pełnowymiarowych oknach edytora.

- W menu **Okno** kliknij polecenie **Nowe okno**.

   Zostanie dodane nowe wystąpienie edytora z kartami.
