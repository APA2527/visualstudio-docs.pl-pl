# <a name="visual-studio-feedback-options"></a>Opcje opinii programu Visual Studio

Po co wysyłać do nas opinie? Ponieważ traktujemy opinie klientów poważnie; to napędza wiele z tego, co robimy. Oto jak się z nami skontaktować, abyśmy mogli skierować Twoją opinię do właściwej osoby lub zespołu.

## <a name="report-a-problem"></a>zgłaszanie problemu

::: moniker range="vs-2019"

Jeśli wystąpią problemy&mdash;z programem Visual Studio,&mdash;takie jak awarie, niska wydajność lub nieoczekiwane zachowanie, poinformuj nas o tym za pomocą narzędzia **Zgłoś problem.** W programie Visual Studio wybierz ikonę opinii w prawym górnym rogu lub wybierz pozycję **Pomóż** > **wysłać opinię Zgłoś** > problem z**paska** menu.

![Ikona Wyślij opinię w programie Visual Studio 2019 IDE](./media/vs-2019/send-feedback-icon.png)

Następnie program Visual Studio otwiera interfejs [społeczności deweloperów,](https://developercommunity.visualstudio.com) w którym można zalogować się, aby zgłosić problem. Aby uzyskać więcej informacji, zobacz [Jak zgłosić problem z programem Visual Studio](how-to-report-a-problem-with-visual-studio.md).

::: moniker-end

::: moniker range="vs-2017"

Jeśli wystąpią problemy&mdash;z programem Visual Studio,&mdash;takie jak awarie, niska wydajność lub nieoczekiwane zachowanie, poinformuj nas o tym za pomocą narzędzia **Zgłoś problem.** W programie Visual Studio wybierz ikonę opinii obok pola wyszukiwania lub wybierz pozycję **Pomóż** > **wysłać opinię Zgłoś** > problem z**paska** menu.

![Ikona Wyślij opinię w programie Visual Studio 2019 IDE](./media/send-feedback-icon.png)

Następnie program Visual Studio otwiera interfejs [społeczności deweloperów,](https://developercommunity.visualstudio.com) w którym można zalogować się, aby zgłosić problem. Aby uzyskać więcej informacji, zobacz [Jak zgłosić problem z programem Visual Studio](how-to-report-a-problem-with-visual-studio.md).

::: moniker-end

## <a name="suggest-a-feature"></a>proponowanie funkcji

::: moniker range="vs-2019"

Jeśli masz pomysł lub sugestię, aby ulewiedzieć programu Visual Studio, daj nam znać za pomocą narzędzia **Zaproponuj funkcję.** W programie Visual Studio wybierz ikonę opinii w prawym górnym rogu lub wybierz pozycję **Pomóż** > **wysłać opinię** > **Zaproponuj funkcję** na pasku menu.

Następnie program Visual Studio otwiera interfejs [społeczności deweloperów,](https://developercommunity.visualstudio.com) w którym można zalogować się, aby udostępnić swój pomysł. Aby uzyskać więcej informacji, zobacz [Sugerowanie funkcji programu Visual Studio](suggest-a-feature.md).

::: moniker-end

::: moniker range="vs-2017"

Jeśli masz pomysł lub sugestię, aby program Visual Studio lepiej, daj nam znać za pomocą narzędzia **Podaj sugestię.** W programie Visual Studio wybierz ikonę opinii obok pola wyszukiwania lub wybierz pozycję **Pomóż** > **wysłać opinię** > **Udostępnij sugestię** na pasku menu.

Następnie program Visual Studio otwiera stronę [społeczność deweloperów,](https://developercommunity.visualstudio.com) na której można się zalogować, aby udostępnić swój pomysł. Aby uzyskać więcej informacji, zobacz [Sugerowanie funkcji programu Visual Studio](suggest-a-feature.md).

::: moniker-end

## <a name="improve-the-documentation"></a>Ulepszanie dokumentacji

Istnieją dwa sposoby, które mogą pomóc nam poprawić dokumentację:

* Użyj narzędzia odpowiedzi **Czy ta strona jest pomocna?**
* Użyj swojego konta GitHub, aby przekazać opinię na temat dowolnej strony programu Visual Studio w [docs.microsoft.com](/visualstudio/). Aby to zrobić, wybierz przycisk **Wyślij opinię o** > **tej stronie** u dołu dowolnej strony dokumentacji.

## <a name="contact-microsoft-support"></a>Kontaktowanie się z pomocą techniczną firmy Microsoft

Aby uzyskać informacje pomocy technicznej programu Visual Studio, zobacz [cyklu życia produktu & stronie obsługi.](/visualstudio/releases/2019/servicing/) Aby uzyskać pomoc dotyczącą innych produktów i usług firmy Microsoft, zobacz [Pomoc online firmy Microsoft.](https://support.microsoft.com/)

> [!NOTE]
> Pomoc techniczna poza terenem Stanów Zjednoczonych i Kanady może się różnić. Aby uzyskać listę kontaktów regionalnych, zobacz [Witryny firmy Microsoft na całym świecie](https://www.microsoft.com/worldwide/).

W przypadku większych organizacji, które wymagają zarządzanej pomocy technicznej bezpośrednio od firmy Microsoft, kontrakty są dostępne za pośrednictwem różnych ofert pomocy technicznej dla przedsiębiorstw. Aby uzyskać więcej informacji, zobacz [Rozwiązania pomocy technicznej firmy Microsoft dla przedsiębiorstw](https://www.microsoft.com/industry/services/support).

Jeżeli produkt jest preinstalowany na nowym komputerze lub urządzeniu, producent sprzętu zapewnia pomoc i obsługę techniczną dla tego oprogramowania. Skontaktuj się bezpośrednio z producentem, aby uzyskać wsparcie.

Usługi pomocy technicznej firmy Microsoft podlegają aktualnej cenie i warunkom. Ceny, warunki i warunki mogą ulec zmianie bez powiadomienia.
