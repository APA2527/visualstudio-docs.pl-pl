## <a name="toolbox-controls"></a>Formanty z przybornika

 Do formularza lub projektanta przy użyciu klawiatury, można dodać kontrolkę w przyborniku.

#### <a name="to-add-controls-from-the-toolbox-to-a-designer-from-the-keyboard"></a>Aby dodać formanty z przybornika do projektanta z klawiatury

1.  Na pasku menu wybierz **widoku**, **przybornika**.

2.  Użyj kombinacji klawisza Ctrl + Strzałka w górę lub Ctrl + klawiszy strzałek, aby poruszać się między sekcje w przyborniku bieżącej karcie.

3.  Użyj strzałki w górę lub **Strzałka w dół** kluczy można przenieść między formantami.

4.  Po wybraniu formantu za pomocą klawisza Enter.

 Formant został dodany do formularza lub projektanta.

## <a name="dialog-box-options"></a>Opcje okien dialogowych

 Można przechodzić między opcje w oknie dialogowym i zmienić ustawienia opcji za pomocą klawiatury.

#### <a name="to-set-dialog-box-options-from-the-keyboard"></a>Aby ustawić opcje okien dialogowych z klawiatury

1.  Użyj **kartę** lub **SHIFT + TAB** przenoszenia w górę i w dół do formantów w oknie dialogowym.

2.  Aby zmienić ustawienia opcji:

    -   Przyciski radiowe, można użyć **Strzałka w górę** i **Strzałka w dół** Aby zmienić wybór.

    -   Dla pola wyboru Użyj **spacja** zaznacz lub usuń zaznaczenie.

    -   Użyj listy rozwijanej **ALT** + **DOWNARROW** Aby wyświetlić elementy, a następnie użyć **UPARROW** i **DOWNARROW** zmiany wybrany element.

    -   W przypadku przycisków, naciśnij klawisz **ENTER** do wywołania.

    -   Dla siatki użyj klawiszy strzałek w celu przejścia. Użyj listy rozwijanej w siatkach, **SHIFT** + **ALT** + **DOWNARROW** Aby wyświetlić elementy, a następnie użyć **UPARROW**i **DOWNARROW** można zmienić wybranego elementu.

## <a name="window-and-file-navigation"></a>Okno i plik nawigacji

 IDE udostępnia kilka metod umożliwiające przechodzenie między otwarte okna narzędzia i zarządzania dokumentami, za pomocą klawiatury. Można również przenosić i dokowanie okien narzędzi w różnych lokalizacjach za pomocą klawiatury.

#### <a name="to-navigate-among-windows-and-files-in-the-ide-from-the-keyboard"></a>Aby poruszać się między systemem windows i pliki w IDE z klawiatury

-   Aby poruszać się między pliki w edytorze lub projektanta, wybierz polecenie CTRL + TAB klucze do wyświetlenia Nawigator IDE z **pliki Active** wybrane. Wybierz klawisz Enter, aby przejść do pliku zaznaczony.

-   Aby poruszać się między zadokowanych narzędzia systemu windows, wybierz Alt + F7, aby wyświetlić Nawigator IDE z **aktywnego okna narzędzi** wybrane. Wybierz klawisz Enter, aby przejść do okna zaznaczony.

#### <a name="to-move-and-dock-tool-windows-from-the-keyboard"></a>Aby przenieść i dokowanie okien narzędzi z klawiatury

1.  Przejdź do okna narzędzia, które chcesz przenieść i ustawić fokus.

2.  Na **okna** menu, wybierz opcję **Dockable** opcji.

3.  Naciśnij klawisz **ALT** + **miejsca** , a następnie wybierz **Przenieś**.

     Zostanie wyświetlone dokowania dokowania.

4.  Użyj **strzałka** kluczy, aby przenieść okno do nowej lokalizacji.

     Wskaźnik myszy jest przesuwany w oknie używanej **strzałka** kluczy.

5.  Po osiągnięciu nowej lokalizacji, użyj **strzałka** klucze do Przesuń wskaźnik myszy nad poprawne część dokowania.

     Konspekt okna narzędzia pojawia się w nowej lokalizacji dokowania.

6.  Naciśnij klawisz **ENTER**.

     Okno narzędzia zostanie zablokowany na miejscu na nową lokalizację dokowania.
