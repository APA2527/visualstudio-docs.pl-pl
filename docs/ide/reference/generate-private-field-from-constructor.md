# <a name="generate-private-field-from-constructor"></a>Generuj pole prywatne z konstruktora

Ta Refaktoryzacja mają zastosowanie do: 

- C# 

**Co:** Generuj pole prywatne z konstruktora. 

**Kiedy:** Chcesz szybko dodać pole prywatne z konstruktora.

**Dlaczego:** Pisanie prywatnych pól może być czasochłonne i powtarzane. Korzystanie z tego refaktoryzacji jest szybkie i sprawia, że program jest bardziej niezawodny.

## <a name="how-to"></a>Porady 

1. Umieść kursor na nazwie parametru w konstruktorze.

2. Naciśnij klawisz **Ctrl**+ **.** Aby wyzwolić menu **szybkie akcje i operacje refaktoryzacji** .
   
3. Wybierz opcję **utworzenia i zainicjowania pola**.
