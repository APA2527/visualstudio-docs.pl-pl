# <a name="code-snippet-picker"></a>Selektor wstawek kodu

Edytor kodu programu Visual Studio zapewnia **selektor wstawek kodu** pozwala, za pomocą kilku kliknięć myszą, aby wstawić gotowych do użycia bloków kodu do aktywnego dokumentu.

Procedury, aby wyświetlić **selektor wstawek kodu** zależy od języka.

- Visual Basic – kliknij prawym przyciskiem myszy w dowolnym miejscu w edytorze kodu można wyświetlić menu skrótów, a następnie wybierz pozycję **Wstaw fragment kodu**.

- C#-Kliknij prawym przyciskiem myszy w dowolnym miejscu w edytorze kodu, aby wyświetlić menu skrótów, a następnie kliknij przycisk **Wstaw fragment kodu** lub **Otocz**.

- C++ — **selektor wstawek kodu** nie jest dostępna.

- F#**Selektor wstawek kodu** nie jest dostępna.

- JavaScript — kliknij prawym przyciskiem myszy w dowolnym miejscu w edytorze kodu wyświetla menu skrótów, a następnie kliknij przycisk **Wstaw fragment kodu** lub **Otocz**.

- XML - kliknij prawym przyciskiem myszy w dowolnym miejscu w edytorze kodu wyświetla menu skrótów, a następnie kliknij przycisk **Wstaw fragment kodu** lub **Otocz**.

- HTML — kliknij prawym przyciskiem myszy w dowolnym miejscu w edytorze kodu wyświetla menu skrótów, a następnie kliknij przycisk **Wstaw fragment kodu** lub **Otocz**.

- SQL — kliknij prawym przyciskiem myszy w dowolnym miejscu w edytorze kodu wyświetla menu skrótów, a następnie kliknij przycisk **Wstaw fragment kodu**.

W większości języków programowania Visual Studio, można użyć **Menedżera wstawek kodu** dodać listę folderów w folderze **selektor wstawek kodu** skanuje w poszukiwaniu plików fragment kodu XML. Można również utworzyć własne fragmenty kodu, aby dodać do listy. Aby uzyskać więcej informacji, zobacz [instruktażu: Tworzenie wstawek kodu](../../ide/walkthrough-creating-a-code-snippet.md).

## <a name="uielement-list"></a>Lista elementów UI

Nazwa elementu

Edytowalnym polem tekstowym Wyświetla nazwę elementu zaznaczonego w **listy elementów**. Aby wykonać wyszukiwanie przyrostowe dla elementu, który chcesz, należy rozpocząć, wpisując jej nazwę w tym polu. Dodawaj do momentu żądany element jest zaznaczony na litery **listy elementów**.

Lista elementów

Lista fragmenty kodu są dostępne do wstawiania lub Podaj listę folderów zawierających fragmenty kodu. Aby wstawić framgent kodu lub rozwinąć folder, wybierz element, a następnie naciśnij Enter.
