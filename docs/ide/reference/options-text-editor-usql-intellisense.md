# <a name="options-text-editor-u-sql-intellisense"></a>Opcje, Edytor tekstu, U-SQL IntelliSense

Użyj **IntelliSense** Strona opcji, aby zmodyfikować niektóre ustawienia edytora tekstu do języka U-SQL. Dostępu do tej opcji strony, wybierz **narzędzia** > **opcje**, a następnie wybierz **Edytor tekstu** > **U-SQL**  >  **IntelliSense**.

## <a name="intellisense-settings"></a>Ustawienia funkcji IntelliSense

Zaznacz pole wyboru, aby umożliwić **Quick Info** lub **Intellisense**. Szybkie informacje wyświetlają pełną deklarację, po umieszczeniu wskaźnika myszy na zmiennej.

## <a name="completion-lists"></a>Listy uzupełniania

- **Pokaż listę uzupełniania po wpisaniu znaku**

   Gdy ta opcja jest zaznaczona, IntelliSense wyświetla automatycznie na liście uzupełniania po rozpoczęciu wpisywania. Jeśli nie zaznaczysz tej opcji, uzupełnianie przez funkcję IntelliSense będzie nadal dostępna, z poziomu menu IntelliSense lub naciskając **Ctrl** + **spacja**.

- **Umieść słowa kluczowe w listach uzupełniania**

   Gdy ta opcja jest zaznaczona, IntelliSense zawiera słowa kluczowe na liście uzupełniania.

- **Umieść fragmenty kodu w listach uzupełniania**

   Gdy ta opcja jest zaznaczona, IntelliSense zawiera fragmenty kodu na liście uzupełniania.

## <a name="selection-in-completion-list"></a>Wybór na liście uzupełniania

- **Potwierdź, wpisując następujące znaki**

   To pole zawiera znaki, które powodują sugestii listy uzupełniania obecnie podświetlony do zatwierdzenia. Można dodać lub usunąć znaki z tej listy.

- **Potwierdź, naciskając spację**

   Po wybraniu tej opcji należy zatwierdzić sugestii listy uzupełniania wyróżniony, naciskając klawisz spacji.

- **Dodaj nowy wiersz na końcu pełnego wpisanego wyrazu Enter**

   Po wybraniu nowego wiersza jest automatycznie dodawany i kursor zostaje przeniesiony do nowego wiersza po wpisaniu wszystkich znaków dla sugestię listy uzupełniania.
