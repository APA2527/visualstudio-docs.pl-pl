# <a name="options-text-editor-file-extension"></a>Opcje, edytor tekstu, rozszerzenie pliku

To okno dialogowe Opcje umożliwia określenie sposobu obsługi wszystkich plików z określonymi rozszerzeniami plików przez zintegrowane środowisko programistyczne programu Visual Studio (IDE). Dla każdego **wprowadzonego rozszerzenia** można wybrać środowisko edycji. Dzięki temu można wybrać edytora IDE lub projektanta, w którym zostaną otwarte dokumenty określonego typu. Aby wyświetlić te opcje, wybierz polecenie **Opcje** z menu **Narzędzia,** rozwiń węzeł **Edytor tekstu** i wybierz polecenie Rozszerzenie **pliku**.

Po wybraniu opcji "z kodowaniem" zostanie wyświetlone okno dialogowe przy każdym otwarciu dokumentu tego typu, który umożliwia wybranie schematu kodowania dla tego dokumentu. Może to być przydatne, jeśli przygotowujesz wersje dokumentów projektu do użycia na różnych platformach lub w różnych językach docelowych.

## <a name="uielement-list"></a>Lista UIElement

**Wewnętrzny**

Wpisz rozszerzenie pliku, którego środowisko edycji w IDE, które chcesz zdefiniować.

**Edytor**

Wybierz edytor IDE lub projektant, w którym zostaną otwarte dokumenty z tym rozszerzeniem pliku. Po wybraniu opcji "z kodowaniem", po otwarciu takiego dokumentu zostanie wyświetlone okno dialogowe umożliwiające wybranie schematu kodowania.

**Dodaj**

Dodaje wpis, który zawiera określone **rozszerzenie** i **środowisko edycji** do listy rozszerzeń.

**Usuń**

Usuwa zaznaczony wpis z listy rozszerzeń.

**Lista rozszerzeń**

Wyświetla listę wszystkich rozszerzeń, dla których określono środowisko edycji.

**Mapowanie plików bez rozszerzenia do**

Wybierz tę opcję, jeśli chcesz określić sposób obsługi plików bez rozszerzenia przez IDE.

**Opcje plików bez rozszerzeń**

Udostępnia tę samą listę co **Edytor**. Wybierz edytor IDE lub projektant, w którym zostaną otwarte dokumenty bez rozszerzeń plików.
