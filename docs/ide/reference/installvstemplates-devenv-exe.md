# <a name="installvstemplates-devenvexe"></a>/InstallVSTemplates (devenv.exe)
Rejestruje szablony projektów lub elementów, które znajdują się w  *\<ścieżki instalacji programu Visual Studio >*\Common7\IDE\ProjectTemplates\ lub  *\<ścieżki instalacji programu Visual Studio >*\Common7\IDE\ItemTemplates\, dzięki czemu są one dostępne za pośrednictwem **nowy projekt** i **Dodaj nowy element** okien dialogowych.  
  
> [!WARNING]
>  Ten przełącznik jest obsługiwana tylko w przypadku tworzenia partnera Visual Studio i nie jest dostępna w wersji Express. Devenv musi działać jako administrator, aby można było używać [/Setup (devenv.exe)](../../ide/reference/setup-devenv-exe.md) i [/installvstemplates (devenv.exe)](../../ide/reference/installvstemplates-devenv-exe.md) przełączników. Aby uzyskać więcej informacji, zobacz [uprawnienia użytkownika](../../ide/user-permissions-and-visual-studio.md).  
  
## <a name="syntax"></a>Składnia  
  
```  
devenv.exe /InstallVSTemplates  
```
