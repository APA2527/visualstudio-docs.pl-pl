# <a name="go-to-line-dialog-box"></a>Idź do linii — Okno dialogowe

**Przejdź do wiersza** okno dialogowe pozwala na przechodzenie do określonego wiersza w aktywnym dokumencie. Aby uzyskać dostęp do tego okna dialogowego, otwórz dokument do edycji, a następnie wybierz **Edytuj** > **przejdź do** > **przejdź do wiersza** lub naciśnij **Ctrl**  + **G**.

![Okno dialogowe Przejdź do wiersza w programie Visual Studio](media/go-to-line-dialog-box.png)

## <a name="line-number-1---n"></a>Numer wiersza (1 - \<n >)

W **numer wiersza (1 - \<n >)** wprowadź numer wiersza w aktywnym dokumencie, który chcesz przenieść. Wprowadzona liczba muszą należeć do zakresu od 1 do ostatni numer wiersza w bieżącym dokumencie.
