# <a name="how-to-display-line-numbers-in-the-editor"></a>Jak: Wyświetlanie numerów wierszy w edytorze

Numerowanie wierszy można wyświetlać lub ukrywać w kodzie.

## <a name="display-line-numbers-in-code"></a>Wyświetlanie numerów wierszy w kodzie

1. Na pasku menu wybierz pozycję**Opcje** **narzędzi** > . Rozwiń węzeł **Edytor tekstu,** a następnie wybierz język, którego używasz, lub **wszystkie języki,** aby włączyć numery wierszy we wszystkich językach. (Lub wpisz **numer wiersza** w polu wyszukiwania i wybierz **pozycję Włącz lub wyłącz numery wierszy** z wyników).

2. Zaznacz pole wyboru **Numery wierszy.**

   ![Opcja wyświetlania numerów wierszy w edytorze VS](../../ide/reference/media/line-numbers-option.png)

> [!TIP]
> Numery wierszy nie są dodawane do kodu; są one tylko dla odniesienia. Jeśli chcesz drukować numery wierszy, w oknie dialogowym **Drukowanie** zaznacz pole wyboru **Dołącz numery wierszy.**
