# <a name="options-text-editor-c-formatting"></a>Opcje, edytor tekstu, C#, formatowanie
Użyj **formatowanie** okno dialogowe strony właściwości można ustawić opcje formatowania kodu w edytorze kodu. Aby uzyskać dostęp do tego okna dialogowego, kliknij przycisk **opcje** na **narzędzia** menu, rozwiń węzeł **Edytor tekstu**, rozwiń węzeł **C#**, a następnie kliknij przycisk  **Formatowanie**.  
  
> [!NOTE]
>  Okna dialogowe i polecenia menu mogą się różnić od tych opisanych w Pomocy, w zależności od ustawień aktywnych lub wydania. Aby zmienić ustawienia, wybierz **Import i eksport ustawień** na **narzędzia** menu. Aby uzyskać więcej informacji, zobacz [personalizowanie środowiska IDE programu Visual Studio](../../ide/personalizing-the-visual-studio-ide.md).  
  
## <a name="general-settings"></a>Ustawienia ogólne  
 Ogólne ustawienia mają wpływ na sposób edytora kodu dotyczy opcje formatowania kodu.  
  
## <a name="uielement-list"></a>Lista elementów UI  
  
|Etykieta|Opis|  
|-----------|-----------------|  
|**Automatycznie Formatuj instrukcję ukończone.**|Po wybraniu formatuje instrukcje po zakończeniu zgodnie z opcjami formatowania wybrane dla edytora kodu. Wyczyść to pole, jeśli nie chcesz, aby zmienić instrukcje w edytorze kodu.|  
|**Automatycznie Formatuj blok ukończone w}**|Po wybraniu formatów kodu bloki zgodnie z opcjami formatowania wybrane dla edytora kodu, jak zakończyć blok kodu. Wyczyść to pole, jeśli nie chcesz, aby zmiany bloków w edytorze kodu.|  
|**Dostosuj wcięcie przy wklejeniu**|Po wybraniu formatuje wkleić do edytora kodu aby zmieścić wybrany dla edytora kodu opcje formatowania tekstu. Wyczyść to pole, jeśli nie chcesz wkleić tekst, który ma zostać zmieniony.|  
  
## <a name="preview-window"></a>Okno podglądu  
 **Wcięcia**, **nowych wierszy**, **odstępy**, i **zawijanie** opcje okienka wyświetlania okna podglądu. Okno podglądu przedstawiono wpływ poszczególnych opcji. Aby użyć okna podglądu, wybierz opcję formatowania. Okno podglądu przedstawiono przykład wybranej opcji. Po zmianie ustawienia, na przykład wybierz lub wyczyść pole wyboru, okna podglądu aktualizacji pokazanie efekt nowe ustawienie.  
  
## <a name="remarks"></a>Uwagi  
 Opcje wcięcia na **karty** stron dla każdego języka tylko określić, gdzie edytora kodu umieszcza kursor po naciśnięciu klawisza ENTER na końcu wiersza. Wcięcie opcje w obszarze **formatowanie** stosowane, gdy kod jest formatowana automatycznie, na przykład podczas wklejania kod do pliku podczas **dostosować wcięcie przy wklejeniu** jest zaznaczona i kiedy trwa bloku sformatowany został wpisany ręcznie.
