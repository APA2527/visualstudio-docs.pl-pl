# <a name="options-text-editor-c-formatting"></a>Opcje, edytor tekstu, C#, formatowanie

Użyj **formatowanie** Strona opcji, aby ustawić opcje formatowania kodu w edytorze kodu. Aby uzyskać dostęp do tej strony opcji, należy wybrać **narzędzia** > **opcje**, a następnie wybierz pozycję **Edytor tekstu** > **C#**  >  **Kodu stylu** > **formatowania**.

> [!NOTE]
> Okna dialogowe i polecenia menu mogą się różnić od tych opisanych w Pomocy, w zależności od ustawień aktywnych lub wydania. Aby zmienić ustawienia, wybierz **Import i eksport ustawień** na **narzędzia** menu. Aby uzyskać więcej informacji, zobacz [personalizowanie środowiska IDE programu Visual Studio](../../ide/personalizing-the-visual-studio-ide.md).

## <a name="general-settings"></a>Ustawienia ogólne

Ogólne ustawienia mają wpływ na sposób edytora kodu dotyczy opcje formatowania kodu.

## <a name="uielement-list"></a>Lista elementów interfejsu użytkownika

|Etykieta|Opis|
|-----------|-----------------|
|**Automatycznie Formatuj podczas pisania**|W przeciwnym wypadku **Formatuj instrukcję;** i **sformatuj blok w}** opcje zostaną wyłączone.|
|**Automatycznie Formatuj instrukcji.**|Po wybraniu formatuje instrukcje po zakończeniu zgodnie z opcjami formatowania wybrane edytora.|
|**Automatycznie sformatuj blok w}**|Po wybraniu formatów kodu bloki zgodnie z opcjami formatowania wybrane edytora zaraz po wykonaniu bloku kodu.|
|**Automatycznie Formatuj przy powrocie**|Po wybraniu formatowania tekstu podczas **Enter** zostanie naciśnięty, aby zmieścić wybrany edytora opcje formatowania.|
|**Automatycznie sformatuj przy wklejeniu**|Po wybraniu formatowania tekstu, która zostanie wklejona do edytora, aby zmieścić wybrany edytora opcje formatowania.|

## <a name="preview-window"></a>Okno podglądu

**Wcięcia**, **nowych wierszy**, **odstępy**, i **zawijanie** opcje okienka wyświetlania okna podglądu. Okno podglądu przedstawiono wpływ poszczególnych opcji. Aby użyć okna podglądu, wybierz opcję formatowania. Okno podglądu przedstawiono przykład wybranej opcji. Po zmianie ustawienia, zaznaczając pole wyboru, aktualizacje okno podglądu pokazanie efekt nowe ustawienie lub przycisku radiowego.

## <a name="remarks"></a>Uwagi

Opcje wcięcia na **karty** stron dla każdego języka tylko określić, gdzie edytora kodu umieszcza kursor po naciśnięciu **Enter** na końcu linii. Wcięcie opcje w obszarze **formatowanie** stosowane, gdy kod jest formatowana automatycznie, na przykład podczas wklejania kod do pliku podczas **automatycznie sformatuj przy wklejeniu** jest zaznaczone, gdy blok formatowanego został wpisany ręcznie.
