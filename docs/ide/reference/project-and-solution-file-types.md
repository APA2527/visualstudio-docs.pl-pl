# <a name="project-and-solution-file-types"></a>Typy plików projektu i rozwiązania
[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]obsługuje wiele typów plików. W danej instalacji zainstalowanych składników określić typy plików są obsługiwane. Ten temat zawiera listę typów plików rozwiązań i projektów, które są obsługiwane w niektórych typowych instalacji. Aby uzyskać informacje o innych typów plików wyszukiwania przy użyciu rozszerzeń nazw plików dla każdego typu.  
  
## <a name="solution-files-sln-and-suo"></a>Pliki rozwiązania (sln i .suo)  
 [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]wykorzystuje dwa typy plików (.sln i .suo), aby przechowywać ustawienia specyficzne dla rozwiązania. Te pliki, nazywane pliki rozwiązania dostarczać informacje potrzebne do wyświetlenia interfejsem graficznym do zarządzania plikami Eksploratora rozwiązań. Umożliwiają one skoncentrować się na swoje projekty i cele końcowego, a nie na samego środowiska zawsze, gdy powrócisz do zadań związanych z projektowaniem.  
  
|Rozszerzenia|Nazwa|Opis|  
|---------------|----------|-----------------|  
|.sln|[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]Rozwiązania|Organizuje projektów, elementy projektów i elementów rozwiązania w zakresie.|  
|.suo|Opcje użytkownika rozwiązania|Przechowuje informacje o poziomie użytkownika dostosowania wprowadzone Visual Studio, takich jak punkty przerwania.|
