# <a name="move-type-to-namespace"></a>Przenoszenie typu do przestrzeni nazw

Ten refaktoryzator ma zastosowanie do:

- C#

**Co:** Przenieś typ do obszaru nazw.

**Kiedy:** Chcesz przenieść typ do innego obszaru nazw lub folderu. 

**Dlaczego?** Chcesz refaktoryzować części rozwiązania i mieć szybki sposób, aby przenieść typ do innego obszaru nazw lub folderu. 

## <a name="how-to"></a>Porady

1. Umieść kursor w nazwie klasy.
2. Naciśnij **klawisze Ctrl**+**.** , aby wyzwolić menu **Szybkie akcje i Refaktoryzowania.**
3. Wybierz **pozycję Przenieś do obszaru nazw**.

   ![Przechodzenie do refaktoryzacji przestrzeni nazw](media/move-to-namespace.png)

4. W oknie dialogowym, które zostanie otwarte, wybierz docelową przestrzeń nazw, do której chcesz przenieść ten typ. 

   ![Wybieranie okna dialogowego obszaru nazw](media/select-target-namespace.png)
