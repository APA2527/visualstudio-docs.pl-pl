# <a name="move-type-to-namespace"></a>Przenoszenie typu do przestrzeni nazw

To Refaktoryzacja dotyczy:

- C#

**Co:** Przenieś typ do przestrzeni nazw.

**Kiedy:** Chcesz przenieść typ do innej przestrzeni nazw lub folderu. 

**Dlaczego:** Chcesz refragmentować części rozwiązania i mieć szybki sposób przenoszenia typu do innej przestrzeni nazw lub folderu. 

## <a name="how-to"></a>Porady

1. Umieść kursor w nazwie klasy.
2. Naciśnij klawisz **Ctrl** + **.** Aby wyzwolić menu **szybkie akcje i operacje refaktoryzacji** .
3. Wybierz pozycję **Przenieś do przestrzeni nazw**.

   ![Refaktoryzacja przejścia do przestrzeni nazw](media/move-to-namespace.png)

4. W otwartym oknie dialogowym wybierz docelową przestrzeń nazw, do której chcesz przenieść typ. 

   ![Okno dialogowe Wybieranie przestrzeni nazw](media/select-target-namespace.png)
