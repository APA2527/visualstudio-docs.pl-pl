# <a name="options-dialog-box-text-editor--javascripttypescript--linting"></a>Okno dialogowe Opcje: Edytor \> tekstu \> JavaScript/TypeScript Linting

Użyj **linting** strony okna dialogowego **Opcje,** aby ustawić opcje analizowania kodu w Edytorze kodu. Aby uzyskać dostęp do tej strony, na pasku menu wybierz pozycję **Opcje narzędzi** > **Options**, a następnie rozwiń pozycję **Edytor** > tekstu**JavaScript/TypeScript** > **Linting**.

## <a name="eslint-settings"></a>Ustawienia ESLint

Te opcje umożliwiają włączenie statycznej analizy kodu JavaScript i TypeSCript oraz wybranie analizowanych plików. Aby uzyskać więcej informacji na temat ESLint, zobacz [ESLint.org](https://eslint.org/).

### <a name="uielement-list"></a>Lista UIElement

|Opcja|Opis|
|------------|-----------------|
|**Włącz ESLint**|Gdy ta opcja jest zaznaczona, Edytor kodu umożliwia analizę statyczną kodu.|
|**Lint wszystkie pliki zawarte w projekcie, nawet zamknięte pliki**|Gdy ta opcja jest zaznaczona, zamknięte pliki są analizowane, chyba że diagnostyka jest zgłaszana tylko dla otwartych plików.|

## <a name="global-eslint-config-options"></a>Globalne opcje konfiguracji ESLint

Ta opcja umożliwia skopiowanie lokalizacji globalnego pliku konfiguracyjnego ESLint. Ponadto jeśli wcześniej zmieniono lokalizację, można zresetować plik do jego domyślnej lokalizacji.
