# <a name="options-text-editor-javascript-linting"></a>Opcje, Edytor tekstu, JavaScript, Zaznaczanie błędów

Użyj **Zaznaczanie błędów** strony **opcje** okno dialogowe, aby ustawić opcje do analizowania kodu w edytorze kodu. Dostępu do tej strony, na pasku menu wybierz **narzędzia** > **opcje**, a następnie rozwiń węzeł **edytora tekstów** > **językaJavaScript**  >  **Zaznaczanie błędów**.

## <a name="eslint-settings"></a>Ustawienia ESLint

Te opcje umożliwiają Włącz analizę statyczną kodu JavaScript i wybierz pliki, które są analizowane. Aby uzyskać więcej informacji na temat ESLint zobacz [ESLint.org](https://eslint.org/).

### <a name="uielement-list"></a>Lista elementów interfejsu użytkownika

|Opcja|Opis|
|------------|-----------------|
|**Włącz program ESLint**|Gdy ta opcja jest zaznaczona, Edytor kodu umożliwia statycznej analizy kodu.|
|**Wszystkie pliki zawarte w projekcie, nawet lint zamknięte plików**|Gdy ta opcja jest zaznaczona, zamkniętych plikach są analizowane, chyba że Diagnostyka tylko są zgłaszane dla otwartych plików.|

## <a name="global-eslint-config-options"></a>Opcje globalne konfiguracji ESLint

Ta opcja pozwala skopiować lokalizacji plik konfiguracyjny ESLint globalnego. Ponadto wcześniej zmieniono tę lokalizację, można zresetować go do domyślnej lokalizacji.
