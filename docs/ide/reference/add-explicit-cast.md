# <a name="add-explicit-cast"></a>Dodawanie jawnego rzutu

To generowanie kodu dotyczy:

- C#

**Co:** Umożliwia automatyczne dodawanie jawnego rzutu do wyrażenia na podstawie użycia.

**Kiedy:** Należy dodać jawne rzutowania do wyrażenia i chcesz poprawnie przypisać go automatycznie.

**Dlaczego?** Rzutowanie jawne można dodać ręcznie do wyrażenia, jednak ta funkcja dodaje go automatycznie na podstawie kontekstu kodu.

## <a name="how-to-use-it"></a>Korzystanie

1. Umieść swoją cieszkę na błędzie.
2. Naciśnij **klawisze Ctrl**+**.** , aby wyzwolić menu **Szybkie akcje i Refaktoryzowania.**
3. Wybierz **pozycję Dodaj jawne rzuty**.

   ![Dodawanie jawnego szybkiego rzutu w programie Visual Studio](media/add-explicit-cast.png)
