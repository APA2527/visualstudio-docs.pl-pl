# <a name="code-analysis-project-designer"></a>Analiza kodu, Projektant projektu

Zawiera narzędzie do analizy kodu, które można wybrać do uruchomienia w kodzie. Narzędzie raportuje informacje o zestawach, takie jak naruszenia reguł programowania i projektowania określonych w wytycznych dotyczących projektowania platformy Microsoft .NET.

## <a name="uielement-list"></a>Lista UIElement

Włącz analizę kodu\

Zasady projektowania\
