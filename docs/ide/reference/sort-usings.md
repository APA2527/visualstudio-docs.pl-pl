# <a name="sort-usings"></a>Sortuj deklaracje Using

Ta Refaktoryzacja mają zastosowanie do:

- C#

**Co:** Sortuj deklaracje Using.

**Kiedy:** Aby posortować `using` dyrektywy w górnej części pliku, aby były w kolejności alfabetycznej. 

**Dlaczego:** To ułatwia można znaleźć przy użyciu dyrektywy.

## <a name="how-to"></a>Instrukcje

1. Wybierz **Edytuj** z paska menu.
2. Wybierz **Intellisense** > **Sortuj deklaracje Using**.
