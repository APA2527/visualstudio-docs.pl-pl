# <a name="international-settings-environment-options-dialog-box"></a>Ustawienia międzynarodowe, środowisko, opcje — Okno dialogowe
Na stronie Ustawienia międzynarodowe można zmienić domyślny język, jeśli masz więcej niż jedną wersję językową zintegrowane środowisko programistyczne (IDE) na komputerze jest zainstalowany. Dostęp do tego okna dialogowego, wybierając **opcje** z **narzędzia** menu, a następnie wybierając **ustawienia międzynarodowe** z **środowiska** folderu. Jeśli ta strona nie ma na liście, wybierz **Pokaż wszystkie ustawienia** w **opcje** okno dialogowe.  
  
> [!NOTE]
>  Opcje dostępne w oknach dialogowych i nazwy i lokalizacje poleceń menu, które zostanie wyświetlone, może się różnić od co to jest opisany w pomocy, w zależności od wersji lub aktywne ustawienia. Aby zmienić ustawienia, wybierz **Import i eksport ustawień** na **narzędzia** menu. Aby uzyskać więcej informacji, zobacz [personalizowanie środowiska IDE programu Visual Studio](../../ide/personalizing-the-visual-studio-ide.md).  
  
 **Język**  
 Wyświetla listę dostępnych języków dla wersje językowe zainstalowany produkt. Ta opcja jest niedostępna, jeśli nie masz więcej niż jedną wersję językową zainstalowana na tym komputerze. Jeśli wiele języków, produktów lub mieszanym językiem instalacji produktów należy udostępnić środowiska, wybór języka jest zmieniana na **taki sam jak Microsoft Windows**.  
  
> [!CAUTION]
>  W systemie z wieloma językami zainstalowanymi narzędzia kompilacji Visual C++ (cl.exe, link.exe, nmake.exe, bscmake.exe i powiązane pliki) nie dotyczy to ustawienie. Te narzędzia Użyj wersji dla ostatniego język zainstalowany. Narzędzia kompilacji dla wcześniej zainstalowany język zostaną zastąpione, ponieważ nie używają modelu satelitarnej biblioteki DLL narzędzia kompilacji Visual C++.
