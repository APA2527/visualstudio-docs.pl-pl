# <a name="diff"></a>/Diff
Porównuje dwa pliki. Różnice są wyświetlane w specjalnym oknie Visual Studio.  
  
## <a name="syntax"></a>Składnia  
  
```  
devenv /Diff SourceFile, TargetFile, [SourceDisplayName],[TargetDisplayName]  
```  
  
## <a name="arguments"></a>Argumenty  
 `SourceFile`  
 Wymagany. Pełna ścieżka i nazwa pierwszego pliku do porównania.  
  
 `TargetFile`  
 Wymagany. Pełna ścieżka i nazwa drugiego pliku do porównania  
  
 `SourceDisplayName`  
 Opcjonalny. Wyświetlana nazwa pierwszego pliku.  
  
 `TargetDisplayName`  
 Opcjonalny. Nazwa wyświetlana drugiego pliku.
