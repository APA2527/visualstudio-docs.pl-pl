# <a name="-devenvexe"></a>/? (devenv.exe)
Wyświetla komunikat z listą wszystkich `devenv` przełączników, krótki opis każdego z nich.  
  
## <a name="syntax"></a>Składnia  
  
```  
devenv /?  
```
