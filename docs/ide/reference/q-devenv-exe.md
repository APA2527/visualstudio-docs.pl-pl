# <a name="-devenvexe"></a>/? (devenv.exe)

Wyświetla okno komunikatu z listą wszystkich przełączników `devenv`, z krótkim opisem każdego przełącznika.

## <a name="syntax"></a>Składnia

```shell
devenv /?
```
