# <a name="extensions-and-updates-environment-options-dialog-box"></a>Rozszerzenia i aktualizacje, środowisko, opcje — okno dialogowe
Ta strona służy do ustawiania opcji jak Visual Studio przeprowadza aktualizacji i sposobu rozszerzenia są aktualizowane i odnalezione.  
  
### <a name="automatically-check-for-updates"></a>Automatyczne sprawdzenie dostępności aktualizacji  
 Po zaznaczeniu tej opcji, Visual Studio Okresowo sprawdzaj aktualizacji do samej siebie, zainstalowanych zestawów SDK i narzędzia i rozszerzenia i wyświetlanie powiadomień na pasku menu, gdy są dostępne aktualizacje. Aby uzyskać więcej informacji, zobacz [powiadomienia](../../ide/visual-studio-notifications.md).  
  
### <a name="automatically-update-extensions"></a>Automatycznie aktualizuje rozszerzeń  
 Po zaznaczeniu tej opcji, aby rozszerzenia zaktualizowane bez monitowania. Aby uzyskać więcej informacji, zobacz [Znajdowanie i rozszerzenia przy użyciu programu Visual Studio](../../ide/finding-and-using-visual-studio-extensions.md).  
  
### <a name="load-user-extensions-when-running-as-administrator"></a>Ładowanie rozszerzenia użytkownika podczas uruchamiania jako administrator  
 Aby uzyskać więcej informacji, zobacz [Znajdowanie i rozszerzenia przy użyciu programu Visual Studio](../../ide/finding-and-using-visual-studio-extensions.md).  
  
### <a name="additional-extension-galleries"></a>Dodatkowe galerii rozszerzeń  
 Funkcja przedsiębiorstwa, która umożliwia obsługę galerie rozszerzeń zastrzeżonych. Aby uzyskać więcej informacji, zobacz [galerie prywatnego](../../extensibility/private-galleries.md).
