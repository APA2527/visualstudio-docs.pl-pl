# <a name="options-dialog-box-environment--extensions"></a>Okno dialogowe \> Opcje: Rozszerzenia środowiska

Ta strona służy do ustawiania opcji, w jaki sposób visual studio wykonuje aktualizacje i jak rozszerzenia są aktualizowane i odnajdowane.

## <a name="automatically-check-for-updates"></a>Automatyczne sprawdzanie dostępności aktualizacji

Po zaznaczeniu, Visual Studio będzie okresowo sprawdzać aktualizacje dla siebie, zainstalowane zestawy SDK i narzędzia i rozszerzenia i wyświetlać powiadomienia na pasku menu, gdy aktualizacje są dostępne. Aby uzyskać więcej informacji, zobacz [Aktualizowanie programu Visual Studio](../../install/update-visual-studio.md).

## <a name="automatically-update-extensions"></a>Automatyczne aktualizowanie rozszerzeń

Po zaznaczeniu tej kontroli aktualizacje rozszerzeń są dokonywane bez monitowania. Aby uzyskać więcej informacji, zobacz [Znajdowanie i używanie rozszerzeń programu Visual Studio](../../ide/finding-and-using-visual-studio-extensions.md).

## <a name="load-per-user-extensions-when-running-as-administrator"></a>Ładowanie rozszerzeń użytkownika podczas pracy jako administrator

Aby uzyskać więcej informacji, zobacz [Znajdowanie i używanie rozszerzeń programu Visual Studio](../../ide/finding-and-using-visual-studio-extensions.md).

## <a name="additional-extension-galleries"></a>Dodatkowe galerie rozszerzeń

Funkcja Enterprise, która umożliwia obsługę galerii zastrzeżonych rozszerzeń. Aby uzyskać więcej informacji, zobacz [Prywatne galerie](../../extensibility/private-galleries.md).
