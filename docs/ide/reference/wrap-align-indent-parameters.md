# <a name="wrap-indent-and-align-parameters"></a>Opakowywanie, wcięcia i parametrów wyrównanie

Ta Refaktoryzacja mają zastosowanie do:

- C#

- Visual Basic

**Co:** Umożliwia opakować, wcięcia i parametrów wyrównanie.

**Kiedy:** Masz deklaracji metody lub wywołania, który ma wiele parametrów.

**Dlaczego:** Odczytywanie długą listę parametrów jest łatwiejsze w przypadku, gdy są one opakowane lub wcięte w zależności od preferencji użytkownika.

## <a name="how-to"></a>Instrukcje

1. Umieść kursor na liście parametrów.
2. Naciśnij klawisz **Ctrl**+**.** wyzwalacz **szybkie akcje i Refaktoryzacje** menu.

   ![Opakowywanie, wcięcia i parametrów wyrównanie](media/wrap-parameters.png)

3. Naciśnij klawisz **Enter** do akceptowania refaktoryzacji.

   ![Opakowywanie, wcięcia i stosowane parametrów wyrównanie](media/wrap-parameters-completed.png)
