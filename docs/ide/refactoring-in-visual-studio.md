# <a name="refactoring-in-visual-studio"></a>Refaktoryzacja w programie Visual Studio
Refaktoryzacja to proces modyfikowania kodu w celu ułatwienia utrzymanie, zrozumieć i rozszerzenie, ale bez zmiany jego zachowania. Niektóre typowe *refaktoryzacje* obejmują:  
  
1.  podzielić długie funkcje na mniejsze  
  
2.  Zmiana nazwy zmiennych, typów, funkcji lub parametrów  
  
3.  zmiany kolejności parametrów  
  
4.  Usuwanie lub zmiana kolejności przy użyciu dyrektyw  
  
 Ponieważ logiki refaktoryzacji jest ściśle związany składni języka, każda usługa języka w programie Visual Studio udostępnia refaktoryzacji możliwości. Aby uzyskać więcej informacji zobacz następujące strony:
