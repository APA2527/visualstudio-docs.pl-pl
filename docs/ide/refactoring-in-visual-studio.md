# <a name="refactor-code"></a>Refaktoryzacja kodu

Refaktoryzowanie jest procesem modyfikowania kodu w celu ułatwienia obsługi, zrozumienia i rozszerzenia, ale bez zmiany jego zachowania.

## <a name="programming-languages"></a>Języki programowania

Różne operacje refaktoryzacji są dostępne dla różnych języków programowania w programie Visual Studio:

- Strony w tej sekcji spisu treści obejmują refaktoryzacji dostępnych dla języka C# i Visual Basic. Niektóre przykłady to [Wyodrębnij refaktoryzatoring metody](reference/extract-method.md) i [Przenieś typ do pasującego refaktoryzacji pliku](reference/move-type-to-matching-file.md).
- Aby uzyskać informacje na temat refaktoryzacji kodu języka C++, zobacz [Kod zapisu i refaktoryzacji (C++)](/cpp/ide/writing-and-refactoring-code-cpp).
- Refaktoryzowanie wsparcie dla języka F# jest dostarczany przez [Visual F# Power Tools](https://marketplace.visualstudio.com/items?itemName=FSharpSoftwareFoundation.VisualFPowerTools), rozszerzenie programu Visual Studio innej firmy.
