# <a name="personalize-the-visual-studio-ide"></a>Personalizowanie środowiska IDE programu Visual Studio

Program Visual Studio można spersonalizować na różne sposoby, aby jak najlepiej obsługiwać własny styl i wymagania dotyczące programowania. Wiele ustawień przemieszcza się z tobą w&mdash;różnych wystąpieniach programu Visual Studio, zobacz [Ustawienia zsynchronizowane](../ide/synchronized-settings-in-visual-studio.md). W tym artykule pokrótce opisano różne personalizacje i gdzie można znaleźć więcej informacji.

> [!NOTE]
> W tym temacie stosuje się do programu Visual Studio w systemie Windows. W przypadku programu Visual Studio dla komputerów Mac zobacz [Dostosowywanie środowiska IDE programu Visual Studio dla komputerów Mac](/visualstudio/mac/customizing-the-ide).

## <a name="default-settings"></a>Ustawienia domyślne

Można wybrać domyślną kolekcję ustawień, która optymalizuje visual studio dla typu rozwoju. Aby uzyskać więcej informacji, zobacz [Ustawienia środowiska](environment-settings.md).

## <a name="general-environment-options"></a>Ogólne opcje środowiska

Wiele opcji personalizacji jest dostępnych w oknie dialogowym [Opcje środowiska.](../ide/reference/general-environment-options-dialog-box.md) Dostęp do tego okna dialogowego można uzyskać na dwa sposoby:

- Na pasku menu wybierz pozycję**Opcje** **narzędzi** > , a jeśli nie jest on jeszcze rozwinięty, rozwiń węzeł **Środowisko.**

- Naciśnij **klawisz Ctrl**+**Q**, wpisz **środowisko** w polu wyszukiwania, a następnie wybierz z wyników pozycję Środowisko **> Ogólne.**

> [!TIP]
> Po wyświetleniu okna dialogowego Opcje można nacisnąć klawisz **F1,** aby uzyskać pomoc dotyczącą różnych ustawień na tej stronie.

## <a name="environment-color-themes"></a>Motywy kolorów środowiska

Aby zmienić motyw kolorów między jasnym, ciemnym i niebieskim, wpisz **środowisko** w polu wyszukiwania, a następnie wybierz **polecenie Środowisko > Ogólne**. W oknie dialogowym **Opcje** zmień opcję **Motyw koloru.**

Aby zmienić opcje koloryzacji w edytorze, wpisz **środowisko** w polu wyszukiwania, a następnie wybierz polecenie **Środowisko > Czcionki i Kolory**. Zobacz [Jak: Zmienianie czcionek i kolorów](../ide/how-to-change-fonts-and-colors-in-visual-studio.md).

### <a name="main-menu-casing"></a>Obudowa menu głównego

Możesz zmienić wielkość liter menu głównego między **tytułową** literą ("Plik") a **wszystkie wsiewowe** ("PLIK"). Wpisz **środowisko** w polu wyszukiwania, wybierz opcję **Środowisko > Ogólne**, a następnie zmień opcję Zastosuj styl przypadku tytułu na pasek **menu.**

### <a name="customize-menus-and-toolbars"></a>Dostosowywanie menu i pasków narzędzi

Aby dodać lub usunąć elementy menu lub paska narzędzi, zobacz [Jak: Dostosowywanie menu i pasków narzędzi](../ide/how-to-customize-menus-and-toolbars-in-visual-studio.md).

## <a name="window-layouts"></a>Układy okien

Można zdefiniować i zapisać wiele układów okien i przełączać się między nimi. Na przykład można zdefiniować jeden układ do kodowania i jeden do debugowania. Aby rozmieścić pozycje i zachowanie okien oraz zapisać układy niestandardowe, zobacz [Dostosowywanie układów okien](../ide/customizing-window-layouts-in-visual-studio.md).

## <a name="external-tools"></a>Narzędzia zewnętrzne

Można dostosować menu **Narzędzia,** aby uruchomić narzędzia zewnętrzne. Aby uzyskać więcej informacji, zobacz [Zarządzanie narzędziami zewnętrznymi](../ide/managing-external-tools.md).
