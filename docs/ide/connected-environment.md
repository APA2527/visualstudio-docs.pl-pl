Program Visual Studio jest połączonych środowiska:

- Możesz [Zaloguj](../ide/signing-in-to-visual-studio.md) dla programu Visual Studio, aby umożliwić dostęp do wszystkich usług, które są skojarzone z kontem użytkownika, a także wszystkich kont, które są połączone z tym kontem. Ustawienia konta są przenoszone z Tobą wszędzie tam, gdzie możesz użyć programu Visual Studio.  
- Visual Studio [powiadamia](../ide/visual-studio-notifications.md) Jeśli dowolny składnik instalacji, takich jak zestaw SDK zawiera dostępna jest aktualizacja.

- Strona początkowa jest zaktualizowano o najnowsze dewelopera z innej witryny sieci Web.  
- itp.
