# <a name="f1-help"></a>F1 pomoc

Wygląda na to, że nie mogliśmy znaleźć tego, czego szukaliście, a my przepraszamy. Możesz pomóc nam poprawić, przesyłając opinię za pomocą przycisków na dole tej strony.

## <a name="cursor-position"></a>Położenie kursora

Jeśli szukasz pomocy F1 na słowo kluczowe lub nazwa interfejsu API, upewnij się, że kursor jest umieszczony wewnątrz lub po prawej stronie słowa przed naciśnięciem **Klawisza F1**.
