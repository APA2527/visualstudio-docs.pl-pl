# <a name="f1-help"></a>F1 — Pomoc

Wygląda na to, że nie udało nam się znaleźć tego, czego szukasz, i Przepraszamy. Możesz pomóc nam ulepszyć program, pozostawiając Opinie na przyciskach w dolnej części tej strony.

## <a name="cursor-position"></a>Położenie kursora

Jeśli szukasz pomocy F1 dla słowa kluczowego lub nazwy interfejsu API, pamiętaj, aby umieścić kursor wewnątrz lub po prawej stronie słowa przed naciśnięciem klawisza **F1**. Jeśli to nie zadziała, wybierz z poniższej listy [przydatne linki](#useful-links) lub przeszukaj https://docs.microsoft.com .
