# <a name="javascript-f1-help-for-visual-studio"></a>Pomoc języka JavaScript F1 dla programu Visual Studio

JavaScript jest językiem skryptowym, który może być osadzony w stronach sieci Web i innych aplikacjach. W przypadku starszych wersji programu Microsoft Edge i przeglądarki Internet Explorer implementacja języka JavaScript firmy Microsoft jest zgodna z specyfikacją języka ECMAScript w wersji 5. Udostępnia także dodatkowe funkcje, które nie są uwzględnione w standardach ECMA.

Aby uzyskać pomoc referencyjną dotyczącą słów kluczowych i elementów kodu języka, zobacz:

- [Powiadomienia MDN Web docs](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference) , aby uzyskać pomoc dotyczącą języka JavaScript. W polu wyszukiwania wpisz nazwę szukanego słowa kluczowego.
- [Dokumentacja języka TypeScript](https://www.typescriptlang.org/docs) w celu uzyskania pomocy dotyczącej języka TypeScript.
- [Strona wyszukiwania](/search) w witrynie docs.Microsoft.com, jeśli szukasz pomocy w oficjalnej dokumentacji firmy Microsoft.

> [!IMPORTANT]
> Firma Microsoft jest częścią wysiłków dla całej społeczności, aby [Dokumentacja powiadomienia MDN w witrynie](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference) sieci Web była zatrzymywana. Wszystkie 500 stron dokumentacji interfejsu API języka JavaScript firmy Microsoft zostały przewoźny od [docs.Microsoft.com](https://docs.microsoft.com) do ich odpowiedników powiadomienia MDN. Nasze starsze informacje o rozszerzeniach JS zostały również przełączone do celów historycznych i obsługi.
