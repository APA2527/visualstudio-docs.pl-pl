# <a name="understand-build-platforms"></a>Opis platform kompilacji

Można przechowywać różne wersje właściwości rozwiązania i projektu, które mają zastosowanie do różnych platform docelowych. Na przykład można utworzyć konfigurację debugowania, która jest przeznaczona dla platformy x86 i konfiguracji debugowania, która jest przeznaczona dla platformy x64. Można szybko zmienić aktywną platformę, dzięki czemu można łatwo tworzyć wiele konfiguracji.
