# <a name="optimize-visual-studio-performance"></a>Optymalizacja wydajności programu Visual Studio

W tym artykule przedstawiono kilka sugestii próby, jeśli okaże się, że program Visual Studio działa powoli. Można również wykonać przyjrzeć się [i porady dotyczące wydajności programu Visual Studio](../ide/visual-studio-performance-tips-and-tricks.md) więcej masz sugestie dotyczące poprawy wydajności.

## <a name="upgrade-to-visual-studio-2017-version-156-or-later"></a>Uaktualnienie do wersji Visual Studio 2017 15,6 lub nowszej

Jeśli obecnie używasz programu Visual Studio 2015, Pobierz [programu Visual Studio 2017](https://aka.ms/vsdownload?utm_source=mscom&utm_campaign=msdocs) bezpłatnie do zapoznaj się z jego lepszą wydajność. Rozwiązania załadować dwóch do trzech razy szybciej w Visual Studio 2017 r. o ulepszenia wydajności w innych obszarach zbyt. Visual Studio 2017 jest side-by-side zgodny z programem Visual Studio 2015, aby nie dopuścić do przeoczenia żadnych podejmując.

Jeśli obecnie używasz programu Visual Studio 2017 r, upewnij się, że używasz wersji 15.6 lub nowszej. Dane pokazują, że rozwiązań załadować dwie lub trzy godziny szybszy w wersji 15,6. Pobierz go [tutaj](https://aka.ms/vsdownload?utm_source=mscom&utm_campaign=msdocs).

## <a name="hardware"></a>Sprzęt

Jeśli myślisz o uaktualnienie sprzętu, dysków półprzewodnikowych (SSD) ma więcej wpływ na wydajność niż więcej pamięci RAM lub szybszy Procesor.

Jeśli dodasz dysków SSD, aby zapewnić optymalną wydajność Zainstaluj system Windows na tym dysku, w przeciwieństwie do stacji dysków twardych (HDD). Lokalizację dysku rozwiązań programu Visual Studio nie wydaje się niezależnie od tego, jaka.

Ponadto nie należy uruchamiać rozwiązanie z dysku USB. Skopiuj go na dysk twardy lub dysk SSD.
