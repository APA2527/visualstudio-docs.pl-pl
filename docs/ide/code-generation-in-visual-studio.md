# <a name="code-generation-features-in-visual-studio"></a>Funkcje generowania kodu w programie Visual Studio

Istnieje wiele sposobów, które program Visual Studio mogą pomóc Ci Generowanie, naprawianie kluczy i refaktoryzacji kodu.

## <a name="features"></a>Funkcje

- Możesz użyć [fragmenty kodu](../ide/code-snippets.md) do wstawienia szablonu, takie jak [Przełącz](/dotnet/csharp/language-reference/keywords/switch) bloku lub [wyliczenia](/dotnet/csharp/language-reference/keywords/enum) deklaracji.

- Możesz użyć [szybkie akcje](../ide/quick-actions.md) do generowania kodu, takich jak klasy i właściwości lub wprowadzanie zmiennej lokalnej. Umożliwia również szybkie akcje na [poprawienia kodu](../ide/common-quick-actions.md), na przykład usuwania niepotrzebnego rzutowania i zmienne, lub Dodaj sprawdzenia wartości null, przed uzyskaniem dostępu do zmiennych.

- Możesz [refaktoryzacji kodu](../ide/refactoring-in-visual-studio.md) Aby zmienić nazwę zmiennej, zmienić kolejność parametrów metody lub zsynchronizuj typu przy użyciu jego nazwy pliku, kilka.

> [!NOTE]
> Każda usługa języka w programie Visual Studio udostępnia własnego kodu możliwości generowania, więc niektóre funkcje są dostępne tylko w C#, a niektóre są dostępne w obu C# i Visual Basic.
