# <a name="code-generation-features-in-visual-studio"></a>Funkcje generowania kodu w programie Visual Studio

Istnieje wiele sposobów, które Visual Studio może pomóc w generowaniu, naprawianie i refaktoryzuje kod.

## <a name="features"></a>Funkcje

- Za pomocą [fragmentów kodu](../ide/code-snippets.md) można wstawić szablon, taki jak blok [przełącznika](/dotnet/csharp/language-reference/keywords/switch) lub deklaracja [wyliczenia.](/dotnet/csharp/language-reference/keywords/enum)

- Szybkie [akcje](../ide/quick-actions.md) służy do generowania kodu, takich jak klasy i właściwości lub wprowadzić zmienną lokalną. Można również użyć szybkich akcji, aby [poprawić kod,](../ide/common-quick-actions.md)na przykład, aby usunąć niepotrzebne rzutowania i nieużywane zmienne lub dodać kontrole wartości null przed dostępem do zmiennych.

- Można [refaktoryzować kod,](../ide/refactoring-in-visual-studio.md) aby zmienić nazwę zmiennej, zmienić kolejność parametrów metody lub zsynchronizować typ z jego nazwą pliku, aby wymienić tylko kilka.

> [!NOTE]
> Każda usługa języka w programie Visual Studio zapewnia własne możliwości generowania kodu, więc niektóre funkcje są dostępne tylko w języku C#, a niektóre są dostępne zarówno w języku C# i Visual Basic.
