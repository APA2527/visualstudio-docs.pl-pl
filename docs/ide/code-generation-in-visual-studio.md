# <a name="code-generation-features-in-visual-studio"></a>Funkcje generowania kodu w programie Visual Studio

Istnieje wiele sposobów, które Visual Studio może pomóc Generowanie napraw i zrefaktoryzuj kod.

- Można użyć [wstawki kodu](../ide/code-snippets.md) do wstawienia szablonu, takie jak [przełącznika](/dotnet/csharp/language-reference/keywords/switch) bloku lub [wyliczenia](/dotnet/csharp/language-reference/keywords/enum) deklaracji.

- Można użyć [szybkie akcje](../ide/quick-actions.md) do generowania kodu, takich jak klasy i właściwości lub wprowadzić zmiennej lokalnej. Umożliwia również szybkie akcje na [poprawić kod](../ide/common-quick-actions.md), na przykład aby usunąć niepotrzebne rzutowania i zmienne, lub aby dodać sprawdzenia wartości null przed uzyskaniem dostępu do zmiennych.

- Możesz [zrefaktoryzuj kod](../ide/refactoring-in-visual-studio.md) Aby zmienić nazwę zmiennej, zmienić kolejność parametrów metody lub synchronizowana kilka jego nazwę, typ.

> [!NOTE]
> Każda usługa języka w programie Visual Studio udostępnia własny kod możliwości generowania, niektóre funkcje są dostępne tylko w języku C#, a niektóre są dostępne w języku C# i Visual Basic.
