# <a name="code-generation-features-in-visual-studio"></a>Funkcje generowania kodu w programie Visual Studio

Istnieje wiele sposobów, aby Visual Studio generowanie kodu dla Ciebie w edytorze. Korzystając z tych funkcji generowania kodu, można oszczędzić czas i naciśnięcia klawiszy, zmniejszyć liczbę błędów składni i zachowanie spójności w kodzie.

Niektóre funkcje programu Visual Studio z generowaniem kodu zostanie to [wstawki kodu](../ide/code-snippets.md) i [szybkie akcje](../ide/quick-actions.md) ![małych ikon żarówki](media/vs2015_lightbulbsmall.png).

Niektóre typowe zadania generowania kodu dostępne za pośrednictwem [szybkie akcje](../ide/quick-actions.md) są:

* Generowanie klas, metod, właściwości, itd.

* Implementowanie interfejsów lub klas abstrakcyjnych

* Wprowadzenie do zmiennej lokalnej do wyrażenia złożone

Ponadto wpisując niektórych znaków, które można wykonywać następujące czynności:

* Generowanie [bloków w formacie XML komentarz]() dla tego kodu, które później mogą być przetwarzane do automatycznego tworzenia dokumentacji

* Generowanie [zastąpienie metody]() podpisów

Ponieważ logiki generowania kodu jest ściśle związany składni języka, każda usługa języka w programie Visual Studio udostępnia własny kod możliwości generowania.
