# <a name="code-generation-in-visual-studio"></a>Generowanie kodu w programie Visual Studio
Visual Studio może pomóc Generowanie fragmenty kodu źródłowego, dzięki czemu można skupić się na rzeczywistym pracy.  Niektóre typowe funkcje generowania kodu:  

* Generowanie klas, metod, typy itp.
* Generowanie komentarze dokumentacji XML
* Implementowanie interfejsów lub klas abstrakcyjnych
* Wprowadzenie do zmiennej lokalnej do wyrażenia złożone

Ponieważ logiki generowania kodu jest ściśle związany składni języka, każda usługa języka w programie Visual Studio udostępnia własny kod możliwości generowania. Aby uzyskać więcej informacji zobacz następujące strony:  
  
* [Generowanie kodu (C#)](../csharp-ide/code-generation-csharp.md)
* [Generowanie kodu (Visual Basic)](../vb-ide/code-generation-vb.md)
