# <a name="install-dotfuscator-community-edition-ce"></a>Instalowanie programu Dotfuscator Community Edition (CE)

Program Visual Studio 2017 wprowadzono nowe środowisko instalacji o niskim wpływie na.
W rezultacie system Dotfuscator Community Edition (Dotfuscator CE) nie jest instalowany domyślnie.
Jest łatwa instalacja programu Dotfuscator CE, nawet jeśli zainstalowano już program Visual Studio.

> [!NOTE]
> Oprócz wersji programu Dotfuscator CE są dostarczane z wersjami produktów Visual Studio firmy PreEmptive Solutions okresowo zawiera zaktualizowane wersje na jej witrynie internetowej.
> Jeśli chcesz pobrać **najnowszej wersji** bezpośrednio, zamiast instalować ją z poziomu programu Visual Studio, **[kliknij tutaj, aby przejść do strony pobierania narzędzia Dotfuscator] [ download]**.

## <a name="within-visual-studio"></a>W programie Visual Studio

Zaciemniacza kodu Dotfuscator CE można zainstalować w środowisku IDE programu Visual Studio:

1. W **Szybkie uruchamianie** paska wyszukiwania (Ctrl + Q), typ `dotfuscator`. <br/> <br/> ![Szybkie uruchamianie](media/install_from_vs_12.png) <br/> <br/>
2. Na pasku szybkiego uruchamiania powoduje pokazano, w obszarze *zainstalować* nagłówka, wybierz **PreEmptive ochrona — Dotfuscator (poszczególnych składników)**.
   * Jeśli widać, w obszarze *menu* nagłówkiem **— ochrona PreEmptive — narzędzia Dotfuscator**, a następnie zaciemniacza kodu Dotfuscator CE jest już zainstalowana. Aby uzyskać szczegóły użycia, zobacz [stronie wprowadzenie pełnego przewodnika użytkownika programu Dotfuscator CE][get-started].
3. Instalator Visual Studio, który spowoduje uruchomienie okna, wstępnie skonfigurowaną do zainstalowania programu Dotfuscator CE.
   * Może być konieczne podanie poświadczeń administratora, aby kontynuować.
4. Zamknij wszystkie wystąpienia programu Visual Studio IDE. <br/> <br/> ![Kliknij przycisk Zainstaluj](media/install_from_vs_345.png) <br/> <br/>
5. W oknie Instalatora programu Visual Studio kliknij *zainstalować*.

Po zakończeniu instalacji można uruchomić przy użyciu zaciemniacza kodu Dotfuscator CE. Aby uzyskać więcej informacji, zobacz [stronie wprowadzenie pełnego przewodnika użytkownika programu Dotfuscator CE][get-started].

## <a name="during-visual-studio-installation"></a>Podczas instalowania programu Visual Studio

Jeśli nie zainstalowano jeszcze programu Visual Studio 2017, możesz pobrać Instalatora z [witryny sieci Web programu Visual Studio][2017-install].
Po uruchomieniu wyświetli opcje instalacji dla wybranej wersji programu Visual Studio.

![Opcje instalacji](media/install_ui.png)

Jako poszczególnych składników programu Visual Studio 2017 można zainstalować programu Dotfuscator CE:

1. Wybierz **poszczególne składniki** kartę.
2. W obszarze *kodu narzędzia*, sprawdź *PreEmptive ochrona — Dotfuscator* elementu.<br/> <br/> ![Poszczególne składniki](media/install_individually_12.png) <br/> <br/>
3. *Podsumowanie* panelu Wyświetla *PreEmptive ochrona — Dotfuscator* w obszarze *poszczególne składniki* sekcji. <br/> <br/> ![Okienko podsumowania](media/install_individually_3.png) <br/> <br/>
4. Skonfiguruj dodatkowe ustawienia instalacji odpowiednie dla danego środowiska.
5. Gdy wszystko będzie gotowe do zainstalowania programu Visual Studio, kliknij przycisk *zainstalować* przycisku.

Po zakończeniu instalacji można uruchomić przy użyciu zaciemniacza kodu Dotfuscator CE. Aby uzyskać więcej informacji, zobacz [stronie wprowadzenie pełnego przewodnika użytkownika programu Dotfuscator CE][get-started].
