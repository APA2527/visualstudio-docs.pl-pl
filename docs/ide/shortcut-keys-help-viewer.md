 **Zarządzanie oknem**  
  
|||  
|-|-|  
|**Naciśnięcie klawisza**|**Akcja**|  
|CTRL + L|Resetowanie układu podglądu pomocy do domyślny układ i zamknięcie wszystkich kart tematu.|  
|Ctrl + Tab|Pierwsze naciśnięcie klawisza przenosi fokus do **wybór karty** menu. Następny naciśnięcie klawisza przenosi fokus w elemencie menu u góry, a kolejne naciśnięcia klawiszy fokus elementów menu w kolejności od góry do dołu. Po aktywowaniu elementu menu klawisza Enter zgłasza, które element aktywnej karty.|  
|Ctrl + Shift + Tab|Pierwsze naciśnięcie klawisza przenosi fokus do **wybór karty** menu. Następny naciśnięcie klawisza przenosi fokus na najniższy element menu, a kolejne naciśnięcia klawiszy fokus elementów menu w kolejności od dołu do góry. Po aktywowaniu elementu menu klawisza Enter zgłasza, które element aktywnej karty.|  
|ALT + I, Alt + S, Alt + C, Alt + F, Alt + M|Tych klawiszy skrótów nie działa, gdy są niezadokowanego karty nawigacji i zarządzanie zawartością.|
