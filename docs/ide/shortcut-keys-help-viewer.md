 **Zarządzanie oknem**

|||
|-|-|
|**Naciśnięcie klawisza**|**Akcja**|
|**CTRL**+**L**|Resetuj **podglądu pomocy** układ domyślny układ i zamknięcie wszystkich kart tematu.|
|**CTRL**+**kartę**|Pierwsze naciśnięcie klawisza przenosi fokus do **wybór karty** menu. Następny naciśnięcie klawisza przenosi fokus w elemencie menu u góry, a kolejne naciśnięcia klawiszy fokus elementów menu w kolejności od góry do dołu. Gdy element menu ma fokus, **Enter** klucza ułatwia, które element aktywnej karty.|
|**CTRL**+**Shift**+**kartę**|Pierwsze naciśnięcie klawisza przenosi fokus do **wybór karty** menu. Następny naciśnięcie klawisza przenosi fokus na najniższy element menu, a kolejne naciśnięcia klawiszy fokus elementów menu w kolejności od dołu do góry. Gdy element menu ma fokus, **Enter** klucza ułatwia, które element aktywnej karty.|
|**ALT**+**I**, **Alt**+**S**, **Alt**+**C**, **Alt**+**F**, **Alt**+**M**|Tych klawiszy skrótów nie działa, gdy są niezadokowanego karty nawigacji i zarządzanie zawartością.|
