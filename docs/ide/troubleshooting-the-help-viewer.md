# <a name="troubleshooting-the-help-viewer"></a>Rozwiązywanie problemów z Podglądem Pomocy
W tym temacie opisano problemy, które mogą wystąpić w programie Podgląd pomocy.  
  
## <a name="audio-doesnt-work"></a>Audio nie działa.  
 Podgląd pomocy nie zawiera odtwarzacz audio. Jeśli pobieranie zawartości, która zawiera audio i nic się nie dzieje po wybraniu **odtwarzanie**, zainstalować odtwarzacz audio.  
  
## <a name="search-doesnt-work-in-windows-server-2008-windows-server-2008-with-sp1-or-windows-server-2008-r2"></a>Wyszukiwanie nie działa w systemie Windows Server 2008, Windows Server 2008 z dodatkiem SP1 lub Windows Server 2008 R2.  
 Funkcje wyszukiwania i filtrowania w Podglądzie pomocy wymagają usługi Windows Search do zainstalowania i na. Domyślnie ta usługa jest wyłączona w systemie Windows Server 2008, Windows Server 2008 z dodatkiem Service Pack 1 (SP1) i Windows Server 2008 R2.  
  
#### <a name="to-activate-windows-search-service"></a>Aby aktywować usługę wyszukiwania systemu Windows  
  
1.  Uruchom Menedżera serwera.  
  
2.  W okienku nawigacji po lewej stronie wybierz **ról**.  
  
3.  W okienku Podsumowanie ról, wybierz **Dodaj rolę**.  
  
4.  Wybierz rolę usługi plików, a następnie wybierz **dalej** przycisku.  
  
5.  Wybierz usługę roli wyszukiwania systemu Windows.
