# <a name="use-office-functionality-inside-of-visual-studio"></a>Użyj funkcji pakietu Office w Visual Studio
  Podczas tworzenia projektu poziomie dokumentu, dokument i skojarzonej aplikacji są obsługiwane w programie Visual Studio umożliwiające projektowanie i pracować bezpośrednio z dokumentu. Jeśli masz Microsoft Office aplikacji Otwórz w programie Visual Studio on zazwyczaj działa zgodnie z oczekiwaniami. Jednak niektóre funkcje aplikacji jest inny lub jest niedostępny.  
  
 [!INCLUDE[appliesto_alldoc](../vsto/includes/appliesto-alldoc-md.md)]  
  
## <a name="document-protection"></a>Ochrona dokumentu  
 Microsoft Office Word i Microsoft Office Excel oferują funkcje ochrony, których można używać w projektach dokumentu. Jednak jeśli dokument jest chroniony dokument jest otwarty w programie Visual Studio, jego może uniemożliwić wprowadzania pewnych zmian w projekcie. Aby uzyskać więcej informacji, zobacz [ochrona w rozwiązaniach na poziomie dokumentu dokumentu](../vsto/document-protection-in-document-level-solutions.md).  
  
## <a name="information-rights-management"></a>Zarządzanie prawami do informacji  
 Zarządzanie prawami do informacji (IRM) jest dostępna w programie Microsoft Office Word i Microsoft Office Excel. Usługi IRM można łatwiej uniemożliwić osobom nieupoważnionym wyświetlanie lub zmienianie poufne informacje. Jednak usługi IRM można również zablokować kodu uruchomienie. Aby uzyskać więcej informacji, zobacz [Zarządzanie prawami do informacji i rozszerzenia kodu zarządzanego ― omówienie](../vsto/information-rights-management-and-managed-code-extensions-overview.md).  
  
## <a name="password-protection"></a>Ochrona za pomocą hasła  
 Dokumenty programu Microsoft Office Word i skoroszytów programu Microsoft Office Excel można ustawić tak, aby nie można otworzyć przez osobę, która nie może określić hasło. Ochrona za pomocą hasła różni się w programach Word i Excel i może mieć wpływ na procesie tworzenia aplikacji. Aby uzyskać więcej informacji, zobacz [ochrona za pomocą hasła w dokumentach pakietu Office](../vsto/password-protection-on-office-documents.md).
