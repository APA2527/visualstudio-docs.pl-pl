# <a name="walkthroughs-using-excel"></a>Wskazówki dotyczące korzystania z programu Excel
  W poniższych przewodnikach przedstawiono sposoby używania dostosowań na poziomie dokumentu i dodatków narzędzi VSTO do automatyzowania Microsoft Office programu Excel i dostosowywania interfejsu użytkownika.

## <a name="document-level-walkthroughs"></a>Instruktaże na poziomie dokumentu
- [Przewodnik: Utwórz pierwsze dostosowanie na poziomie dokumentu dla programu Excel](../vsto/walkthrough-creating-your-first-document-level-customization-for-excel.md) — pokazuje, jak utworzyć podstawowe dostosowanie na poziomie dokumentu dla programu Excel.

- [Przewodnik: Utwórz niestandardową kartę przy użyciu projektanta](../vsto/walkthrough-creating-a-custom-tab-by-using-the-ribbon-designer.md) wstążki pokazuje, jak dodać niestandardową kartę do wstążki skoroszytu programu Excel.

- [Przewodnik: Zbieranie danych przy użyciu formularza](../vsto/walkthrough-collecting-data-using-a-windows-form.md) systemu Windows ilustruje użycie formularza systemu Windows w celu zebrania danych wejściowych użytkownika, a następnie wysłania danych wejściowych do komórki w arkuszu programu Excel.

- [Przewodnik: Zmiana formatowania arkusza za pomocą kontrolek](../vsto/walkthrough-changing-worksheet-formatting-using-checkbox-controls.md) CheckBox pokazuje podstawy używania pól wyboru w arkuszu programu Excel, aby zmienić formatowanie.

- [Przewodnik: Wyświetlanie tekstu w polu tekstowym w arkuszu za pomocą przycisku](../vsto/walkthrough-displaying-text-in-a-text-box-in-a-worksheet-using-a-button.md) pokazuje podstawy korzystania z przycisków i pól tekstowych w arkuszach programu Excel.

- [Przewodnik: Program przed zdarzeniami formantu](../vsto/walkthrough-programming-against-events-of-a-namedrange-control.md) NamedRange zawiera instrukcje krok po kroku dotyczące <xref:Microsoft.Office.Tools.Excel.NamedRange> dodawania kontrolki do arkusza i programu do jego zdarzeń.

- [Przewodnik: Proste powiązanie danych w projekcie](../vsto/walkthrough-simple-data-binding-in-a-document-level-project.md) na poziomie dokumentu przedstawia podstawowe informacje dotyczące powiązania pojedynczego pola danych w bazie danych SQL Server z nazwanym zakresem w programie Excel.

- [Przewodnik: Złożone powiązanie danych w projekcie](../vsto/walkthrough-complex-data-binding-in-a-document-level-project.md) na poziomie dokumentu pokazuje podstawowe informacje o powiązaniu wielu komórek w arkuszu programu Excel do pól w bazie danych Northwind SQL Server.

- [Przewodnik: Utwórz relację wzorca głównego przy użyciu buforowanego zestawu](../vsto/walkthrough-creating-a-master-detail-relation-using-a-cached-dataset.md) danych pokazuje, jak utworzyć relację główną/szczegółową w arkuszu i buforować dane, aby umożliwić korzystanie z rozwiązania w trybie offline.

- [Przewodnik: Aktualizacja wykresu w arkuszu za pomocą przycisków](../vsto/walkthrough-updating-a-chart-in-a-worksheet-using-radio-buttons.md) radiowych przedstawia podstawy zmiany stylów wykresu za pomocą przycisków radiowych w arkuszu programu Excel.

- [Przewodnik: Powiązywanie danych z kontrolkami w okienku](../vsto/walkthrough-binding-data-to-controls-on-an-excel-actions-pane.md) akcji programu Excel zawiera opis sposobu dodawania formantów, które są powiązane ze źródłem danych, do okienka Akcje w programie Excel.
