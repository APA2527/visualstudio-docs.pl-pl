# <a name="work-with-visio-documents"></a>Praca z dokumentami Visio
  Tematy w tej sekcji Obejmij procedury krok po kroku i przykłady kodu za pomocą modelu obiektów programu Microsoft Visio pakietu Office do pracy z dokumentami w projektach pakietu Office.

|Zadanie|Procedura|
|----------|---------------|
|Utwórz pusty dokument programu Visio lub nowego dokumentu, który jest oparty na istniejącego szablonu programu Visio, wzornika lub dokumentu.|[Instrukcje: Programowe tworzenie nowych dokumentów programu Visio](../vsto/how-to-programmatically-create-new-visio-documents.md)|
|Otwórz istniejący dokument programu Visio i określ opcje dla odczytu/zapisu, tylko do odczytu, a zadokowany.|[Instrukcje: Programowe otwieranie dokumentów programu Visio](../vsto/how-to-programmatically-open-visio-documents.md)|
|Zamknij aktywny dokument programu Visio.|[Instrukcje: Programowe zamykanie dokumentów programu Visio](../vsto/how-to-programmatically-close-visio-documents.md)|
|Zapisywanie dokumentu o takiej samej nazwie lub nową nazwę.|[Instrukcje: Programowe zapisywanie dokumentów programu Visio](../vsto/how-to-programmatically-save-visio-documents.md)|
|Wydrukuj cały dokument lub określonej strony.|[Instrukcje: Programowe drukowanie dokumentów programu Visio](../vsto/how-to-programmatically-print-visio-documents.md)|
