# <a name="error-messages-office-development-in-visual-studio"></a>Komunikaty o błędach (Programowanie Office w Visual Studio)
  Każdy komunikat o błędzie zawiera informacje o możliwych przyczynach błędu i potencjalnych rozwiązań lub obejściach w celu rozwiązania problemu.

 Informacje o komunikacie o błędzie można uzyskać, korzystając z następujących sposobów.

|Metoda|Opis|
|-|-|
|F1|Jeśli wystąpi błąd, możesz nacisnąć klawisz **F1**. Spowoduje to otwarcie odpowiedniego tematu komunikatu o błędzie w dokumentacji online.|
|Wyszukiwanie pełnotekstowe|Możesz użyć wyszukiwania pełnotekstowego przy użyciu wyrazów z pierwszego zdania w tekście komunikatu o błędzie. W menu **Pomoc** kliknij polecenie **Wyszukaj**.|
