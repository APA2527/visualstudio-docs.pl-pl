# <a name="project-upgrade-options-dialog-box"></a>Uaktualniania projektu, opcje— Okno dialogowe
  Użyj **uaktualniania projektu** stronę, aby określić, czy program Visual Studio automatycznie zmieni projektów dodatku VSTO pod kątem wersji pakietu Office, który jest zainstalowany na komputerze dewelopera.  
  
 Aby uzyskać dostęp do tej strony, na **narzędzia** menu, kliknij przycisk **opcje**. Rozwiń węzeł **narzędzia Office** folder, a następnie kliknij przycisk **uaktualniania projektu**.  
  
## <a name="uielement-list"></a>Lista elementów UI  
 **Zawsze uaktualnienia do zainstalowanych wersji pakietu Office**  
 Ta opcja jest wyłączona dla [!INCLUDE[vs_dev12](../vsto/includes/vs-dev12-md.md)] ponieważ projektów pakietu Office w [!INCLUDE[vs_dev12](../vsto/includes/vs-dev12-md.md)] można kierować tylko [!INCLUDE[office14_long](../vsto/includes/office14-long-md.md)] lub Microsoft [!INCLUDE[Office_15_short](../vsto/includes/office-15-short-md.md)] aplikacji. Aby uzyskać więcej informacji, zobacz [uaktualnianie i migracja rozwiązań pakietu Office](../vsto/upgrading-and-migrating-office-solutions.md).
