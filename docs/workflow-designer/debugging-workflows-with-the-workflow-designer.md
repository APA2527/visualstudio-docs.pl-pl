# <a name="debugging-workflows-with-the-workflow-designer"></a>Debugowanie przepływów pracy za pomocą projektanta przepływów pracy

Projektant przepływu pracy systemu Windows umożliwia debugowanie przepływów pracy i działań niestandardowych. Proces i zachowanie są podobne do domyślnych [!INCLUDE[vs2010](../misc/includes/vs2010_md.md)] debugera. Tematy w tej sekcji opisano sposób wykonywania tradycyjnych debugowanie procedur, a także debugowanie XAML i wyrażenia.
