# <a name="using-themes-in-workflows-legacy"></a>Używanie tematów w przepływach pracy (starsze)
W tym temacie opisano sposób używania tematów w [!INCLUDE[wf](../workflow-designer/includes/wf_md.md)] używając projektanta przepływów pracy starszej wersji systemu Windows, przeznaczonego dla [!INCLUDE[netfx35_long](../workflow-designer/includes/netfx35_long_md.md)] lub [!INCLUDE[vstecwinfx](../workflow-designer/includes/vstecwinfx_md.md)].

 Można dostosować wygląd przepływy pracy przy użyciu starszej wersji projektanta przy użyciu [!INCLUDE[wf2](../workflow-designer/includes/wf2_md.md)] motywów. Motyw definiuje kolory tła i pierwszego planu, style, ikony i inne elementy wizualne przepływu pracy. Możesz zapisać kompozycje do ponownego użycia przez inne przepływy pracy.

 Możesz tworzyć i edytować motywów przy użyciu [okno dialogowe konfiguracji motywu (starsze)](../workflow-designer/theme-configuration-dialog-box-legacy.md). Aby otworzyć to okno dialogowe, wybierz **Utwórz nowy motyw** na **przepływu pracy** menu lub kliknij prawym przyciskiem myszy projekt przepływu pracy powierzchni i wybierz pozycję **Utwórz nowy motyw**.

 Można wybrać domyślne i zapisane przy użyciu niestandardowych kompozycji [kompozycje, projektanta przepływów pracy, opcje — Okno dialogowe (starsze)](../workflow-designer/themes-workflow-designer-options-dialog-box-legacy.md). Aby otworzyć to okno dialogowe, kliknij przycisk **wybierz motyw niestandardowy** na **przepływu pracy** menu (lub kliknij prawym przyciskiem myszy powierzchnię projektu przepływu pracy i kliknij przycisk **wybierz motyw niestandardowy**), a następnie wybierz pozycję **projektanta przepływów pracy** opcji. Umożliwia wybranie motywu i kliknij przycisk **OK**.
