# <a name="workflow-designer-ui-help"></a>Pomocy interfejsu użytkownika projektanta przepływów pracy

Ta sekcja zawiera tematy pomocy kontekstowej dla projektanta przepływu pracy systemu Windows. Podczas pracy w Visual Studio 2010, są dostępne w tych tematach interfejsu użytkownika, naciskając klawisz **F1** kiedy jest otwarte okno dialogowe projektanta przepływów pracy.

Ta sekcja zawiera również kilka tematów związanych z nawigacji i komunikaty o błędach.
