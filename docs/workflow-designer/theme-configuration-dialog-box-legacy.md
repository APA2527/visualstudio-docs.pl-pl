# <a name="theme-configuration-dialog-box-legacy"></a>Okno dialogowe konfiguracji motywu (starsze)

W tym temacie opisano sposób użycia **konfiguracji motywu** okno dialogowe w starszej wersji projektanta przepływów pracy programu Windows. Starsze projektanta przepływów pracy należy używać wtedy, gdy konieczne objęcie .NET Framework w wersji 3.5 lub WinFX.

Motyw definiuje kolory tła i pierwszego planu, style, ikony i inne elementy wizualne przepływu pracy. Możesz zapisać kompozycje do ponownego użycia przez inne przepływy pracy.

Możesz tworzyć i edytować motywów przy użyciu **konfiguracji motywu** okno dialogowe. Aby otworzyć okno dialogowe, wybierz **Utwórz nowy motyw** na **przepływu pracy** menu lub kliknij prawym przyciskiem myszy projekt przepływu pracy powierzchni i wybierz pozycję **Utwórz nowy motyw**.

W poniższej tabeli opisano elementy interfejsu użytkownika **konfiguracji motywu** okno dialogowe.

|Element interfejsu użytkownika|Opis|
|----------------|-----------------|
|**Nazwa motywu:**|Nazwa, która identyfikuje motywu w [kompozycje, projektanta przepływów pracy, opcje — Okno dialogowe (starsze)](../workflow-designer/themes-workflow-designer-options-dialog-box-legacy.md). Zmienić nazwę jest generowany dla nowej kompozycji.|
|**Lokalizacja motywu:**|Nazwa pliku i ścieżka pliku motywu. Nazwa pliku modyfikowalne jest generowany dla nowej kompozycji na podstawie nazwy wygenerowanego motywu. Jeśli zmienisz nazwę wygenerowanego motywu, można zmienić nazwę pliku, aby dopasować nazwę motywu.|
|**...**|Kliknij, aby wybrać lokalizację do zapisania pliku motywu przepływu pracy, który używa wtm rozszerzenia nazwy pliku. Wybrana ścieżka będzie widoczna w **lokalizacji kompozycji** pola tekstowego.|
|**Wybierz projektanta i Konfiguruj właściwości:**|Okienka po lewej stronie zawiera listę działań, dla których można dostosowywać motyw w widoku drzewa. Wybierz działanie, w widoku drzewa i właściwości motywu działania będą wyświetlane w okienku właściwości, które ma prawa w okienku widoku drzewa. Kliknij przycisk Właściwości, aby zmienić jego wartość.|
|**Wersja zapoznawcza**|Kliknij, aby wyświetlić okno podglądu skutków zmiany właściwości.|
