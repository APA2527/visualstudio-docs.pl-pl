# <a name="how-to-add-activities-to-the-toolbox-legacy"></a>Porady: dodawanie działań do przybornika (starsze)

W przypadku kompilowania rozwiązania przepływu pracy z projektanta przepływów pracy starszej wersji systemu Windows, przeznaczonego dla programu .NET Framework w wersji 3.5 lub WinFX, działań niestandardowych można dodać do projektu przepływu pracy i ich projektantów umieszczone w **przybornika** dla łatwy dostęp. Możesz także dodać działania bezpośrednio do **przybornika** z biblioteki dołączanej (dynamicznie DLL).

## <a name="to-add-an-activity-to-the-toolbox-from-a-dll"></a>Aby dodać działanie do przybornika z biblioteki DLL

1.  Kliknij prawym przyciskiem myszy powierzchnię okno przybornika w obszarze **Windows Workflow**, a następnie kliknij przycisk **wybierz elementy**.

2.  W **wybierz elementy przybornika** okno dialogowe, kliknij przycisk **składniki elementu System.Activities** , a następnie kliknij pozycję **Przeglądaj** z po prawej stronie dolnej części okna.

3.  Wybierz bibliotekę DLL z katalogu pliku, który zawiera implementację działań niestandardowych do dodania do **przybornika**, a następnie kliknij przycisk **Otwórz**.

4.  Kliknij przycisk **OK** aby zakończyć dodawanie działań do przybornika.
