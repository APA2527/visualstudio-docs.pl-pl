# <a name="collection-activity-designers"></a>Projektanci działań Collection

Kolekcję, którą Projektanci działań są używani do tworzenia i konfigurowania działań, które umożliwia zarządzanie kolekcjami w zawarte w tej sekcji i wymienione w poniższej tabeli tematach wskazówki dotyczące sposobu używania projektantów Designeractivity przepływu pracy i szablonu.
