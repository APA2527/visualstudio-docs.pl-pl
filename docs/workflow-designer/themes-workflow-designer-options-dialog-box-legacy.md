# <a name="themes-workflow-designer-options-dialog-box-legacy"></a>Kompozycje, projektanta przepływów pracy, opcje — Okno dialogowe (starsze)

W tym temacie opisano sposób użycia **opcje kompozycje, projektanta przepływów pracy** okno dialogowe w starszej wersji projektanta przepływów pracy programu Windows. Starsze projektanta przepływów pracy należy używać wtedy, gdy konieczne objęcie .NET Framework w wersji 3.5 lub WinFX.

Motyw definiuje kolory tła i pierwszego planu, style, ikony i inne elementy wizualne przepływu pracy. Możesz zapisać kompozycje do ponownego użycia przez inne przepływy pracy. Możesz wybrać domyślne i zapisane niestandardowych kompozycji przy użyciu **opcje kompozycje, projektanta przepływów pracy** okno dialogowe. Aby otworzyć to okno dialogowe, kliknij przycisk **wybierz motyw niestandardowy** na **przepływu pracy** menu. Lub kliknij prawym przyciskiem myszy powierzchnię projektu przepływu pracy i kliknij przycisk **wybierz motyw niestandardowy**.

W poniższej tabeli opisano elementy interfejsu użytkownika **opcje kompozycje, projektanta przepływów pracy** okno dialogowe.

|Element interfejsu użytkownika|Opis|
|----------------|-----------------|
|**Nowy**|Kliknij, aby otworzyć [okno dialogowe konfiguracji motywu (starsze)](../workflow-designer/theme-configuration-dialog-box-legacy.md) do utworzenia nowego motywu.|
|**Modify...**|Kliknij, aby otworzyć [okno dialogowe konfiguracji motywu (starsze)](../workflow-designer/theme-configuration-dialog-box-legacy.md) do modyfikowania wybranego motywu.|
|**Przeglądaj**|Kliknij, aby wyszukać plik motywu przepływu pracy, który korzysta z rozszerzeniem nazwy pliku wtm.|
|**Wybierz motyw**|Nagłówek kolumny Nazwa motywu listy motywów. Wybierz motyw z listy, kliknij przycisk **Modyfikuj** można edytować, lub kliknij przycisk **OK** do używania wybranego motywu przepływu pracy.|
|**Opis**|Motyw nagłówek kolumny opis listy motywów.|
|**Lokalizacja**|Nagłówek kolumny Lokalizacja motywu listy motywów.|
