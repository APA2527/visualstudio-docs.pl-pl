# <a name="state-machine-activity-designers"></a>Projektanci działań automatu stanów
[!INCLUDE[wfd1](../workflow-designer/includes/wfd1_md.md)]zawiera szereg projektantów działań, których można użyć podczas tworzenia przepływów pracy. Ta sekcja zawiera projektantów działań dostarczane przez system, używany do tworzenia i Zarządzaj przepływami pracy komputera stanu. Poniższe tematy opisują te projektantów i zawierają wskazówki dotyczące sposobu ich używania.  
  
## <a name="in-this-section"></a>W tej sekcji  
 [Obiekt StateMachine](../workflow-designer/statemachine-activity-designer.md)  
 Zawiera kolekcję stanów i modele przepływów pracy za pomocą modelu maszyny znanego stanu.  
  
 [Stan](../workflow-designer/state-activity-designer.md)  
 Reprezentuje stan, w którym mogą mieć automatu stanów.  
  
 [Stan końcowy](../workflow-designer/finalstate-activity-designer.md)  
 Reprezentuje stan, w którym kończy się wystąpieniem komputera stanu.  
  
 [Przejścia](../workflow-designer/transition-activity-designer.md)  
 Reprezentuje przejście między dwoma stanami.
