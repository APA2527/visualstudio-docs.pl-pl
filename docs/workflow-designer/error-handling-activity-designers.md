# <a name="error-handling-activity-designers"></a>Projektanci działań Obsługa błędów

Projektanci działania obsługi błędów służą do zarządzania obsługi błędów w Projektancie przepływów pracy systemu Windows. Tematy zawarte w tej sekcji zawierają wskazówki dotyczące sposobu używania projektanta przepływów pracy działania i projektantom szablonu.
