# <a name="develop-apps-with-the-workflow-designer"></a>Programowanie aplikacji za pomocą Projektanta przepływu pracy

Projektant przepływu pracy jest projektantem wizualnym i debugerem dla graficznej konstrukcji i debugowania aplikacji [Windows Workflow Foundation](/dotnet/framework/windows-workflow-foundation/index) (WF) w programie Visual Studio. Umożliwia tworzenie złożonej aplikacji przepływu pracy, biblioteki aktywności lub usługi Windows Communication Foundation (WCF) za pomocą szablonów i projektantów działań.

## <a name="install-windows-workflow-foundation"></a>Instalowanie fundacji przepływu pracy systemu Windows

Aby użyć szablonów projektów przepływu pracy w programie Visual Studio, należy najpierw zainstalować składnik **Fundacji przepływu pracy systemu Windows.**

1. Otwórz Instalatora programu Visual Studio. Szybkim sposobem jego otwarcia jest **wybranie** > opcji Narzędzia**Pobierz narzędzia i funkcje** w programie Visual Studio.

1. W Instalatorze programu Visual Studio wybierz kartę **Poszczególne składniki.**

1. Przewiń w dół do kategorii **Działania deweloperów,** a następnie wybierz składnik **Podstawa przepływu pracy systemu Windows.**

   ![Składnik Podstawy przepływu pracy systemu Windows dla programu Visual Studio](media/windows-workflow-foundation-component.png)

1. Wybierz **pozycję Modyfikuj**.

   Program Visual Studio instaluje składnik **Fundacji przepływu pracy systemu Windows.**
