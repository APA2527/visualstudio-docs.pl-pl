# <a name="legacy-designer-for-windows-workflow-foundation-ui-help"></a>Projektant starszych pomocy systemu Windows Workflow Foundation interfejsu użytkownika
Ta sekcja zawiera tematy pomocy kontekstowej dla starszych projektanta przepływów pracy systemu Windows w [!INCLUDE[vs2010](../misc/includes/vs2010_md.md)]. Gdy należy określić za pomocą starszej wersji projektanta [!INCLUDE[netfx35_long](../workflow-designer/includes/netfx35_long_md.md)] lub [!INCLUDE[vstecwinfx](../workflow-designer/includes/vstecwinfx_md.md)].

 Można uzyskać dostępu do tych tematów z [!INCLUDE[vs_current_long](../misc/includes/vs_current_long_md.md)] interfejsu użytkownika, naciskając klawisz **F1** podczas starszych [!INCLUDE[wfd2](../workflow-designer/includes/wfd2_md.md)] jest otwarte okno dialogowe.
