# <a name="legacy-designer-for-windows-workflow-foundation-ui-help"></a>Projektant starszych pomocy systemu Windows Workflow Foundation interfejsu użytkownika

Ta sekcja zawiera tematy pomocy kontekstowej dla starszej wersji systemu Windows projektanta przepływów pracy w Visual Studio 2010. Jeśli potrzebujesz objęcie .NET Framework w wersji 3.5 lub WinFX za pomocą starszej wersji projektanta.

Są dostępne w tych tematach interfejsu użytkownika programu Visual Studio, naciskając klawisz **F1** gdy jest otwarte okno dialogowe starszych projektanta przepływów pracy.
