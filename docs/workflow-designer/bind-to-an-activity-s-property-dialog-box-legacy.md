# <a name="bind-to-an-activity39s-property-dialog-box-legacy"></a>Powiąż z działaniem&#39;s właściwości — okno dialogowe (starsze)
W tym temacie opisano sposób użycia **powiązać z właściwością działania** okno dialogowe w starszej wersji projektanta przepływów pracy systemu Windows. Użyj starszego [!INCLUDE[wfd2](../workflow-designer/includes/wfd2_md.md)] konieczność docelowy: [!INCLUDE[netfx35_long](../workflow-designer/includes/netfx35_long_md.md)] lub [!INCLUDE[vstecwinfx](../workflow-designer/includes/vstecwinfx_md.md)].

 Typ wystąpienia właściwości zależności może być powiązana do publicznej właściwości lub zdarzenia innego działania. Aby uzyskać więcej informacji na temat powiązanie działania, zobacz [przy użyciu właściwości zależności](http://go.microsoft.com/fwlink?LinkID=65007).

 Wybierz właściwość, aby powiązać przy użyciu **powiązać z właściwością działania** okno dialogowe. Otwórz okno dialogowe, klikając wielokropek **[...]**  na końcu wybranej właściwości pola tekstowego w **właściwości** okna, ani klikając ikonę Niebieski wykrzyknik, który jest wyświetlany obok nazwy właściwości w przeglądarce właściwości.

 W poniższej tabeli opisano elementy interfejsu użytkownika **powiązać z właściwością działania** okno dialogowe.

|Element interfejsu użytkownika|Opis|
|----------------|-----------------|
|**Dowiąż do istniejącego członka**|Wybierz element członkowski, który chcesz utworzyć powiązania w okienku widoku drzewa. W okienku poniżej widoku drzewa wyświetlane komunikat, który wskazuje, czy element członkowski jest prawidłowym typem powiązać lub nie. Kliknij przycisk **OK** powiązać wybranej prawidłowego elementu członkowskiego.|
|**Dowiąż do nowego członka**|Tworzenie nowego elementu członkowskiego pola lub właściwości do powiązania. Wprowadź **nową nazwę elementu członkowskiego**. Wybierz, czy program do tworzenia właściwości zależności lub pole publiczne przez wybranie **utworzyć pole** lub **Tworzenie właściwości**. Kliknij przycisk **OK** do utworzenia nowego elementu członkowskiego.|
