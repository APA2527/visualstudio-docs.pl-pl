# <a name="integration-of-xml-literals-with-xml-schema-explorer"></a>Integracja literałów XML z Eksploratora schematu XML
Visual Basic obsługuje literałów XML, co oznacza, że fragmenty XML można zastosować bezpośrednio w kodzie języka Visual Basic. Aby uzyskać więcej informacji, zobacz [literały XML-Przegląd](http://go.microsoft.com/fwlink/?LinkId=140325).  
  
 Jeśli plik XSD, w projekcie języka Visual Basic zawiera literał XML, można wyświetlić schematu XML w Eksploratora schematu XML. Aby wyświetlić zestaw schematu, skojarzony z literał XML, kliknij prawym przyciskiem myszy w węźle XML literał XML lub importu przestrzeni nazw XML, a następnie wybierz **Pokaż w Eksploratorze schematu**.  
  
 ![Literały XML języka Visual Basic; Eksploratora schematu XML](../xml-tools/media/vbxmlliteralswithxmlschemaexplorer1.gif "VBXMLLiteralsWithXMLSchemaExplorer1")  
  
 Spowoduje to otwarcie Eksploratora schematu XML równolegle z pliku programu Visual Basic.
