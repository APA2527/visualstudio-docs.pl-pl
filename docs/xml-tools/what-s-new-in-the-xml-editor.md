# <a name="what39s-new-in-the-xml-editor"></a>Co&#39;s Nowość w edytorze XML
Następujące funkcje są nowością w programie Visual Studio 2010.  
  
## <a name="xslt-intellisense"></a>XSLT IntelliSense  
 Funkcja XSLT IntelliSense umożliwia nazwy zestawu atrybutów funkcja automatycznego uzupełniania; tryby szablonu i nazwy; i nazwy parametrów dla określonego trybu lub określony nazwany szablon. Aby uzyskać więcej informacji, zobacz [funkcje IntelliSense w edytorze XML](../xml-tools/xml-editor-intellisense-features.md).  
  
## <a name="xslt-profiler"></a>Profiler XSLT  
 Profilera XSLT tworzy szczegółowe wydajności XSLT raporty, które ułatwiają mierzyć, oceniać i określać problemy związane z wydajnością w kodzie XSLT. Profilera XSLT zawiera również wskazówki przydatne dla optymalizacji arkusza stylów XSL i XSLT. Aby uzyskać więcej informacji, zobacz [wskazówki: profilera XSLT](../xml-tools/walkthrough-xslt-profiler.md).  
  
## <a name="xslt-hierarchy"></a>Hierarchia XSLT  
 Narzędzia hierarchii XSLT pozwala dodać punkty przerwania w arkuszach stylów dołączone i/lub wbudowanych szablonów zasad. Aby uzyskać więcej informacji, zobacz [wskazówki: za pomocą hierarchii XSLT](../xml-tools/walkthrough-using-xslt-hierarchy.md).
