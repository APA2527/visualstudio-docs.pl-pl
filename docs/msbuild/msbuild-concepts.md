# <a name="msbuild-concepts"></a>Koncepcje MSBuild

MSBuild udostępnia podstawowy schemat XML, którego można użyć do kontrolowania sposobu tworzenia oprogramowania platformy kompilacji. Aby określić składniki w kompilacji i jak mają być budowane, należy użyć tych czterech części MSBuild: właściwości, elementy, zadania i obiekty docelowe.

| [Zadania MSBuild](../msbuild/msbuild-tasks.md) | Pokazuje, jak utworzyć jednostkę kodu wykonywalnego, który może być używany przez MSBuild do wykonywania operacji kompilacji niepodzielnej. |
