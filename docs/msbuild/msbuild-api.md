# <a name="use-the-msbuild-api"></a>Korzystanie z interfejsu API MSBuild

MSBuild udostępnia powierzchni publicznej interfejsu API, dzięki czemu program może wykonywać kompilacje i inspekcji projektów.

Dokumentację interfejsu API MSBuild można znaleźć na stronie [Microsoft.Build namespaces](https://msdn.microsoft.com/library/mt476050(v=vs.110).aspx).
