# <a name="logging-in-msbuild"></a>Logowanie w programie MSBuild

Rejestrowanie umożliwia monitorowanie postępu kompilacji. Rejestrowanie przechwytuje zdarzenia kompilacji, komunikaty, ostrzeżenia i błędy w pliku dziennika.
